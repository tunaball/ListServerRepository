Is he in single user mode or did he ssh in?

If single user mode, be sure to mount the drive first. The command to do
that is shown after starting up in single user mode, right after the note
about running fsck.

In either case, I would
	cd /Users/<his username>
first. (Where <his username> is his short username).
This will put the file in his home directory.
