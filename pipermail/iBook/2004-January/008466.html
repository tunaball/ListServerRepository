About a month ago, shortly after I purchased a G4 iBook, my G3 800 started
smelling - it smelled like something was burning inside, so I sent it to
Apple (AppleCare contract), but they found nothing. So I sent it back, same
thing. But it still smells.

This morning, while waiting for an AC technician, I found a thread on the
Apple discussion board about this problem - it seems that some iBooks'
keyboards start emitting what smells like a burning odor; it is suggested
that this is actually coming from the glue on the sticker under the
keyboard.

I did a test today, removing the keyboard, and letting the iBook run
Seti at Home for two hours until the fan came on - no smell. So it is clearly
coming from the keyboard.

Anyone else have this problem?
