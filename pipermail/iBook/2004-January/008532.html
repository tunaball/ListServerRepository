My understanding is that open-source DVD software, because it was
written by reverse-engineering the DVD standard, ignores DVD regions.
Therefore, you could try using one of them. (I don't know what the legal
status is - it probably varies from country to country).

One I recommend is ogle. It runs on Linux, but I don't know if it runs
on mac osx. Another popular one is xine.

If you want to try these, you should use 'fink', which is a project to
port open source software to mac osx. 

Good luck,
-Angus
