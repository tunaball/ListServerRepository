When you say full, I'm supposing you mean the hard disk is full. One 
thing that OSX does is fill up your computer with many  many language 
files. They take up many many megabytes of space. You only need the 
languages you use, right? For me, that is US English. Search via Google 
for something called Delocalizer. Run it (I've run it twice after OS 
updates). It will present a list of language files that you can remove. 
Select all checks all of them. It automatically assumes you want to 
keep US English on your machine and won't try to delete those files. It 
will run for about 15 minutes or so and delete literally dozens of 
files, some of which are simply huge. The recoup of space is amazing.

>> other program's memory needs.  A multitasking OS at work may be
