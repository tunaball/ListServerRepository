Well, in the sense that there's no software I'm aware of that truly 
works well and allows you to ring up actual phones, no. There is a 
product I've downloaded but not tried called Mac2Phone, it is Java so 
it is VERY slow to load. Again, I haven't tried it yet. You can get it 
from Mac2phone.com

I have successfully used Netphone to talk to people overseas. It is not 
a "telephone" solution, just a "voice conferencing" solution. The other 
person has to be running the program at the same time, and you need to 
know their IP address. It seems to work very well, particularly on a 
fast connection, with only the tiniest of delays. You can get an 
evaluation version from haxialsoftware.com.
