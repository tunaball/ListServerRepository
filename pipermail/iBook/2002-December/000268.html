In a word....don't follow through on this plan...okay, so that wasn't a
single word. I have the same machine as yours (more memory though) and I
found the following:

With Jaguar and VPC I was okay with Win98 SE although Win95 was much more
responsive. Anything newer than Win98 in Jag just wasn't usable IMHO. Now in
OS 9 I was quite happy with Win2000. I never bothered to install WinXP in
VPC but on my Gateway Win2000 is a tad faster than WinXP so I'd expect
similar results in VPC.

There really is that big a difference between OS X and OS 9 when it comes to
VPC performance. Mostly it is because of the way OS X multitasks. With OS 9
the front most application is able to virtually take over the computer
leaving no processing time for other applications. This isn't true for OS X.
