About this posting...

This script does *NOT* download music from Mac to MD via USB.

All it does is automate the process of naming pre-existing tracks on an 
MD.

i.e. to use it, you'll have to manually record onto an MD (completely 
separate to the script), and then use the script to title the tracks.

I'm sorry if my earlier posting was misleading - I should have written 
it more carefully.

Cheers,
-Angus
