I believe the default virtual memory size is 80MB (OS 10.1 &
10.2, anyhow), so your 67,108,864 would be non-standard and low.
If the 1st swapfile (number 0, like yours) gets filled up, I
believe additional ones are allocated 80MB at a time.  If you have
a lot of RAM (384 is not "a lot"), depending on what software
you are using you may never be using the swap file anyhow.
