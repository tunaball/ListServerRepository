On 12/05/2004, at 9:11 AM, Angus Wallace wrote:

> There is a program called Gnucash that is free:  www.gnucash.org
>
> Probably the easiest way to use it is to install it using Fink.
>
> Fink is a utility to allow one to install Unix, Linux and FreeBSD 
> software on Mac. Once Fink, The Apple Developer Tools, and Apple X11 
> are installed, it's simply a matter of going into the Terminal, and 
> typing
>
> >> fink install gnucash
>
> The fink program will do the rest. Pretty cool, huh?
>
> I've installed several programs (gimp, octave, freeciv) like this 
> without (m)any problems.
>
> Email if you have any questions about this.
>
> Cheers,
> -Angus
