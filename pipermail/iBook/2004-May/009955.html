I have tried this several times with varying success. 
It usually involves selecting and copying all files
from the cd to the hard drive.  Then you install the
software from the hard drive.  This is usually the way
that works for me.  

The only other option seems to be software like
virtual drive (pc only?) that creates a virtual (cd)
drive on your hard drive and will disable most if not
all copy protection.  This assumes that you are the
owner of the software and you are not committing
piracy.  HTH
