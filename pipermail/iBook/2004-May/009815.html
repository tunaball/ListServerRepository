I sync the files on my iBook with the files on my eMac using a free
application called SynkX (sorry, I don't have the URL handy).  It compares
and replaces older versions of files in twp separate folders, plus you can
set it up to sync a bunch of folders.  I usually sync my Documents,
Downloads and Music folders.  I've been using it for more than a year and
I'm very happy with it.
