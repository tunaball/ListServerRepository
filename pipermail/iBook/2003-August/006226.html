On 8/30/03 3:13 PM, "Dan Farley" <danfarley at mac.com> wrote:

> What's the best way to move my stuff from my 500 mhz laptop to a used tower?
> 
> Should I do a clean install and just load on all my software - or what about
> cloning my hard drive and moving that to the Tower?
> 
I don't clone to dissimilar computers due to issues involving drivers and
other related complications.

It would be best to install the OS and all updaters on the new tower first.
Then install your software and their updaters. Assuming the tower has
firewire, I'd use firewire target mode to move the data from the notebook(s)
onto the tower.

Note: due to permission issues, after transferring the data to each user's
folders, you'll want to use batChmod, the terminal, or some other utility to
make certain each person's data is properly set with his/her permissions.
