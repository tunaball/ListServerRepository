Your ISP probably provides you with a certain amount of web space. Is 
that enough? (You said that the files are large.) If you make at least 
one directory public, then MSIE, Netscape, etc, can get them. No log in 
should be needed, not even anonymous FTP. Check this:

http://users.rio.com/gslusher/images/

You should be able to see and download any of the files in that 
directory. You should be able to set any directory's permissions, making 
public only those you want to.
