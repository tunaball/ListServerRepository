It's usually best to install software from the master disks--be sure that 
you have any upgrades, as well. In OSX, I understand, this is especially 
true, as there may be system files that you don't see. I wouldn't clone 
one drive, as there may be files that are specific to a particular type 
of Mac.  

An easy way to connect the two is through an Ethernet crossover cable--I 
don't know if your Macs allow the firewire target disk mode. Then, set up 
file sharing on the two computers.
