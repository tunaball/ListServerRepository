Author of original iBook (2002/Radeon graphics) Dual Monitor Mod has 
Applescript Available - The author of the original iBook (Radeon 
graphics model) dual monitor OF mod sent a note he's posted his own 
Applescript enabler:

"Hi Mike,

I updated my website with instructions on how to revitalize dead iBooks...

I also wrote my own AppleScript to enable screen spanning and 
clamshell mode as a new feature. It's in English and should work with 
every machine that could use that patch before (clamshell mode 
obviously can only be used with iBooks). It checks if you have a 
supported machine. (iMac G4s with GeForce4MX's (not Geforce2MX) and 
later eMacs w/Radeon 7500 are other machines mentioned previously)

If you don't have one it will refuse to run.

I didn't release a script or a program to automate the process before 
because I thought that a one-click process would lower the inhibition 
threshold to use a patch that might cause problems. However, since 
there now is one out there (that is not available in English and 
didn't include any warnings) I decided to release my own which I 
think does a better job. The patch is available upon request from 
http://macparts.de/ibook

Best regards -Klaus"

(english page is at http://www.rutemoeller.com/mp/ibook/ibook_e.html)
