> I'm puzzled. How does a "conduit" for entourage help you sync your 
> Palm? It opens up the USB port, or wot?

In the Palm OS, I think a "conduit" is a set of software rules to 
define how different types of data are handled and exchanged among the 
different applications that can use them. When there isn't a "conduit" 
to exchange data between an application and a Palm Computer, it 
probably means there's no way for the application to seamlessly make 
sense of it.
