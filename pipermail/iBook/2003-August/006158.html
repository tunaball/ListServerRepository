An encrypted disk image would solve problem 1.

Using Carbon Copy Cloner you can back up to a disk image. Keep it on 
the external drive and have it as big as you like (or need) and mount 
it when you need it. If anyone stole the drive all they would see if 
they mounted it was a hard drive with a large disk image file on it 
that they wouldn't be able to mount or crack into.

I use disk images like this, and on the system drive for storing 
sensitive info. I can keep one disk image on my system drive that 
contains all my sensitive folders and I only mount it when I need to 
access it. At all other times it's just sitting on the disk. You need a 
mixed case multiple word passphrase to access it and I don't keep any 
of my disk images passphrases on my keychain.
