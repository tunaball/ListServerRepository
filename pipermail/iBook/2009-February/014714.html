I do love my iBook so far. I decided to go with 10.4.11. The latest 
version of Firefox lets you zoom in and out of web pages, which helps 
TREMENDOUSLY with the 800x600 resolution on the Web. Previous versions 
only let you make text smaller or bigger, but images stayed the same. 
But, you need 10.4 to run this version. I'm going to max out the ram at 
576 (64 + 512 chip) and see how it handles 10.4. So far it can be a bit 
sluggish at times, but it's still usable.

Thanks to everyone for their comments and suggestions, I appreciate it.
