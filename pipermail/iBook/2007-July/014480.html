I'm not sure it will work that way, but it is worth a try. The target  
computer in this case (MacBook) will mount on the host computer  
(iBook), but I think only the hard drive (MacBook) will show up on  
the iBook. Only the master ATA drive will be available to the host  
computer. Check the tip near the bottom of this link:
