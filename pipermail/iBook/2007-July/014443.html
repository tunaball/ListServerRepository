Janice,

Start the MacBook from the OS X DVD while the iBook is in Target Disk  
mode. When you start the installation from the DVD (or CD) this way,  
the target disk (iBook) should show as a destination disk.

If you are seeing the OS X install disk on the desktop, then you may  
have started from the MacBook HD.
