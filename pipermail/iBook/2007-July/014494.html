I pretty sure I've read that all 802.11n routers slow approx. 20-25% mixed
mode.  You get right at (or slightly over) 100 Mb/s in 802.11n mode when
using the 5 GHz band and 75-85 Mb/s in mixed 802.11/b/g/n mode.

If I recall correctly the speed drop is caused by mixed mode requiring use
of the old 2.4 GHz band.  At 5 GHz the N routers are able to transform 2
channels into a single big data pipe, but if implemented at the 2.4 GHz band
"channel bonding," as its called, interferes with bluetooth, which operates
in the same band.

Still 80Mb/s is pretty decent speed.
