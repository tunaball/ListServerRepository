Lots of folks have had trouble with this. See MacFixit:
http://www.macfixit.com/article.php?story=20040705075638430

which talks about an Apple KnowledgeBase Doc:
http://docs.info.apple.com/article.html?artnum=25788

which gives these steps
1.  	 If the Java 1.4.2 installer is open, quit it.
2.  	Click the Finder icon in the Dock.
3.  	From the Go menu, choose Go to Folder.
4.  	Type: /Library/Receipts and click Go.
5.  	Find, then move the package file "QuickTimeJavaUpdate.pkg" to the 
desktop
6.  	Install Java 1.4.2
7.  	After installation, move QuickTimeJavaUpdate.pkg back into 
/Library/Receipts

Hope this works for you.
