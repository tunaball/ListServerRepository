I left out some potentially crucial consumer information. The Belkin FM 
transmitters use the headphone port on the iPod, so you can plug your 
remote control in, and then plug the TuneCast (I finally remembered the 
name) into the remote control head, thus allowing use of the remote 
control. The iTrip plugs into both the headphone port and the remote 
control port on the top of the most recent generation iPod, so the 
remote can't be used with it. I've never really bonded with the remote 
anyway, so I don't miss it. My friend with the TuneCast uses his 
remote, so this is an important feature to him. The TuneCast also works 
with all generation iPods. I can't say for sure if the iTrip only works 
with the "docking" iPods or if it can be turned and work with the older 
ones.

And that's pretty much all I have to say on that topic.
