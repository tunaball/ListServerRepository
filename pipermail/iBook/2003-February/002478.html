Don't make life so difficult by doing it all manually. Just run Software
Update. It will do a good job of handing the installation for you. You may
be unsure what updater you need, but it knows exactly.

If you are the type that needs to keep a copy of the installer, then after
you've launched Software Update and checked the items you want to install,
simply go to the Update menu and choose "Download checked items to desktop".
Obviously not everyone is aware of this. This will allow you to keep the
updaters. Once Software Updater has completed the download(s), then tell
Software Update to install it/them. Or run them manually yourself.

Also, if you need information about what's new in each update, then click on
it in the Software Updater window and read the description provided.
