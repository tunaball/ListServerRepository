This sure sounds like a permissions problem or a .plist (preference) 
file problem.

Try deleting com.apple.iCal.plist and com.apple.iChat.plist from the 
Home/Library/Preferences folder.

If that doesn't solve the problem, try running Disk Utility and "Repair 
Permissions." (I can't imagine that they wouldn't have done this at the 
Apple Store, but hey it doesn't hurt to try).

If that doesn't fix it, let us know.
