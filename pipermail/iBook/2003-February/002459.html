The software update for 10.2.4 REQUIRES 10.2.3. So, if you don't have 
that installed software update installs 10.2.3 instead of 10.2.4.

There is an installer on the apple web site for 10.2.4 that will update 
many earlier jaguar versions to 10.2.4.

As to how long an install takes, it does tell you what the size of the 
install is. Thus you multiple that by your minute per megabyte figure. 
My dialups take about 5 minutes per megabyte.

And so, the 10.2.4 download took about two hours.
