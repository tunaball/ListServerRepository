One other note, to make sure everything runs smoothly, I always launch the
Disk Utility and repair permissions right after installing an OS upgrade. I
don't know if this truly helps or not, but I have yet to experience a kernel
panic or any other system problems since going with OSX (starting with
v10.1.1).

You might also start up from a current OSX install disk and run the disk
repair utility from there (different from repairing permissions).
