> In Panther there are two ways to connect to a server and which way you 
> use
> affects the behavior.
>
> 1) command K and type in the server address: afp://193.32.11.32 for 
> example
>    doing this you'll see the computer mounted on the desktop and the 
> sidebar
>
> 2) use the network browser and double click on a server
>    connect this way and you DON'T see the server mounted on the desktop
>
> Using the second method, if you re-open the network browser window 
> you'll
> see that the computer you are connected to as an icon that is slightly 
> more
> blue in color. That is the only hint that you're connected to that 
> computer.
> If you select that icon, you can eject it using command E or from the 
> file
> menu of the Finder
