*** Your iBook network preference is setup to use the build-in modem to get
out on the net. What you need to do is setup a new Network profile in order
to activate your build-in network card.

Go to Apple menu and into the network configuration pane. There you have to
create a new profile, select the Network card and deselct the modem.

Sorry, I can't be more specific as I'm on a French iBook.

While you at it, you might even create a third Network profile where you
would leave everything blanc and deselect the modem and the network card.
You could call it "On the road", "or on the move". The point is to not to
use  both the modem and the network card when you're far away from a phone
or network connection. It helps you save a bit more battery juice.
