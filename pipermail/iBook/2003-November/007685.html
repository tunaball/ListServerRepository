Not too familiar with the G4 iMac but I would have thought that they used
Airport Extreme? Will an older Airport Card fit?

When you do get the cards I would suggest using the AirPort Setup Assistant
as it will ask you how you want to use the cards and set them up for
you...Don't sweat trying to pre investigate how and what to configure...it
is all fantastically easy...

BTW, If you are using panther this should be a dream to do...Doing a clean
install earlier I thought I would need to put in quite a few details and
configure mostly by hand...it was a pleasant surprise when it didn't even
blink and auto configured all settings and even showed me the latest updates
for Panther before finishing the install...
