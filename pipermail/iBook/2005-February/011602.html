There is a program called Screen Spanning Doctor that will let you 
use the external display and LCD as separate displays or mirrored. 
The nice thing about this is that the OS already supports it and all 
the program does is to set some NVRAM parameters that enable the 
function.  I've been using it on my iBook to watch a DVD on the TV 
and still use the laptop display for the usual stuff.
