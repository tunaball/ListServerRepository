I have Virtual PC 7 on my ibook G4 933 to run AutoCad 2002Lt.

There is a definite speed issue when using the emulator, but I'm able 
to work on variety of dwg files at work
and email or use idisk to open the files at home.

Just keep in mind that the laptop will be running the OS9 system then 
Virtual PC then Windows 95/98/2000/XP OS (depending on what version 
Virtual PC you can run on the clamshell) and then your LoopCad 
application,
So max out your memory!!!

Virtual PC is now a Microsoft product.

I don't need it for word documents, I have Office X for Mac.
But there's no reason I couldn't load up Microsoft Office for Windows, 
and run it under the Virtual PC.
