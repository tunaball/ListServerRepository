Basically like OS 9, you can do a clean install of OS X over another OS X
installation. (I had to do it with Jaguar when an update went wrong.) It
will create a folder called Previous System and put the old System & Library
into it. Some programs put items into the root Library's Application Support
and other folders. You'll either have to manually move them or reinstall the
applications involved - they include Adobe products, Stuffit, Keynote, Palm,
and Stone. (It isn't a complete list.) I'm of two minds as to whether
manually copying them is a good idea. I've settled the matter by keeping a
clone of my System on an external drive.
