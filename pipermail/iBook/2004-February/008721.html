G'day everyone,

I recently discovered a very interesting project. It's called Darwine
(http://darwine.sourceforge.net), and it's an attempt to port the gnu
program, Wine (www.winehq.org), to Mac OSX.
Wine is a program that allows windows applications to run on x86 *nix
platforms (such as Linux, FreeBSD, etc), but it doesn't yet run on Mac
OSX - it needs PC hardware. Thus, Darwine will allow Wine to run on
PowerPC hardware (eg. G3, G4, G5)

*
What this amounts to is that, when more mature, Darwine will allow
people to run windows applications on Mac OSX. 
*

Darwine is a gnu project, so it needs volunteers to help out with
development and testing. At this stage, it's still targeted at
developers, but I reckon it's a project that's _definitely_ worth
keeping an eye on!
