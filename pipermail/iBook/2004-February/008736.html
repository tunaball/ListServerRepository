According to Apple the following iBooks use the A/V
cable to output to TVs.

I have used it on my ibook 500(cable came with) it
works great for dvds.



Compatible computers

*	iBook (FireWire)
*	iBook (Dual USB)
*	iBook (Late 2001)
*	iBook (14.1 LCD)

The Apple A/V cable came with the iBook (FireWire)
computer. The Apple A/V cable can be used in
conjunction with the composite audio/video port to
connect the computer to a television. This connection
requires the optional Apple AV Cable, which is
available from The Apple Store
(http://www.apple.com/store/), or from an Apple
reseller. 

NTSC Resolutions

*	640 by 480 60 Hz (NTSC)
*	800 by 600 60 Hz (NTSC)
*	1024 by 768 60 Hz (NTSC)
PAL Resolutions

*	640 by 480 50 Hz (PAL)
*	800 by 600 50 Hz (PAL)
*	1024 by 768 50 Hz (PAL)
