I leave it this way by default - if a spam email gets by the filter and 
loads it makes a request to a remote server to load the image and thus 
the spammer knows your address is valid and marks you up for 20x more 
spam.

Nasty stuff.

I set up a filter that transfers all html mail directly to the junk 
folder no matter who it's from. I check there every so often to see if 
there are any messages from non-spam sources. 99% of the time, it's all 
spam.
