I've found that odd things happen if I remove things while my iBook is 
asleep - even something as innocuous as my headphones. It usually 
causes the display to cycle inputs - ie, as if it detects that it's 
attached to a television so it sets itself to the "1024 50Hz Pal mode" 
setting, then back to the regular 1024 LCD. The result of this is that 
iTunes doesn't unhide unless you quit and restart it and my desktop 
picture doesn't show unless you change the tile/centre option in the 
display prefs.

I try to remember to unplug things before sleeping. The power cable 
doesn't make a difference though, you can take that out if the machine 
is asleep.
