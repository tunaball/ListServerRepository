Yep.

When the Bell South installer finally arrived to install my DSL, I 
faced the same problem described by the original poster.

All you need is to be able to connect to a properly install modem. If 
necessary, borrow a windows machine.

The Mac provides all of the networking choices and you just have to 
fill them in.

There may be an issue with using the ethernet card number but you can 
find that in Apple System Profiler.
