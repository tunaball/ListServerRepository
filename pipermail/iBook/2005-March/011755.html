On 20/3/05 8:42 pm, "Larry Kollar" <kollar at alltel.net> wrote (in part):

> You might need to run update_prebinding. You can do that from the
> Terminal (the command is update_prebinding -root  /  -force) or use a
> GUI tool like Onyx to run it for you.

To add to Larry's good suggestion:

If you have anything pre-10.3 then the suggestion is a good one. 10.3 and
later generally do not need this as whenever any apps are installed
(especially Apple or OS updates) this is done automatically during the
'optimizing' end period of the installs. However, it can't hurt if it has
been a while since you have installed an update.
