time whenever a new set of models arrive.
Went to the site of Xbench, particularly the Comparison Site at 
http://ladd.dyndns.org/xbench/, to see if it was worth upgrading.
Typically the G3/800/640 has a benchmark of something like 50 up to 60, 
while G3/900 is up to 65 and the G4/800/640 is about 80 up to nearly 
90. Powerbook G4/800 (Ti) (I don't know the PB models so well, maybe 
this is not the best model to compare with) shows more like 90 up to 
100.
For each model there is a quite large range of benchmark values, but I 
think the data can be used to indicate the relative performances, there 
are quite much data for each computer model.
