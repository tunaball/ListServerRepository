After suffering through several more freezes, long repeated restarts, and HD CD checks which proved negative, I started looking at what else was new, software-wise that might be causing my problem. I had recently downloaded Cunning Fox which permitted me to not only watch my memory usage, it also has an application "kill" function which immediately turned off any selected running app which saved a lot of time if I wanted to quit or if a restart were necessary for software installation. As soon as I went back to the book method of quitting an application instead of using Cunning Fox, the freezes stopped. At least so far...
