Kathryn Antoinette Seth wrote:

> After buying & receiving my clamshell, I was so eager to use & set  
> it up to receive and send email, that I plumb forgot to change the  
> other person's ( short name ), so what it looks like now is: My  
> name _ _ _ _ & so on / the faded out ( short name ) of the other  
> person...
> only administrators can change an accounts name, & even they Can't  
> change an accounts short name after the account has been created.

The easiest (and recommended) way to handle this situation is:

1) Create a new user account (with the right short name & everything)  
using System Preferences -> Accounts
2) Copy everything over from the old account to the new one
3) Delete the old account

I believe that it's possible to directly change a short name, but  
it's a lot more hassle than it's worth.
