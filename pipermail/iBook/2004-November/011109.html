Todd Doerpinghaus wrote:

> Does anyone know a way to get an ibook to print in
> classic without AppleTalk?  They won't enable it so I
> am stuck until I can find a workaround.

Here's a workaround:

	1) Print to a PostScript file.
	2) Use "ps2pdf" (included in Panther, get GhostScript for Jag) to make 
it a PDF.
	3) Open the PDF in Preview and print it.

Or... I think Panther's Preview might be able to open the PostScript 
file directly. Try that if you have Panther.
