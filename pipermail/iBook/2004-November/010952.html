Nancy,

I have the same iBook and at 640MB it does just fine for me. Now, this 
is my second machine and I'm not doing photo or video editing or 
anything super memory intensive. I have run PhotoShop Elements and it 
does just fine on files up to 120MB while having Safari, Mail, iCal and 
Address Book all open and Airport connection open. Now , if this was my 
main computer OR I intended to run PS CS I would go for the 1GB.

As for buying ram, I've had good luck with OWC. I have purchased 4 
different ram upgrades over the last 3 years and I've been very 
pleased.

JeffW.
