David Harris wrote:

> I need to reset the systems administration in my 900 Mhz iBook as I  
> am in the process of selling it for personal reasons.  What is the  
> correct process to do this?

First, if you haven't already done it, create an Administrator  
account. I use "Administrator" for a long name, and "admin" for a  
short name. Give it a password that's easy to remember & remind the  
buyer to change it.

Once you've done that, and have all your stuff off the iBook, delete  
all the other accounts. If the buyer isn't Mac-savvy, walk him/her  
through creating a new account (and changing the admin password).  
Panther & Tiger especially make it easy to work normally with non- 
admin accounts -- use Fast User Switching to cut over to the admin  
account for installing software & doing system maintenance, and work  
in the normal user account otherwise.
