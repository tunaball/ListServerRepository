Given the specs of the PowerBook I'm guessing it is a first generation 12"
model. If so, I'd go with the iBook. You get a bit more with the PowerBook
but the iBook is newer and not really bare bones. All it really needs is a
bit more RAM which you can add with the $200 you save. When it comes to
notebooks age is a serious factor in my mind new trumps used. The only
factor that might push me to the PowerBook would be if it has AppleCare.
