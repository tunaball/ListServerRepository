Preview definitely doesn't contain OCR tools. Try to select the second 
piece of text from this sample file: 
<http://scruss.com/dump/preview_ocr_test.pdf> -- it's an embedded bitmap.

I've never found the free OCR tools to work well, and I've spent many 
years looking. OCR got real expensive since ScanSoft bought up all the 
patents. I use ReadIRIS, and it works pretty well: 
<http://www.irislink.com/>.

It pains me to say it, but the OCR built into the Microsoft Office 
Document Viewer is quite good.
