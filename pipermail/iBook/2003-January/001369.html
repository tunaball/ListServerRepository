A USB hard drive is infinitely slow, about the same speed as a Zip 
Drive or floppy, due to the nature of the USB bus which is maybe 50 
times slower than Firewire which is slower than the internal hd.

If you do get it to work it will be real slow. Whether you have timing 
issues, I don't know.

At the first setup I was backing up a Filemaker server to an external 
usb drive, about 5+ minutes. When they switched to a Tower I had 3 
addtl internal drives installed for backup purposes, a backup every 30 
minutes to a new folder. Time for each backup dropped to 10 SECONDS.

Since OS X uses Virtual Memory, I would suspect you wouldn't care for 
the performance on a USB drive.
