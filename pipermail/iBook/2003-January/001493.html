One great resource that I always recommend to new Mac users with lots of
basic questions is to use your Help Center, already installed on your
machine. 

While in the Finder (your desktop), it's under the Help menu. If you're in
an application and choose help from the Help Menu, it will open up help with
info specific to that application.

One nice thing about the Help Center is that if you're online it will
retrieve the latest help info directly from Apple.
