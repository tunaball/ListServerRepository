Try using fsck to fix any problem that may have been caused by foolish 
'tinkering'...

Start up and hold down Command+S to drop into the UNIX repair program. 
You should see a black screen and really old fashioned white text. At 
the bottom of the text is the instruction on how to type the fsck 
command. If you get this far and if you are lucky, the hd will return.
