A PDF file is still not an image anymore than a Word document or 
AppleWorks document is an image. For example, one can select and edit 
text in a PDF--try that with a JPEG, for example. A PDF is somewhat like 
an Illustator file, but not exactly. 

It is only the latest upgrade of Graphic Converter that has added this 
feature, as I found when I got the 4.5.2 update. In any case, one can 
bypass the problem by using Snapz Pro X, which also gives you a lot more 
capabilities.
