Yep, Apple's Address application is limited and has some short comings. 
Versiontracker.com lists various third party freeware and shareware 
apps to deal with this.

However, let me upgrade my suggestion to using Filemaker for your 
addresses. It is far more flexible than any application I've seen and 
you can customize it to do what you want. It's easy enough to learn so 
that you wouldn't spend much time getting started. It also has 
templates that might be all that you need.

All you need to do is export your addresses as a tab delimited text 
file and Filemaker will create a new file for you using that data and 
you are off and running.
