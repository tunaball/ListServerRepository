> Recently Mail has been unexpectedly quitting on me. It seems to happen

> Delete,----<poof!>----Mail quits, giving me the Unexpected Quit
> message. It doesn't affect anything else.
>
>   I'm on Jaguar 10.2.3, with Apple Mail 1.2.3 (v551), both running on 
> an
> ibook 500MHz dual usb with 640 mb.
>
Try the following in this order:

1. Rebuild the affected mailbox (the command is the last one under the 
"Mailbox" menu).

2. If that doesn't fix it, move the file "com.apple.mail.plist" to the 
trash (don't empty the trash just yet). The file is in your Home 
folder->Library->Preferences folder. Then re-launch Mail.

3. If that doesn't solve the problem, try running Disk Utility and opt 
to "Repair Permissions." See if any of the repaired permissions involve 
Mail.

4. If that doesn't solve the problem, download the free "Jaguar Cache 
Cleaner" from versiontracker.com or MacUpate.com and run it. Restart 
the machine.

Let us know if that solves the problem.
