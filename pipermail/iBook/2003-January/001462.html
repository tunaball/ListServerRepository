It depends on what kind of a server application and operating system 
(Mac OS 8/9 or X) you are running. For some apps, extra RAM is *the* 
best way to accelerate performance, typically for use as a cache.

Are you just running personal file sharing? Extra RAM will probably not 
help.

If you are running a web server such as WebSTAR, you can allocate extra 
RAM which is used to hold a copy of files already served. Successive 
requests will be served from the RAM cache rather than from disk (disks 
are way, way slower than RAM). This same thing goes for database 
servers, such as 4D Server or SQL servers like FrontBase.
