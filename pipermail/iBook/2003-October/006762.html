As politicians might say on the witness stand, I have refreshed my 
memory and will now state:

If you buy OS X as the standalone package for $129 or less, you will 
not receive OS 9 with it. If you are going to install it on a Mac with 
OS 9 installed, you are OK. If not, you will need to buy OS 9, perhaps 
OS 9.2.

If you buy OS X installed on a new Mac, OS 9 will be installed with it 
and available for reinstall on the system CDs shipped with the Mac.
