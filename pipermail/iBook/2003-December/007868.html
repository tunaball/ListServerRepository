It's difficult to say what the reason for your 2nd problem could be. It's
not a definite hardware failure so it might be the software installed. As a
rule of thumb, always do a clean install when you get a computer, even if
it's new.

To install, boot on the CD and format your disk before installing the
system. You will use the disk utility, which you can access from the menu,
if i remember well.

Then try to use your iBook and see if you still have problems. get back to
the list if you do.
