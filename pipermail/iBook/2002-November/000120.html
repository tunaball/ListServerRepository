the boot check is for the startup blocks of data ("is there a bootable 
system here?"). fsck is a File System Check (thus the name) that 
examines the directory thoroughly (rather a lot like what DiskWarrior 
does as part of its magic).

Running Disk Utility does much the same thing as running fsck, but Disk 
Utility does not run at boot.
