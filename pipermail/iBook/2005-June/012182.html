I've used it on my iBook G4 14" (933Mhz) -- the dual monitor mode  
works great -- I've run a 1600x1200 external and the 1024x768  
internal w/o any issues

However, DONT install the patch that lets you run the iBook with the  
lid closed -- this will defeat the passive cooling through the  
keyboard surface area.
