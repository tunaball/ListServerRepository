G'day Michelle,

I'd guesstimate 25 mins for a 4MB file with a 56k modem.

You need iTunes to access the Music Store (presumably you have it).  
Click on Music store - you can browse the songs, and don't have to  
buy anything.

Note that it's not available everywhere. If you live in Canada or the  
US, or almost anywhere in Western Eurpoe you can get it I think. It's  
not available in Australia yet.. :-(

iTunes store uses DRM that restricts what you can do with a file once  
downloaded. My understanding is that you are able to burn the file to  
an audio cd, and put it on 'limitless' ipods... you should double  
check this, because these conditions can change.

There are utilities to allow you to strip the copy protection from  
itunes music store (ITMS) files. I've never used them, but one name  
is Hymn. Try googling that if you're interested.


HTH,
-Angus
