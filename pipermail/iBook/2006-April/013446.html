On Apr 3, 2006, at 6:00 pm, Vern Andrews wrote:

> I don't know which e-mail address I used, so I can't unsubscribe. How 
> do I cancel the subscription? Thanks.

Take a look at the full headers for this message - or any other message 
sent to the list. There will be several beginning "Received:" that 
trace the route taken by the message as it ping-pongs from server to 
server. In one of these headers you should find the email address you 
subscribed from.
