I looked into this a while ago and discovered that OS X does not support
using shared windows printers directly. I believe Panther will address this,
but someone who is more familiar with the soon-to-be-released Panther will
have to confirm this.

With Jaguar you still have a couple of options. While not terribly
difficult, they are not trivial for the average non-computer aficionado.
Also, your mileage may vary greatly:

1. You can try to get it working using Postscript emulation & Unix print
services. Check out the following link for step by step instructions:
http://iharder.sourceforge.net/macosx/winmacprinter/

2. You can try the combination of Gimp-Print and Samba. You can download
Gimp-Print from www.sourceforge.net or www.versiontracker.com

3. Buy a print server and move the windows printer to the network.

After much trial and error with software only solutions, I eventually bought
an inexpensive 1-port print server and set it up using IPP. Now all the
computers in my house (3 laptops, 3 desktops, running OS 9, OS X, Win XP,
and Linux) all talk to my Photosmart printer just fine. I use the drivers
from Gimp-Print for my OS X Powerbook and Linux desktop.


Hope this helps,
Barry
