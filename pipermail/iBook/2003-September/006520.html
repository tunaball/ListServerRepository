Following-up my own message. Bad form, but not as bad as
leaving a mistake uncorrected....
 
> install GhostScript and print a PostScript file from Preview or 
> Quicktime. Go to the command line and type "ps2pdf -dPrinted file.ps 
> file.pdf" and see if it does any better.

That command line should have been:

    ps2pdf -dPDFSETTINGS=/printer file.ps file.pdf

Sorry about any confusion.

Replace "/printer" with "/prepress" if you want maximum sharpness & don't care how big the PDF gets.

See http://ghostscript.com/doc/AFPL/8.00/Ps2pdf.htm for all sorts of details.
