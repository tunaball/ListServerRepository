Actually, according to most of the reading that I have done, OS-X is 
supposed to be using the same installation regardless of the hardware. 
I have loaded a Mac OS-X system on my iPod and have booted various 
computers (white and clamshell iBooks, Beige, B&W Mac, Cube) and not 
have had any problems (I installed the system by cloning a new 
clamshell install).

Still, I agree with the belief that the best way to avoid problems is 
to do a ground up installation from the original System and application 
install disks. I just feel safer and things just SEEM to run smoother.
