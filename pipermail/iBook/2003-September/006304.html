Not with a G3 processor.

To use an external DVD burner you need to use apple's DVD studio pro to 
make the disk (iDVD only works with internal burners). Both iDVD and 
DVD Studio Pro require a G4 to run (single 733 or Dual 400 minimum).

The iBook wasn't designed for DVD work, although it works quite well 
with iMovie if you export back to the camera via firewire.
