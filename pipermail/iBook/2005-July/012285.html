I had a similar problem not to long ago, it turns out there was a  
corrupt file attachment. Try limiting the attachment down load size  
to a small number(preferences) then once it downloads the messages,  
delete the attachments from the server or If you can access the  
account using a different client try removing the message/attachments  
from the server.
That solved my problem.

good luck
