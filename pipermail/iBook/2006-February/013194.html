Hi,

We started using this at work and got pretty amazed.

http://synergy2.sourceforge.net/

In short: "Synergy lets you easily share a single mouse and keyboard
between multiple computers with different operating systems, each with
its own display, without special hardware. It's intended for users
with multiple computers on their desk since each system uses its own
monitor(s)."

So you can use your iBook and your windows machine at work (if
applicable) and use only one mouse and one keyboard. Great open source
tool :-)
