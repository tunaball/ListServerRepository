Join the iBook Brigade - we work anywhere. Just be sure you stack far enough

Been there done that. When my local bank changed its website and could only
be used via Exploder I visited the bank and was told that internal research
showed we Mac users were a minority. Too small to be reckoned with. I asked
if the research had looked beyond mere numbers and got a blank look so I
expanded on the idea. Say Mac web users make up 5% of the bank's web usage
but because we tend to be upscale we actually account for a much larger
proportion of profits. Had anyone compared Mac users to account activity,
loans, savings, etc? Within a week the website had a link for Mac users to
access the old system while they made changes to the new one. Whenever I see
needless, as one website calls it, Macrimination I fight back.
