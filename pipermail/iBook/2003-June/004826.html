OS X sees a return to the days when you search for files in the Finder; 
either by bringing up "Find" using Command-F (where you can also search 
by content), or with the interface that's an optional part of every 
Finder window.

Sherlock still does in OS X what it did in OS 9; provide a means to 
find all sorts of different information (schedules, listings, pictures, 
phone listings, and so on) quickly and easily.
