If nothing else works, you should be able to start up from CD and
reinstall.  You will need to make sure that your personal data aren't
bounced, though.  If you have no important data, then it is pretty
simple.  If you *do*, then the easiest thing to do (probably) is to
boot off CD and copy your files to an external device (e.g., a
Firewire drive).

Another less serious thing you could try would be to boot to single
user mode and run fsck, but it sounds like this "time synchronizer"
thingus would still fail.  There are instructions for doing this
somewhere in the list archive.
