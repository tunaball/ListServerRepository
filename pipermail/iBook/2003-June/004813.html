That said, my computers only have 8 other "user names" besides my own 
-- but then I'm the only person using the machines.

So if you're the only person using the machine and you have 16 
accounts, it sounds like you either have another "active user" or there 
was one left over from before you got the machine (I see this a lot of 
"floor models" and "demos" etc).

I agree with Allan's advice, it's unwise to delete any users unless you 
are sure you know what you're doing. Otherwise disaster may follow.
