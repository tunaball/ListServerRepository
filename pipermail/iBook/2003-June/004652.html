The Apple thing is a separate lawsuit.

The Open Group, which owns the name "Unix" is suing Apple for claiming 
OS X is "Unix-like" without paying the $110,000 licence fee to use the 
name, and submit OS X to the array of tests they have to ensure that it 
meets the Unix standard and can thus carry the name.

The fact that The Open Group's own website lists OS X as a true Unix 
vendor doesn't seem to be a factor.

They just want Apple to pay up.
