Not always - There are several ISPs here in the UK that flat out refuse 
to help you if you have a Mac, and some that won't let you connect with

tell you "err, it should work, but I don't know about Macs.. err, 
reinstall Windows on it maybe?".

My cable provider doesn't officially support OS X, but it does support 
OS 9. Fortunately, they don't use any special software for connection 
so I can just use OS X's built in networking stuff.
