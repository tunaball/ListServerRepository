> From: tinman <heart93 at woh.rr.com>
>
> I have heard a lot of good things about Cocktail. The problem that I 
> have is
> I did not install the BSD subsystem (which Cocktail apparently 
> requires)
> when I installed Jaguar. Is there any way to install this without 
> completely
> reinstalling Jaguar?

Yes.

1. Reboot from your OS X CD.
2. Run the installer as though you were going to re-install all of OS X.
3. When you get to the part where you can "customize" the install, turn 
OFF everything but BSD subsystem.

(depending on how old your OS X CD is, you may want to run Software 
Update afterwards to see if anything "new" shows up -- or possibly the 
whole 10.2.6 Combo Update package again, just to be on the safe side)
