I think it depends on how you handle the iBook. If you carry it around 
a lot, you're more likely to have the logic board problem. If you pick 
it up one-handed, it can flex just a bit and cause problems. If you use 
it on your lap mostly, you're more likely to have problems than if you 
leave it sitting flat on a table most of the time.

-pam (yes, I figured these things out the hard way - I've had 7 logic 
board failures with 4 different ibooks)
