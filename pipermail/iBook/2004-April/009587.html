The latest version of Outlook Express for Mac is 5.0.6, a free download from
microsoft.com.mac

I would suggest you update your OE.

OE has a Junk Mail Filter which is off by default.

You need to create a Junk Folder and a rule to shift all the mail labelled
as Junk to this Folder.
