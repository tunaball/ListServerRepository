I upgraded my icebook to a 40gb 5400rpm toshiba with 16mb cache.  It ran a
little warmer, but worked well.  You need a slim 12.5mm (I think) hd for an
icebook. Try www.xlr8yourmac.com for info on upgrades.

It was rather complicated, but I found a site with all the info I needed.  I
saved it as a file for IE.  I also found some tech docs that helped.  I can
e-mail them to you if you like.
