You should only have to type in a MAC address if you have the "restrict 
access to these mac addresses only" option turned on... however, a mac 
address is only 12 hex digits like 08:1c:09:f9:b1:dc (sometimes they 
have : in them, sometimes they dont) and not typically a 40 digit hex 
number... that might be the WEP code that you would need on all the 
client computers.
