All Macs have ethernet these days.

If you are using OS X:

You need to make sure your ethernet port is turned on.

Go to System Preferences-> Network -> select the "Show" pulldown ->  
"Network Port Configurations" and make sure that the "Built-In  
Ethernet" box is checked also click and drag "Built-in Ethernet" to  
the top of the list.
   Deselect connections you are not using- such as modem or airport.

Choose "Apply Now".

Now go back to the "Show" pull down and select "Built-In Ethernet"  
and choose the various tabs and you'll have to fill in the blanks  
from here...
