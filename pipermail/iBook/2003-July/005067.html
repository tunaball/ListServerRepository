I believe the best Get Info Ram allotment is 25 Megs. TCP should be 
used. Filesharing turned off since Filemaker Server handles its own 
business. Virtual memory off.

desktop is enough to grind the server to a crawl. Filemaker Server 5 
does not run well in the background.

It seems rather odd to dedicate one g4 tower with three 140 gig drives 
to running 400 megabytes of databases but those were the only available 
add in drives and were really cheap.

Also, don't put it on a server, just a plain vanilla tower.
