More freeware.
This one's probably already on your mac.
iTunes.

Set a stop/start time in the song's "get info" box.
Then, in the "Advanced" menu, select "Convert Selection to mp3". It will
make a new file that is reduced in size based on your start/stop times.

I don't know how well the quality compares. Sounds fine with my lo-fi
music & crappy ears. ;)

Allan

On Thu, 24 Jul 2003, Joe Jones wrote:

> Use Audacity.
>
> It's a freeware, open source app available on versiontracker and
> similar download sites.
>
> I won't edit an mp3 file directly, but it will convert it into an
> uncompressed format for editing. You can then export the smaller chunks
> as either wav or aiff files that can be made into mp3s again by iTunes.
>
> Obviously, this means that the file will have been compressed into mp3
> twice, once to make the original large file and a second time to make
> the smaller ones. Choose a larger bitrate for the smaller files to try
> and minimise quality loss, although it shouldn't be too bad.
>
> http://www.versiontracker.com/dyn/moreinfo/macosx/15063
>
> That's the link to the program you need.
