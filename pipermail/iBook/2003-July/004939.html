It might not be the base station. I was having similar trouble, 
including the Admin Utility not seeing it. A few days ago, I realized 
that when I was hung, my son (on the G3 Ethernet'ed to the base) was 
still able to get online. Turning off the AirPort card on the iBook, 
then turning it back on, fixes the problem.

Not sure what's causing the problem for me, unless it's moving between 
home (wide-open) and work (WEP). I only have the hanging AirPort card 
problem at home, so I suspect that the iBook or the AirPort card is 
still trying to use WEP to access a base that doesn't need it.
