First I've never tried this.

Secondly use a common type face, truetype or one of the newer faces, 
and create a few styles. Word saves styles in both documents and in the 
application preferences so you can experiment here. You can set point 
size and leading.

I guess I would first check to see how the styles are being saved and 
would suspect that saving the style in the document might be the answer 
as the pc and the mac word default styles might differ and there is 
your trouble.

Does that make sense? You have three sources of style sheets, pc word, 
mac word and the document. Also you of you are using the font menu for 
styling rather than a define style selection, that may vary on the two 
platforms.

Fonts are also a problem. Various fonts with the same name perform 
differently and i don't want to spend an hour discussing that.
