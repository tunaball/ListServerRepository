No, Diskwarrior does nothing for permissions in X at this time. The 
X-Version might...

Permissions are options and values stored inside files. These values 
and files can become corrupted.

Peermission Fixer changes the values stored in the files. Fixing the 
permissions resolves a large number of problems that depend upon the 
correct permission setting.

Diskwarrior builds a new directory with the addresses to the files.

Think of a permission as a key to the front door and the directory as a 
set of cards listing who lives in a house by the address. Problems 
occur if either one is mixed up but the problems are different.
