possibilities: Your OS X install is hosed or when you tell the system 
to start in OS X it is ignoring your instructions. Try this: In the 
System Folder (that's the OS 9 System Folder) open the preferences 
folder and find the Startup Disk pref. Kill it. Then try selecting OS X 
as the startup disk and reboot.
