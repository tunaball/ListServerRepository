scan sites like version tracker and rez excellent? there are a few little
apps that can help (i.e. shadow killer (kills shadows), etc. look in system
prefs... delete any start up items you don't need use, etc. go to network
and delete any connections you don't use (or make sure your main connection
is the 1st in the list and turn the others off). go to displays and set for
color in thousands). go to energy settings and set for max performance, no
drive sleep, etc. go to applications/utilities folder, use disk first aid to
fix the permissions (need to boot from disk and use ripthe app from there?)
rip the book apart and put a 5400 hd in there (i am not sure if the ibook
can really take advantage of this... larger hd buffer would be plus). if you
are not using airport turn it off. sell the 256 chip and get a 512.... all i
can think of... learn the terminal and you can shut off stuff like dns, etc.
...i think.
