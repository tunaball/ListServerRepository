That's single user mode. Be careful, you are logged in as root.

Run /sbin/fsck -y until it reports that no problems are found. No need to do
the /sbin/mount -uw, just type reboot and it will reboot. If that doesn't
help, try booting into Open Firmware (start up holding Command-Option-O-F),
and type "reset-all" + return. It should reboot after that.

If that still doesn't work, try a safe boot; hold down Shift while booting.
If that doesn't work either, chances are you'll have to reinstall your OS.
