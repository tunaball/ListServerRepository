I've been using this router for about a year, and it works fine with 
the Mac. The setup software is accessed via browser, so it doesn't 
matter if you're on Windows or a Mac. I have two wired computers 
connected directly to it (ethernet) and have used up to 4 different 
PowerBooks and iBooks with it wirelessly.

I was never able to get the print server working, but I think that's 
because it wasn't a postscript printer. USB printer sharing solved my 
problem.
