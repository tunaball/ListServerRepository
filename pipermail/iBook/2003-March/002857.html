You should go read http://www.macattorney.com/tutorial.html.   Speed 
can be optimized by using Fix Preferences.

As for removing OS 9, there's little point.  It doesn't take up that 
much disk space on a modern iBook.   And you don't have to run Classic 
if you don't want to.

Start-up programs are in the Login panel in the System Preferences.   
But unless you've added a lot, there won't be much there that isn't 
already important.

Some programs have their own Uninstaller.  Spring Cleaning X 5.04 is a 
Universal Uninstaller.  I tried it under OS 9 and it made me nervous.  
Ask someone who's used it lately.
