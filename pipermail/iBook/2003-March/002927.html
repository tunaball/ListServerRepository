I already had an ethernet router ( Asante, very mac friendly ) but put 
off going wireless because of the expense of the Airport base station. 
I bought a Linksys  wireless router because it was cheap but it's setup 
is a Windows only program. Because I only wanted the wireless 
connection and didn't need it's router function I just plugged it into 
the router and turned it on. No setup at all. I ran Airport setup on my 
iBook and it works perfectly. My point is; software is not an issue if 
all you need is the wireless access like I did. If you need the router 
function I'm ignorant. My Asante setup was done through a web browser 
so any computer can configure it, if the SMC works this way I bet it 
will work fine.
