Which version of Mac OS X is it? If you have Jaguar, I would recommend 
an archive install. Mac OS 9 knows nothing of Mac OS X permissions, and 
is exactly why you can do things like boot into Mac OS 9 and delete the 
/mach_kernel file.

Try this ... boot into Mac OS X, holding Command-S for single-user 
boot. After running fsck, carefully type the following:

/sbin/mount -uw /

then press return. Note the space between the w and the /

Then, type:

rm /Users/UserName/Library/Preferences/com.apple.loginwindow.plist   
<press return>
rm /Users/UserName/Library/Preferences/com.apple.finder.plist   <press 
return>
rm /Users/UserName/Library/Preferences/loginwindow.plist   <press 
return>

Where "UserName" is obviously your user name. ;o)

(NOTE: you don't have to type the entire thing -- you can press tab to 
complete after typing a few characters. Be careful -- make sure that it 
completes to the intended file)

then, for good measure, type:

rm /Library/Preferences/com.apple.loginwindow.plist  <press return>
rm /Library/Preferences/com.apple.windowserver.plist  <press return>

then type exit.
