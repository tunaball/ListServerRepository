> I'm starting to get spam to the (custom) email address I use for this
> list - I don't use the email address for anything else. Is anyone else
> getting it? Can anything be done?

Sure. If you're only using this address for the mailing lists in  
question, you can use a tool like mailfilter -- http:// 
mailfilter.sf.net -- to delete mail from your ISP's inbox before you  
download it. I use an Automator workflow that runs mailfilter then  
tells Mail to get what's left; it catches a significant amount of  
spam and virus attempts that wouldn't work on a Mac anyway but waste  
bandwidth to download.
