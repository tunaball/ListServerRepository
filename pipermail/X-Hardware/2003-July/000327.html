I've had extremely good luck with VueScan
(third party software) for running many different
scanners under Mac OS 9, OS X, Linux and Windows.

can be found here..
http://www.hamrick.com/vsm.html

looks like AGFA is supported..
http://www.hamrick.com/vuescan/vuescan.htm#supported

I run an older Epson 1200s as well as an
old Minolta Scan Dual.  Both are SCSI, both
work great under Mac OS X with VueScan.

IMHO VueScan is better than the original OEM
drivers that came with the scanners (only work
in OS 9)

hope this helps..
