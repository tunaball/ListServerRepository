>On 28-Nov-03, at 12:48 PM, Florin Alexander Neumann wrote:
>
>> HFS+, but they can handle ISO9660, which is most likely the format 
>> your Windows CD is in.
>
>That makes sense. So then what is the format that Panther burns them 
>in? (when it says that it is cross-platform compatible) HFS?

I believe that the Finder burns hybris ISO9660/HFS+ volumes.  The problem with OS 8.0 is that it will most likely see what it thinks is the HFS volume and try to mount that, which it can't.

If you have Toast, but an ISO9660 volume or specificaly specify an HFS(plain) volume.

If not, use the Panther Disk Utility to create an HFS disk image.  Mount it, prepare it, then burn the CD from the image using Disk Utility.
