On Friday, Nov 28, 2003, at 09:32 Canada/Eastern, Evan Holt wrote:

> Has anyone noticed that CDs burnt in Jaguar or Panther aren't readable 
> on old Performas? I can burn a CD in Windows and it will read on the 
> Performa, so I don't think it has to do with the drive itself.

Most likely, it's a file system issue. What Performas are you talking 
about? What version of the OS? What format do you write your CDs in? 
Versions of the Mac OS prior to 8.1 cannot handle HFS+, but they can 
handle ISO9660, which is most likely the format your Windows CD is in.
