Keep in mind for the older G3's like the beige mini-tower in question, 
the entire OS X has to be in the first partition and that partition has 
to be less that 8 GB. So if you put it a big HD and want to use it to 
boot OS X you have to make the first partition less than 8 GB, 7.9 GB 
being the safe size to put in. One tech note that addresses the issue 
can be found at <http://docs.info.apple.com/article.html?artnum=25249> .
