I also have used IOxperts cam drivers.  I have a
Logitech Pro 3000.  the only hangup is that ichat AV
is designed to work with FW cams only.  The solution
for me was to get ichatusbcam.  It's shareware ($8 I
think) and is now compatible with Panther.  I found it
on versiontracker.com.  To make it run really well, it
is suggested you also install APE, application
enhancer.  The links to this were included with
ichatusbcam.
