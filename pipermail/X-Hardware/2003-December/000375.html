On Dec 17, 2003, at 8:28 PM, Evan Holt wrote:

> Is there a web page that lists the maximum hard drive size for each 
> line? I'm trying to figure out the maximum drive size for the G3 B/W. 
> MacTracker doesn't list all of them, and Apple's site does't turn up 
> much.

For a B/W G3 its 128 GB unless you add a different controller. To use a 
bigger drive without adding a different controller card, you need a G4 
with mirrored drive doors or later. See 
<http://docs.info.apple.com/article.html?artnum=86178>
