Congrats on the new box! Yes, using FW target mode would work just 
fine. There are many and varying views on the best way to do so. Here's 
my method that has worked fine for me

G4 - ensure that you don't have any 'haxies' installed (additional 
Preference Panes such as WindowShadX), if you do, remove them.
G4 start up normally
G5 start in TD mode
CCC all the 'User Stuff' to the G5
Dismount G5 HD from G4 desktop
Restart G5 normally.

You should be OK.
