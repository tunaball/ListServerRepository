Two things probably need to be done... you need the nVidia 3.0 video  
drivers and the latest 1057.019 rom update for you GF3 card!

The nVidia driver can be downloaded from Apple's site:

http://docs.info.apple.com/ 
article.html?artnum=120110&SaveKCWindowURL=http%3A%2F%2Fkbase.info.apple

You will need a program called "TomeViewer" to extract the files as the  
installer will only run on a 4x AGP mac and above (Digital Audio).

The nVidia ROM update can also be downloaded from Apple's website and  
it specifically talks about resolutions with Apple's larger monitors so  
it might help as well:
