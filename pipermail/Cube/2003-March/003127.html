Before tearing of hair and gnashing of teeth try going to the extension
manager and selecting OS 9.x all. Then duplicate and call it test or
something. Reboot and see what happens with Word. It should reinstall what
it needs into the new 'set'. Does something similar to a clean install but
not quite and intrusive. I'm not sure what AOL will do but you might have to
reinstall that too. However I'd work with Word for a bit before AOL is
installed and see if it crashes. Try to duplicate what you were doing when
it crashed. If it doesn't crash then it's not memory settings because when
it starts up and reinstalls it's running factory settings. If it does crash
I'd go to OS 9 minimum extensions. Try again, crashes then it could be RAM
and I'd take the machine down to the minimum that I had and go from there.

A clean install might be in the cards after all that work and finally a back
up, format and reinstall. Even running Disk First Aid when booted from the
Install CD will help along with zapping the PRAM and rebuilding the desktop.

Good luck
