OK, well, didn't know that.  That was the 'bad RAM' hypothesis.  
Several other hypotheses are available:

Overheating hypothesis:  Do you have thermoindock running? I assume the 
machine has no fan.  Any environmental change e.g. books around the 
computer?  How long does it run before it panics?

Disk full hypothesis: Make sure there's plenty of free disk space.  How 
much RAM is in the machine?  Is it possible to configure VM in Classic? 
  Have you tried booting OS 9 and seeing how well it runs?

Compatibility hypothesis: It only bombs in Classic?  Do you have 9.2.2?

USB hypothesis: Any weird stuff on the USB bus?  Try simplifying it if 
so.

I won't mention the bad power hypothesis...

Sorry I can't come up with more, I'm a programmer not a tech support 
guy, but nobody else seemed to be responding.  Hopefully some more 
Cubers will step up to the plate...
