I had the same problem with an early install of OSX, the only thing that
cured it for  me was complete re-install.

Because the core OS is based on UNIX it "should" be Very stable, what you
are experiencing is not what one would tend to expect.

Other things to rule out first:
Remove all other peripherals so you just have a stand alone system, i.e.
cube screen keyboard mouse etc
Try a restart now
If you are still having problems examine the following:

Intermittent fault with the logic board/dc supply
Hard disk file corruption, especially taking note of the various file trees,
if they are screwed, files just start disappearing never to be seen again.


Although personally I would tend to think you have had a corruption in the
install process.

Another option, if you have "upgraded" from os9.x to OSX then when you
reboot hold down the option key this should give you the option to boot into
either OS


See how you get on with that lot first.
