Couple of questions.
1) Can you print a test page?
	If you can't have you tried reinstalling the drivers per Samsung's
directions?
2) Can you print a text page?
3) Can you print a RTF/DOC file that is only text?
3) Can you describe what you mean as a simple graphic?
4) Does the resolution of the item being printed match the print device?
5) Have you checked the Virtual Memory and Disk Cache settings?
	Yes you may have 1.5g RAM but some apps don't behave well if they don't
see these
