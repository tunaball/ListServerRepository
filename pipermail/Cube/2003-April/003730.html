This is for sure your ram.

Take out all the ram except the original 64 and install OSX and run it. 
Re-install the ram one at a time and find out which one it doesn't like.

OS 9 was very dumb about ram. OS X however is vry picky. Make sure you get 
better ram after. Kingston, Corsair, PNY, Crucial...etc.
