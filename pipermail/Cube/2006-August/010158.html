>
> On Jul 2,In previous versions of OSX Mail, there was a little  
> spinning cartwheel that showed activity - downloading, sending,  
> etc.  Now it's gone (sometimes shows up during a send if the file  
> is big - but never for "Get Mail").  I liked it - How do I get it

Apple may have had a problem trying to show too much information with  
1 indicator. It's now been split up so that each inbox and sent  
folder in the sidebar shows it's own indicator. You can also see  
details of what is going on in the Activity Viewer window.

I liked the earlier icon in the toolbar because it was a handy  
shortcut to the Activity Viewer.
