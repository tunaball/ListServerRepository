Marty,

When you had OS X on the cube previously and removed it did you format the
drive. If so which format option, HFS or HFS+. X won't install on a HFS.
Also it won't install from from a slave drive Knowledge base # 106728. Don't
know if the internal Optical drive is set to master but from this article I
guess it should be master.

Also remove any other devices plugged in via USB or FireWire, other than the
keyboard and mouse of course :).

The other things I'd try, zap the PRAM in case some code has been stored
there causing problems. Rebuild the desktop in OS 9.

The other option is try another drive in the cube though I don't know if the
girlfriend would be willing to lend you her drive from the dual G4.

Had a look at apple's error code list which is pretty generic and universal
as far as I know. They don't list a 0 error of course but all the errors in
the lower numbers relate to drive problems or seem to relate to drive
problems. 

The other thing you could do is get a firewire cable, take the cube to the
Girlfriends and put it in Target mode and copy your data over to the G4.
Then do a format, zero data, and start from scratch. Lots of bonuses there,
data backed up, fresh hard drive to install onto, and quality time with the
girlfriend. Zeroing data takes some time.... Good Luck
