I'm not sure this is exactly what you want, but if you put your e-mail program in the main applications folder, not the user applications folder, then anyone who uses your computer can use the e-mail program and access all the e-mail. If you want to be able to use the same e-mail address and have separate mail folders you could do that by setting up a filter on "any recipient" contains "wife name" or "your name" and sending the msgs. To individual personal folders. This would mean the 'kids' could access it too.

I you can set up groups, which could be you and the wife, with access limited the group, but I've never done that, so not sure how it's done.
