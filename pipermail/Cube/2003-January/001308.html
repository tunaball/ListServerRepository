To use iPod has a hard disk:

1. Connect it to your computer
2. Open iTunes. 
3. Select iPod in the iTunes source list.
4. Click the Options button (Figure 1).
5. Select "Enable FireWire disk use." Selecting "Manually manage songs and
playlists" in the Options window also enables iPod to be used as a hard
disk. 

To make it bootable, install a system on it and select it as a startup
device when you want to boot from it (by selecting it in the startup disk
preference pane or holding down the OPTION key at startup to select it from
the startup volume manager.
