Never had this on a client but on OS X Server I have and fixed it by booting
into Single User mode (Cmd-S) and typing the commands below, be warned, it
opens up every file to all users and obviously this is a security problem.

/bin/fsck -y; /bin/mount -vw /; chmod a+rwx /;

It can happen when you set the sharing access for Everyone to 'None'.
Probably else how as well.
