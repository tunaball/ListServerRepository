Entity Marc A. Le Pape spoke thus:

> What would be really neat however would be to tell us, if you know, how
> to assign Applications to an independent partition as well, so you
> would not have to re-install applications when you erase your
> osx_applications partitions.
That's not recommended. Software Update looks on the specified disk
regardless of redirects. Also, some applications require special system
drivers or extensions and simply copying the application to the new
Applications folder will break those apps.
But if you don't mind that, it's possible to copy apps in bulk to the
"Applications" folder.
