You did not elaborate on what your specific problem is, so it's hard
to recommend a 'work-around'.

I'm running PalmDesktop4.1 (you're running 4.01).  So get the latest
version.  Also, you did not say which version of iSync Palm Conduit
you are running, but it should be v1.2 (892k).  I tried re-installing
the original iSync Palm Conduit I was running with Jaguar and it did
not work.  With Panther you need v1.2.

I'm running iSync with Palm on both my Cube and my iBook, both running
Panther, with no problems.
