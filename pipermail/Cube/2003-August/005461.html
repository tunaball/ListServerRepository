The first thing you can do is run a file system check
and fix, by booting with CMD and S held down. Follow
the exact file path which is something like sbin/fsck
-y  and hit enter. Note the space. When it runs it
will fix files and directories, but if it shows some
problems after that then run it again and again. When
it shows system OK then type reboot and hit return.
