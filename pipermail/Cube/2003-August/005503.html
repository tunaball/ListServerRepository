You don't have to totally "leave" Mac OS. As you know, fink runs in 
tandem -- or as  an application with OS X. Easiest way to do what you 
want is install fink commander and then tell it to install the kde 
bundle.

Or, if you have fink already running, you can use the Terminal to do:

sudo fink install bundle-kde

Either way, the interface will ask you to make choices about 
dependencies or whatever, but you'll have kde when you're done.
