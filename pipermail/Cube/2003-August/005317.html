On Monday, August 4, 2003, at 02:00 AM, J. Torrey wrote:

> Sometime a few months ago, my Finder windows started not respecting 
> the location, size, and layout (column views) that I defined to them.  
> When opening new windows, they used to respect my preference; but now, 
> they don't.  Is there a preferences file I need to delete to help my 
> Cube remember this information again?

The view and position for the new windows are set according to the last 
finder window being closed.
Close all finder windows except one. Set the view and size on that one. 
Sloce it. Every new finder window will open with those settings.
