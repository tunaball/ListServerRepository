On Wed, 16 Mar 2005, Melvetta Overturf wrote:

> I have a G4 Cube and also a  Power Mac G4 dual processor unit. I am
> wondering how to install OS 9 on it. It has OS X 10.3.8. Any help would
> be greatly appreciated. Mel

Assuming the DP G4 can boot OS 9, and the hard drive has OS 9 drivers
installed, just boot off the OS 9 install CD and install it (just don't
format the drive first, unless you want to blow away 10.3.8...)

If it can"t boot OS 9, obviously stop here. It can't be done.

If your drive doesn't have OS 9 drivers installed (sorry, don't know how
to check, it is an option when you format the drive!), then you have to
reformat your drive and select that option in disk utility.

HTH
