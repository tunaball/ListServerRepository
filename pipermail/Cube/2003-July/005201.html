> Where is the best place to get iDVD from.  I only have dial up so

iDVD is part of the OS X install discs. As soon as you install the MCE 
Superdrive  (internal superdrive that is) you should be able to install 
iDVD. The install routines will only allow you to install iDVD when it 
discovers an internal Superdrive. Alternatively you can use Pacificst 
(see www.versiontracker.com) to extract the relevant iDVD bits from the 
install disc.
