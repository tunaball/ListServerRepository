Guys,

When I replaced my 20Gb. for a 80Gb Seagate Barracuda hard disk, beside 
the jumper stuff, I had to start externally from my old 20HD with an 
external Firewire case and from there initialized with Disk utility the 
internal 80Gb. Disk. If the disk is initilized in a external device, 
like a external Firewire case and then reinstalled inside the Cube it 
doesn't work !
When the disk is initilized been internal it works without any problem.

I also experimented this problem (Apple Logo and never ending startup) 
with another cube from a friend with the same kind of disk, but maybe 
because of the DC board didn't support the Disk internally we finally 
changed it externally in a FireWire case. Anyway mine is working until 
now perfectly.

Maybe this can help.
