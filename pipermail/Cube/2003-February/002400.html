It is something else about that RAM that is bothering your Cube, not 
the pc133 spec. Actually, there are some PC133 RAM modules on the 
market that fail to failback to PC100, but if the RAM conforms to the 
PC133. it _must_ work as PC100 memory.
It is kind of a lottery really, but if you buy it from a reputabel 
source you should have no problems. Or usually ask if you can return 
them in case they don't work, as you obviously did.
