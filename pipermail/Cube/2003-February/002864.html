It sounds like you have some kind of TCP/IP configuration mess-up.  The
one-way pinging is a sign.  This could mean any kind of problem, like a bad
subnet address, etc.  Are you using DHCP?  Is the router configured to
supply addresses to the ethernet?

You don't happen to have an Airport hub, do you?  I had a problem once where
the Airport hub was set to supply addresses to the Ethernet, and so was the
router going to the DSL, so machines that got addresses from the Airport
could sort of work, but not the others.  It was a mess.  The solution was to
change the Airport hub to not supply addresses to the Ethernet but only to
the wireless users.
