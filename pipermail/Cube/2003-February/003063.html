> hey gang,
> 
> I have lost the cute little site Icons that appear next to the web
> address in Safari. Any Ideas how to get them back. Also does anyone
> know how to get those on you own personal web page.

They're called "Favicons", and are really just ".ico" (Windows icon) files.
Use GraphicConverter to make a 16x16, 256 colour (or maybe wev-safe 216
colour?) image file, then include this line in your web page:

<LINK REL="SHORTCUT ICON" HREF="<your_favicon>.ICO">

For more information, check here:

<http://www.htmlgoodies.com/tutors/favicon.html>
