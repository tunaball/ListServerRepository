My best guess is memory.  People tend to forget how picky OSX can
be about memory.  And even though you may have been running OSX
before the upgrade, you are running faster now, so more of a chance
for marginal memory to make a difference.

If you have more than one memory SIMM, I'd swap them and then check
to see if your symptoms stay the same.  Or is it failing in some
different way now?  If the problem goes away, you're good to go.
If it changes, then see if you can isolate the bad SIMM by running
with one SIMM at a time.

That's what I'd do....
