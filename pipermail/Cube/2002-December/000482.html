On 12/29/02 3:33 AM, simon.meyer at iteams.org typeth:

> I'm trying to network my cube with my upgraded B&W G3.
> Does anyone know of a mac compatible PCI card solution for my G3 or even an
> external solution?
> 
> Thanks, Simon

The Proxim SkyLine PCI card is an option:
<http://www.proxim.com/products/all/skyline/pn476/>

But with the following caveat:

"The following systems will fail with no video output symptom if a Skyline
Wireless PCI Card is installed:
*    G3 beige desktop/300, Machine ID: 510, ROM $77D.45F2
*    G3 beige tower/300, Machine ID: 510, ROM $77D.45F2
*    G3 beige tower/333, Machine ID: 510, ROM $77D.45F2
*    G3 beige desktop/266, Machine ID: 510, ROM $77D.45F2
*    Blue and White G3, Machine ID: 406, ROM 77D.44F1

(If you think you have one of above systems, you can find out for sure by
running the Apple System Profiler utility software and by looking at the
'Machine ID' and 'ROM revision.' If both the Machine ID and ROM revisions
match, then you have one of these systems and the Wireless PCI card will not
work in these machines) "

Failing that, there are several USB Wireless adapters you can use.
