On 2003-06-10 09:56, Andra and Reid wrote:

> Does anyone have a suggestion on how to reinstall cube system soft
> including the firmware by not using the cube but another mac?

You can install OS X (any version) on another computer and just swap hard
drives; OS X installs are generic.

You can do the same with OS 9, but it isn't advised, since OS 9 installs are
specific to the system they are created on. Of course, if you have a second
Cube with the exact same configuration...

You cannot do a firmware install for a Cube on another computer. That is
because firmware is more part of the hardware than the software, really. It
sits in the machine's (non user-accessible) ROM.
