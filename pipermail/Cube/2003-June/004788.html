After you install the 10.2, run the Software Updater to see what it 
wants to install first. Make sure that you use the 10.2.6 COMBO updater 
directly from 10.2.0 and not incremental updaters. It was recently 
discovered that the use of incremental updaters yields a different and 
not as stable system.
