Not necessarily stock...but a 450 with 1.5gb ram, radeon 7500, DVD, and 
a 80gb HD, runs 10.3 just fine.

However, the only really cool features I could see using are expose 
(Which is VERY COOL if you have a lot of windows open...I can't believe 
how easy it is to use and manage!) and Activity monitor, which is just 
like the "top" command in terminal, but also sports pie graphs to show 
you how much memory is in use/available as well as disk space and 
network resources. Very nifty tool. You can also choose a variety of 
icons for that app to show system load in the dock.

I'm sure with time I'll appreciate it more, but it does seem a bit 
buggy and I definitely will stick with 10.2.6 until 10.3 is final.
