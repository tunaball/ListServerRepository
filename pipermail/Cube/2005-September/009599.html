Apple removed the API for some early G4 CPU's (like the Cube's), because
they were grossly inacurrate. Something about a 24 deg. Celsius variance,
meaning that a CPU reporting 20 deg. C might well be running hotter than one
reporting 67 deg. C.

Sadly, this also removes the possibility to do relative temperature
measurements n the same CPU, which is what the OP intended to do.
