Bruce, 
 
You don't specify which OS X you're using and if OS 9 resides in the
same partition as OS X (but assuming it is)... AFAIR (and I'm still
using 10.3.9), a 'security Update' was needed to run 'classic mode'
without glitches. Please check if you have OS X up-to-date in Software
Update. 
 
1st hint: Open Disk Utility (in Applications/Utilities), select your
startup disk and hit 'Repair Permissions' button. Next go to Disk
Utility menu bar... File>Fix OS 9 Permissions. 
2nd hint: Clean every preference that is classic related. Go to
HD/System Folder (with X logo)/Library/Classic and trash 'Classic'
folder contents. 
 
Now open your Classic preference pane and let it show your OS 9 system,
in the 'Advance' button, select 'Turn Off Extensions' and hit 'Start
Classic' button. 
 
Hope this helps you out,
