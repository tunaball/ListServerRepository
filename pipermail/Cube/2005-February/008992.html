John (and all the other good souls out there):

In order to make a .toast file targeting the CD should be as easy as:

1. Insert the CD in your optical drive and let it appear in the desktop.
2. Open Roxio Toast and select 'Disc Image' as format.
3. Point your select destination to 'Hardware Diagnostic' CD inserted and mounted on desktop.
4. Press 'Record' button.

In order to make a .dmg file targeting the CD should be as easy as:

1. Insert the CD in your optical drive and let it appear in the desktop.
2. Open Mac OS X 'Disk Utility' application.
3. Click on the CD icon in the left side navigator bar.
4. Press 'New Image' and give it a name and path.

Hope this helps,
