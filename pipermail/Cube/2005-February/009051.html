Marco: Thanks for the info. Mine was bought after Jan 2001 and definitely came with the CD that misteriously vanished from my CD archive.
If you're planning on upgrading your Cube (PowerLogix, Sonnet and/or GigaDesigns), it is mandatory that you upgrade its firmware version to the latest, 4.1.9f.

If you have another Mac (or a friend of yours) with FireWire, you could try the FW Target Disk Mode trick. Connect both Macs using a FW 6-6 pin cable. Boot your freind's Mac and when prompted, boot the Cube pressing the keyboard's "T" key. That will put your Cube under Target Disk Mode. Now run the upgrade from your friend's Mac and point the Cube as the destination hard disk to run the firmware upgrade. You should get no further errors.

Hope this helps,
