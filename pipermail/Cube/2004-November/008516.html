First off, yes you can clone between different types of hardware. Provided
the OS version you are using is compatible with both (i.e. You can't clone a
10.2 system, for example, to a machine that shipped with 10.3 or later)

Second of all, if you are using Carbon Copy Cloner, you MUST reformat the
target disk prior to cloning the source over to it otherwise you will get
exactly the result you encountered. If you want it to be bootable, you must
completely reformat, with Disk Utility, then clone.

Hope this helps,
