Steve -- have you tried adding the freeware application Safari 
Enhancer?  It enables the DeBug option on the menu bar which allows you 
to select a browser "User Agent"  that may be needed for the particular 
website that you are visiting.  These choices are available:
	Mozilla 1.1
	Netscape 7.0
	Netscape 6.2.2
	Netscape 4.79
	Mac MSIE 5.22
	Windows MSIE 6.0
	Konqueror 3

Safari Enhancer 2.0.6 can be found at      
http://www.lordofthecows.com/safari_enhancer.php
