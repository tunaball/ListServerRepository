There you have it -- I couldn't find Dashboard because
I didn't look hard enough. But spotlight didn't find
it either :~}

The main UI changes post installation that I recall (I
am not in front of the machine now; using a PC at
work):

1) Spotlight on the top menu bar, right corner

2) The System Preferences is generally the same but
there is a "show all" button and the arrangement of
the top menu seemed different (don't completely
recall) 

3) The dock seemed to have been slightly tweeked in
background transparency,

4) There is a new "Tiger" screen background available.

5) It seems like some of the options for showing
things on the task bar were gone. For example, I
couldn't find where to set up PPTP vpn connection. But
again, this was about 1am and I didn't really look
hard.

The basic good news is that it installs fine, I am
sure the group will find plenty of details that I
missed.

Will circulate screen shots later...
