One consideration that is always worth making mention, is that when you are
"trying" to install OS X if there is a slightly dubious piece of memory in
the machine OS X is more than likely to knock back the install as it chooses
to fail the memory.

If you have more than 128M RAM that is Apple original memory put that in on
its own and try the install again.

This should over come the reluctance issues that you are having...

OS X is real picky about when it will actually install itself.
