On Jan 26, 2006, at 4:42 PM, Melvetta Overturf wrote:

> Will this work on a cube that the power on button won't work?  
> Thanks, Mel

Yes and no (or visa-versa).

To use Target Disk Mode, both machines need to power on. Only 1 needs  
to actually boot. If you have a problem that prevents a machine from  
powering on, you would need to remove the drive and put it in an  
external case or another system in order to recover the data.
