Hi there Listers,

I've been experiencing exactly the same issue since 10.3 (aka Panther) came out. I've been in touch with people at Iomega through their 'live chat' services and all they tell me is... "Please note that current Iomegaware software is not Mac OS 10.3 compatible".

My workaround(s)...

Boot in a Mac OS 10.2.x (aka Jaguar) and/or Mac OS 9.2.x (aka Dead Cat Walking :) with the latest Iomegaware software installed and read/write protected zips will show up without a glitch.

Note that the latest versions of Iomegaware are available for download through their website: http://www.iomega.com/

Good luck,
