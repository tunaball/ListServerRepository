I've used http://www.nikotel.com/software.htm for audioconferencing between
Mac (450Mhz Cube 10.2.8 and iMic) and PC (Dell PIII-800Mhz.) with a headset
and using DSL 512-512 it have worked good enough for having a good and clear
conversation. With around 2-3 seconds of delay between each other, maybe a
faster Mac and Pc can have better results.

This Free Software is Java based and works great with the latest Java update
from Apple ( http://www.apple.com/macosx/features/java/ ). Also on the Pc is
important to install the latest Java version (
http://www.java.com/en/index.jsp ).

Users need to register for using it, but it remains free for computer to
computer calls unless users deside to buy the option for calling VoiP (Voice
over Internet) phones.
Also uses a AOL messaging inside the application for text chat.


For Mac to Mac the best thing is iChat.

I haven't tried AIM(Pc)-iChat(Mac) with the newest iChat Apple Beta yet.
