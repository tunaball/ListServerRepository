I had that problem twice on my iBook [Once with the OS X.ii.iv update,  
and once with the OS X.ii.viii update]... In both cases, some files  
were corrupted causing the computer to hang on boot-up... Run disk  
utility and see if you don't get some red lines indicating various  
failures... If you do, you'll probably need to run Disk Warrior, as the  
Disk Utility was not able to repair mine... Also, be sure to repair  
Permissions AFTER EACH UPDATE... If you get no red lines, just  
repairing permissions might do the trick...

I hate to tell you this, but one of my failures required that I erase  
my HD and reinstall everything from scratch... You can use OS 9 to  
rescue your files if you don't have recent backups...

Good luck...

Pep
