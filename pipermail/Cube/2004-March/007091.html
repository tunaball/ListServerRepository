iTunes started stalling while playing a tune.  So I switched to another
tune and it stalled, too.  Just for a second or so, but several times
during the song.  The rest of the system seemed ok.

So I tried playing those same tunes from my iPod, via iTunes.  No skip
problem.  Starting to look like a hard disk problem.

I tried logining out and back in again, no change, still skips.  Tried
rebooting, no change, still skips (actually, stalls is a better word for
it because it picks up the tune where it left off, rather than jumping
ahead).

Next I tried shutting down, yanking the core, reseating all the memory
SIMMS (1.5Gig, 3 SIMMS).  Blew out some dust, looked around for heat
issues (none found).  Put it all back together and tried it again.  No
problem this time.  All is well.

Next time you have weird system behavior, try reseating your ram.
