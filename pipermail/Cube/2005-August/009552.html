On 7 Aug, 2005, at 17:04, Ken Johnson wrote:

If you cannot start the Cube, then this procedure will not work, 
because both computers need to be successfully started.  Instead, you 
must remove the hard drive from the Cube and either connect it as a 
second drive on the G4 (pretty easy - just 2 connectors), or install it 
into a Firewire enclosure, and connect that to the G4 using a Firewire 
cable which you buy at a computer store (harder to do, and more $$ 
involved because you must buy the enclosure and the cable).

Absolutely the right answer, Ken.

The first thing to check is always whether the machine will start or 
not. Everything else is totally non-relevant without that!

With a G4 tower, the suspect drive can be set to "Slave" and installed 
as a second drive in the case. If it will start in that location, then 
the data can be transferred to the Master drive without any problem and 
saved for future use.
