Re: New Hard Drive & Cooling Fan

The 7200 speed Barracuda IV is a great drive, and the QUIETEST that I 
have used. Don't waste your time with an 80GB - put a 120GB in now. 
It's as high as the machine will go. If you plan on many LARGE files 
(like video), get the 8MB cache - if not, get the 2MB (it's cooler, 
and more faster with smaller, everyday files). Either way, there is 
NO reason to add a fan if that's all you are changing. Bets tip for 
formatting is to format OUTSIDE of the Cube (in an external case) 
using OSX, then put it in and install the OS.
