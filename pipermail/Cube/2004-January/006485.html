I used to run VPC with Win98.  Win is probably just warning you that 
the hard drive image it runs from is fragmented.  I've gotten the same 
warning and had Win defrag its drive.  It should be seeing your shared 
Mac drive or folder(s) as a network share and probably wouldn't be able 
to see fragmentation there.  I'm not sure about that but remember 
having Win98 wanting to defrag only its own virtual drive.  I forget 
where the drive image is by default, but it's a relatively big file 
with a Windows FAT filesystem in it, kind of like a .dmg file.

I leave my cube on 24/7 myself.  I have dnetc running so it's at least 
doing _something_ all night and the monitor set to power down after 15 
minutes idle to save a bit of power.
