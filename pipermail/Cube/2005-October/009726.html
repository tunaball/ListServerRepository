Mac Cube.

I have shifted from a PC (Shuttle XPC), a bit too noisy, to my  
fanless 1.2 ghz Gigadesigns Cube for running Music Library and other  
recorded music programs, in my living room

Bill McCain's Music Library 2.5 will control an Inday Multimedia DA4X- 
RS, a Toslink optical switch, which allows his program to operate all  
three of my changers.

But the Mac version of Music Library (a Java program) so far doesn't  
seem to control the Inday switch.

per all accounts) an Iogear USB to Serial adapter to run the Inday  
switch.

serial setup.  So far, in Mac OS X Tiger, and the Apple web site, I  
have not found the info I need to successfully configure my Mac.
