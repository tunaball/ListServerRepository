also keep in mind that the quality will be better if you use toast's copy function.  with itunes, if you import the files as anything other than AIFF, you'll be using compression and then decompressing when you burn as an audio cd.  with the copy function, you'll be making a straight copy.

by the way, i use both itunes and toast all the time for audio cds using an external firewire cd burner and have not had any problems.  i'd recommend the lacie 52x.  only about $100 or so now.  i've never had any problems with it and it looks great.

good luck.
