    <DIV>&nbsp;</DIV>
    <DIV>1. Use Disk Utility to eject The first and simplest method&#44; if you only want to unmount a single volume on the disk&#44; is to use Disk Utility&#44; located in Applications/Utilities. In Disk Utility&#44; simply select the volume you want to unmount and click Eject.
    </DIV>
    <DIV>&nbsp;</DIV>
    <DIV>2. Hold Down the mouse button at startup In some cases holding down a connected mouse button at startup will cause a misbehaving optical drive to eject its media. &#40;Unfortunately&#44; this method will not work if you are using a Bluetooth keyboard.&#41;.
    </DIV>
    <DIV>&nbsp;</DIV>
    <DIV>3. Boot into Open Firmware and eject If you have a Mac that will not startup properly and has a stuck disc&#44; try booting into Open Firmware by holding down Command&#44; Option&#44; O key and F key during startup. After booting into Open Firmware&#44; type the command: eject-cd.
    </DIV>
    <DIV>&nbsp;</DIV>
    <DIV>4. Enabling an Eject menu As a side note&#44; you can enable another means of ejecting disks by opening the folder System/Library/CoreServices/Menu Extras and double-clicking the file &quot;Eject.menu&quot;. An eject icon will appear in the menubar that can be used to close and open selected optical drives. &#40;If you no longer want the Eject menubar item&#44; hold down the command key while clicking and dragging the icon out of the menubar space.&#41;.
    </DIV>
