Eric F Crist wrote on Saturday, May 21, 2005:

>With *real* unix systems,  
>you can set and activation/deactivation date, lock an account, etc.

You mean a "real" Unix system like OS X Server?

Go into Workgroup Manager, select an account, go to the Advanced pane, click on Options..., uncheck the box that says "Allow User to log in".

You can also set the account to deactivate on a particular date, after a certain period of inactivity, after n number of failed logins....
