I don't think makewhatis(8) needs to be run for the man page to show up.
By itself, man(1) uses /usr/share/misc/man.conf which defines the MANPATH
as /usr/share/man, /usr/local/share/man, and /usr/X11R6/man.  Since the
man page for ssh(1) is located in /usr/share/man/man1/ssh.1, you can look 
there yourself and see if the file exists.
