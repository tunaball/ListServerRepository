Check the DISPLAY environment variable for xterm and Terminal.app. For me
the default for xterm is ':0.0' while this environment variable is not set
by default in Terminal.app. So, when I ssh to a computer from xterm,
everything forwards appropriately. However, things only forward properly
for me in terminal.app if I manally set the DISPLAY to ':0.0'. I just
edited my ~/.bash_profile to make sure this variable is set whenever I
launch termina.app.
