When you boot into single-user mode, the hard drive itself is
not "ready" to be used by OS X.  At this point, it is mounted
as a read-only volume.  You need to run the following commands:

	/sbin/fsck -y
	/sbin/mount -uw
	/sbin/SystemStarter
