Hello!

With Disktool you can leave one half of your disk empty. pdisk should be 
able to create another partition and label it as some kind of 
UNIX/Linux. Finally Brian Bergstrand's ext2fsx 
https://sourceforge.net/projects/ext2fsx/ can format and mount this ext2 
or ext3 partition.

You could also boot from CD/DVD into some Linux and start to install on 
this empty partition. The most minimal Linux could be deleted afterwards 
with ext2fsx
