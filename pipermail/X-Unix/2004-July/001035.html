Andrew, unless you really want to "play it like the big guys" why do you use
rsync (and I assume you use an HFS-compatible version) and SSH?

I don't mind the Terminal and shell and everything. But for a backup task
I'd go after this differently.

Setup the iMac as a file server via AFP. You can use SharePoints to achieve
significantly more customizability. Reading the manual helps for
SharePoints. Then connect the PowerBook to the iMac which can be as simple
as a click once it's set up.

Now you can use Synk, CCC, rsyncX, psync ... whatever backup software suits
you best.

Just a thought. Of course doing it with SSH gives the task a much more
"serious spin". (c:
