Those names are different from the file name shown by Finder in your 
screen shot. I don't know if this will help you but set the encoding to 
UTF-8 in Terminal => Window Settings => Display. Check also Wide glyphs 
options. And create ~/.inputrc with the following content.

set convert-meta off
set meta-flag on
set output-meta on

Then open a new Terminal window. What happens if you drag and drop the 
file from Finder to the Terminal window or use "ls -v" instead of "ls"?
