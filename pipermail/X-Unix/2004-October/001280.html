I'm not familiar with this use of expr(1).  It doesn't appear to be
a built-in bash command.  For strings, the only use I know of is:

	$ expr expression1 : expression2

For example:

	$ expr foo : '.*'
	3
	$ expr /usr/X11R6/bin/xbiff : '.*/\(.*\)'
	xbiff
