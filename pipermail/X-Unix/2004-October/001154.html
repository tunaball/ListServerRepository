inappropriate. Stan, you do *not* make up the lists rules, face it!

	List rules should be made up and enforced by the list mom, and we 
should all abide by them, regardless of 10million being pro attachments 
and 100000million being against them. Messages sent to this list go 
through someone's private mail server and that someone has the right to 
impose whatever he/she wants (laws not withstanding); if we don't like 
it, we should go somewhere else, plain and simple. If the listmom

somewhere else or quite complaining; should the contrary happen, then 
the ones sending attachments should stop doing so or, just the same, go 
somewhere else (regardless of having 10000000000million years worth of 
UNIX knowledge). But so far the only fact on the table is that no valid 
authority on this list has said anything against sending attachments, 
so others should quit pretending to be authorities and demanding a stop 
to this behavior.

	I mean, how *hard* can it be to configure even the most simplistic
