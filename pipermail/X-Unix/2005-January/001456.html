> Finder, and also does not show in Terminal when I type "ls -al".  I

It's stored in the disk label.

What you are seeing is actually the partition name, not the disk name.

For example, this is the first internal drive on my system, which has 
two partitions:
BenNevis and Loch.

dun> diskutil list disk0
/dev/disk0
    #:                   type name               size      identifier
    0: Apple_partition_scheme                    *55.9 GB  disk0
    1:    Apple_partition_map                    31.5 KB   disk0s1
    2:         Apple_Driver43                    28.0 KB   disk0s2
    3:         Apple_Driver43                    28.0 KB   disk0s3
    4:       Apple_Driver_ATA                    28.0 KB   disk0s4
    5:       Apple_Driver_ATA                    28.0 KB   disk0s5
    6:         Apple_FWDriver                    256.0 KB  disk0s6
    7:     Apple_Driver_IOKit                    256.0 KB  disk0s7
    8:          Apple_Patches                    256.0 KB  disk0s8
    9:              Apple_HFS BenNevis           27.9 GB   disk0s9
   10:              Apple_HFS Loch               27.9 GB   disk0s10

As I recall, you can't change the name without re-partitioning under OS 
X.
