On Mar 22, 2005, at 4:51 AM, Kuestner, Bjoern wrote:

> I actually prefer maildir over mail, at least in theory:
> - incremental backups are much easier
> - large mbox files can be a pain

Indeed.  I used to think that mbox was the best format, and it is 
certainly better than 99% of the others out there, because it is a 
plain text format.

But maildir's one email = one file is better for the reasons stated.  
It's much easier to search for matching messages in scripts, etc.  Plus 
if there is  corruption (I've got some mbox files with megabytes worth 
of NULLs in them) it is contained to a single message.
