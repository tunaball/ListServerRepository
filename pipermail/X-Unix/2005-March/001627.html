On 16 mars 05, at 03:12, Mark Philip wrote:

> Hi all,
> I've got a directory called Timothy Ng, which contains lots of JPEG 
> images. I wish to compress these images into a ZIP file, put the ZIP 
> file into another directory (say Desktop) and then email this file to 
> a user who is running Windows XP. I used to do this a while ago but 
> have forgotten how to do this using TAR. Can anyone help me out?
Right click in the Finder on that directory and choose "Make archive of 
$NAME_OF_DIRECTORY". You'll get a zip file. If you'd rather get a tar 
file, do a man tar in the terminal.
