In BBEdit, you can set the Find string to:

(LogFilePath.+cobra\\[^\\]+\\)([^\\]+)(\\.*\r(?:.+\r)+?)(LogFilePath2.+corvette\\[^\\]+\\)([^\\]+)\\


Set the Replace string to:

\1\2\3\4\2\\
