>> On Jan 31, 2008, at 4:51 PM, Eric F Crist wrote:

>>> you can do this really easily with a short command in sed,  
>>> provided this text only appears in this one place.
>>>
>>> sed -i ".bak" s/W3SVC1823/W3SVC1822/g <filename>
>>>
>>> This will replace all instances for the first argument with the  
>>> second argument.  <filename> can contain wild cards, etc.
>>>
>>> If you need something more specific, just add text in both  
>>> arguments to make it unique.
>>>
>>> sed -i ".bak" s%ogFilePath2=file:///d:\Webtrendslogs\corvette 
>>> \e0182201\W3SVC1823\*.log%ogFilePath2=file:///d:\Webtrendslogs 
>>> \corvette\e0182201\W3SVC1822\*.log%g
>>>
>>> You'll note that I've changed the delimiter to a % instead of  
>>> the /, as I'm using the forward and backward slashes in the  
>>> matched text.
>>>
>>> HTH
>>>
>>> Eric
