I have two messages in my inbox today about the Wanadoo Livebox, which 
uses the Broadcom 6345 chipset.
There are a number of routers using this chipset sold running Linux, 
and the drivers for the ADSL modem & wireless card are in all cases 
closed.

For the Airport Extreme there is a project underway to reverse-engineer 
the chipset, however.
http://linux-bcom4301.sourceforge.net/
