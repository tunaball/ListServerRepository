On 03 Aug, 2004, at 10:46, Peter Schumacher wrote:

>  how can check who's logged into an Airport (standard, not extreme)?

There are a number of utilities for any unit which supports SNMP.

See the notes for the Wireless host monitor.
http://edge.mcs.drexel.edu/GICL/people/sevy/airport/

I haven't played with it for a while, but to run it:

java -jar HostMonitor.jar or simply double click on the .jar file.

I have two versions -- one works and one does not ... don't know which 
is which.
The version 2.1 is the one that works with the Extreme.

There are a couple of others floating around, but I don't have the URLs 
handy.
