There are two different versions depending upon which base station you 
are using.

Verson 1.1 should support all 3 base stations. (I'm running it under 
10.3.4 with both a Graphite and an AEBS... However, it does not know 
about the express, unless the MIB is unchanged.

Beyond that, it depends on which interface you are using.

If you are using the 802.11 interface (10.0.1.1) all you need is the 
community name ... aka password.

However, if you are trying to use the IP address you need to first 
enable WAN management by checking the "Enable remote configuration" 
(which is actually the default) on the "Base Station Options) popup 
from the  "AirPort" tab in the Airport Admin Utility.

(Note that the SNMP management checkbox also needs to be checked for a 
response from either interface.) And if I remember correctly, you MUST 
assign a password to the base station if either option is enabled.
