On Thu, 2006-02-23 at 02:14 -0500, Jeff Porten wrote:
> Client just asked me how to set up his Panther system so that his / 
> Library/Logs folder is rotated daily -- as it stands, these files can  
> grow quite large.
> 

afaik, you should be looking at periodic ("man periodic"). periodic
usually already runs such tasks at a very unearthly hour. Since, most
normal humans sleep at 3am at night, these tasks almost never get run.

You might want to set these to at a time when the system is actually up.
Or even better, modify the scripts to run as a startup item. Go play :)
