DON't set up a normal user, unless you want to send the password around 
the world ...


To set up an anonymous ftp access, you simply have to add an ftp user 
to NetInfo and then enable ftp in the system preferences.

The following link will assist you in doing so:

http://macosx.org/software/applications/anonftp.html

Bassically, it tells you to set up a user called 'ftp' and a group 
called 'ftp'. You can forget the stuff about creating and populating 
the bin and lib directories, these are NOT necessary for the 10.3.3 
version.
