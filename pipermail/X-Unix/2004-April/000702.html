On Mon, Apr 19, 2004 at 07:02:20AM -0700, Jerry Krinock wrote:

> I see that tclspice is *not* listed on the Fink "Packages" page:
> 
> http://fink.sourceforge.net/pdb/list.php
> 
> Does that mean that installing Fink cannot help me with tclspice itself?

Not necessarily.  While fink may not provide you with a turnkey
install of tclspice, it can help you with dependent libraries.
For example, readline is in fink.  So, you would install whatever
dependencies fink provides for you and then manually install
the others into /sw.  Finally, you would install tclspice into
/sw.

At least that is what I would do.
