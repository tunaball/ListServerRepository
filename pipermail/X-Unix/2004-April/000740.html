If you start Auto-replying to lists, make sure you have a bomb-safe 
house, as a lot of people will then get quite angry!

When that is said, a simple rule in mail can do the trick.

Make sure the Match rules contain:

- Header X-Listserver or Header X-Autogererated should NOT be pressent.
- Message is addressed to you.

Make sure it's evaluated AFTER the SPAM rule or other rules filing your 
List-mails.

Then make it Reply with a message...

The better way would be to make the Vacation rule fire off an 
AppleScript than does the processing, and can add the FROM address to a 
aray so each FROM only get's a Vacation message ONCE.

Hope it helps...

PS: Why even send out vacation messages for a 3 day leave?
