Richard,

There's a couple of things you can try.  First, I would try the  
following:

1) Open a terminal window and type 'sudo su', and enter your  
password.  This will put you into the root account.
2) Type the following, exactly: find / -name "X11.app"
     This command is going to do a find on the entire root tree, and  
everything mounted beneath it.  It is going to look for a file called  
X11.app.

If this doesn't work, do like you mentioned and utilize Pacifist.   
I'm not sure where Mac OS X registers it's installed applications,  
but you could do some research on this and unregister X11, which  
would allow you to reinstall it.
