You're used to the concept that "executable" implies "binary".
In Unix, this is often not the case.  There are cases where
executable files are text files.  In these cases, the text files
are simply (not always "simple") scripts executed in another
environment (e.g. shells, scripting languages like Perl, Python,
TCL, and Ruby).
