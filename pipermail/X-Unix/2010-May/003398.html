Alexandre Gauthier writes:
> I am still curious as to if these manipulations create a software raid
> array through OS X, or if that somehow hooks in the hardware and does it
> there...

	It is done in hardware. Now you see 1 single drive
instead of 2 drives. The RAID card has the write-through cache
and even a keep-alive battery so that the RAM in the cache
behaves in a non-volitle manner unless, of course, the battery
is allowed to discharge such as if the system were shut down for
some extended time period.
