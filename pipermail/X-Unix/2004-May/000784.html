On 13 May 2004, at 16:50, Simon Forster wrote:

> Similar question to the one I posted earlier but perhaps more 
> concisely defined. How do I get lukemftpd to return the external IP 
> address when setting up a passive ftp session? At the moment it's 
> returning its local/internal IP number.

Answer:

- Create file /etc/ftpd.conf

- Add line:

advertise all <host>

where <host> is either the host name or external IP address.
