>> I would suspect because the file command doesn't have file info for 
>> OS X?
>
> From the man page it looks like it has a database.

The "file" command does indeed use a database of sorts, and you can even
look at it, and add to it yourself if you like.  It is called 
/etc/magic,
and it is a simple ASCII file that can be edited / appended to with a
regular text editor or other text manipulation tools.

Read the comments at the top for more info on how to add entries for
specific file types.
