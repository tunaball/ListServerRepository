Hi,

Being that this is a unix group, I assume that many of you use a text
editor.  Even more probable is that this editor is a vi derivative or
emacs.  If vi, then perhaps you're using vim that comes with the os or
fink or compiled yourself.  Just thought I'd let you know that the
author of vim has a system for accepting support.

This support does more than help Bram, you get to vote on a subset of
features for upcoming vim versions...and one of the options is to make
it better on MacOS X!

http://www.vim.org/sponsor/index.php
