On 1/22/04 16:12, "Doug McNutt" <douglist at macnauchtan.com> wrote:

> .dmg  .smi   are two Apple disk image formats that are rather new.
> 
> How long will they be supported? Is it reasonable to use them for such things
> as archiving on a CD-ROM?

That's a good question. I wouldn't, at least not for anything that is not
OS-specific. (Apps that run on OS X would be fine, but I wouldn't put
graphics or text files in them.) In addition, if you have one sector on a CD
that's screwed, you're liable to not be able to open the disk image file at
all.
