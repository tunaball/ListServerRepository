A while ago I asked how to install a local IMAP server.

I was pointed to DarwinPorts, because fink doesn't have Courier-IMAP (yet).

Well, there's a simpler solution: Postfix Enabler supplies a nice little GUI
to enable Postfix for sending mails and as an IMAP server as well.
http://www.macupdate.com/search.php?keywords=postfix&os=
Cost is $10.

The readme isn't clear about what it really does regarding the IMAP server.
But it appears that OS X Panther has Cyrus pre-installed and Postfix Enabler
just turns it on.
