A while ago somebody asked for a solution to zip files from the command
line.

German c't-magazine published the following solution:

ditto -c -k -X --rsrc YourFile ArchiveName.zip

No issues known with Panther's Finder.

For Jaguar, they say, StuffitExpander can unzip the archives. However if
applications are zipped then the application type is lost which breaks the
app's neck. )c:
