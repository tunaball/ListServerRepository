On Nov 8, 2004, at 23:49, Mark Philip wrote:

> Hi all,
> From my PowerBook, I wish to connect to my home iMac via the Internet 
> and remotely control it via VNC. I've set up the VNC server on the 
> iMac already.
>
> This is the situation:
>
> Hardware:
> PowerBook (External Wireless or wired network) -> Internet -> Cable 
> Modem -> Linksys Wireless Router -> iMac (Ethernet connected)
>
> Software:
> PowerBook: 	1) Jaguar
> 			2) VNC client
>
> iMac:		1) Jaguar
> 			2) VNC Server
>
> My concern is:
> 1)  PowerBook is assigned an IP from wireless network or DHCP server 
> whilst on Internet.
> 2) The iMac is assigned a 192.168.1.102 address from the Linksys 
> router.
>
> How can I get to the iMac from the PowerBook?

1) Determine the public (WAN) IP address of the Linksys Wireless Router.
2) Configure the LWR to forward TCP port 5900 (VNC default) to 
192.168.1.102.
3) Point PowerBook VNC client at public IP address of LWR.
