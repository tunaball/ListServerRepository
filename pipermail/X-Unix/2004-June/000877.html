That only works if it's the default site on the server. If the server 
is using virtual hosts, you have to use the host.domain.com name.

Make sure the DNS server was restarted / reset after the zone record 
was updated. If it's a unix DNS server, ask them to restart the name 
server rather than reset it (restarting should flush the server's 
cache).

supposed to point to? Some of us who've done much work with DNS (it was 
one of my specialty gigs at an ISP where I worked for years) can tell
