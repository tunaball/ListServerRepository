Not a lot of help to you now, but for the future...

I use SilverKeeper, a free utility from LACIE. It can be scheduled to 
perform a task on computer startup, and I have it backup my Mail folder 
to a folder called Mail_current, which I keep on another partition for 
extra safety. The backup is incremental so the number of files saved 
each time is manageable.
