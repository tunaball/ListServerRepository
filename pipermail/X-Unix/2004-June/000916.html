>Basically, in a _real_ UNIX environment, you can use the password file and

"Real" UNIX environments are for hackers.

Just use niutil to set the password to '*'.  Account disabled.  (e.g. /users/root passwd).  Something like:

    niutil -createprop / /users/whomever passwd '*'
