On Jun 16, 2004, at 8:02 PM, R. Welz wrote:

> Hello.
> How can I connect to a linux system running XFree86 4.3?

Besides using the X display option you can also use ssh:

1) Start X11 on you Mac.
2) From an X11 xterm: ssh -X <remoteHost>		# That's a capital X.
3) After logging in to the remote host start an X program, it should 
show up on your Mac.
