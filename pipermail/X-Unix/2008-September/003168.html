My Leopard is also missing a man page for 'strings'. For other 
versions of strings
   strings -a file
searches the entire binary. If it thinks it's an app, only the data 
segment is searched.
   strings -t x file
shows the offset where the string is found. 'x' can be 'd' for 
decimal, 'o' for octal, 'x' for hex.

Searching with perl is another possibility.
