	It's well known that some times man pages don't really help much, but in this case it's a bit hard to overlook:

WHO(1)                    BSD General Commands Manual                   WHO(1)

NAME
     who -- display who is logged in

(...)

     -r    Print the current runlevel.  This is meaningless on Mac OS X.


	So I seriously doubt you'll get any info from who(1) that might interest you. All in all, runlevels are a SYSV UNIX thing (later on implemented by Linux as well), while Mac OS X is derived from BSD UNIX, the opposing  branch in the infamous UNIX wars back in 'them years.
