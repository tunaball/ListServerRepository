In case you wanted to know,  you could gzip anything that's not  
already gzipped by doing a similar command:

perl -e 'foreach $file ( `find . | grep -v .gz` ) { chomp ( $file =  
$file ) ; if ( -ne "$file.gz" ) { print "gzipping file: $file; gzip  
$file; } }'
