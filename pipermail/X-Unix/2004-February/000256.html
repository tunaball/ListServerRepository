Skip,

To follow filipp's post, try restarting and holding down the shift key 
at boot.  This will put you into Safe Mode and any kernel extensions 
will be disabled.  If you have installed software recently try to 
uninstall it or disable any kernel extensions that were added/tweaked.

You may need to reinstall the OS if you can't get this fixed.  Archive 
and Install SHOULD work, unless there's a problem with your hardware.
