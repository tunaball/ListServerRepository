I've observed two annoying anomalies with Address Book and Mail.app

Mail 2.0.1 (730)
Address Book 4.0.1 (472)
OS X 10.4.1

I've encountered an issue with Address book where the groups keep  
vanishing as soon as you
edit an entry or export an address from Mail.App to the Address Book.

So far, I can get them back by re-synching with .mac, but as soon as  
I go to edit an entry, they vanish again.

I'd also love to know what the little circling arrows next to "All"  
means... the implication is that it is syncing, but I suspect that it  
really just means that it's busy.

The second issue is with mail.app -- it double counts the number of  
incoming messages -- randomly.

I suspect it happens if I launch mail.app and don't read all of the  
new messages in a given mail box, quit and then re-launch mail.app.  
[I run my own IMAP server and use procmail to sort the incoming  
messages into multiple inboxes.]

Doing a quick search on some of the boards, I have seen a couple of  
people comment on the groups issue in Address book, but no answers.
