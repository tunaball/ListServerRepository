If you happen to have Stuffit Deluxe installed, they include a  
command-line "stuff" and "unstuff" tool. Generates actual stuffit  
files (with res fork).

I can't remember if version 9.x includes them or not. V 9 does omit  
some of the Finder driven (drop) tools that came with v 8.x, so I  
ended up installing 8 again, and then 9, in order to have them all. I  
don't remember if v 9 includes the command-line tools or if I had to  
install 8.x first to get them back after a system restore.

Cool thing is the "unstuff" CL tool will handle anything that Stuffit  
Expander will, like .zip and .tar, .rar, etc...
