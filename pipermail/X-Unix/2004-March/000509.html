The list of command line apps I use every day goes on for a while, but 
then I'm a unix geek by trade. There are many things that lend 
themselves to a CLI based solution, but these days there's often a GUI 
way of doing any given task. This means pretty much that whether you 
use the CLI or the GUI is a matter of choice, which seems like a good 
thing to me.

Do you have a particular task you'd like to move to the command line, 
or are you just reveling in how far the Graphical User Interface has 
come? Or perhaps you'd like to get more familiar with the command line?
