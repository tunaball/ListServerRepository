> Where did you get info that psync does a remote backup?  I 
> have only used the command line version, 
> I have never seen mention of remote backups?

It says "psync is designed to allow novice computer users to backup either
one part of their hard drive or the entire hard drive to another computer."

Of course that "entire hard drive" can be a showstopper for some.

> Not to mention, psync copies entire files even if only a few bits have
> changed, this will take longer than I want to deal with.

That, too, can be a major pain if you deal with lots of large files that
change frequently, e. g. disk images

Another work around would be to install an independent second version of
rsync which is probably better supported than the one which comes with OS X:

http://www.macupdate.com/info.php/id/9066
