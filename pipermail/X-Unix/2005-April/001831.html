Russell,

Well if you have OS X server it has a built in email server.

But if you don't (and indeed even if you did) I heartily recommend 
Eudora Internet Mail Server (EIMS):

<<http://www.eudora.co.nz>>

I've installed this product on sites where people either had nothing 
before or indeed had used the earlier OS X mail server.

Its very stable very efficient (i.e. processes mail without needing a 
super machine - e.g. one 30 person site is running it off a Bondi 
Blue iMac) and has excellent support.

There is a mail list you can subscribe to as well.

Usual disclaimer about not receiving any kickbacks for recommending 
it - just a very satisfied customer with several very satisfied 
clients.
