On Apr 13, 2005, at 13:49, William H. Magill wrote:

> MIT (I think it was) was responsible for hybridizing csh and sh into 
> what became the dominant interactive shell in the Unix world, tcsh... 
> but it still had scripting issues.

A version of the tcsh man page (for tcsh-6.12.00) indicates that J.E. 
Kulp (IIASA in Laxenburg, Austria) was the first to add job control and 
directory stack features (pushd/popd) to csh, which modifications made 
their way to Ken Greer at HP.  Ken start his added file name completion 
in 1981.  It snowballed from there.

It was ported to Unix running on a Mac more than once!
