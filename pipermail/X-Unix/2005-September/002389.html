Sounds like the output from Emailchemy. It'll produce an mbox file full of
RFC822-style messages.

I wrote a small python program to separate such a file once. I used the
built-in library functions to read each chunk as an actual message, giving
me access to all the headers and letting me cope with weird character sets.
I don't know if I still have that script around anywhere, but if you've any
python experience it'd pretty easy to cobble something together.
