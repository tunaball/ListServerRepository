> Couldn't this same thing be accomplished with a shell script calling the

> find . -name ".DS_Store" \

> -or -name ".Trashes" \
> -or -name "__MACOSX" \
> -maxdepth 1 \
> -delete
>

portable, and it is ready to build on it.

But if that shell script is enough for you that works of course.
