I believe the correct syntax would be something like;

ssh -2  -C -X -l <username> <IP Address>

At least that is how I have been using it on this MacBook for the last  
year. And this includes running X11 apps like, Synaptic, Moneydance,  
FireFox, & GQView, just to name a few.

Since the upgrade, I've had to drop NFS mounting my debian machine,  
and started using Samba. Much more reliable.
