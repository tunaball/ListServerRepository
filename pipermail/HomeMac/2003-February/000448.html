MacJanitor deletes temp internet file caches making things quicker.

SwapCop only will work if you have partitioned your drive or have a
second drive. It moves the Virtual memory cache to a less busy, 
dedicated disk area.

Moo folder changes certain window properties and lets you prioritize 
functions and force the system to utilize the memory instead of virtual 
memory. It also optimizes when finishing installation speeding things 
up a bit more.
