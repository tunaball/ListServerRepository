This is using Disc Burner. I have never used Disc Burner, but it's not
really setup to do Window's compatible discs as far as I know.

If you want to do cross-platform discs, you should get a copy of Toast.
Its reasonably priced and very powerful.

The other thing that Disc Burner does is single session CD's. If you only
put 100M of the 600M available space on the CD, you're done. With Toast,
you can create a multi-session CD and add more pictures later.
