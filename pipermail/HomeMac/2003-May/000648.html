I think that a lot of people are still holding back from OS X because they
have some programs that they need to use on OS 9.  But after two years, we
probably have most of the programs that we're going to get for X.  And X is
definitely the future (assuming you want to stay with Macs).  Not all
programs have been ported to OS X, but (such as in the Fontagrapher case)
there are often good alternatives available.  (Even in the case of
Microsoft Office, there are supposed to be programs that can read the MS
Word format.  I don't know how good they are, though.)

So maybe we should have a discussion thread in which people ask whether OS
X supports the features that they need the most, or what other alternatives
are available.  I think that (upgrading costs aside) most people would be
able to start using OS X if they had all the information they needed about
what else was out there.
