Note that it's really the same as what Greg recommended. The difference 
is that sometimes your Mac can take a while before it finds and 
displays in the Connect dialogue other machines; it's faster if you 
type the correct address. Next time, you won't need to type in the 
address; you will be able to choose it from the server list. You can 
also add the XP machine to your favourites, or make an alias and place 
it somewhere conveniently.
