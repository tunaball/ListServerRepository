Have you booted to your Jaguar disk #1, then run Disk Utility, Disk First 
Aid?  If not, then try that.  Under Disk First Aid run "Verify Disk 
Permissions" then run "Repair Disk Permissions".

Remember, do not run this from your OSX boot HD.  Stick in your disk #1 of 
Jaguar, and run it from the CD, not the HD.

I hope this helps.
