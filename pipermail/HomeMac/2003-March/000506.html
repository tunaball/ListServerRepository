Eudora is obviously a Carbonized application. (This means it cannot run
on 68K Machine by the way.) If CarbonLib has never been on your 575 then
it has never run _this_ version of Eudora, maybe some older version, but
clearly not this version.

As an aside, you do need to reboot after changing CarbonLib.

My conclusion at this point is that Eudora is not compatible with
CarbonLib 1.0.4. You will need to downgrade Eudora or upgrade the OS.
