>Is this error message coming from Eudora?  If I re install Eudora, will I 
>loose all my past email or will it over write it?

You do not need to reinstall Eudora. CarbobLib is an library file in the
system folder that is needed by newer versoins of software. Simply search
your drive for Carbonlib and see if it has been inadvertantly moved to a
different folder. If you find it, drop it on the System folder and it
will ask you if you want it put into the correct folder.

If you dont find CarbonLib then you can visit the Apple web site and find
a copy of it in there software downloads area. You might even be able to
use Software Update and see if it will download a copy for you.
