How long is long? I experimented and URLs that were so long that they 
had to wrap to another line were  still fine when I received the test 
message that I sent to myself.

I don't know how widely this is honored but I've read that when a URL 
is bracketed by "< >" characters that is supposed to tell an Email 
program that the contents inside that bracket should remain intact, 
i.e. never chopped into multiple lines.  However in my experiment it 
didn't matter to my Mail.app (receiving) that the long URL was not 
inside "< >". Maybe that could affect how some Win mail package would 
handle receiving a long URL.
