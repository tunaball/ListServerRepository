I don't think there is any problem with having both OS9 and OSx 
versions of packages installed on the machine. I have some of that; 
it's just a matter of HD space.

Probably one should run the OS specific versions of system level 
utilities, i.e. when booted under OS9 run the OS9 version (of course) 
but when booted under OSX run the OSX native version and not an OS9 
version in the classic environment.

Otherwise for ordinary applications the classic environment works well. 
Some of the new features of OSX, however, are only available for 
applications running as native OSX programs. One of the specific OSX 
features I use often is the ability to print but then save as PDF.
