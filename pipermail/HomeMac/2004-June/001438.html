On Sunday, Jun 20, 2004, at 01:55 Canada/Eastern, Pat Murray wrote:

> I got file the other day that when expanded had .chm appended to it.  
> [...] The same file opened automatically with Microsoft HTML help on a 
> Dell computer. Am I missing something from my Microsoft Office suite?

No. Several file types use the .chm extension, but the most common is 
Microsoft's compiled help. It's fairly common on Win, but not normally 
used on Macs.

> Is there some simple conversion utitlity available? [...]

There are quite a few utilities on PC. There aren't all that many on 
the Mac. Here are some:

Tubby <http://mikebultrowicz.com/software/tubby/> (free)
CHM Viewer 
<http://www.jouledata.com/DesktopDefault.aspx?tabindex=1&tabid=21> 
(shareware)
xCHM <http://xchm.sourceforge.net/> (free)
chmtools <http://fink.sourceforge.net/pdb/package.php/chmtools> (CLI; 
free)
