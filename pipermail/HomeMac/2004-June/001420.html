A few questions:

Does your www host site have PHP? It's common to have a web page use 
PHP and html forms to collect information and then send that as an 
Email. Here's a link to an article about that (Google found many 
articles about this):

<http://www.netspade.com/articles/php/form-mail.xml>

Do you want the www page to display a form which actually is a PDF with 
editable fields? If so I don't know how to create that kind of PDF 
except by using Adobe Acrobat.

If you have access to a Mac that runs OSX then it is easy to create 
basic PDF files. The print output from any OSX application can be saved 
using the "save as PDF" button in the print window.
