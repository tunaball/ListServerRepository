You're dreaming. :-)

But there is a solution. Open the Memory Control Panel (on 9 only!) while
holding down the option key. You will get an extra radio button that will
let you disable the startup memory test. 

AFAIK, there is no way to do this from X or from the start up screen.

There is a list of all of the known startup key combinations at 

<http://davespicks.com/writing/programming/mackeys.html>

If you notice any that are missing, I am sure Dave would love to know.
