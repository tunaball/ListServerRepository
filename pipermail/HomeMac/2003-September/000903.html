On Sunday, Sep 28, 2003, at 17:37 Canada/Eastern, John Baltutis wrote:

> AFAIK, when booted into X, there are no memory checks.

All Macs do a POST (power-on self-test) when cold-booted. This test 
includes basic RAM checking, and it's part of the  BootROM firmware; 
it's done irrespective of which operating system will be subsequently 
loaded. Once this is done and control handed over to Mac OS X, no 
further RAM checks are done -- or, at least, none is mentioned in 
Apple's tech docs.
