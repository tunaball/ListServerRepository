There are a couple of reasons why an image might bounce back to its original 
window.  The main one I have ever found is that there is a difference in the 
"modes" for each window.  For example, if the first one is Indexed Color and 
the second one is RGB (or CMYK or whatever) then it won't allow the image to 
move.

Usually if one is RGB and the other CMYK it will just transfer the color 
mode.


Let us know how it turns out.
