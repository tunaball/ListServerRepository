(1) Copy protection schemes don't apply to files copied from a Zip disk.

(2) They do apply when attempting to duplicate copy-protected audio or 
data CDs -- which is not applicable here. Moreover, the number of 
copy-protected CDs is relatively small. I have an extensive audio CD 
collection, but none is copy-protected; and of the data CDs, only the 
Final Cut 3 install CD is copy-protected. (Note that in the current 
version of Apple's Disk Copy, audio CD duplication is intentionally 
disabled -- more than likely, pressure from the music industry.)
