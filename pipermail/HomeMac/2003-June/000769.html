Its not hard to get what you want:
   - partition the drive any way you desire
   - install BOTH OS X and OS 9 on the OS X partition
   - copy the OS 9 System folder to the OS 9 partition
   - delete he OS 0 System FOlder from the OS X partition
   - use the Classic pref pane to tell OS X where to find the OS 9 System Folder
