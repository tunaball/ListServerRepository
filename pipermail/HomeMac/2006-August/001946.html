I have no idea if the USB WiFi will work with the Mac. Some googling
might get you some information to find out if anyone else has done it.

The unit will not show up as USB. USB is just part of the transport. It
will show up as an AirPort interface in the Network Preferences Panel.
To see if anything shows up, open the Network Preferences panel. Select
"Network Port Configurations" from the Show menu. Look through the list
of interfaces and see if there's anything resembling the USB interface.
