George,

Assuming you DON'T have a hardware problem (bad memory, etc.) here's what
"I" would try (in this order): (your mileage may vary)
1. Make a backup copy of your quicken data file and store it in a safe
location
2. Delete quicken prefs; restart quicken to see if resolved
3. Reinstall quicken (you should be able to do this without harming your
data file); restart quicken to see if resolved
4. Try running quicken with a new data file to see if you're dealing with a
corrupted data file (just move the old data file temporarily and start
quicken, choose new)
5. Try to open data file on another mac with another quicken installation
(could be difficult, especially considering the personal nature of the data)
