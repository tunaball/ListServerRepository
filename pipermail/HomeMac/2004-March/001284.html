Error -127 is a system message indicating a menu error ["Could not find 
HMenu's parent in MenuKey (hMenuFindErr)", to be specific]. Hence, the 
problem has nothing to do with the "usual suspects" invoked when a file 
can't be deleted.

Either some system resources are corrupt, or, more likely, this is 
caused by a 3rd party extension or utility. This is where you should 
begin the troubleshooting -- IMHO, of course.
