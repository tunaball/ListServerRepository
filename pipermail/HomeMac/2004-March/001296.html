Hi Debbie,
A couple of suggestions.
Do you have an installation CD. Insert that and use the C key. See if the machine boots 
up off of the CD. The Cd should have a Utilities function on it. That would confirm that 
it's the Norton Cd and not the machine. If that is the case, confirm that you are running 
9.2.x. Use System Profiler or whatever it's called under 9. My recollection is that 9.2 is 
used only as a classic system folder on machines running X(perhaps others could 
respond to this point). You should be running 9.1.x on a none X machine. If you do 
have X than your Norton CD may be out of date. Also if your not running 9.2 but really 
9.1 you may have an incompatible version of IE.
