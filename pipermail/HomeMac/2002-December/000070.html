Denise.

wma stand for Windows Media Audio. If you have the Windows Media Player 
in your computer you can play it. You must either drag and drop the 
file on the Windows Media Player Application, or launch the Windows 
Media Player and from the File menu choose Open. I tried it on a wma 
file and it works.

You can download the free Windows Media Player for OSX or 9.xxx here: 
The Classic version is for 9.xxx
