You might not have been doing anything. You might need to turn off the
index in the background preference (Preferences, Indexing & Spelling,
Automatically index messages in the background; turn this off). There is
a bug in the v-twin engine on OS-X that causes it to crash if it is run
in the background. This forces you to index when you want to search
(which I have had crash on rare occasions) or to manually index.

This is the only major problem that I can remember right off-hand.
