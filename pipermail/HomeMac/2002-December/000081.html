Memory is allocated to a program when it starts up. When you close the
program it may not " give back " all the memory to the system heap. I think
this is called " memory leak " which could cause your out of memory errors.
The best way to get back all the memory is to restart. I know this can be
frustrating in the middle of web surfing..it has happened to me a few times.
I had a freeware program called " Fix Heap " which reclaimed memory but it
doesn't seem to work on 9.2.2. It worked well on 8.X. You have to close all
open programs - but you don't have to restart. There may be newer programs
that do the same as Fix Heap.

A restart now and then can fix a lot of problems.

Good luck.
