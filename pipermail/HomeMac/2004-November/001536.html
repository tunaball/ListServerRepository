The answer is yes - almost all ATA drives now sold can be used in 
either a Mac or Windows machine.

As for installing a hard drive into an external case - make sure the 
external case will be able to connect with your mac (firewire 
preferably), open up the case, attach the drive to the case using the 
mounting screws or whatever is supplied, plug in the power and data 
cords, close it back up and format it using disk utilities. Almost 
takes longer to type than to do.

Enjoy.
