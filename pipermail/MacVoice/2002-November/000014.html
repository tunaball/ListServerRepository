It sounds like this is related to the other bug under Jaguar that 
causes problems if new words are found when documents are analyzed on 
Save if the speech data is also saved.  Since I'm running Jaguar 
these days I now open the Voice Center drawer and click on the 
Assistant icon before opening SpeakPad to begin dictation.  I find 
this has eliminated any problems that occur if ViaVoice needs to run 
either the Analyze My Voice or Analyze my Documents assistant.  Yes 
this leaves a big white rectangle in the center of your screen, but 
other windows can cover it up--and IMO it's worth it to avoid 
problems until IBM comes up with a fix.  You may want to try this 
procedure yourself.  BTW there can still be memory issues under OS X; 
but the user cannot control how much memory an application uses. 
Also, I have 768 MB in my Mac so that gives me a little "elbow room."
