Charlie,

No speech recognition program would allow you to do that. In order to 
maintain the best possible accuracy they all are designed so that 
microphone levels can be adjusted based on the current conditions at 
the time of running the microphone setup so the clearest possible 
signal is available for processing by the recognition engine. This 
way the best possible signal is guaranteed no matter what voice model 
is being used.
