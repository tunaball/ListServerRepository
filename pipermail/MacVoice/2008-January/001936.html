But there should still be a keyboard option for everything. Those of us
with memory problems can then use the keyboard for these simple tasks
and let Dictate do the actual serious typing. We don't all have exactly
the same kind of disability, or the same needs, or the same preferences.
I could probably commit to memory five or six commonly used voice
commands, but would be completely incapable of memorising the dozens
needed to command the programme solely by voice. Mac programmes are
famous for giving people the freedom of choice which their PC brethren
lack; don't let Dictate be the exception.
