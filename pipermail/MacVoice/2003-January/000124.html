***FOR IMMEDIATE RELEASE

COMMANDSETS.COM RELEASES COMMANDS FOR MAC OS X's BUILT-IN SPEECH 
RECOGNITION

New Orleans, LA, January 22, 2003 — Speech middleware vendor 
CommandSets.com announced today it has released ten command sets for 
the speech recognition system built into Mac OS X (also referred to as 
PlainTalk™). Combined the sets add almost 700 spoken commands to Mac OS 
X's vocabulary.

The sets include PlainTalk versions of sets previously released for 
iListen and ViaVoice, including BBEdit, iCal, iPhoto, iTunes, Mail, 
Microsoft Excel, Microsoft PowerPoint, Microsoft Word, and Microsoft 
Messenger. New is a set containing over 40 commands for Apple's 
TextEdit program. TextEdit sets for iListen and ViaVoice were also 
released.

The new Command Sets sell for between $7.50 and $25. For a limited 
time, CommandSets.com is giving away the full TextEdit command set for 
either iListen, PlainTalk, or ViaVoice to anyone as a way of building 
traffic on the site and getting people to take speech recognition on 
the Mac for a test drive. There is no obligation to purchase to receive 
the free command set.

Visit http://www.commandsets.com for more details.
