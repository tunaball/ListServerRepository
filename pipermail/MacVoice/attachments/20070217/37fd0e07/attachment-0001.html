<P>I partitioned my drive and loaded Windows XP Pro using Boot Camp.&nbsp; I boot into Windows to use DNS Professional Medical.&nbsp; So far, it has been great and has exceeded my expectations.&nbsp; I don't know if the DNS Professional has a better speech engine or not - I think that the main difference is just the specialized vocabulary that comes with it.&nbsp; I used DNS about 7 years ago for business school (on a Dell laptop) and found that its error rate was too high requiring too many corrections.&nbsp; The current version is a whole lot faster and more accurate.&nbsp; Compared to iListen, I find DNS much easier to use out of the box, or with a little training.&nbsp; Since iListen doesn't offer specialized vocabularies, this is a no brainer for me.&nbsp; Even without the specialized vocabulary, DNS is faster and&nbsp;more accurate with less training.<o:p></o:p></P>
<P>I consider myself a Mac die-hard, but for dictating, I'm going to use DNS in windows because it's so much better.<o:p></o:p></P>
