Just FYI, iListen runs well under both Mac OS 10.1.5, (we don't support Mac
OS X 10.1 - 10.1.4). It also runs well on all version of Mac OS X 10.2.x -
I'd recommend using the latest version of the OS.

You can train a profile for iListen using an Olympus Digital Recorder, you
just have to create a profile where you use the Recorder itself as a
Microphone. The Olympus docs have the information on how to do that. If you
create a profile specifically for your Olympus Recorder, you will do better
than 77%. 

If you are going to MacWorld, stop by our booth, (#4141), to see a preview
of what's coming next with iListen.
