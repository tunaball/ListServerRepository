Here's the Macworld article on how one would use the Olympus recorder.

The trick for the iPod would seem to be  speaking into the Belkin 
mike while having the sound pass through to the Mac. Since the iPod's 
connection is via firewire and not via a sound input jack, making the 
voice profile would be a problem.

If there was a way to get iListen (or Via Voice) to listen to a sound 
file rather than a microphone source, then one would only need to 
transfer the audio to the computer (via firewire) and then play the 
file as the dictation software listens. Easier said than done.
