The data dmg file is pretty much what it says it is - data the program needs to do its stuff. The way you install the stuff on the data disk (or in your case, the data disk image), is to have it mounted on your computer (double-click the disk image so it appears on your desktop like a hard drive), then run Dragon Dictate. The program will then grab the data it needs from the disk image on its own.
