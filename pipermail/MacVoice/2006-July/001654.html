William (and everyone else):

Let's be clear about running Windows on a Mac. You, in fact, do "give  
up" the Mac OS X environment if you are running Boot Camp. While it  
is true you can use Parallels to run Windows in its own window on Mac  
OS X (thus having both operating systems active at the same time),  
you cannot use Dragon NaturallySpeaking to dictate or control any Mac  
applications.

You can, however, use iListen to dictate into a Windows application  
when running Parallels (as we recently found out). There are  
limitations - such as it doesn't look like Correction will work - but  
you can dictate!
