A caveat: I've found that turning on full GUI access with Jaguar
causes crashes -- Acrobat, Finder, Safari, you name it.

Given that, it's quite easy to get the built-in SR to bind a spoken
command to some key-combination (chord).  First, say "Make this 
application speakable", then "Define a keyboard command".  It'll prompt
you for the key combination, and the spoken command.  Basic things
like "Page down", etc are predefined.  "Show me what to say" to see
what's available.
