Mike,

*NEVER* connect a microphone to the computer through a hub If you can 
possibly avoid it. Doing so degrades accuracy at best, and at worst 
renders speech recognition software unusable.

Also, If you are using an Andrea Electronics USB pod make sure that 
iListen is set for AK5370 (the USB pod) as the microphone.
