I have not tried the GN Netcom Satellite wireless microphone with the
iListen software but see no reason why it should not work well as long as
you are using an Andrea USB pod. On all other systems this is as accurate
and noise canceling as our best wired headsets.

Be advised that being wireless allows you to get up and walk around but you
still need a belt pack transmitter with a wire to the headset. This
transmitter sends the signal to the base unit which is connected to the
Andrea USB pod.
