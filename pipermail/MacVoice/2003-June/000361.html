Yes, the Windows software is better and has been for years. I say 
this having used both. But don't even THINK about trying to use it 
with VirtualPC on a Mac. The processor emulation needed to convert 
x86 instructions into instructions the processor in the Mac can 
execute takes up too many CPU cycles for the Windows speech 
recognition software to work.
