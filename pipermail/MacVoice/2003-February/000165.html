MacSpeech Releases New ScriptPaks for Mac OS X

Bedford, NH, February 6th, 2003 - MacSpeech, Inc., the Speech Recognition
authority for the Mac, today announced the release of four new ScriptPaks
for iListen on Mac OS X 10.2.x (Jaguar).

The four ScriptPaks voice enable Menus and functionality in Apple's Mail,
iMovie 3, iTunes 3 and iPhoto 2 iLife applications. The ScriptPaks are
available for $10.00 each, (iListen 1.5.1 or greater required).

"This is the first set of releases for extending our Voice Command and
Control capability on Mac OS X.", said MacSpeech's Donald MacCormick. "We
wanted to start with the iLife applications and grow from there."

ScriptPaks enhance the Voice Command and Control capabilities of iListen by
voice enabling Menus and Specific functionality within the targeted
application.
 
More information about MacSpeech can be found at the company's web site at
<http://www.macspeech.com>.
