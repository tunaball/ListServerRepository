Hello All,

Another application that is going to "track" better is TextEdit. It is somewhat more Accessibility API (the operating system code that lets Dictate track other applications) compliant than most other applications. If you are using it in "normal" mode - not outline, you can generally click with the mouse and type as well and Dictate will keep track of your manual modifications. But do it "gently", rapid clicking here and there, and fast typing can still mess things up.
