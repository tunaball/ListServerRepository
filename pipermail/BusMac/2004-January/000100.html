Sounds like the problem is with your system software.
I haven't migrated to OSX, so my solutions would be "Classic" oriented:
Zap the PRAM, rebuild the desktop, etc.
Reinstall system software if necessary.
If, by chance, you are still using OS 8 (or 9), Conflict Catcher has a
feature that makes system software re installation quick and simple by
automatically reinstalling custom extensions and control panels as they
were.
Again, doesn't sound like a problem with the Kensington software. Would
expect you'd have the same symptoms if you disabled it and just plugged in
the standard mouse. The system software ("Finder" in Classic) is lagging.
