>Use the command Get From Browser under the File menu. If the bookmark file
>is in the usual place, URL Manager will find it when you select the browser
>(in your case, Explorer) from the list. If it has been moved elsewhere, you
>must use the Select Bookmark File command and navigate to the bookmark file
>that you want to import.
