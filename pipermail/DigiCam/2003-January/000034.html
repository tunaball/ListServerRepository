If the camera uses removable memory cards, then a card reader is
absolutely the best way to go. You can get the readers for $15-20.
Memory cards aren't that expensive anymore either. Just make sure the
memory card you get is supported by the camera. (It's more of a problem
with smart media than compact flash).
