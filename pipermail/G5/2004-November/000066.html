Ram usage in Mac OS X is downright frightening.  My G5 2x2 came with 
512MB, which was quickly upgraded to 1.5GB, for normal usage, that 
would probably be fine, but my usage isn't exactly normal, and I 
really need more RAM (that is without running all my Adobe apps).  My 
Virtual Memory usage gets up around 8.5GB!  Sadly one thing that 
might help your memory usage is frequent reboots, at least it's worth 
checking to see what happens if you reboot once or twice a day (I 
generally leave mine up for weeks at a time and the memory usage 
slowly grows).

Doesn't the new G5 iMac use the same RAM as the 1.8, 2.0, and 2.5Ghz 
G5 Towers?  If so, don't hold your breath on the price of the RAM 
dropping anytime soon.  I've had my G5 2x2 for just under a year, and 
RAM has remained almost the same for the 1GB kits, and has actually 
gone up slightly!  I'm concerned that the RAM is going to be like 
that for the PowerMac 8500's where it remained expensive for half of 
forever :^(
