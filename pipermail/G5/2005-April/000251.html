I'm guessing you did an archive-install. Stuffit installs a component  
or two in the root library. If you own either the Delux or Standard  
version just install afresh and all will be well again. I'm not sure  
if AVR works since I don't use it, but I've stuffed and unstuffed  
without issue.

If you just use Expander, download the Standard version of Stuffit  
and install it. (That's the only way to get Stuffit in a DMG) You can  
throw it all away after the install - you just did the install to get  
the resources back in your root Library.
