Apple builds machines from 'known' images.  Those images are only updated
occasionally (say when the OS goes from 10.3.1 to 10.3.2).  The images are
certainly not updated each time a new update comes out.  It would just be
too costly and time consuming to do so.  Additionally... Updates are NOT
applied on the factory floor... So that is why there are all the updates to
apply when you receive a new machine.

This is true of ALL computer manufacturers (that I know of).  Even on the
big boxes (Data Center class hardware from the big boys [IBM, Sun, HP, etc))
you have to obtain and install the latest patches for the OS.

Whenever I obtain a new computer... It's almost the first thing I do...
Apply patches.  Actually... That's the first thing I do whenever I install
additional hardware as well... Go out the the mfg page and fetch the latest
version of their drivers.
