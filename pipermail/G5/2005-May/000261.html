In ther terminal, the "w" command will tell you who has an active shell on
your computer (you will see an entry for every open terminal window, plus
one for the console, which is the 1st user to log into the GUI). Anyone
that connects via SSH should show up here. As a bonus, you can see the
last shell command typed (which would be "w" for you!)

I don't know of a way to see who is connecting via FTP or file sharing
services, but I am sure there is a way...

Example ouput from "w"

 7:15  up 2 days, 10:12, 3 users, load averages: 0.28 0.26 0.23
USER     TTY      FROM              LOGIN@  IDLE WHAT
allan    console  -                Mon21   2days -
allan    p1       -                Mon21       - w
allan    p2       -                Mon21       - ssh xxxxxxxxxxxx.org
