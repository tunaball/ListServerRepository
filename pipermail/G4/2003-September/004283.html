OS X has a built in web server-apache (a very robust/the most used/open 
source solution) to activate it simply go to System Preferences: 
Sharing: "Services" tab and select "Personal Web Sharing" and click 
apply.

the files for your website should either go into :Users: Your_Account: 
Library: Sites  (this is accessed at http://xx.xx.xx.xx/~username )

or here
:Library: Web Server: Documents  (this is accessed at 
http://xx.xx.xx.xx/ )

xx.xx.xx.xx = your ip address

of course none of this will work if you are behind a firewall and dont 
have port 80  forwarded to the server, etc.

if you have anymore questions, ask
