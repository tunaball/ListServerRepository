AppleGMACEthernet Patcher 1.0.1

This utility will only run under Mac OS X 10.2.8 and addresses an 
issue where ethernet access is lost after updating Mac OS X 10.2.6 to 
10.2.8. You should only use this utility if you are having problems 
with your network connection. Fixamac Software cannot be held 
responsible for damage arising from the use of this software. Please 
make a backup of any important files before continuing with the 
installation. The AppleGMACEthernet Patcher will replace version 
1.3.0 of the AppleGMACEthernet.kext file with version 1.2.4 installed 
with Mac OS X 10.2.6. At present, the impact of using an older 
version of the AppleGMACEthernet.kext file with Mac OS X 10.2.8 is 
unknown.

The AppleGMACEthernet Patcher was written to aid with the process of 
restoring network services quickly and easily. The 
AppleGMACEthernet.kext file used in the installation was taken from a 
clean installation of Mac OS X 10.2.6 and has not been modified in 
any way. If you would feel more comfortable performing the downgrade 
manually, you can find details of the procedure here:

http://www.macfixit.com/article.php?story=20030925095222415

Fixamac Software is not charging a fee for this utility. However, if 
you feel that the time and effort put into this software has saved 
you a great deal of trouble, feel free to make a small donation.
