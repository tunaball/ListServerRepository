The recent Version Tracker blurb said
------------------
HP Photosmart 5.5 (Update - 09/16/2003)
printer driver for the photosmart series

http://www.versiontracker.com/dyn/moreinfo/mac/14957
------------------

I read the VT user comments and more than one mentioned problems with the
driver obtained via Version Tracker.  Rather than risk problems I went to
the H-P driver web page and they had the HP Photosmart 5.5 driver available
for download. I downloaded the file and STOPPED the install after reading
the blurb and it said NOTHING about it being for Photosmart printers.

I then called H-P and was advised that if my H-P Photosmart 1115 was
printing, do not run the 5.5 installer.  It has been moved to trash!
