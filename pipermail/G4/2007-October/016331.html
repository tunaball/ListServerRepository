Well any Mac can be made into a server all you need is OS X server  
software.

It sounds like you really need a Network Storage Device, there are a  
few option open to you.

An actual NAS volume (Maxtor 300GB Shared Storage External Hard Drive  
NAS ( H01R300 ) ) Just one example there are lots out there some  
allow for remote access.

The other option is to get one of the new airport base stations they  
have usb and ethernet ports to allow for multiple wired and unwired  
computers, drives and printers to be attached, you can also set up  
remote access.
