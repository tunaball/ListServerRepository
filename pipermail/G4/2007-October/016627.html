The minimum requirements for Toast excluding G3 processors is probably because Apple no longer supports any computers prior to G4.  But if you are running OS 10.3 or above, you could run Toast 7.  I have no personal experience with Toast.  However, if you're still running OS 9.x (you didn't say which OS) you need to see which version of Toast supports your OS version, which is more important than processor speed.

Go here:

http://www.roxio.com/enu/solutions/enterprise_solutions/toast/default.html

It lists the versions available and the requirements.  You should also check
 
http://www.versiontracker.com 

for software info and links to items like this.
