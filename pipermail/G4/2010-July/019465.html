If Classic is starting up that is because you have it set to
do so.

Just go into System Preferences >> Classic >> Start/Stop
and UNcheck "Start Classic when you login".  Might as well also
uncheck "Show Classic status in menu bar"

Then click on Stop - which will stop Classic.

Next time you login Classic will not start - unless you double-click
on a Classic app.

You could delete /System Folder and /Applications (OS 9) if
you really need the disk space.
