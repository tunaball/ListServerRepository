I once had a quite similar porblem, but the Os9 CD only hanged when i 
started the disk utilities.
In the end it was a corrupted partition table on my hard disk. Actually 
it was a bug in disk doctor - there was an endless loop over partition 
entries...
I could fix my problem using a linux for ppc CD. If you are not familiar 
with linux i suggest you try to boot your mac with another hard disk.
