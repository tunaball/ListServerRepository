Hi and welcome to the Mac community and the G4 list.

Technically, the install disc you describe is licensed for the Mac  
Mini hardware only and isn't supposed to be sold/used on another  
system. In many cases, system CDs that are made for specific systems  
(like this one) will not install on other systems. They look for  
specific hardware to key the install and will not properly install on  
another system. Some of the OS9 CDs that were OEM with specific  
systems will install on other hardware, however. Your mileage may vary.

If you are looking for OS 9.x, you can usually find cheap copies on  
eBay. I've also found them at fastmac.com. If you want OSX, you want  
to make sure you're getting a "retail" install CD, which is one that  
is designed to install on any supported hardware. If you have a  
G4/450 tower, you can use any version of Mac OSX up to and including  
10.4.x (Tiger).

Hope that helps.
