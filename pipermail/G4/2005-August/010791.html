   Mostly the only reason you would Partition a drive Dan, was to brake down 
a 100gig drive, for example, to four 25gig drives. This was useful when only 
one computer was in the house hold. Also, IMHO it's a carry over from when 
you were limited to only 2-4gigs on any drive because it was all the system 
could handel, so you had to partition a 10gig drive so you could use it all.

You can partition any hard drive if you want too, but the fast home use 
drives of today don't realy need it unless the system is used by two or more 
in a household, then it makes good sense, most times.
