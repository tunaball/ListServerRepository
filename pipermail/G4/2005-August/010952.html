I did try booting with the os 10 disk, but it didi not see the hard 
drive. Yet, yellow dog see the disk?

Did you try to launch Disk Utility from the OS CD and see if the drive 
would show up in the utility window? A hard drive that has had its' 
formatting altered by Unix will not show up on the desktop at startup 
because it cannot be recognized by the Mac OS.

You have to use Disk Utility or some other formatting software to 
create the directory information that allows the OS to recognize the 
drive as a "Macintosh" drive so an operating system can be installed on 
it. If the drive is not formatted as a "Mac" drive, Disk Utility should 
give you the opportunity to initialize the drive with the proper 
identification data.
