better performance. It is not better at running one application  
unless that application was written to take advantage of multi  
processors. The newer versions of OSX run numerous applications in  
the background, by default, and dual processors can help this a lot.  
I am looking forward to Dual Core processors in all Apples computers,  
so the mentioned advantages are even in single processors.
