> It should be usable as an internal, assuming it has the correct port 
> for plugging the internal cable into it.
>
> AFAIK, it doesn't matter where the HD is physically located. You can 
> select which HD you want to be the boot drive in Sys Prefs > Startup 
> Drive.
>
> You don't need to boot from the HD in question...just install it, and 
> see if it pops onto the Desktop.
