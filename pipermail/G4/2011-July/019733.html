Some thoughts I have:
	
	What browser are you using? Have your tried a different browser? Have you tried to update your browser?
	Do you have a java script blocking plugin installed on your browser? Many links like what you state are javascript launched. If scripting is blocked then the link will fail
	Have you cleaned your browser cache and tried again?
	This might seem silly, but is a new window opening behind your browser and your not seeing it?
	Have you made any changes to your computer which is coincidental to this problem?
	
	Finally I find too many commercial sites are written specifically for Microsoft browsers using proprietary features. And if they are using a .NET or Silverlight code then you could have problems. So does the link to log in look like Flash? It might be Silverlight instead. If I remember Microsoft has a Silverlight plugin for Macs. I can't comment on that plugin as I would cut my left arm off before I installed it.
