If you're lucky a connector associated with the disk is loose.  Open the machine up and have a look. Wiggle the power and data connectors and see if it makes a difference.

It's ok to start up with the access door open. Try that and see if the disk spins. Put your hand on it and you should feel the disk inside rotating.

Is the disk warm to the touch? It should be if it's getting power.

I sure hope you have a good backup. A disk is where you store information until it fails.
