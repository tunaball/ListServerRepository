This memory as I have found with google searches, is an ECC memory. If so then 
no it will not work as G4's are not compatible with ECC type memory. (not that 
I have ever heard of) This kind of memory is used in high end work stations 
and servers.

Check the specs that came with your memory and see if it's Error Correcting or 
not
