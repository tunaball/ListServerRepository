If two Macs both have FireWire, you can connect them together
by putting one in "Target Disk" mode (hold the T key while it is booting
until the FireWire icon appears).  From that point on that Mac will act
as an external FireWire disk until it is rebooted.

"Firewire target disk mode . It allows you to mount one Mac on another as
if the first Mac were a Firewire hard drive . It's a feature of the Mac
and is not dependent on the version of Mac OS X that's installed."

See the following for details:

   <http://support.apple.com/kb/HT1661>

VERY useful for copying files across, either manually or using
/Applications/Utilities/Setup Assistant
