John:

Now that I understand the way you use Mail, I doubt that Drag and Drop  
to the list in the left sidebar will work. But try anyway.  My next  
suggestion is to create a different name for "The G4 List" mailbox.   
Perhaps this sounds like a quirk, but there could be some something  
about the name that the system does not like.  Oh! by the way, can you  
put one of these list messages into any other mailbox?  If so, then a  
new name should work.  If a new name does not work, I would try  
putting the Mail Preferences file into the Trash.  That is  
"com.apple.mail.plist" in your home>Library>Preferences.  Beyond that,  
I would consider establishing a new, "Troubleshooting" user account to  
see if the problem replicates itself there.  If the same problem  
occurs, you know the problem exists in the System.  If the problem  
does not occur, you know that it is a User problem.
