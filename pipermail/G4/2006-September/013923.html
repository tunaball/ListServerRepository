I would suggest that first, you unplug the DVD from the firewire and then
check if full sleep will occur, if not then remove the PCI firewire 800 card
and try again. Some PCI cards do not support sleep on the Macintosh
platform.

I have a Memory card reader that plugs into the firewire port on my G4, and
after I got and used it the first time, sleep would not happen. So I trouble
shot the problem for a day or so and then I happen to remove the Memory card
reader and the platform would go to sleep. To double check I reconnected the
Memory card reader, no sleep, removed the Memory card reader and sleep was
Okay.

Give it a try.
