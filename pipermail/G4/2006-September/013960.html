I had almost the same problem, do a finder "Find" and search for 
"Invisible Items"

Then sort the results by "Size"

When I did that I found that I had an invisible file, {which had been 
created by CCC when I was doing a learning session,) that was consuming 
almost 50% of my disk.

Then do a get info and change the file to a visible file and delete the 
file to regain a large portion of your drive.

If you don't make it a visible file you won't be able to delete it!
