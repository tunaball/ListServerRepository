Brandon,

If all else fails, think about getting an Apple external USB modem. 
I don't know if they require USB 2.0, or if they will work with USB 
1.1, but at any rate, it sould pay you to get a USB 2.0 PCI card with 
several USB 2.0 ports, as you will probably want to use some of them 
with OS X.

I believe that the Apple external modem is about $49, and a 
Macintosh-compatible PCI USB 2.0 card should run you under $30.
