You have to select the disk in the left-hand column, then you'll see  
a Partition tab to the right.

You also need to be aware that some versions of the motherboard in  
those machines had potential corruption problems with drives other  
than those shipped by Apple. You can read more about this in the B&W  
G3 pages at http://www.xlr8yourmac.com/G3-ZONE/yosemite/. There,  
you'll also find a program that you can use to imperically test for  
the problem.
