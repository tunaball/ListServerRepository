You should be more specific about what you're trying to do.

Unless you have a second drive that can at least _read_  DVD+R discs, you're going to be copying the contents of the DVD+R disc to a hard drive and then from there onto a DVD-R disc. So the difference in the kinds of discs should be irrelevant, since the Pioneer 111D can handle both formats.

(If the disc you're trying to copy is a video DVD, you should probably copy it using MacTheRipper, since that program (it's free!) can compensate better for disc errors than the Finder can.)
