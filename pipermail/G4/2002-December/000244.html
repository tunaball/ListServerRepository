Assuming the G4 is from a Quicksilver 800:

I know both of these machines use the same bus speed, but perhaps you 
have to set the multiplier jumpers to accomodate the different 
processor. I'd check the xlr8yourmac Web site for G4 upgrade 
information - these kinds of details are covered.

You say it was working for the seller's DA G4? Graphics card conflict 
seems unlikely, but worth trying another card I suppose. HTH.
