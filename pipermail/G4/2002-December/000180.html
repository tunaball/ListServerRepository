>> I want to network a Titanium powerbook to a desktop G4 350.
>> I have the crossed cable. I just dont know the format for DNS numbers to
>> use
>> in TCP. Or is there another way?
>
> On your Titanium, assuming you are using OS 9, go to the Finder Help and
> ask for something like Network.  The system will lead you by the nose.  You
> should not have to worry about DNS numbers.  In the TCP/IP Control Panel you
> will choose Using DHCP Server.  In the AppleTalk Control Panel, you will
> choose connect via Ethernet.  In the File Sharing Control Panel you have a
> whole range of choices, including security features if you want them.
>
> In OS X, it is supposed to be easier.
>>
>> Also would anybody know if its possible to network an old IICi to these
>> machines mentioned above?
>
> Yes.  This involves a LocalTalk to Ethernet Bridge.  Instead of a crossover
> cable, use a hub for the three machines.  You may have to get a PhoneNet
> connector for the IICi.
>
> Look at Apple's web site for Networking.  Also, look at:
> http://www.threemacs.com./network/index.html for info.
> http://www.farallon.com/ for bridge.
