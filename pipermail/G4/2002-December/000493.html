You didn't indicate which OS (9 or X), but ALWAYS, ALWAYS, ALWAYS, for any
software search, go to http://www.versiontracker.com, choose the appropriate
OS and perform a search.  Try a couple of different searches because authors
use different words to describe their software.  Try 3 searches: one for
"audio", then one for "sound", then one for "record".  You should get a good
list of potential software to use.  If you click on the name of the software
in the search results, you will be taken to a page that better describes the
software as well as some reviews by users.
