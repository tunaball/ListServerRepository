Take a look at NeoOffice -- you can read reviews about it at:

      http://www.versiontracker.com/dyn/moreinfo/macosx/22021

or go directly to Trinity Software for more information:

      http://www.planamesa.com/neojava/en/download.php

Both PowerPC and Intel/Mac versions are available.

It's free and has a powerful word processor, spread sheet,  
presentation program (a la PowerPoint), template maker, drawing  
program, database, etc.

NOTE:  This is Beta software -- might have a bug or two.  So far, my  
usage has been pretty reliable.
