Creating an alias  will work but is not at all necessary. After you  
move the folder to a new location and restart the app, it will try to  
find the folder in the original spot. If it can't a window will open  
that asks if you want to create a new library or open an old one. If  
you choose open, you show it where the folder is now located. It  
remembers that until you make further location changes or open the  
app with the Option key depressed. This feature allows you to keep  
and maintain several different libraries.
