> As my G3 iBook is running 10.3.9,
> would that then run on a G4 if it was of an age that it would have been
> installed with Panther 10.3.9 when it was new?

Only if the unique G4 components are included. If you want to transfer the G3
stuff to the G4, your best bet, IMHO, is to use the G4's restore disk, wipe the
G4, restore the software, hook up the G3 in target disk mode
<http://docs.info.apple.com/article.html?artnum=58583>, and migrate
<http://docs.info.apple.com/article.html?artnum=25773 > the G3 stuff to the G4.
Another thing you can try is to clone the G3 to a bootable, external FireWire
HD, ensure that it can boot the G3, hook it up to the G4, and see if you can
boot up the G4 with it.
