This is not completely true. You can boot into single user mode and  
run fsck, but Apple says that it's preferable to use Disk Utility so  
if you don't have an install disc or alternate boot disk you should  
boot into safe mode. In safe mode disk repairs are done as part of  
the boot process.

http://docs.info.apple.com/article.html?artnum=106214
