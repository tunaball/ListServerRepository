Any version of 10.3.x should install on your machine. You've already  
said your firmware is up to date (4.2.8 yes?) so that's not the  
problem. The first thing that comes to mind is checking your memory.  
If I recall correctly 10.3 was much more picky about memory than  
previous releases. If you have more than one stick try one at a time  
and see if you still get the panic. Another option is to get memtest  
for OS X at http://www.memtestosx.org/. Do you have the Apple  
Hardware Test disc for your machine by any chance?

Do you have access to another machine so that you could test the  
install disc?
