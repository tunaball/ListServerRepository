Pat,
This is normal if you are not using the special version of 9.2.2 that 
shipped with the MDD, the problem is that the MDD post-dates the release 
of 9.2.2 and has a different ATA controller, as the minimum you require 
the 'OS ROM' file from a working MDD system folder to access the ATA-100 
bus, I can send you a copy of the one from mine if you like as I have 
just been through the same problems, however I believe that various 
other files were also altered for the MDD, I should know more in a few 
days as I have a set of MDD install disks coming from the states that 
should sort out all my problems.
Regarding the 250GB HDD, the MDD has got large drive support (one of the 
few G4's that does!) and it can see all of the drive natively, if it is 
not seeing it all then the drive may have been formatted on an older 
machine (or possible with an early version of OSX, I don't know if all 
versions of X had large drive support), certainly under 10.4x I can use 
my whole 250GB Seagate.
Let me know if you need any help and I will try and do what I can.
Ben.
