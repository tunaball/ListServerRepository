Yes, Leopard install disks are universal, so they should work on
either PPC or Intel machines. The problem with the target disk mode
is that there are two different disk partitioning schemes. Basically:
- Intel machines will boot from PPC-style partitions but will not
  install to them
- PPC machines will not boot from OR install to Intel-style partitions

What that means is that if you try to install from an Intel machine
to a PPC machine in target disk mode, you will first have to wipe the
partition clean and create an Intel-style partition. The PPC machine
will then not boot from it.

It should work the other way though (installing from a PPC machine to
a PPC-style disk partition on an Intel machine in target disk mode).
