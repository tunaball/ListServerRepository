>A bit of behind the scene detective work has paid off.
>
>My solution was,
>
>1. Export the bookmarks from within Chimera (HTML file).
>2. Replace Favorites.html file within ~/Library/Preferences/Explorer folder
>by the above file.
>3. Quit Safari. Delete the following file and 2 folders...
>~/Library/Preferences/com.apple.Safari.plist
>~/Library/Safari/
>~/Library/Caches/Safari/
>4. Restart Safari and the new files would now be under imported IE
>bookmarks.
>
>Yep, a very promising app with some holes eg. No Keychain integration as far
>as I can tell.
