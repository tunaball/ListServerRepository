There is definitely a USB modem available. 

I am holding one sold by Boca Research, Inc., of Boca Raton, Florida. The 
model number is MD56EUSB, Tidalwave 56K USB Fax/Modem. The FCC Reg Number is 
IEJUSA-33695-M5-E. In the lower right corner of the label, it reads: P/N 
LC092.

It very closely resembles the older style Global Village modems in shape and 
connector layout except that it has no power input socket or on/off rocker 
switch.

I haven't tested this one yet, having received it only last week in a 
shipment of donated computer equipment, but hope to run it through sometime 
this week or next. If I got a donated item like this, there must be more 
somewhere.
