Ultimately these all lead back to the Linksys router. You can log into the
Linksys router via HTTP (your web browser)  and do all of the initial
configuration from ANY of the machines, but if you feel more comfortable
using the help guides from Linksys for Windows then by all means do it
there. 

Actually if you use the guides for Windows, they are all pretty much the
same. If you know where the corresponding preference panels are on the Mac
then a lot of the same lingo will apply for the Mac setup. Things like DNS,
TCP/IP, DHCP, static IP, subnet, router.

Good luck.
