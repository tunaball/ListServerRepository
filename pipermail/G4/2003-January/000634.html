>> Try this:

>> Do what you have to do to deactivate the Ghostprint and Gimp-Print

>> Open Print Center
>> Hold down the "Option" key and click "Add" printer
>> In the pop-up, select "Advanced"
>> Under "Device" select "USB"
>> Under "Device Name" type "Stylus 600"
>> Under "Device URI" type "file://dev/null"
>> Under "Printer Model" select "ESP" (they misspelled)
>> Select one of the (old) Epson Stylus printers in the bottom pane.
>> Click "Add"
>>
>> This uses the Unix CUPS printing module built into OSX Jaguar.  It is 
>> very
>> similar to Gimp-Print.  You can also configure it through your web 
>> browser
>> by going to this address:  http://127.0.0.1:631
>>
>> CUPS already has a version of Ghostprint installed to support it.
>> Hopefully, the new installation did not adversely affect it.  You can 
>> read
>> about CUPS here in the manual installed by Jaguar onto your computer:
>> http://127.0.0.1:631/sum.html#1_1
