Michael Carroll,

The next time you make an administrative change to the Vanity HQ web page,
you may want to add a caution blurb that the FCC does not currently support
online applications via Macintosh computers (even with OSX) or PC uses via
an AOL connection or from behind a firewall.

I wasted several hours over Sat/Sun trying to file an online FCC Form 605
application and on Monday the Technical Support lady at the FCC said I
needed to file using a PC as none of the Apple Macintosh operating systems
are currently supported. She went on to explain it is a Java script issue.
They are not picking on Apple as they do not support AOL users or PC users
behind firewalls. 

Thanks for providing the very Vanity HW web page for use by the amateur
community.
