Winsor,

There is a version of URL Mgr Pro that does work with Safari, that 
allows you to set Safari as the default browser. Although shared menus 
is not yet part of Safari, you can start up URL MGr Pro and, by 
clicking on its icon in your dock, bring up your URLs in a list taken 
from your default document.

Check it out at their web site!
