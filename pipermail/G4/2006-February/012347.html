It is more dependent on what codecs you have loaded into WMP.  If you  
have the latest version, you might have to visit Microsoft's website  
and see if there are new codecs available.

You should be able to "Get Info" on the movie if it loads and it will  
tell you how it is encoded (e.g., WMV3, XVID, etc).  Sound is usually  
MP3, AAC or WMA...
