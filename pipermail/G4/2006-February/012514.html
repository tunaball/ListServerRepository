Michael,

Try System Preferences/Network/Built-in Ethernet/Configure/Configure  
IPv6 and Select 'OFF'.

This fixes a delay that was introduced with the latest OS X update on  
some machines, but I've also seen it fix network speed issues in  
earlier versions of the OS. Worth a try.
