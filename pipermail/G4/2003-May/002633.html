I agree that it is likely the drive is malfunctioning, however, there 
is one other possible explanation. If there is a program logging a 
benign error to the console on a recurring basis (e.g. every second or 
two), it could make the same kind of sound. Open the console and check 
if you see a repetitive error. If so, you can use the information to 
determine which application is doing it, and hopefully correct it. If 
you don't see anything like that in the console log, the disk is 
probably going bad.
