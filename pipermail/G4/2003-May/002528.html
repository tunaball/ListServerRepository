>
><< Since the most current mirrored door G4's will not boot from an OS 10.2 CD,
>how do you run the Disk Utility to repair permissions, etc?
>  >>
>
>What have I missed here? If a MDD G4 won't boot from an OS-X v10.2 CD, then
>how in the world can owners do an OS installation or upgrade?
>
>There has to be some way to boot to an external device, doesn't there? Kindly
>explain to me how this is done without being able to boot to an OS-X v10.2 CD.

One way would be to boot from your Software Restore Disk and look for 
a Utilities Folder with a copy of Disk Utility. It won't be the 
latest version, but is a starting point. An alternative would be to 
connect your mac by firewire cable to a functional Mac running OS 
10.2.6 and start it in the target disk mode. You could then run Disk 
Utility  from the original computer on the target computer. Another 
way would be to install a fully upgraded system on a firewire hard 
drive and boot from it using the target disk mode.

Check out http://docs.info.apple.com/article.html?artnum=60985 if you 
want to play with the target disk mode.
