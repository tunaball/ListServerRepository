Boot into OS 9.2.2 (use a bootable CD from DiskWarrior, Tech Tool Pro,
whatever) and find the TexTedit .plist file (Property List file) in your
user>Library>Preferences. Trash it, Empty the Trash, reboot in OS X and try
loading TextEdit again. Should do it.
