Jay,
To install Mac OS X on a Beige G3, you must install on a partition that 
is 8GB or less. So on your 20GB drive, you would need to create a boot 
partition that is 8GB or smaller and a second partition with the 
remainder. The OS must be installed to the 8GB partition. Some quirk 
with the boot ROM and IDE controller requires the boot volume to to be 
less than 8GB. You can read that in the "Before You Install" booklet 
under Hardware Compatibility Issues. This is probably why you are not 
having success cloning a bootable drive.
If you install a third-party IDE controller, this limitation goes away. 
I have a 40GB boot drive on an Acard Ahard 66 in my Beige.
Hope this helps,
Mel
