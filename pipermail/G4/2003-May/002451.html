You should be at 9.2.2 for Classic to work properly with 10.2.5 (soon 
to be 10.2.6). 9.2 is a very OLD system version and definitely not the 
version you should be running. Boot into Native 9.2 and run your 
updater or simply download the 9.2.2 updater or run the 9.2.1 updater 
that came with 10.1 and then run the updater. There is a lot of 
difference between 9.2 and 9.2.2. A LOT. For one thing, Classic will 
boot way faster with 9.2.2 installed.

If you mean that you can't start and run Classic from 10.2.5 AT ALL, I 
had that problem and wound up reinstalling 10.2 and 10.2.5 Combo 
updater to fix that. But get the 9.2 to 9.2.2 first to see if that 
fixes your problem.
