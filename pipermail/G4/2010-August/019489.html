Greetings Charles,

You should either get AppleJack-1.6 and install it on the bad HD, or  
select the bad HD as the startup drive and then hold down the "Shift  
Key" after you select restart.

Holding down the shift key will cause the platform to preform a fsck  
before startup, a fsck is a disk rebuild/cleanup operation.

The Applejack single use application does the same and a lot more.
