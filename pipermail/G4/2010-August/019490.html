Hi, guys.  I'm a bit late to the party, but I'll throw in my 2 cents  
anyway.

When I have those "restart required" messages start on my G4's, I  
always start with the Open Firmware resets.  It usually takes care of  
the problem.

On startup, immediately after the chime, press and hold 4 keys  
simultaneously;  the letter "o", the letter "f", the command (apple)  
key, and the option key.

The system will boot into open firmware and the message "release keys  
to continue" (or something close) will appear on the screen.

Type the following three commands, one at a time, exactly as shown  
within the quotes, and hit the return key after each:

"reset-nvram" (return)
"set-defaults"  (return)
"reset-all"        (return)

If you type the line correctly the system will respond with "ok".   
Make sure you get an "ok" on each line before continuing.

After the "reset-all" command, the system will reboot and bring up  
the normal operating system.

Good luck.  I'd also be interested in seeing the problem line(s) from  
the log and the proposed solutions.

Dan
