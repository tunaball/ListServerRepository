If I am correct, you will NOT be able to start your G4 from an  
external USB device. It has to be a FireWire device in order to be  
used for a startup device for the G4 models. Only machines with an  
Intel chip can use an external USB device to start up from.

The external USB device WILL be able to be used as a back-up drive  
with no problems and can be used as an external iTunes storage location.
