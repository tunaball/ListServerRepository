> Try looking in the finder in Computer, (press apple(key)+ Shift + C 
> while in the finder) Is it there and not showing up on the desktop. If 
> yes then you need to adjust the finders general preferences.
>
> If it is still not there try looking in the Apple System profiler. 
> Does the computer see the usb zip drive if not try to get it to show 
> up there. Maybe up-plug and re-plug in the drive or plug it in a 
> different usb port. If the drive is showing up in the usb profile you 
> may want to try and mount the disk manually by going in to the disk 
> utility.  If that does not work try restarting with the drive 
> unplugged and then plug it in once booted.  If nothing works then 
> maybe you have a corrupt driver. I would try downloading and 
> installing the latest driver stuff from iomega. 
> http://www.iomega.com/software/ioware402mac.html
