>10.3.4 Shut Down command. the machine shuts down then after a few seconds it
>starts up again. The only way (other than pulling the power plug) I can get
>it to shut down is to remove the iMic.
>

running, and I'd also check that you don't have it set to be the 
default sound input or output device.
