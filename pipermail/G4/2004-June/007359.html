I don't want to be negative, but the firewire enclosure will handle the 
drive interface internally and doesn't require a separate PCI card.  I 
think the issue is really that the Firewire enclosure you are using 
doesn't support larger hard drives (e.g., it is not the correct 
revision of ATA to handle larger than 128Mb drives).

Do you know what revision the LaCie case controller is (ATA-4, ATA-5, 
ATA-6?)?

If I remember correctly, you need at least an ATA-5 controller to 
handle 128Mb+ size drives (based on the bit width of the addressing).  
If it is not the correct revision you can try to get a new enclosure or 
get a PCI internal controller as a previous person mentioned.
