It will depend an how your modem is setup

ATT uses a PPPoe protocol, so you will need username and password.  
Those could however be stored in the modem.

You did not say what your current settings are on the DP 450

If it is set  to automatic using DHCP, you will have no problems the  
new machine will connect just fine.

If the user id an password are stored on your DP you will need them on  
the new machine.
