>Here's the big picture. This machine was given (free!) to a friend 
>of mine who is a Graphics Artist Illustrator. It was given away

>Current Problem.)
>His parents generously gave hiim $1200. for a new computer, to

Why won't OS X install? I have a Sawtooth and have had OS X running 
on it on and off since 10.1, and full time since 10.2. Unless there's 
something wrong with the machine or it has a ridiculously small 
amount of memory or disk space, OS X should run on it just fine. 
That's the first thing you need to figure out, and then worry about 
upgrades. If OS X doesn't run now and it's not because of memory or 
disk space, an upgrade probably won't help.
