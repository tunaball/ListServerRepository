If your using the Mac Finder, to mount FTP volumes, be aware that Mac OS X
only supports read access to FTP Volumes. I always like Transmit for FTP.
If you are having problems with Transmit check your Firewall settings
under System Preferences. Make sure they are set to allow FTP access.
