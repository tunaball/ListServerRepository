Greetings Steve ( + )!( + )

Open your log files and see what they say.
  >Applications >Utilities >Console
Once you have opened the Console file click on the "Show Log List"  
icon which will then give you a list of log files that can be reviewed.

You can then inspect the logs, All Messages, Console Messages and  
other log files.
One that you might want to take close notice to is the system.log,  
however if you haven't cleared the log files for a long time you might  
want to clear all files and then try opening Safari again and see what  
files are generated.
