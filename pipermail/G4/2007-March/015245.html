Nice guess Rich, however all that Apple did was copy a patch that was 
issued by the University of New Hampshire, and other user groups about

For OUR Win-Tel brethren, the following patches are also available at:
====================================================
Windows XP SP2						Apply update, see KB931836 **
Windows XP SP1						Upgrade to SP2 first, then apply update
Windows Server 2003 				Apply update, see KB931836 **
Windows 2000							Update available only under Extended Hotfix Support 
**
Mac OS X 10.4.6 or later				None - but updating to 10.4.8 is recommended
Mac OS X 10.4 - 10.4.5				Update to 10.4.8
Mac OS X 10.3.9 and earlier		Install the CIS Mac OS X TimeZone update
Unix/Linux								Check Unix distribution's website for update
Outlook 2007							Run "Time Zone Data Update Tool" **
Outlook 2003, XP, 2000				Run "Time Zone Data Update Tool" **
Entourage 2004						Install Office 2004 Update 11.3.3 **
Other Devices (Blackberry, etc.)	See below, and check manufacturer's 
website
