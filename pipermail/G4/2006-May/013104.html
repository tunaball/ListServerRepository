Also Carol just for your information fonts are located in many places  
in OSX.

Macintosh HD/Library/Fonts
Macintosh HD/System/Library/Fonts
Macintosh HD/Users/your user folder/Library/Fonts

Each Users Home Directory has a Library folder which also contains a  
Font folder, but those fonts are only available for those individual  
users. Fonts that are loaded in the:

Macintosh HD/Library/Fonts
Macintosh HD/System/Library/Fonts

are available to all. If you run Classic, OSX will also be aware of  
any fonts that are running in the Classic environment. Which would be:
Macintosh HD/System Folder/Fonts.

Hope this helps, Wayne.
