Bob, I did not make the posting you mention but I suspect your inquiry has
something to copying files.  You can drag copy the OS9.2.2 System Folder
from one drive to anther but you cannot drag copy OSX. To move OSX you need
something like Carbon Copy Cloner

I am not sure what you are tying to do.  Sounds like you have an OSX only G4
and want to install 9.2.2 so you can run Classic or boot into OS9.   If that
is the case you should be able to drag copy the System Folder from your
other drive to the OSX drive.  Just any old OS9 will not run in the Classic
mode.  I think it needs to be 9.2 at least.
