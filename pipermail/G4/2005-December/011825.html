Michael,
Why not first try upgrading to the last (and most stable) of the System 7
versions?

Go here:
http://www.info.apple.com/support/oldersoftwarelist.html

Then press apple key and "F" and paste this line in the search field:
Macintosh/System/Mac_OS_7.6.1

It will take you down the page to all the downloads for 7.6.1
Don't forget to run Drive Setup to update the hard disk drivers after you do
the install. Otherwise you might end up with the flashing question mark.

Contact me off-list if you have questions. I was an avid user of System 7,
until OS 8 was released, but that was quite awhile ago, so my memory will be
a bit rusty. I'm glad to help where I can though.
Cheers,
John
