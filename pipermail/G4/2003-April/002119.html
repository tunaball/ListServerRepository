You can only install Mac OS 8.6 if it was included at the time of purchase:
see <http://docs.info.apple.com/article.html?artnum=25114>. I don't know if
the sawtooth fits that criteria. If you're having problems with OS 9, I
suggest that you reinstall OS 9, updating to 9.2.2 as you do. Got to
<http://kbase.info.apple.com/cgi-bin/WebObjects/kbase.woa/wa/assisted> and
do a search for G4 and when the page comes up, click on the slow
performance subsection. You might find some clues there.
