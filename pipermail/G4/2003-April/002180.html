Ruby,

Try creating a new User and see if the problem disappears. If so, then go to
http://www.ifthensoft.com and download Dragster. Install Dragster in your
original User and use Dragster to delete your user GlobalPreferences.

This usually solves the problem. You'll have to reset several of your System
Preferences after this, but it should solve your problem.

If you have an app that Unexpectedly Quits after some use, you might try
deleting that app's .plist preferences file. It's best to do this while
booted in OS9, I've heard.
