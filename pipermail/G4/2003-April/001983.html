I have a Superdrive, but I bought an external CD burner because the  
Pioneer A104 only burns CDs at 8x, which got very tedious for my  
nightly backups. If you do a lot of CD burning you could put the Combo  
drive in a Firewire external case and use the Superdrive for burning  
DVDs only, like I do. The current Superdrive burns at 16x, which would  
probably be tolerable however. If you don't worry too much about CD  
burning speed, a Superdrive is a great drive. I have been very happy  
with it for burning DVDs, and it reads CDs fast enough as well.
Mel
