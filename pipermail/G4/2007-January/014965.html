On 01/24/07, Fred Levine <flevine at comcast.net> wrote:
>
> Can someone please tell me how the heck to set the column view in the finder
> window as the DEFAULT setting? Every time I change the view and close the
> window, it reopens in icon view. Small, but annoying problem.
> Thanks in advance for any help.

Open a Finder window, set it up the way you want, click CMD+J, and select All
Windows. BTW, newly opened Finder windows, IIRC, always open in icon view.
Click CMD+3 to set it to column view and the setting should hold.
