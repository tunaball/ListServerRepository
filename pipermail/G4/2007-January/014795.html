Thanks Richard, David and Harry for your help. I have trashed my 
Quicktime and reinstalled it. Initially it seemed to work without the 
jerky intermittent playback. Then I realized that my tablet/mouse combo 
(Wacom Graphire 3-usb version) is probably the cause. If only quicktime 
is running and I only click the mouse button once to start the video, it 
plays completely start to finish. If I move the mouse or for some reason 
click anywhere (even on the quicktime window); the jerky problems start 
to happen. Ive discovered that theres an updated driver software for 
this but I need to find a normal usb mouse before I can uninstall the 
original driver.

This is the Wacom info I found:

The Graphire3 Driver supports the Graphire3 tablets on Mac OS X v.10.2.6 
or later and supports BOTH the native and classic environment of OS X. 
It includes all supported languages. NOTE: If you have recently 
completed an operating system upgrade first uninstall your current 
tablet driver then install version 4.86-6. Complete instructions 
available in FAQ 335 or Keyword Tiger in the FAQ search field
