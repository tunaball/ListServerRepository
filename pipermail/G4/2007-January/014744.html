Using the built-in controller you can only address from 0 to 128 GiB  
(137 GB) of the drive, partitioning won't get around this limit. The  
driver Bill mentioned will, but it has it own limitations - see it's  
READ ME for details.

As S. said the problem might be the drive size jumper, but no  
manufacturer I know of ships their drives with this set so I doubt  
that this is the problem.
