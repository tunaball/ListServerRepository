If you can't boot up at all to get to the System Profiler, simply  
remove all the sticks of RAM.  Then take one stick and put it in the  
RAM slot, and boot it up.  If it boots up correctly, that RAM is  
good.   Power it down, take out that RAM stick, put in another, boot  
up, etc.  Eventually you'll find the offending RAM module.
