Yaz, would you obtain a copy of "Applejack" from:

http://mac.softpedia.com/get/System-Utilities/AppleJack.shtml

Once you have installed "applejack" restart your computer, holding down 
the "Cmd, Opt" along with the "P" & "R" keys to effect a PRAM reset. 
You need to keep holding down these keys until you have heard the 
"BONG" at least three "3" times, and after the 3rd "BONG" hold down the 
"Cmd" and "S" key which will cause the computer to boot into the 
"Single User Mode" You will end up with a black screen with white 
writing and there isn't any "GUI" interface.

Once the "Single User Mode" has completed loading you should see a 
statement regarding "applejack" and if you would like to know more 
about applejack you can now type "man applejack" which will bring up 
the applejack manual.

However if you know interested in knowing the details about applejack, 
then type "applejack AUTO restart" which will activate the applejack 
application to run in the Automatic mode (with deep cleaning) and when 
finished to restart.

If this doesn't fix your problem with the "System Profiler" it surly 
won't hurt the platforms operation.
