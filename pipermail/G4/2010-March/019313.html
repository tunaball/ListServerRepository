Hi,

I run Leopard on my PowerMac G4 Sawtooth.
It has 1GHz cpu and 1.3GB RAM. Performance is acceptable,
in fact I don't notice any difference from Tiger for the
things I do.

Your memory size is a little small for Leopard. I would
suggest at least 1GB.

Also you will need a graphics card that is quartz extreme/
core animation capable for some applications: Time Machine
(TM is one of the main reasons to use Leopard IMO), Front Row,
or DVD Player. The stock ATI Rage 128 card will not do it.
I added a Radeon 9800 Pro and it works fine.

Eric S.
