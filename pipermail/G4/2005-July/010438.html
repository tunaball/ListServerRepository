> Launch Safari 2.0 (Tiger)

The OP was about Java in Panther.

> [...] I conclude that Safari uses whichever plug-in is required, but 
> it's
> basic setup is to use version 1.4.2.

The conclusion is mistaken. Prior to Tiger, Safari does not use a Java 
plug-in. Under Tiger, the only Java plug-in Safari uses is J2SE v5 (if 
it is installed). See "Java Deployment Options for Mac OS X".

<http://tinyurl.com/98u5w>

> See <http://docs.info.apple.com/article.html?artnum=301382> for 
> details on
> installing 1.4.2 in 10.3.9

Apple KB doc ID301382 is not about installing Java 1.4.2, but about an 
update which fixes a JSA bug in Mac OS X v10.3.9. There are no notes on 
installing Java 1.4.2 in Panther; at any rate, the OP already has the 
latest version installed.

> see
> <http://docs.info.apple.com/article.html?artnum=301073> for details on
> installing Java 2 SE 5.0 Release 1.

J2SE v5 is only supported in Tiger.
