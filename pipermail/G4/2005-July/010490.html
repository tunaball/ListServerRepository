Hello All,

I got my extra HD installed and so far everything is running fine ...  
but this G4 came with TIGER installed and it seems that everytime I  
try and install some of my other apps, like Mouseworks, it asks for a  
password that I do not have.


If you have the OS installation CD/DVD, boot to it and go to "Change  
Password" in the menu when it becomes available. Do this before  
clicking on the "Install" button!!!!!!

At that point, you should be able to create a new password for the  
root user (you). Just be sure to remember it for future use.
