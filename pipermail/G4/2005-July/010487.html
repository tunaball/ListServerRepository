Oh, oh
Go to the System Preferences and click on "Security" to see if the 
"Master password" has been set. If it's not set then just enter 
"return" when asked for the "password"
That is unless it is asking for the "user" password. If it's the user 
password then just create a new new user and use the "new user" as your 
login. Then delete the old user, however be sure that there isn't some 
data or application that are in the old user's login.
