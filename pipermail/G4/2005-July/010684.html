Dennis,

I am using this scanner but with 10.3.8 so I'm can't be sure if this 
will be of any help.  The driver I am using is  "CanonPl CS-U 3.8X" 
which is a plug in to use with graphic editors.  I had a very difficult 
time getting it to work and I'm not sure exactly what the problems 
(plural) were but I now have it working with both Photoshop CS 2 and 
Photoshop Elements 2.  You may have to move it around in various plug 
in or import/export folders to get it to work and you may have to use 
file info to set the application  "to open with". 

I have been unsuccessful in getting the plug in to work with Graphic 
Converter. but I haven't tried that hard since it is working with the 
other apps.  The plug in was downloaded from the Canon site.  If you 
need it I can look up the URL.

When installing this plug in the installer will look for suitable 
locations and give you a list.  You should be able to add locations 
that it doesn't find.  I think this is the add install location button. 
You then select the install location you desire to use.  It has been 
about a year and I don't remember all the details.
