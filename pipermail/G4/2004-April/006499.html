"Mac OS X 10.3 Help
I can't use some characters in file and folder names

You cannot use a colon (:) in the name of files and folders. Mac OS X 
uses this character to separate folders in pathnames.

In addition, some applications may not allow you to use slashes (/) in 
the names of items." Straight from 
apple.<http://docs.info.apple.com/article.html?artnum=151944>.

Mac os x  is much less pickier than windows when it comes to file 
names, no list......... I guess that is why we don't care to take
