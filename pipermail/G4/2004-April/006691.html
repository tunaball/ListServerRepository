Just for kicks I went to this site - you might have missed this 
statement:

This website is best viewed with IE 5.0 or greater or Netscape 4.7. 
Netscape 6 or 7 will not view properly

Translated it means they are using IE specific extensions that only M$ 
will support using its IE platform.  Safari is based on the Mozilla 
browser as is Netscape.  They only support non-platform specific 
extensions (as far as I know), so if the people who code the web site 
decide to limit their audience to only M$ enthusiasts there is little 
you can do aside form complain or suck it up and launch IE.

The bright side is you just have to run another program - if you were 
running windows you'd be sweating out the next virus alert....
