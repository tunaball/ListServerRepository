>
>Is there a definative list that shows
> what keyboard characters are allowed for file names on Macs?

AFAIK, the only forbidden character is the colon; from Mac Help:

"You cannot use a colon (:) in the name of files and folders. Mac OS X uses
this character to separate folders in pathnames. In addition, some
applications may not allow you to use slashes (/) in the names of items."
