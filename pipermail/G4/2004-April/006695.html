It's because they're using a MicroSloth version of JavaScript to build the
page's frames. If you select View->View Source, you'll see this: "This page
uses frames, but your browser doesn't support them. Please return to our
main page http://amsatnet.com/fright.html."

You'll also note this disclaimer on the original page: This website is best
viewed with IE 5.0 or greater or Netscape 4.7. Netscape 6 or 7 will not
view properly.

So it isn't just Safari.that has problems. Send them a critique at <mailto:
kk5do at amsat.org>.
