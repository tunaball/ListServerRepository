Yup.  I have been using RDC to run a Windocs server in my basement that I use for file serving.  It works just fine on my G3 iBook or G4/500.  You can save your session parameters with the dialogue box so that you can just start up by launching the saved session instead of configuring each time...also allows you to turn off certain features to speed up the connection.
