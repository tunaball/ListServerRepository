I've been in and out of this group so I've lost some of the continuity of

The trigger seems to be Netscape or Internet Explorer. I was able to use IE

I'd like to continue to use OS 9.2.2 because many of my core apps won't work

if the problem can be overcome. I tried the Iomega 120 GB but had difficulty
