I have a Motorola wireless print server and have the same problem; I 
don't think this is peculiar to Belkin print servers.  Unfortunately, My 
GigE has started locking up shortly after booting, so printing with it 
is the least of my worries.

ISTR reading that it's an issue with printer drivers, and that you need 
to set up a dummy driver on one of the PCs.  The dummy driver looks like 
a PostScript printer, but it does something like translating the print 
job and forwarding it to another print driver.  You share the dummy 
print driver and have the Macs print in PostScript to that.  That 
requires you to have a Windows computer running whenever you want to 
print, though, which defeats the purpose of having a print server.

I hope you find a better answer!
