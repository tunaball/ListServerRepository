What type of network -- mac or pc.  If it is a pc network, Windows does not
like Appletalk, which I believe you will need to communicate to the
Laserwriter.  There is a commerical product called MacLan which can be run
on a Windows network, permitting a pc and other computers to see an
Appletalk device  -- computer or printer.

However, MacLan costs about $200, which is more than the cost of a new
printer!!!
