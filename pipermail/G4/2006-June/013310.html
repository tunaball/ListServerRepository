This probably means that OS X's maintenance scripts aren't running  
regularly so the system's log files are being rotated and cleaned up.  
This is likely the case if you turn off or sleep your computer  
everyday, since these scripts are timed to run early in the morning.  
One way to deal with this is to use a program like Onyx (http:// 
www.titanium.free.fr/), which is my favorite amongst the free  
programs of this type.
