According to the specs it will not read a DVD +R



Specification DVR-104

Record formats supported
DVD: DVD-R(for General use only), DVD-RW
CD: CD-R, CD-RW
Read formats supported
DVD: DVD-ROM single layer/dual layer, DVD-R (all types), DVD-RW
CD: CD-ROM Mode 1, CD-ROM XA Mode 2 (form 1, form 2), CD-TEXT Photo  
CD (single and multiple session), CD-DA (Audio CD), CD-Extra, CD-R,  
CD-RW (Support AM2)
Write speeds
DVD-R: 2X CLV, 1x CLV
DVD-RW: 1X CLV
CD-R: 8X CLV, 4X CLV
CD-RW: 4X CLV
Read speeds
DVD-ROM (Single): 6X CAV
DVD-ROM (Dual): 2X CLV
DVD-R, DVD-RW: 2X CLV
CD-ROM: 24X CAV
CD-R, CD-RW: 24X CAV
CD-Audio: 10X CAV
Video-CD: 4X CLV
Access time (Random Average)
DVD: 200 Msec
CD: 180 Msec
