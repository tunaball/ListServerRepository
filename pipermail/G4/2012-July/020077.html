Making a PDF in Pagemaker requires a post-script printer to be active in the chooser.  Then select acrobat distiller in the print dialog box.  This will let you make a ".ps" file that still has to be distilled in Acrobat Distiller, which will make a PDF.  Lots of whoops to jump through to get these puppies made.

If this is done, many times when I am having difficulties I'll quit classic and from the Preferences>Classic pane... then, I'll rebuild the desktop (2nd tab), and just rebuild the disc that Classic is on.  Sometimes you may have to rebuild 3, 4, or even 5 times to get everything working properly.

This inlcudes fonts, the chooser, and other things you may be using in Classic to help PM.

Hope this helps.

>>> The Open Firmware resets have solved a multitude of strange problems for me over the years. They can be in addition to resetting the  PRAM.
>>> 
>>> On startup, just after the chime, press and hold the following four (4) keys:
>>> 
>>> 1.  the command key (the Apple  key)
>>> 2.  the option key
>>> 3.  the letter "o" key and
>>> 4.  the letter "f" key.
>>> 
>>> Hold them all down until a message appears saying something along the lines of "release keys to continue".
>>> 
>>> One at a time, type the following three commands, including the dash, but not the quotes, and press the return key
>>> 
>>> "reset-nvram"  return
>>> "set-defaults"  return
>>> "reset-all"    return
>>> 
>>> The machine will flash the rom (rewrite it to original specs) and reboot.  Check to see if you are still having problems.  Let us know.
