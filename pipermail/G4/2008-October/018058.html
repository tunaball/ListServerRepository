When I was using the Acard ATA card, all of my drives were inside.   
Now that I switched to SATA, the drives are in the server drive case  
and I used a couple of these <http://www.pugetsystems.com/

some long SATA cables between the two.  I had two free slots open in  
the DA, so I didn't have to worry about using up the PCI slots.
