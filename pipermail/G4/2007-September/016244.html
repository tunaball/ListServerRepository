Hi Joe,

For the hard drive, I'd say power it off, unplug it, and then try
plugging it in and turning it on, and then try it in reverse if it
doesn't work that way. Another thought is leaving it on and plugged in,
and then restarting the system. If you can have the drive show up in
Disk Utility, have it verify or repair (if you do select repair, it will
automatically verify it in the process) and see if it checks out. If it
does end up making a repair, have it check again and make sure it's OK.

As for deleted files, it's hard to know how many potentially important
files have been lost, if that's indeed what's happened, so your best bet
is to reinstall the applications if possible. Check your system discs
for packages that will install just your iApps. Hopefully your system
CDs or DVDs will have that available. If your available documentation
can't give you a solution, well it'll be up to someone with more
knowledge of this particular problem than I have.

Eric
