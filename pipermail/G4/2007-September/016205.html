Hi Russell,

The dialog is coming up because your current password does not match  
the password for your Keychain.

You have two options:

1. Reset the password in the Keychain
This is easy if you actually know the password.

2. Delete the login Keychain
Delete the current login keychain and log out. The next time you log  
in, you'll create a new one that will have the same password so that  
the dialog doesn't come up.
