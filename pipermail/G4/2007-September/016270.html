One other issue:

I assume that, because you have never had a Mac before, you won't be running any old "legacy" software. Although Apple has always been very good about making most everything backward compatible, there was a major change with the release of OSX. OSX is really a completely new and different OS from all the previous Mac systems. OS1-OS9 software won't run on OSX and OSX software won't run on older systems. If, for some reason, you are going to use any OS9 software you will need OS9.2.2 installed on your Mac in addition to OSX. Your Mac will boot into OSX and always run in OSX unless you try to launch an OS9 app. At this point, your Mac will automatically go into the "Classic" mode which is an emulation of OS9. When you switch back to some other app or to the finder, your Mac will automatically switch back to OSX.

If you're never going to use any old, pre-OSX software, don't bother installing OS9.2.2.
