Disk First Aid is a Mac OS 9 or earlier disk utility. It can't help 
with problems with one file or one application.

> [...]  the Disk First Aid on the Jaguar install disks bleats that it 
> isn't
> compatible with Classic (?) - Why would it be?

Because Classic is an emulator which allows Mac OS 9 and compatible 
applications to run under Mac OS X. One of the limitations of this 
emulator is that it doesn't have low-level access to hardware. 
Therefore, Mac OS 9 extensions and applications which require hardware 
access, such as disk drivers, disk utilities, etc., either cannot run 
or can't be used in Classic.
