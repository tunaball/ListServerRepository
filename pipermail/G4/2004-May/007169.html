If you use 10.3 you already have the ability to zip files, but the Mac 
OS does not call it zip. Right click on a file and you will see an 
option that says "Create Archive of ..." the file you clicked on. This 
archive is a zip file. Very handy for sharing stuff with Windows using 
friends. You can even select multiple files and create one archive 
(zip) file of all of them.
