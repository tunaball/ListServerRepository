If your scanner AND printer are ethernet compatible, you should be able to 
access each one individually from either G4 when they are connected to an 
ethernet hub as long as you have the proper software installed on each G4.

By using a hub instead of trying to "share" the devices, each device becomes 
a part of the network and is accessible to any compatibly configured computer 
on the network.
