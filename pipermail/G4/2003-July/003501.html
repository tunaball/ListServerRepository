CL2 is faster than CL3 for memory access. I would put your ram in the J21 
slot. If you are having problems with it make sure you have the firmware 
patch from like 3 years ago installed. 

Honestly unless it starts acting flaky I wouldnt worry about it. 

I have PC133 CL2 in mine but that is only because it was like 5 bucks more 
and i only paid ~50/stick for 512M..
