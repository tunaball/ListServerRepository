If you buy a G4 MDD 2003 there apparently is no way to install a 
different version (a different language version for instance) of Mac OS 
9 on it than the one that came with it. This is because Apple ships this 
hardware with restore disks that are different from retail Mac OS 9 
install disks (even if they are the same version number) and Apple 
refuses to sell these restore disks separately. So if you buy such a Mac 
you are stuck with the language version you buy:-(

Because I figured I could install an english Mac OS 9 myself from a 
retail CD I ordered a G4 in the basic (dutch) configuration (I was told 
options meant delay). This Mac is on its way now but hasn't yet arrived. 
Now I found out this was going to be a problem..  So now I have the 
choice: cancel the order and order an english version G4 and have even 
more delay or be stuck with a language version I don't want. Thanks Apple!

Or does anyone see a easier/quicker way out? My intention is to have an 
international english dual boot G4 with proper/legal Mac OS 9 restore 
disks. Any suggestions are welcome,
