I recommend switching to Eudora 6.0.2. Free download and use it in 
the sponsored or light modes until you can afford to register it. It 
works on Mac OS 9 and above. In the paid mode you get Spam filtering 
and a couple other gee whiz capabilities. 6.0.2 with import all your 
current Eudora mail. My daughter uses 9.2.2 and Panther, and I use 
Panther. When in need we can read each other's mail just by using the 
same set of filters and trashing the email we don't want to store in 
the mail folders. Much faster than accessing the same mail via 
Adelphia's web mail site.

I have tried all the OS X capable mail programs and always seem to 
come back to Eudora.
