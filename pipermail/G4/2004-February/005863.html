Anne,
Panther was when journaling support was added to the filesystem.  It was alrady
there for OSX Server. And some enterprising folks figured out how to turn it on
in Jaguar. But it was not supported by Apple.   If you have journaling enabled
that might be a problem.  You should be able to turn journaling off and then use
your version of Disk Warrior.

Anyone else have any thoughts?

> Answer
> 
> 
> All versions of DiskWarrior will rebuild the directory structure of Mac
> formatted disks running Mac OS 7.1 - Mac OS X, including Mac OS X 10.2

> the ability to fix problems introduced to the file system by Mac OS X.
> 
> DiskWarrior v3.0, released in May 2003, runs natively in Mac OS X.
