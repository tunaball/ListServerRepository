You could, if running Safari, periodically empty its cache. Alternatively, you
could enable Safari's Debug menu: quit Safari, launch the Terminal app, type in
this command, hit the return key, relaunch Safari,  select Debug->Always check
for world leaks, and see what's reported:

	defaults write com.apple.Safari IncludeDebugMenu 1

To turn off the Debug menu, use this:

	defaults write com.apple.Safari IncludeDebugMenu 0
