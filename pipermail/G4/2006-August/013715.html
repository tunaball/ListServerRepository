These should get you started:
<http://www.apple.com/support/tiger/install/>,
<http://www.macmaps.com/upgradefaq.html>, and
<http://www.thexlab.com/faqs/installswupdates.html>.

Bottom line. Make a bootable backup, run Disk Utility and Apple Hardware Test
(Tiger is finicky about RAM specs), disconnect peripherals except keyboard and
mouse, install Tiger using the update previous version option, run Software
Update, apply the updates, run it again until there are no more,  after
restart, run Disk Utility and repair permissions, and update any 3rd-party
applications.
