you can try booting in "open firmware"
reboot, hold down the "CMD-OPT-O-F" keys. O is for the letter O.
At the prompt type: reset-nvram
when it completes type: reset-all
the last command will reboot the system. This command line input has the
same effect and does the same thing a cuda switch would.
