True. Low level access to hardware is intentionally isolated in favor 
of overall system stability. The current generation of disk tools for 
OS X don't have all the functionality that some of the earlier ones 
did. That was a tradeoff Apple made when it adopted Unix for the base 
system. It's a tradeoff that I am pleased with, but as always, 
different users have different needs.
