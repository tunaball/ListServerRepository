Some early G4s had this problem. Most often it was caused by either a bad
external USB device sending an errant signal at startup or an extension
problem. Kensington mouse drivers for example. Unplug USB hubs and other
devices except for the keyboard, and see if the problem disappears.

Try restarting with the shift key held down and see if that works.
Run disk first aid (in your utilities folder).

Try these tips first. If they don't work you might have a damaged preference
file or the mouse wire is damaged. The early oval USB mice had a high
failure rate where the wire entered the mouse.

Let me know how you make out.
