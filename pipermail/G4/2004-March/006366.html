To answer my own question, it seems Eudora Pro 3.0 can
save mail out as TextEdit files very easily. It can also do
this in bulk, in other words, you can select a batch of
messages and save them out as text, although they
will string together like Digest mail.

TextEdit is very cool also as you can then Find/Replace
to put visual dividers in between messages.
