Machines that used the existing hardware architecture (mostly prior to 
the FireWire 800 introduction) would boot OS 9 because the ROMs on the 
motherboards were consistent and compatible with OS 9. Since Apple 
discontinued development of OS 9, they apparently did not want to 
expend the considerable effort to make motherboards with new hardware 
features (e.g. FW 800, USB 2, Bluetooth) dual boot OS 9 and X (not to 
mention that it speeds up migration to OS X).
