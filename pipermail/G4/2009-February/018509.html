Suite, which contains mainstream graphics applications like  
Photoshop. Illustrator, InDesign, Bridge (an application which links  
them all together) and a handfull of multimedia and web design  
applications.

Almost everybody that does graphics will have CS3 or now,  CS4. At  
least almost all Mac people will use CS. Many PC folks use COREL as a  
cheaper alternative, but that has been phased out for MAC.
