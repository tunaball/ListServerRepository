Bill did you read "Everything You Need to Know to Install Tiger" from

http://oreilly.com/pub/a/mac/2005/04/29/tiger.html

Your problem might be your firmware revision level.
You'll find you Boot ROM Version by clicking on the "Apple" in the  
upper left hand corner and select the "About This Mac" menu. When that  
opens select the "More Info..." button and first window that opens  
will have:

Hardware Overview:

   Model Name:	Power Mac G4 (AGP graphics)
   Model Identifier:	PowerMac3,1
   Processor Name:	PowerPC G4  (3.3)
   Processor Speed:	1.2 GHz
   Number Of CPUs:	1
   L2 Cache (per CPU):	256 KB
   L3 Cache (per CPU):	2 MB
   Memory:	1.75 GB
   Bus Speed:	100 MHz
   Boot ROM Version:	4.2.8f1
   Serial Number:
   Sales Order Number:

Which can be very useful
