The MDD restore disk will only work with 10.2.8 or below, Apple have 
released a patch to allow you to use it with 10.3 / 10.4 but I cannot 
find the link to it at present.
As others have stated, the retail version of 9.22 will not work properly 
as it is missing some drivers and a ROM file.
You may be able to use Pacifist to extract the 9.2.2 system directly 
from the CD.
