I came across this little tidbit the other day and you might want to  
give it a try, it was stated that it might be helpful when you  
couldn't get your system to boot.

1. Start-up in single user mode
2. enter the following:

sudo nvram boot-args=""
reboot

I don't have any problems booting but I've tried the command sequence  
and it does cause a reboot.
