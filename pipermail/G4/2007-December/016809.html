> i would like to back up my installation discs in case they get lost

With a CD burner in your machine, use a disc burning program like  
Toast to make a copy of the installation disc. This works with OS-9  
and the CD versions of OS-10.

With later versions of OS-10, you need a DVD burner because of the  
amount of data in the installation disc.
