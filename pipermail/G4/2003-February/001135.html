You could also try the trick to restart in OS9 instead. There is very 
little security under 9 so one can really mess up the files belonging 
to OSX if one wants to, but in this case you anly want to delete one 
broken file, so if you don't want to get into the terminal and that 
stuff, this is a fairly simple way to do it.

The broken file will not be placed on the desktop under OS9, but in 
the folder desktop in your user-folder. Look for it there, or do a 
search for the name.
