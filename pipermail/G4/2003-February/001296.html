Have you tried using the built-in search function to find it?
Simultaneously hit "Command" key (it has an Apple or a Cloverleaf on it) and
"F" key, point it to your System disk and enter "Favorites" in the keyword
field.

In MacOS 9, System -> Preferences -> Explorer -> Favorites.html
In MacOS X, Home -> Library -> Preferences -> Explorer -> Favorites.html
