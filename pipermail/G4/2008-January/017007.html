I have had the same problem when using my usb 2 card on OS 10.3.9.  However, I found that the problem only seems to occur with bus powered devices connected to he usb 2 card.  I have two printers that are self powered and there is no problem with them connected.  I use the built in usb 1 connection for my keyboard and mouse and if I need to use a bus powered device on the usb 2 card I disconnect before sleeping.  A bit of a nuisance but better than a hard shutdown with the startup button on my 1.25 GHz DP MDD.  I haven't seen the problem since upgrading to OS 10.4.11 but I haven't really tested all combinations.
