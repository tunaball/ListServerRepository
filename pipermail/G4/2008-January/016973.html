Greetings,

If you can boot into the OS 10.3 install disk, you should be able to  
open the Disk Utility that is part of that CD, then run the Disk  
Format option for your HD, Next try installing the OS 10.3 operating  
system. Or perhaps you might want to run "Repair" first and see if  
there is problems with the disk.
