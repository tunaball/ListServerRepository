To be precise, there is no such thing as SATA II, as an official
standard. "SATA II" has come to be used by drive manufacturers to
indicate support for a 3 Gb/s transfer rate instead of SATA's
original 1.5 Gb/s. Other than that, the drives are probably the same.
See the following web page for more information:

http://www.sata-io.org/namingguidelines.asp
