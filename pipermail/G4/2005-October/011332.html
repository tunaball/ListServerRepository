External HAD Formatting Question:

>You might want to consider using their latest which I think is NTFS.

YOU DO NOT want to format as NTFS unless you want to only be able to 
READ the drive on the Mac. The only format that will allow you to 
read and write the drive on both the Mac and a Windows machine is 
FAT32 (which is what you get if you select MS-DOS as the format 
option under MacOS X. One other GOTCHA with NTFS is that it is not 
even readable under MacOS X if you select the Compressed Option (ie; 
The drive must be left as uncompressed if you want to read the 
contents under MacOS X).
