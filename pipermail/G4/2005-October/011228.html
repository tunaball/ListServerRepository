It's not really very hard, Open your System Preferences and select the 
Sharing folder.

Select the Service tab if it isn't already selected. Then choose the 
service that you want to share. Next click the "Personal File Sharing 
"Start" button.

Next select the Firewall tab and click the "Start" button to tell the 
firewall to allow access from the outside world.
