> under a table, and buried under my router, Airport base station and  
> an external drive I was too lazy to install internal. :-)

>  I have OSXvnc set to start when the G3 starts up (even though I  
> normally leave G3 running). If I need to do something, I start

> together.

>
> OSXVnc can be found at: sourceforge.net/projects/osxvnc
>
> This was the best solution I found for cheap remote access.
