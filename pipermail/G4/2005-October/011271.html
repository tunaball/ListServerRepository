Hector is right, that usually works. And by the way, did you use Disk

Here is something else that might help you (or others). If you go to
www.versiontracker.com and search for Print Center Repair, you can download
and try that utility. It has solved some problemsfor me. Here is the
description:


Product Description:
Print Center Repair is designed to rectify situations where Print Center
will not open, printers cannot be added, and numerous other errors. Although
Print Center Repair cannot fix every printing problem, it is capable of
resolving most of the common ones. Full instructions are included.

Print Center Repair allows the user to do the following:
Verify the root user
Verify and recreate the daemon user
Verify and repair Print Center file permissions
Verify and repair the /System/Library/Printers directory permissions
Reset Print Center
Verify and repair the /private/var/spool directory
Delete CUPS spool files
Verify and repair the /private/tmp directory
Verify and repair the CUPS directories
Verify the status of the cupsd process
View and modify the hostconfig file
Delete hidden temporary files
Delete Print Center preferences
Delete custom preset preferences
Delete the pre-installed drivers
Install PPD files
