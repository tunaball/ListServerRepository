It's not a patch. It doesn't change or fix anything.
It's an application that checks if your machine can
run a dual processor.

It's downloaded as UniNorthAISCChecker.sit which is a
compressed stuffit file. After unstuffing it's
UniNorthAISCChecker.app application file.

Click on the downloaded file, go to "get info" in the
file menu and then change "open with" so it opens with
stuffit.
