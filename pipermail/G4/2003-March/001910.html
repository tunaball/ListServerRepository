Richard -

Steve's response:

<a href=http://www.yournamehere.org>Your Text Here</a>

Is accurate, however, there's more to it than that as most browsers use a
<meta> tag to do this.

The best way to see this in context is to surf to a page that redirects you
to another, then (before the timer kicks in) to switch your view to
"Source".. This works in both IE and Netscape.

 Then you can see and copy some HTML code that works. The code is pretty
straightforward. You shoulkd also see how to put a timer in that
automatically will shift the user to the new page. You may need to add a
text link so that users with really old browsers can find the new address.
