Melissa Crowe wrote:

Sorry for this dumb question. How does one close the cd tray from the 
keyboard on a Sawtooth G4 450 running OS 9.2


Melissa,

This is a tip for the Quicksilver G4, but it might help you. Go to this
webpage:

http://www.bella-usa.com/Support-EZKeyboardQA.htm

In the answer to the first question, there is some info about Eject Extras,
which contain an app called Eject. If you do not have this app on your OS 9
install CD, click on the link toward the end of the answer to the first
question, and download the Eject Extras folder, and install the Eject app,
or the Eject Control Strip. See if one of those helps you.
