in system preferences -> energy saver -> options

make sure that "restart automatically after a power failure" is not 
selected. at one point i had this selected (forget which version of os 
x) and i could never shutdown, the system thought it was experiencing a 
"power failure" and would start right back up.

this might not help, but starting simple is usually a good thing.
