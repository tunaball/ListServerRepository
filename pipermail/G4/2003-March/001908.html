If you are using a postscript capable laser printer, it might be that the printer does not have the fonts you used in your document. You might need to download the fonts you have used in PM6 to the printer so they can be available for the printer to use.

It looks to me like your printer is using the default font setting when it can't find the exact font you used in the document.
