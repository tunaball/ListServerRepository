The first thing you should do. before trying to repair the disk, is move
files/folders to another disk, one at a time. If successful (i.e., no
crash), then erase that file/directory from the faulty disk. Keep doing
this until you've reclaimed 10-20% of free disk space. Then try repairing
it. Use Disk First Aid/Disk Utility to repair it, booting with the install
CD (you can't repair a boot volume). Then try moving the rest of your stuff
to another drive. If you can do that, then you should be able to erase it,
reformat it, and that should isolate the bad blocks and fix the bad files.
If it doesn't work, then you might try Disk Warrior or Norton Utilities.
