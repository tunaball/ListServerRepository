Keith,

Buffering can be adjusted in Windows Media Player, but I've never  
played with it so I can't guarantee any significant level of  
success.  To do this, open up the Windows Media Player application  
from your Apps folder (or wherever else it may be).  Then click on  
the Windows Media Player menu and select Preferences.  Under the  
Network Settings tab, you'll see a section for Buffering.  There, you  
can stipulate how many seconds you want the player to wait while  
buffering before it begins playing.  Try 60... see how that works.
