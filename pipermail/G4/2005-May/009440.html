Although Appleworks included an earlier version 
of MacLink, it is still possible to buy the stand 
alone program MacLinkPlus Deluxe® from Dataviz. 
The most recent version is 15, the program is 
upgraded frequently as new file formats are 
introduced. First time cost is ~$80; the upgrades 
do cost ~$40, but it is only necessary to upgrade 
if you run into a file you can't convert. I've 
been able to skip several versions (I think I 
went from 10 to 13) and there is no penalty for 
doing so.
This might prove a better solution than switching 
to other database and spreadsheet programs, 
especially if Appleworks still meets most your 
needs.
