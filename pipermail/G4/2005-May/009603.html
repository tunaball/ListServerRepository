The message may have a different wording, depending on which version of 
the OS you are using (clearly not Tiger). Such details are relevant.

The message says that the OS recognizes a drive is attached, but it 
does not provide a disc recording driver for it. Consequently, 
applications which use the DiscRecording framework (e.g., iTunes, 
Finder) cannot burn with that drive. Other applications (e.g., Toast, 
DragonBurn, cdrdao) can, because they don't use this framework.
