>  Would this apply to stand alone hard drives as well?

Anything connected via something *other* than the onboard 40-pin IDE 
interfaces will be limited by whatever mechanism, interface and 
standard is used to hook it up. Example: a modern 2 TB external USB 
hard drive will work fine. Slow, but fine.

OSX's own limit on maximum drive size is thirty-two terabytes. I 
don't think you'll have issues with that :D
