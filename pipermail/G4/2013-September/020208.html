MacTracker is a wonderful free reference which lists every Mac
model and its characteristics.  It is my reference whenever
I am asked about a specific Mac computer.

Which version you should download depends on with version of
Mac OS X you are running.

The older versions for OS 9 and OS X 10.0 through 10.5 can be
downloaded from here:

   <http://www.mactracker.ca/archive.html>

The latest version, for OS X 10.6.8 or later on Intel-Macs:

   <http://www.mactracker.ca/index.html>
