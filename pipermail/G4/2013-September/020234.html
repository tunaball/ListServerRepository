FWIW, this is what I did. I have a wonderful G4 Quicksilver 2002  
running Tiger 10.4.11 and it performs beautifully. The main problem  
was speed and browser problems with the internet. If I upgraded to a  
G5 with an intel processor, I would have to buy all new versions of my  
applications. Instead, I purchased a refurbished G5 with the old apple  
processor. I think they stopped producing these in 2007. The cost for  
the refurbished G5 was only $150,00. It came with Leopard OS. The  
speed is fantastic with minimal browser problems. My guess is that I  
have a few more years before it becomes like the G4. As far as the G4  
goes, I hooked it to the G5 via a firewire cable, turned on file  
sharing and screen sharing and, with both machines turned on I can  
access all the files on the G4. I loaded all my old apps onto the G5  
and they work just fine.
