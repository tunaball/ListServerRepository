On 08/30/07, "Petter Adsen" <petter.adsen at gmail.com> wrote asking:

> anyone got any idea what might be wrong, things I can try, or pointers to
> documentation I should read that could help me?

More than you ever wanted to know:
<http://docs.info.apple.com/article.html?artnum=60285>. Ignore the fact that it
addresses network server issues. It's applicable to clients, also.

Boot the machine and when it goes into OF, type these in, one at a time, and
hit the return key after each:

reset-nvram
set-defaults
reset-all

The last reboots the machine.
