I suggest that you look into using ISO 9660 as the format of the CD that you burn. If you burn as a Mac format CD, the PC will not be able to read the CD at all.

ISO 9660 is the hybrid format for multi-platform CDs. If you use Roxio Toast, it will show up as "Mac OS Extended and PC (Hybrid) CD" and will be readable by both machines.
