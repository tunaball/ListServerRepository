> You could always try spoofing that web site by choosing another user 
> agent
> (i.e., telling it your using another browser) in Safari. To activate 
> the
> Debug menu, quit Safari, launch the Terminal application, type the 
> below
> command in the Terminal window, and press return:
>
> 	defaults write com.apple.Safari IncludeDebugMenu 1
>
> Quit Terminal, launch Safari, and note that Debug is now the last Menu
> item. Select User Agent from that menu and you can select one of these
> emulations:
>
> Mozilla 1.1
> Netscape 7.0, 6.22, and 4.79
> Mac MSIE 5.22
> Windoze MSIE 6.0
> Konqueror 3
>
> To disable the debug menu, type this in Terminal:
>
> 	defaults write com.apple.Safari IncludeDebugMenu 0
