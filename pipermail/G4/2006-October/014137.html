> Also, did you confirm that the DVD drive was Mac OS compatible

Patchburn is a utility that will solve the booting problem with many  
third-party drives. I've used it on several machines to recognize  
various brands of CDR and DVD drives. Available here:
<http://www.patchburn.de/>

Shareware, so toss the author a couple of bucks if it works for you.  
According to the webpage,
:PatchBurn is a tool to patch existing CD/DVD-drivers (under Mac-OS X  
10.2.x) or to generate and install new device profiles (under Mac-OS  
10.3.x and later):

Make sure you use the correct version for your OS.
