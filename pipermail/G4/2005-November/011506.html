A codec is an extension of sorts that allows the machine to decode a  
particular format of file.

For your files, it seems that the required codec is included in the  
VLC (and likely Mplayer) software, but QT does not share codecs with  
these programs.  In order to allow QT to play the clips you have two  
choices:

1) Find out what type of codec is being used (using Get Info, for the  
Video track format) and locate a compatible codec from the web and  
install it (e.g., DivX).  You can first try http://versiontracker.com  
to locate and download these to some extent.

or

2) Transcode the file from the existing format to a QT compatible  
format (like MOV).  For this, you can use a program like ffmpegX  
which you can also download from versiontracker.

(or 3, just use VLC or Mplayer and avoid the hassle since both  
support full screen natively and for free)...
