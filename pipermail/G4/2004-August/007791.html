On 08/06/04, "Daniel Brieck Jr." <djbrieck at mac.com> wrote:
>
> Does anyone know how to get the mac os x welcome setup  and
> registration to come up. I want to get rid of my older imac but I don't
> want to reinstall an older os with no updates just to get the welcome
> and registration first run stuff to show up. I have tried deleting the
> user folder and Library preferences with no success. There has to be
> some trick to get this to happen again.

When you get the machine setup the way you want, delete this file:
/private/var/db/.AppleSetupDone, then shutdown. The next boot will recall
the setup assistant.
