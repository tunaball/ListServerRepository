Greetings Inga ( + )!( + )
You should be able to correct the incorrect file information if you run the
"Disk Utility"
First select the disk in question and then select "First Aid"

Next click on the button "Repair Disk Permissions" and as long at the file
that was created when OS X was first installed everything should be
corrected.

You might want to run "Repair Disk Permissions"a least three (3) times
because sometimes the first and second pass might uncover (repair) some
items that that need to be passed over again before they are set to the
correct status.

If everything is not corrected then you might want to run the
above procedure from the install disk.
