It looks like it's a 2003 model (according to LowEndMac the 2004
had a 167MHz bus):

http://lowendmac.com/imacs/emac-2003-800-mhz-1-ghz.html

This should be able to boot 9.2.2 natively. But be careful,
because according to the Apple OS Compatibility chart if
the model is "eMac (ATI Graphics) - CD-ROM and Combo Drive"
it will only boot the disk that was shipped with the system
and not a retail disk:

http://support.apple.com/kb/HT1835
