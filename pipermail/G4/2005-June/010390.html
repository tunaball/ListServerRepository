You will first need to upgrade the firmware on the
Pioneer drive itself. Use this link for help on doing
that.

http://homepage.mac.com/mps/OSXDVR109Flasher155FW.zip.

NOTE: The Retail Boxed and XL models use different
firmware than the OEM (BK) models - firmware in the
downloads is for the OEM DVR-109. 

Then you need to run patchburn, make sure you have the
latest versin of patchburn.

Get that from here.
http://www.patchburn.de/
