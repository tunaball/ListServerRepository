Hi Dan,

What that means is that the alias in the Dock is not associated with 
the correct file. Drag it into the Trash, then do a Command F search 
everywhere for "Safari". If it's on your computer, you should find 
either an application or a package called "Safari".

If it's an app, then create an alias and drop that onto the dock. If 
it's a package, double click it to install, then create the alias etc.

If nothing Safari turns up then go to

http://www.apple.com/support/downloads/safari.html

to download a copy.

Regards,

Alastair.
