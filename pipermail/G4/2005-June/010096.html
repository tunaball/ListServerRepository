I just got a replacement Mercury Extreme 1.2. The first one I  
received failed the post IE.. no level 3 cash. They (OWC) were more  
than prompt getting me a replacement. The new one failed as well. So  
feeling nothing to loose, I set the jumpers for 1.33mhz. The machine  
would not boot so I shutdown, reset the jumpers to 1.2, as shipped,  
started up and lo and behold there was the level 3 cashe in the  
profiler and all post's passed. Go figger. I am going to give this 25  
days and see if things stay normal and if not I am going to trade for  
a Sonnet. I have heard a few folks complain about the Mercury  
upgrades but for now mine is doing fine.  Good luck.
