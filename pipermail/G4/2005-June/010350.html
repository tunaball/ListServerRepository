I had to reload "OS X" from the original disk's once and when I 
selected the "Mac OS X Update combined  10.3.x" it wasn't the 10.3.9 
combined, my machine (G4 AGP) had all kinds of problems.
So just be aware, that if after running the "Combined" up-date, things 
don't seem to be OKay, you might have to reload from disk and then go 
to apple.com and get each one of the updates and install them one at a 
time. Be aware that there are also some security up-dates and perhaps I 
didn't install them in the correct order in relation to the "combined" 
up-date. That is assuming that the individual up-dates are still on he 
apple WEB site. They seam to have a habit of removing items.
