some switches do not include software needed to dial PPPoE  (PPP over 
ethernet) which is needed to operate the DSL modem.  i would suggest 
spending $50 and obtaining a DSL/Cable router which says it supports 
PPPoE and using that, or getting the manual for the linksys switch and 
see if it supports PPPoE dialing.

The only reason i bring this up is because i see you specifically 
mentioned it was a workgroup switch, and i believe linksys does not 
build the PPPoE capabilities into these switches.
