I have a 15" TiBook. The problem is, apple built into the firmware that 
these machines will not boot off a 9.1 or earlier CD. 9.2 is iffy. They 
can only boot off of MacOS X and an already pre-installed Mac OS9 
system folder. If you want to boot off an OS9 CD try making a .dmg 
under MacOSX and install macOS9 to it. Then burn it to a CD and you can 
boot.
