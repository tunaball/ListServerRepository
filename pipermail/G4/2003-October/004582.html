This was just posted on Macintouch.com

[21:05 ET]  Dan Casali notes that Apple has now acknowledged a very serious
Panther FireWire 800 bug that could destroy your files:

Apple has identified an issue with external FireWire hard drives using the
Oxford 922 bridge chip-set with firmware version 1.02 that can result in the
loss of data stored on the disk drive. Apple is working with Oxford
Semiconductor and affected drive manufacturers to resolve this issue which
resides in the Oxford 922 chip-set.
  In the interim, Apple recommends that you do not use these drives. To stop
using the drive, you should unmount or eject the disk drive before doing
anything else. Please check this web page for further updates.

http://www.apple.com/macosx/firewire800specialmessage.html
