I believe Disc Burner writes in HFS (someone correct me if I'm wrong), 
Which is an Apple file system. Toast however, can burn a "Hybrid" CD or 
DVD which can be read by any operating system. Disc burner I believe 
burns CD's that can be read in OS9 or X. I however do not know about 
Windows. Toast I know can burn for windows.
