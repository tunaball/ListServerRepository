Jay,

When you boot into Single-User Mode, after the on-screen text entry stops,
type:

/sbin/fsck -y

This tells the system to run a File System Check on the boot drive partition
and automatically answer Yes to all queries. If any problems are found and
reported, then run /sbin/fsck -y again, until no problems are reported. Then
type:

reboot

and the system will reboot into OS X.
