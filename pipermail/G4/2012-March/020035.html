I don't see a reason in this list of messages to explain why your system
would crash, or from running Camino. 

The messages about FSResolveAliasWithMountFlags means that you have
startup items which are no longer present. If you open Users in
Preferences, you can find the startup items listing. There are some
references which no longer exist. Just remove those references

I guess you use to have a program called RarMachine, which appears to be
no longer installed. There is a reference to a missing plugin. According
to your posting, RarMachine was the last logged item

while I have never done this before, you might wish to rebuild your
Spotlight database. If you open Terminal input this command:
	sudo mdutil -E /

Although I can't see how that has anything to do with a crash. Perhaps
someone else has some ideas?
