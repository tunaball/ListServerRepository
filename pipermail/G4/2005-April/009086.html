On 04/05/05, Michael Lawrence <michael at videosonics.com> wrote:
>
> I was hoping it possible to merge two system disks. Basically I have two
> system disks on separate machines, and I would like to know if it is
> possible to move the applications from one of the system disks to the other,
> thereby creating one system disk with the application from both.
>
> Can this be done? Any suggestions would be most appreciated.

This can be done by dragging and dropping applications from one to the
other (determining which is newer, in case of duplicates, is left as an
exercise for the reader), ensuring that the appropriate application support
and enhancers are also copied from one /Library to the other's /Library,
and then ensuring that the appropriate stuff is moved between the users
folders on each.

Or, you can pop for something such as You Synchronize ($20) or Synchronize!
Pro X ($100), for example.
