Sorry but that concept of reliability is based on flawed logic. (By the 
same logic, buying multiple Lotto tickets would greatly increase my 
chances of hitting it rich) For the RAID to fail, only one drive has to 
fail. So the MTBF for the Array is the same as any single unit. Adding 
more components doesn't increase the likelihood of any single unit 
failing (as long as heat is being dealt with properly).
