> Is your MDD a Firewire 800 model? If so, you will not be able to boot
> up natively in OS 9.
>
> Only the Firewire 400 MDD's without Firewire 800 can boot natively in
> OS 9. If your MDD is a Firewire 400 only model, then you need to get
> the gray colored software install disks that were shipped with the
> Firewire 400 MDD's. The retail edition of OS 9 will NOT  run on
> MDD's. I own 3 Firewire 400 dual boot MDD's and found out the hard way.
>
> You might be able to find the Firewire 400 MDD software install disks
> on Ebay.

Call AppleCare and request replacement discs for your machine. You'll need the
serial number, but if they're available, should only cost S&H.
