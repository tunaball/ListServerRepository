Yes, the space should be there.

sudo /etc/daily
sudo /etc/weekly
sudo /etc/monthly

sudo is a command which runs programs as another user - root, a.k.a. 
superuser, by default. /etc/... are the scripts (programs) to run. In a 
terminal window type "man sudo" for more info.
