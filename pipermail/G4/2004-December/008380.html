The ports are TCP/IP ports - if you have an XP firewall, they may need 
to manually opened.

As for the password, i don't think (but haven't tried in years) that 
samba works well if you have a NULL password.  Also, the password is 
case sensitive but the username and workgroup are not.

If you look at that website, there are also commands to do command line 
(from Terminal or XTerm) smb status and connections described well.

Lastly, did you install the BSD subsystem?  That might also make a 
difference.
