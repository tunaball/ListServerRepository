As CJ said there should be no problem running OS 9.2.2 on any G3 Mac. 
I have done it repeatedly with out OS X on the system.

If you can I would recommend that you find a complete OS X.1 system 
package, you can find it on the net go to a friend house to get on 
line, use the system 9disk to boot your machine and backup all the 
data you need to.  Then reformat the disk, install 9.X that comes 
with the package, if it is 9.2 then go to the net and download from 
the Apple software update site 9.2.1 & 9.2.2.

The other problem that you have is not enough RAM, if 128 is all you 
have, you should have at least 256 Meg to run 9.2.2 and better yet is 
1gig.

Hope this helps.
