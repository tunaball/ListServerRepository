don't want to say i am correct but you may want to think about the
following.....
toast 5 you need to compress before you burn so that is why vcd burn o.k.
toast 6 is built for dvd burning so a file that is in dv format would need
to be processed as it burns to a lesser format.
seeing dv of any quality has compression then we will always get errors when
we push envelops of conversions.
