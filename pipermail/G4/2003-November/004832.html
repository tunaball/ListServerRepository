> 1)  Create a user for yourself - make sure it has administrator privileges
>     System Preferences -> Accounts -> New User ...
> 
> 2)  Delete the previous user
>     System Preferences -> Accounts -> Delete User ...
> 
> 3)  Change the name of the system
>     System Preferences -> Sharing
