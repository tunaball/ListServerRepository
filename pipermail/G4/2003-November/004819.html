My Brother 1670N postscript printer uses Appletalk over ethernet. I 
have both under Jaguar and under Panther have needed to be sure to 
define this printer, specifying the correct PPD. I think I got this PPD 
originally from Brother's www OSX support page. The location for these 
is /<bootvol>/Library/Printers/PPDs/Contents/Resources/en.lproj/ .

Specifically I'm talking about the printer setup utility:
add printer
choose Appletalk
choose printer from list (I choose the "Postscript Printer" item)
choose Printer Model: Brother (in my case, i.e. I specify here, I don't 
use "automatic")
choose appropriately from "Model Name" list (this is where the action 
is, I think)
click "Add"

This selection is key to getting the actual features the printer has, 
duplex, different dpi settings, etc.

However all is not perfect with my Panther printing. For my 
applications that are printing, e.g. Mozilla, Discus CD label printer, 
everything seems 100% as expected. On the other hand Apple's Mail 
program, which I'm using as I type this, won't print. It tries. The 
printer starts blinking its incoming data light. But then nothing 
actually prints!
