On 11/14/03 11:56, "Nat" <nbaker at discover-net.net> wrote:

> My number pad from my keyboard has been working sporadically and it seems
> when it would be working for a few days, it would stop working on a restart.
> I don't know what would trigger it but suddenly it would work again.  I
> finally got tired of it and tried another keyboard but it wouldn't work.
> Now I've tried every keyboard in the house, none of them work (just on the
> number pad) and tried the same keyboards on another computer and they're
> fine.


I solved my own problem.   I just found it on MacFixit forums.  People are
going to want to know this just in case, because it's goofy.  I've noticed
my Universal Access prefs get changed somehow all the time.  Sticky Keys
turns itself on and it's annoying so I go in and turn it off.  Well, it
turns out that under the Mouse tab, "Mouse Keys" turns itself on also, so
when it's on, the number pad doesn't work.
