Since it looks like nobody is real interested in
owning a "Yeager" Duo 280 Prototype, I've thought
about using it as an e-book reader if that screen
rotation software will work on it. :)

But what I haven't been able to find is reader
software for some of the e-book formats. :P HTML
and plain text will of course work just about
anywhere,
possibly RTF too. But I didn't see anything for
Adobe's e-book format for use on a 68k Mac, and there
are some other formats out there that are made for
use on Palm, Pocket PC and dedicated e-book reader
devices, some of which there are Windows reader apps
for or for PPC Mac OS or OSX.

If I have to find some ways to "hack" the other
formats
into plain text, I'll do that, but it'd be nicer to
just have software that'll handle the files in their
original format.
