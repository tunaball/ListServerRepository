For use on System 7.5.2 and older, MacPortrait
requires a display enabler.

I found it here
http://www.adi.nl/nl/support/olddrivers.html
with the complete MacPortrait 6.0 installation.

The later version of MacPortrait linked on the list
works great* on my 280 after dropping in the display
enabler, which is an Apple extention from the
AppleVision 1.0 update.

*Great meaning it works in the Dock. Undocked it pops
an Error -18 and doesn't work. :( The MacPortrait
control panel says I should be able to flip it to
400x640 16 greys but System 7.1 insists there's
something it doesn't like about it when undocked.

I haven't tried version 6.0 yet. Anyone out there
with a 280 that wants to try it or has any insight
as to why it works OK docked but not when undocked?

I don't see how it could be a RAM issue since it
uses more when docked for rotating the screen's
higher resolution plus 16bit color. Thousands of
colors at 480x640, neat, but not portable. ;P
