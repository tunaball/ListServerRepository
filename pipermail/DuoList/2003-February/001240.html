>> Has anyone thought about trying to modify a TFT display from

>It's possible.  I haven't been able to find the pinout diagram for this
>model (there is a Sharp LCD module that uses nearly identical model number
>but uses VGA frequency.)  You would need 3 4bit ADC chips to convert the Mac
>video signal to the digital signal for the LCD and it would only run as Mac
>monitor at about 35KHz.
