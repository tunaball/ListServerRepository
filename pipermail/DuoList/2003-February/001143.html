Your best bet for reformatting the hard drive is to
use an old Apple external CD ROM like the 600e.

This way you can boot from a system CD and reformat
and reinstall the OS. I am not that famaliar with the
upgrade card, but it might need an extension file to
make the card work. Look for and copy any system files
that have the upgrade mfg's name on it.

As for networking, if the you can't use ethernet. You
can always use to the apple talk network boxes. Slow
but the will always work.
