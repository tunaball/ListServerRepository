Conflict Catcher was a program designed to eliminate conflicting 
startup items (Extensions, Control Panels and Preferences) so you can 
avoid this problem. So can you get the machine to start up with the 
extensions turned off? What OS version are you running? What 
non-standard (didn't install with Apple's OS) Control Panels and 
Extensions are you running?
