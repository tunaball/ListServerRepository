I have similar problem when I upgraded my 2400 to MacOS 9.1, the 
external CD ROM that used to work with the previous OS and a third 
party drive, FWB CD-ROM Toolkit is not mounted on the desktop.

The problem was solved by installing the Silverlining Pro 6.4.5. This 
program can scan any hardware and mount it on the desktop. There is 
no eject button in the Silverlining control panel nor it is not 
possible to eject the CD by thrashing but to shut down the 2400 and 
manually eject the CD ROM to remove the disc.
