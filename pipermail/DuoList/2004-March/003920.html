Hi Paul,
I was having very slow page loading on my 2300c using
Netscape Communicator 4.01. I shut off Java in the
Netscape options and it works better now. (I was also
getting a lot of Java errors since Java has changed a
lot since the 4.01 version came out.) I'm was using OS
7.5.3, now 7.5.5.
I increased virtual memory to 56Meg, with 36M RAM in
the 2300c.
I also changed from MacPPP to FreePPP. I also reduced
the speed setting of the serial port from 115k to 56k
since the 2300c apparently only runs no faster than
56k. That helped a lot!
I was using a USR 56k Sportster with a custom made
cable as described on the 3Com site. I also used the
Sportster high speed driver I found on their website.
With all that done, it is still very slow compared to
my Athlon 1900+ with 512Meg RAM, with the same modem
(no surprise there I guess!)
I hope some of this helps.
