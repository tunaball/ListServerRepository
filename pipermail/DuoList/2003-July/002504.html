With your own domain name and a broadband connection
you can run your own e-mail, web and FTP servers.
It's easiest if you get a fixed IP but there are
services that will supply a URL for HTTP/FTP and
automatically track your non-fixed IP so that changes
are transparent to others. Look up "dynamic DNS".
Some of those services are even free.

Windows NT/2000/XP have e-mail, web and FTP
services "built in" (along with Telnet). You just
have to start and configure them. Linux also has those
services.
