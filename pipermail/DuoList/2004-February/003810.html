To check memory allocation, find the actual application. Click on it, 
then Apple+I for Get Info. You'll see a drop down box, select Memory. 
There will be a "minimum" and "preferred" window. Allocate a chunk of 
memory (way more than the default) and then launch Excel. If it 
launches, you fixed the problem. I'd definitely find a legal, 
licensable copy of Office, though.
