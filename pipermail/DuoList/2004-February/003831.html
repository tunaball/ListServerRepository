Robert,

First, please check if your floppy disk continues to
be readable by your Color Classic and other PC's. 

If you already have the PC Exchange (or File Exchange,
as it is known in latter versions of the Mac OS), open
the control panel and tinker with the settings first.
Then see if the problem persists.

If it does, then get a floppy disk cleaner and clean
your disk drive using the freeware utility TechTool
(now TechTool Lite). Restart and then try again.

If you want to find out if the problem is with your
floppy disk drive, then see how it reads other floppy
disks.

Hope this helps.

Keep rocking,
