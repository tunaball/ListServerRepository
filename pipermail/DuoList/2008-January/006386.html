Two issues:

*The 2300c has no PCMCIA slots, so no USB on that what-so-ever (some models,
such as the 3400c, 2400c actually have Cardbus enabled slots [or have
ability to be converted to use Cardbus] for USB)

*I had a difficult time using Ubuntu (actually I had to use Xubuntu [Ubuntu
with a lightweight XFCE window manager] on my 366Mhz G3 ibook with 512MB
Ram, so I would reconsider putting Ubuntu on a 2300c, with its 100Mhz 603
with very little (or maybe even none) L2 cache. BTW, any machine previous to
the G3 (aka OldWorld machines) do not have the ability to run Ubuntu
straight out of the box, it requires dual booting MacOS and switching to
Linux via a boot manager inside of MacOS.

i'm not saying its impossible to install Ubuntu to a 2300c, but it will NOT
run well with a 603 (with little L2 cache), Ubuntu needs much more RAM than
56MB, and USB is most certainly out of the question.
