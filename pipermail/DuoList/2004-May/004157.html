You need a floppy drive, a Network Boot Disk, a PCMCIA ethernet adapter 
(non card bus to be on the safe side).

When I wanted to do this, I bought a SCSI external hard disk and 
appropriate cable off eBay for not very much money. I asked the vendor to 
install an OS9 system folder and a copy of the OS9 install image onto it 
for me. The Powerbook boots from the external hard disk, and I can prep the 
internal drive and install a fresh OS image.
