Do you get the blinking floppy disk icon ?

Hmmmm.... you have a difficult case there.  The 2400 has no CD rom, 
and the OS you would want to load isnt on floppy.  You dont have 
ethernet on board.

There are really 2 ways you can load an OS:

1)  Use an external CD Rom hooked through the SCSI port.  Boot off of 
the OS CD ROM and install

2) remove the HD and put into another machine where you can install 
the OS there (not recommended unless you are familiar with the 2400 
take apart)
