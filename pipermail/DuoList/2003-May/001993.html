        http://docs.info.apple.com/article.html?artnum=58172

        What you have on the Duo is an Apple Express Modem.  The modem
needs the old Apple Express Modem software so the "internal modem" shows
up in the Modem control panel.

        Apple, however, renamed the software "Apple Telecom" from "Apple
Express", so it's rather hard to find.

        The instructions on the Web page (above) explain that the Easy
Install won't work on a 68K Macintosh (your Duo).  You can still get the
modem software installed, however, by choosing Custom Install and only
selecting the modem software (not the phone book, the fax software or
anything else).

        You'll have to download all of the disk images and copy them to
your Duo, then mount them all (by double-clicking).  The first disk has
the installer.

        Good luck.
