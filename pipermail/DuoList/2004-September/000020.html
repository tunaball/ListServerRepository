I love Tex-Edit.  The "Classic" version works on system 7.1 up, so should be 
fine on the 280.      http://www.tex-edit.com/

While you're at it, check out Net-Print(CP and extension).  You can set it to 
save selected text from an icon in the menu bar as any text file(including 
Tex-Edit), and can append text to an already existing text file without even 
running the existing file's app.
http://www93.pair.com/johnmoe/
