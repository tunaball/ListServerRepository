I have done it with the dock, with the built in ethernet. It was a no 
brainer activate appletalk over the ethernet  and that was it. if you 
want to go without the dock you will need an  appletalk to ethernet 
bridge device.
However,I cannot remember if  the duo will not be able to read files on 
the eMac, HD format issue I think.

Before you move everything buy a really long cable(100 footer) and try 
it.
