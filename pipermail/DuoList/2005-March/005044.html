You can share files using AppleTalk and Personal File Sharing from an 
OSX machine just like you would from a System 9 (and lower) Mac. 
Similarly, you can mount drives from a System 9 (and lower) Mac over 
AppleTalk on an OSX machine by "browsing" the network (cmd-K).

Bottom line:  sharing files between OSX and lower MacOS machines is not 
a problem at all.
