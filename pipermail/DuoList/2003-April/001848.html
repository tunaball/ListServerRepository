If the only time you've used the Duos built in screen is on the iBook 
adapter then you are probably just seeing the passive matrix screen

The 190 uses the same coax connector as the 5300 and is incompatible 
with the others.

The early iBooks (the toilet seat model) use the same connector as 
the Duos, 1400, 3400, Kanga and G3 series.  Later models use a 
similar style connector but it is smaller in diameter.
