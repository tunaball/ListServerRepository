Suspect hard drive.

Hardware--It could be overheated/ing or suffering other problems (seen
similar before) You can also test this drive via SCSI mode.

Software--HD directory could be messed up, virtual memory (try RamDoubler
8), improper ram allocation to app, corrupted application

Try reinitializing drive, re-install apps and go from there. 48mb ram is
still OK on 8.6.
