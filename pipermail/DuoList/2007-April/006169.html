I don't have a lot of experience with Mac's but I do with memory limited PC's. I would definitely empty the trash bin, and any temporary internet files (if you are using the internet), and other temporary files that you are sure aren't needed. Of course, if you trashed the icon for the hard drive, restore it first; since you rebuilt the desktop you probably have restored the icon already. After you get rid of all the temp files, power down the Mac, wait a bit, and then reboot.
  If you have anything like a "Favorites" internet sites list, try and get rid of the excess never used stuff. This may not solve a more serious problem with the file structure, but it should help otherwise.
  Uninstalling programs you never use will help reduce the garbage on the hard drive.
  Almost forgot, if you use internet, get rid of all the cookies every day. They will slow down your computer a lot.
