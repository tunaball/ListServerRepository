If you've never seen the "secret about box" built into
the 2400 (and all other first-generation PCI machines
such as the 8500), you should definitely check it out.
It's the best easter egg ever. Unfortunately it
requires installing 7.6.1, which I detailed in a
different post, but it's almost worth it.

In 7.6.1, open a Sticky (or SimpleText, and type
"secret about box" (no quotes). Then highlight it, and
drag the whole phrase to the desktop as a text
clipping. Then go "whoa" and start moving the mouse
around. If you want something really cool, move the
mouse quickly back and forth for a few minutes.
