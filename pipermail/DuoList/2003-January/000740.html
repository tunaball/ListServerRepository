	I was watching all the comments on the PC card slot. I also wasn't
interested in the iBook because it didn't have a PC card slot. After
seeing the new 12" G4 I decided to take stock of my PC cards too see
what I would be missing. Turns out I had less than I thought.

	1. Ethernet
	2. Firewire
	3. Modem
	4. USB

	all now built in

	5. PCMCIA HD card 89MB and a 174MB card which I use as emergency drive
	6. IBM Travelstar 8E replaced with 20GB HD

	both of these now can be replace with Firewire external if I wanted

	7. Road Rocket card

	with one of the latest list discussions which discusses using a patch
to use dual monitors. Hopefully this can be used with 12" G4

	I thought I had other PC card but this was all so I see no reason to
have a PC card slot now.

	Ralp Mawyer you talked about a SCSI to Firewire converter I haven't
seen any info on those. I have 2 different USB/SCSI converter would be
interested in a Firewire/SCSI converter.

	One of my thoughts when I got the 2400c was I would like a full size
keyboard. Although I had no problem with the 2400c keyboard. Also using
all the space for a monitor which is what seems like the new 12" G4 does
