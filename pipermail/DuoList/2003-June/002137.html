It works. The primary downside of using a flash card as a "hard 
drive" is that it has a limited number of read/write cycles built in. 
If used like a HD its life expectancy will be significantly less than 
what one might expect when using it to store photos, etc.

Good backup approach, if you can get it to boot on startup.
