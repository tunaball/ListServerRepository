If you see the Happy Mac face on the system folder then it's already 
"blessed" and you don't need to do anything else.

If not, then you need to open the system folder, drag the "finder" or 
the "system" out onto the desktop and then back into the folder. The 
System Folder should now have the "Happy Mac" face on it and your 
system is now bootable and should be selectable as so in the Startup 
Disk control panel.
