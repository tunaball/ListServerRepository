I am guessing but it is probally a driver conflict
because the both want to grab TCP/IP and Appletalk at
startup.

If you really need both at the same time
Try inserting after boot up. Orinoco first, set your
tcp/ip. Then the GV and set for appletalk.

I assume that is why you need both at once.

They cannot be used at the same time for the same
thing. If one is TCP/IPthe other cannot, etc.
