Power supplies are all cross compatible, except older ones might not 
charge the battery quickly and run a 2300 or 280c.  Batteries all are 
all cross compatible except type 3 batts need a software extension if 
you use them in a 210/230 running earlier than 7.6.

If you want to swap out the modem, you have to tear the unit down, 
which isnt difficult, but maybe not worth the effort.
