The processor runs all but OS X well....it will run.

OS 8.6 is arguably the most stable, with a good feature balance. 7.6.1 
is very fast and 9.x brings more features to the table, e.g., iTunes, 
etc.  Create some multiple partitions and try them all out.

Keep in mind that the PC card memory has a fixed number of read/write 
cycles so at some point you will pay a price for using it as a virtual 
memory device..but heck, give it a shot.
