Apple has made Macs with every model capable of accepting maximum 
amount of RAM aside from having base RAM for upgrading purposes.

If you are not sure how much RAM you have in your Mac, choose About 
This Macintosh (Computer) from the Apple menu. If you want to upgrade 
the PowerBook RAM, you have to know the various RAM requirements in 
particular PowerBook model including the base RAM, maximum RAM, type 
and speed of RAM and module sizes of RAM. In every PowerBook you'll 
have one available slot for upgrading the RAM and its RAM modules can 
be installed with one module at a time. In some PowerBook models RAM 
modules can be stacked on top of one another which enables to add two 
modules to the one slot. You can download the Apple Memory Guide 
document from Apple's Support Web site for all the memory 
requirements for all PowerBook models that includes diagrams.
