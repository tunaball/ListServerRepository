I'm using a G3 with Verizon DSL. However, my 2300,
with internal modem, operates okay in the dial-up mode
providing I first disconnect the Verizon DSL modem
from the phone line.  To facilitate this, I've
installed a Radio Shack line selector switch (RG-11
ports) in the phone line which allows me to switch the
line between the DSL modem and the 2300. One thing: To
make the 2300 dial-up work, I had to install a DSL
line filter (the type used on phones) in the 2300
branch.
