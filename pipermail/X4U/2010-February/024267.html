A friend moving form an MDD to a 24" iMac even had trouble using the migration assistant in target disk mode.  We ended up having to use SuperDuper to dupe is old drive onto a FW external, then running migration assistant with that as the FW device worked fine.

I've never seen Migration Assistant work across a network, so if that is what you are trying to do, I add my voice to the chorus suggesting to use target disk mode instead.
