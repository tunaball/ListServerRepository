David: following the Gmail instructions, I was able to set up my gmail  
account as one of four POP mail servers that is handled by Mail. Works  
like a charm.

I found that I can also use my gmail account on my iPhone when I'm on  
the road and using Wifi. The gmail smtp server will send e-mail that  
otherwise would be difficult with my main e-mail account without  
knowing the smtp server on the wireless network I am on.

Good luck.
