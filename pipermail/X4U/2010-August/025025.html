Put ~/Library/Preferences/com.apple.finder.plist in the trash.
Close all open Finder Windows.
Restart.
Open a new Finder window, then using View set the status of pathbar, statusbar, sidebar, and toolbar as desired. Do nothing else in the window.
Close the window; this will write a new plist file with preferences set to what the recently closed window had.
