Not sure what your exact backup system is, but if I were using iMacs,  
I'd have my data files located on external firewire drives. That way  
if the iMac does become a paper weight, even if only temporarily, you  
have your data available. If you data file load is not too great you  
could probably just mirror the internal hd to an external drive.

Of the iMacs I have familiarity with, which only number 6, but all  
late model intel machines, none have had even the slightest burp.  
Anecdotal, but my experience. :)
