At 10:39 PM +1100 27/11/06, Mark Gibson wrote:
>Hi,
>
>Someone at our work has done some tinkering and managed to disable 
>"Bonjour" (open Printer Setup Utility and a printer that other 
>machines detect, this one can't).
>
>Is there some preference somewhere to turn this off / on or have 
>they trashed some file from the Library which will require an OS 
>re-install (I think its in the BSD from memory).

Try this: go to ~/Applications/Utilities/Directory Access.app

There, if Bonjour is listed but has no checkmark, give it a checkmark 
and see if that restores  Bonjour services.  (You may have to click 
the padlock and enter a password in order to change the settings.)
