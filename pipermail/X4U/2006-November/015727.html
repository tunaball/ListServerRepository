parallels is the way to go since it doesn't require you to reboot (think virtual 
pc but at the hardware level). managing the windows image so it can be shared by 
all users is a bit of a hack. see this post on parallels forum regarding sharing 
and binding of the windows machine to active directory:

<http://forums.parallels.com/post29359-8.html>

until we have an alternative (vmware?), parallels is the choice...it's mature 
enough to be deployed.
