On 26 November, 2006, at 5:44:21, nk wrote:

> I'd like to install Tiger on my Pismo, but the the piz doesn't seem  
> to like the Tiger install disc..either it doesn't like the disc, or  
> can't read it...DVD maybe? Pismo only reads CDs

One of the models of Pismo PB did have a DVD reader installed, but  
most of them only had either a CD reader or a CD-RW reader from the  
factory. The DVD equipped model came with the DVD decoder installed  
on the motherboard and did not need the Apple PCMCIA DVD decoder  
inserted in it to play DVDs like the earlier PBs did (Lombard and  
Wallstreet models).

If you can't see the Tiger disc on the desktop, it's because you  
don't have the DVD module. You could always mount the Pismo in Target  
mode to a DVD equipped tower and then install Tiger through it.

I have the DVD module and was able to install Tiger with no problems.  
Just make sure that you have at least 256Mb RAM and a drive larger  
than 20Gb to handle the newer system.

Good luck and happy computing with the Pismo.
