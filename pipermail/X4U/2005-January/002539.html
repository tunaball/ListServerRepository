>I have the strangest problem with Stuffit Expander 9.01 on my iMac with OS X
>10.3.7.
>
>Stuffit expander takes about two minutes to start up; it expands files
>quickly once it is running.
>
>So far I've tried:
>1. repairing permissions (no help)
>2. Downloading Stuffit once again and using the installer to remove Stuffit.
>Then I've deleted the plist files in my ~/Library/Preferences folder. Then
>reinstalled Stuffit Expander. (no help)
>3. Tried running Stuffit Expander from a brand new user set up just to test
>this; new user had identical problem -- so it is not related to anything
>installed in my user directory.
>
>I have NOT had any other problems with other programs - does anybody have
>any suggestions?


Once the Stuffit application FINALLY loads go under the preferences 
and uncheck the "Allow Version Checking" under the general 
preferences tab. Once that is unchecked then it will load quickly and 
normally. The version checking takes forever. Why they have done that 
is a mystery to all of us. Stunned software developers might be the 
short answer.
