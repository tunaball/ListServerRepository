I have the .Mac settings in System Preferences set to keep a local copy

iDisk icon in the sidebar did nothing.  I deleted the sidebar prefs and 
restarted the Finder.  I still couldn't seem to access the iDisk.  So I

"You cannot connect to this server because it cannot be found on the

http://idisk.mac.com/michaelelliott/ could not be mounted
