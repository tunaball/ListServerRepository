I don't know if this is the best method, but it's one method:

Type the smb pathname into TextEdit:
smb://username:password@Server/Share

Select that text and drag it to the Desktop, which will create a location
file. Rename the file and put it wherever you want it (I put mine in
~/Library/Favorites)

You can then add that file to the Startup Items list in the Accounts pane of
System Preferences.
