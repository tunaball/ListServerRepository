It was called KeyCaps and a Google search found this page which notes  
that "The KeyCaps program is no longer under the Apple menu in O S X.":
     
<http://tlt.its.psu.edu/suggestions/international/keyboards/ 
mackey.html>

Simply: If you don't have different Keyboard Layouts and a menu bar  
item that selects them, you can turn on the new program/utility that's  
called Keyboard Viewer.

You can do this by going to System Preferences ... > International >  
Input Menu.
