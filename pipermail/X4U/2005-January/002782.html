I use Apple's Mail and set up rules. For example I was getting spam 
from Brazil!! So the Rule is any .br in the From sends it to Junk. Then 
I had to add another Rule for a Spanish character (or French character 
like é) that sent all email where the subject (or From or To etc) 
contains any occurance of that character to Junk.
That stopped all those spams. You can do the same for any word or 
letter for one or all parts of an email using Rules in Mail.

Hope that helps.
