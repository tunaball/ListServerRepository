> Now, Word quits as soon as it starts up. I seem to remember having this
> problem the last time I did an Archive and Install and that it was font
> related. I just can't remember what font I needed to activate for Word

You don't say which version of Word but assuming it is part of Office 2004
you'll find all the fonts that Office installed stored in:

~/Applications -> Microsoft Office 2004 -> Office -> Fonts

and it defaults to Times New Roman. You can check what you have installed
against this folder. My understanding is that it is better to use the MS
versions of each font if you want to avoid problems when using Word.

Hope this helps - pb
