check property list (preference) files for corruption:
sudo find /Library/Preferences -name *.plist -exec plutil {} \;;find  
~/Library/Preferences -name *.plist -exec plutil {} \;

show corrupted property list (preference) files:
sudo find /Library/Preferences -name *.plist -exec plutil -s {}  
\;;find ~/Library/Preferences -name *.plist -exec plutil -s {} \;

search entire disk for corrupted property list (preference) files:
sudo find / -name '*.plist' -exec plutil -s {} \;
