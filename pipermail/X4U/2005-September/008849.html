As the others said, you are the connected user (from the G5 at home).  
Very unlikely that anyone even connected while on the wireless  
network, as this would require a user name and password. You do have  
good user names and passwords, right?
To see who is connected at any moment, invoke the terminal and at the  
command line type "who" and return. You should see your short name  
logged on to the console -- that is you using the GUI. Should also  
see your short name logged onto a ttyp1 session -- that is you using  
the cli. Any thing else is cause to worry!
