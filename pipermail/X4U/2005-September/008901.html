First I noticed that iPhoto usually doesn't move things but copies  
them. At least that's what I thought. So your originals should all  
still be there. Maybe there's a setting though to "move" for iPhoto.

Second, iPhoto changes the names. So you cannot compare by name. In  
that case I would have piped the output of a find command to two text  
files and then a diff on the two. But iPhoto changes the file names  
and it also changes the folder structure. (Boy, how I don't want to  
use iPhoto just because of that!)

Third, try GraphicConverter, because it can look for duplicate  
pictures within any folder tree.
