The Office defaults for the Mac are that if a document containing a  
macro is opened, it will ask you if you wish to run the macro or not  
(Macro Protection is on). I think it even has a warning about the  
potential danger of running the macro, but its been such a long time  
since I've seen it that I can't remember the specifics.

So for any of the known viruses for the Mac, the protection is  
already in place.
