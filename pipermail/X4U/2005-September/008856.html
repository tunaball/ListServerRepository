Oliver Marshall said:

>Is there such a thing as a backup package, ideally open source or cheap,
>for Tiger ? Im after a backup package rather than a copying package like
>Rsync or Psync; something that will do incremental, differential, and
>compression, as well as scheduling and ideally backup to ftp.

Free, open source, or cheap?  Maybe one of these is what you are looking 
for:

Phew (FREE)
http://www.substancesoftware.com/phew/

rSyncX (free and open source)
http://archive.macosxlabs.org/rsyncx/rsyncx.html

Deja Vu $25 (highly recommended!)
http://propagandaprod.com/dejavu.html

Carbon Copy Cloner (FREE)
http://www.bombich.com/software/ccc.html

iMsafe $15
http://homepage.mac.com/sweetcocoa/

iBackup (FREE)
http://www.grapefruit.ch/grapefruit/iBackup/

SuperDuper $20 (highly recommended!)
http://www.shirt-pocket.com/SuperDuper/SuperDuperDescription.html

Chronosync, $20:
http://www.econtechnologies.com/site/Pages/ChronoSync/chrono_overview.html

Synk $20 personal/$40 commercial
http://www.decimus.net/synk/

LaCie SilverKeeper (FREE)
http://www.silverkeeper.com/

BUUP4_for_cron ($5)
http://www.m-t-software.com/software-backup.html

Psync (Free and open source)
http://sourceforge.net/projects/psyncx

rBackup (FREE)
http://fletcher.freeshell.org/computers/mac/rbackup

HELIOS has released an HFS-aware version of gnutar. (Command-line 
utility.) 
http://www.helios.de/news/news03/N_06_03.html

MimMac $10
http://www.ascendantsoft.com/

BackupScript (FREE)
http://www.railheaddesign.com/pages/software/backupscript/backupscript.html


BackUp Application and ScreenSaver by LARA-A  FREE
http://users.pandora.be/lara-a/software/backup/index.html

rsyncbackup (FREE) 
A PERL backup script.
http://erlang.no/rsyncbackup/

SyncUpX $15
http://freeridecoding.net/FRC/PROJECTS/02/
is a synchronization utility that provides a graphical interface for the 
rsyncx command-line tool.  Spotlight-integrated, can schedule backups 
using iCal (syncOnStart, autoQuit).

BackupPC ((FREE and open source)
A high-performance, enterprise-grade backup system for backing up to a 
server's disk. Features include clever pooling of identical files, no 
client-side software, and a powerful Apache/CGI user interface.
http://backuppc.sourceforge.net
http://sourceforge.net/projects/backuppc/.

BasicBackup (FREE)
http://www.hillmanminx.net/basicbackup/index.html
