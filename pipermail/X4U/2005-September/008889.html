I suspect that the message headers contain all you're going to get.

Also note that spam headers are almost always forged. The From: header must not to be used because it usually belongs to an innocent victim. The topmost Received from: entry is the only one that is pretty much forge-proof.

Now how does one show headers in Apple Mail.app? I use Eudora. Someone else?
