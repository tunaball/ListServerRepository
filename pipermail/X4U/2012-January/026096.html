Howdy,

Certainly not just you! I not only occasionally have windows that won't activate when clicked but the same thing with files. I have also copied, moved, saved or downloaded files to the Desktop and then end up with a file that has no icon. Also, about 90% of the time, when I mount an external drive with multiple partitions at least one of the Drive Icons is displayed with the default HD icon instead of its correct icon; even happens with flash drives with a single partition or FW HDs.

I believe that there are some real problems with Finder. Keep hoping the next update to Lion will fix things.

Some things have become better with 10.7.2 over earlier versions. I used to have windows change arrangement, size and state of the sidebar and toolbar (I often have these turned off) randomly. This seems to happen much more rarely now.
