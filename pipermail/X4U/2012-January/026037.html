A little late on this, aren't you?  This has been around since  
October of 2010:
http://www.reedcorner.net/guides/macvirus/malware_catalog.php

Koobface/Boonana only seems to be encountered by users if you visit  
questionable Web sites (such as porn sites that are based outside the  
U.S.).  Usually it manifests itself as a request to download a video  
codec that you are ostensibly missing.

There are two very easy ways to avoid this Trojan:

- Don't accept requests to download codecs from Web sites unless you  
are on a Web site that you navigated to specifically for the purpose  
of looking for a codec  to download.

- Turn off Java (not JavaScript, which is different) in your browser  
if you are afraid you might agree to something you shouldn't.   
(You'll never miss it.  Very little on the Web uses Java these days.)
