No, I've looked into this a bit further, and as far as I can figure 
out, the issue had to do with the HFS file system, and I gather that 
the system itself imposed a 2 GB size limit for any file. Since 
Outlook Express put all the mail data into one file, it could reach 
the limitation on file size imposed by the HFS file system.

Anyway, the limitation certainly was NOT 25 MB!
