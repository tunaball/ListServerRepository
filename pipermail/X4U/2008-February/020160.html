I've fixed a number of OS X 10.4 Macs with this hanging-while-booting  
problem simply by booting from a Disk Warrior disk and running the  
application. What Disk Warrior usually finds and fixes is incorrect  
volume information, and a handful of faulty icon stuff. This usually  
is caused by improper shutdowns while programs are still running, in  
my experience.
