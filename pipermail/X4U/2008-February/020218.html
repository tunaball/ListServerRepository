"Best" is subjective. The way I've found easiest is to load the first
VOB file into MPEG Streamclip, answer affirmatively when asked if you
want the whole thing loaded, and then choose the appropriate demux
option from the File menu, based on the audio format. Then, if it
started out as PCM (either AIFF or WAV), just burn it using Toast. If it
was MP2 or AC3, convert to PCM for burning--I generally use ffmpegX, but
there are lots of tools that can be used to perform the conversion.
