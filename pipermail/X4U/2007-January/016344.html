Brett:  Greetings!

I agree that Onyx is a very useful (and free) software.  I also use  
'Cocktail' and 'TinkerTool System' for similar routine maintenance.   
Each has its own quirks and its own strengths.

For example, I like to run 'Cocktail' when I am ready to quit working  
in the evening.  Cocktail allows you to tell it what to do when it  
finishes its work:  Shut Down, Sleep, do Nothing, Log Out, Restart, etc.

All of these are available at 'Version Tracker':

    http://www.versiontracker.com

Version Tracker is itself a very useful piece of software.  Using it,  
you can find just about any ShareWare, freeware, or updates to  
commercial products.
