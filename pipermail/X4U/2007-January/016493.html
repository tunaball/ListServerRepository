The time zone resets itself only when I switch between Windows (via Boot
Camp) and the Mac OS. I leave Airport off in the Mac OS in order to maximize
battery life. I have Windows on the laptop to run an internet explorer for
Windows-requiring medical application that I use for rounding in our
dialysis center (on WiFi), so that I boot into Windows when I go next door
to the dialysis center, back into the Mac OS (and ethernet) when I come back
to the office.

Because Airport isn't turned on in the Mac OS, sometimes when I come back to
the office and reboot into Mac OS I won't have an internet connection.
That's when I see the Mac OS time set to GMT.
