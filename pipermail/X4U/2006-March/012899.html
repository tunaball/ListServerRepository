On 3/2/06 2:54 PM, Allan Rube¹ <allan at nhbungalow.com> wrote:

> Last year I had mlb gameday audio which plays through real player. All was
> fine. I renewed the subscription today. However, when I clicked the button -
> Realplayer did not open nor did the real player file download to my drive.
> Quicktime opened and I got a message - bad url.
> 
> How can I get it so that real player will open instead of quicktime?

It is not uncommon for one media player to override another as the default
player for a protocol. To switch it back to RealPlayer, in RealPlayer's
Preferences, click the Media Types tab and check the box next to the
Real-Time Streaming Protocol.
