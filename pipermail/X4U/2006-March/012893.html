On 03/17/06, Mitchell Senft <ms53 at optonline.net> wrote:
>
> Got a second HD for our G5. I wanted to install OS 10.4 on the new
> drive, using the disks that came with the G5. I was unable to do so.
> Obviously, there's something to prevent it from installing on a drive
> other than the original dive. But my impetus for this was just to
> have a back up drive for such time as the original drive possibly
> breaks down.
>
> So, is there a way to get the OS on the second drive using the disks
> that came with the G5?

Launch Disk Utility, select the HD, select erase, and format/partition it--Mac
OS Extended. That should allow you to install with the disks.
