What you're looking for is EasyFind.

Differently from Spotlight, you can easily set it up for performing 
the search in:
* Only files
* Only folders
* Files and folders
* File contents
* This or that drive, or all
* Package contents
* Invisible Files/Folders
etc.
When you have found what you're looking for, you type Command-R (for 
Reveal in Finder) and the folder opens...
