The unknown user (and group) is a normal one in OS X.

Running this command in the Terminal app, "dscacheutil -q user " reveals:

name: _unknown	uid: 99	gid: 99

Don't muck with it.

Go to <http://discussions.apple.com/> and search for "unknown user". The
resulting posts should point you in the right direction WRT to your query.
Apparently, it's what gets assigned when using the upgrade install option on a
previous Panther or Tiger installation.

On my installation, which was a pristine Leopard OS that I used the Migration
Assistant to transfer from a Tiger installation, my /Applications folder
permissions are: owner me w/R&W, staff w/R&W, and everyone w/R only.
