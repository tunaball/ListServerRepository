	When you're using text to speech, you're not actually using Word  
anymore.  You using Text to Speech.  Therefore, closing Word won't do  
the trick.

	When you want to stop the speaking, the same shortcut key that  
starts the speaking will stop it.  On my computer that's control+s.   
If you used a menu command, there is another menu command next to it  
to stop the speaking.

	This feature is quite useful for proofreading.  Hearing the computer  
speak text helps me catch mistakes I miss when reading it.
