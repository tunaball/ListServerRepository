Here is a lesson learned.

I have no idea if the modified .rules.plist file created below is  
related to the effect I saw a few days later but voodoo is easier to  
explain...

After do the below, I noticed that spotlight did not find anything  
that was in my address book..

After numerous spotlight rebuilds, still no joy.

So, reverted back to the original .rules.plist and still no luck.

Trashed spotlight and address.plist files. Rebuilt spotlight

Still no luck.

Trashed all of the data files in the folder that the .rules.plist  
resides in.

Still nada!

Then, backed up my address file and trashed the original.

Reverted to the saved backup file.

forced spotlight to rebuild.

Voila! Everything is as it was....

An explanation would be nice but not pushing my luck!
