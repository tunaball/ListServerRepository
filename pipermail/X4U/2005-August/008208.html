My take on that is, if your incoming mail filters command Junk Mail to 
go directly into the Junk Mail folder as soon as it identifies them, 
they will show up in that folder identified as Not Read.

The reason is, you never physically opened the file.
If you'll look at the top of the messages pane, the green "click to sort 
by read" status button is still there. If you had opened the message, 
that button would be gone.

Please note, this explanation applies to Netscape 7.2.
If you are using some other mail reader, such as Apple's Mail, I've no 
idea HOW it treats read vs. unread mail...  ;-)
