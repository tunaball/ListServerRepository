Brett,

Well assuming you have the right version of Retrospect (i.e. 
Retrospect Server) you should be OK.

I have a few clients with 6.0.204 or 6.0.212 that are happy backing 
up 10.4.2 Server, 10.3.7 Server, 10.3.9 Server etc.

All you need to do is install the client on the Server (6.0.109 or 
6.0.110) and run Retrospect on a different machine. The Server does 
need to be logged in though (from memory).

Hope that helps!
