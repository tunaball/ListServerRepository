On Aug 29, 2005, at 4:59 pm, Michel Treisman wrote:

> I get mail from the university servers either as webmail or IMAP.  It 
> downloads to Mail.app, or I can log on
> to the server and go to the Herald inbox which holds my mail there.
> For the last two weeks or so some of my mail seems to be delayed.  If 
> I go to the web page I find mail there
> that is dated from days ago. Some has downloaded and been read, some 
> has just stayed there.

Try going to the "Mailbox" menu item & choosing "Rebuild". If this 
doesn't work, create a new IMAP account with the same server settings 
as your original one, then delete the original. This is much faster if 
you're on the same LAN as the server.
