   > How does one discover the true MAC address of the machine?

   It's what's reported in System Profiler. The ethernet ID is the Mac
   Address. If they don't match, it's time to call Apple.

Again, there is no such thing as "_the_ true MAC address of the machine".

There is a MAC address associated with each 802 network _interface_.

So the built-in Ethernet will have one.  If you have AirPort, it will
have a different one.  If you have Bluetooth, it may also have a
different one.  If you have a 2nd Ethernet interface, it will have a
different one.  You get the idea.

Oh, yeah.  All of these can be changed under software control.
