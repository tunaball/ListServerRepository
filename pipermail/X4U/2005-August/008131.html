Using the Finder window search box in Tiger does not limit the search to
filenames (as it did in Panther).

Jim, if your foremost window changes into a 'New Search' window after
pressing Command-F, you can go back to it by pressing the Back button or its
keyboard shortcut, Command-[

whenever I press Command-F, it opens a new window to perform the search.
