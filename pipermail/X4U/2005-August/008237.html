I recently read that one of the drawbacks to the Apple Remote Desktop  
program is that it only works on port 5900 and there is no obvious  
way to change that. Chicken of the VNC has no problems using  
different ports; however, the port you are forwarding on the second  
machine is going to 5901.

An alternative solution would be to use a different VNC server such  
as OSXVNC. This also allows customizing the ports used.
