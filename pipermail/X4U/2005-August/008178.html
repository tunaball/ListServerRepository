>> I'm trying to remotely connect to multiple Apple Remote Desktop  
>> servers on my LAN via VNC, but my  Netgear router seems to only  
>> allow port forwarding of port 5900 to only one IP address. How can  
>> I remotely (i can connect from within the LAN) connect to mutiplle  
>> macs via VNC?

You can work round this via port forwarding - assuming your VNC  
client application allows you to specify different ports.

i.e. If your external IP number is 111.122.133.144 and your internal  
range is 10.0.0.x you map 111.122.133.144:5900 to 10.0.0.1:5900 (as  
you have at the moment), 111.122.133.144:5901 to 10.0.0.2:5900,  
111.122.133.144:5902 to 10.0.0.3:5900 etc. Then you connect to the  
external IP number thru your VNC client using the port number to  
control which machine you're talking to.

Clear as mud?
