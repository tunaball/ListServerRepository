Select the messages you want to archive. Select File->Save As from  
the menu bar. Select your choice of Plain Text, Rich Text, or Raw  
source. Save where convenient.
Raw source preserves the entire message including the headers you  
normally don't see. Rich Text saves it as a nicely readable message,  
w/ the usual headers bolded, and quote coloring preserved.

If you select multiple messages and save as Rich Text, you get a  
single file containing all the messages nicely formatted.
