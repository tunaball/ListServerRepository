On Oct 3, 2008, at 1:45 PM, Zane H. Healy wrote:

> Is it possible to install Mac OS X on an external Firewire HD such  
> that you
> can boot either a PPC or an Intel-based Mac?  I want to build a new  
> utility
> disk that I can use to boot any of my families Mac's and run various
> utilities against them.

 From what I understand, you create a GUID (intel Mac booting)  
partition, load up OS X, and whatever utilities you want on it.

Then make sure it boots an Intel Mac.

Then clone that partition to an APM (PPC Mac booting) partition  
Firewire Drive.
