Here's my understanding from reading online both from Apple and other sources:

Since Lion, 10.7, there are 2 methods of getting OS X: 1 download yourself, then you're left to back up the download for later use and 2. buy a USB drive with the OS X installer pre-loaded by Apple, from Apple.

I use this doo-dad for making my own bootable OS X installer:  Lion Diskmaker  Requires a USB/Flash drive of 8GB or larger. There are published techniques for doing this via Disc Utility, but I never had good luck with that method. I think MacWorld published one such how-to.

To my knowledge there is no legal avenue for purchasing an OS X install DVD from Lion foward.
