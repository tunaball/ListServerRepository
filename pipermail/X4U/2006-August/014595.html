> Did the article say why the Mini's graphics are so much faster  
> given that they use the same graphics hardware?

Yes, that's the whole point of this e-mail thread: The cebop has only  
one RAM slot, so even with the same chipset as the Mac Mini it cannot  
use dual channel RAM access. That's a killer for some types of  
applications that require both the CPU and the GPU to access their  
shared RAM intensively.
