Try using 'Save as....' under the file menu.   I choose Rich Text  
Format and include attachments.  You can select more than one email  
and it saves them all to a file that opens up nicely in textedit.  I  
am not to sure about re-importing to mail but you don't have to to  
read them just use textedit.

or


Put all the messages you want archived into a mailbox and drag it to  
the desktop.   Very easy to reimport.


try out both ways and see which works best for you.
