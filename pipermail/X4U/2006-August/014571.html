> Is there some way to search for duplicate files on 10.4?

Do you judge a  "duplicate" by filename or content?

Do you have specific file types in mind? (text, graphics, music)

The answer to the above questions has an impact on the tools to use.  
For instance if you can assume that file names are identical, then  
the search is much simpler than if you judge by content.

If you can restrict the search to specific types, then there are  
dedicated tools. For instance GraphicsConverter does an excellent job  
to find identical images or even similar images (e. g. the same  
motive but one has a stronger jpeg-compression)
