> Any one have any suggestions where I can find a "cheep" and legal
> version of Win XP Pro?

Look for a copy of Microsoft Virtual PC for Mac 7.0 with Windows XP
Professional.  

Amazon prices:
Virtual PC for Mac 7.0 with Windows XP Professional   $215
Windows XP Professional alone   $280

Second hand, most copies of Virtual PC w/Windows are also selling on eBay
for less than the cost of Windows alone. Throw away the Virtual PC disk and
just use the Windows install CD.

Just make sure that the bundled copy of Windows is the version you want!
