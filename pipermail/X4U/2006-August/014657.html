Instructions for setting up FTP only accounts under Mac OS can be  
found at <http://www.ldml.com/services/support/macosx/ 
ftpUserCreate.html>. To lock it down reasonably well is somewhat  
intricate but I believe it's all there (these instructions were  
written for Mac OS 10.3 but they should work for 10.4 too).

Another poster offered up CrushFTP. We're using it on one box and it  
works quite well - although I found first-time set-up almost as  
complicated as the command line stuff required using the OS's own FTP  
server. Also, we have had some weirdnesses using CrushFTP - nothing  
that couldn't be sorted out with a quick reboot though.
