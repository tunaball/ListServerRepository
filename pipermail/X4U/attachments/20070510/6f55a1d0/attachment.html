None that I know about. I suggest creating a bootable backup/clone (using<BR>
Carbon Copy Cloner, SuperDuper!, etc.) of the PB on to a bootable, external<BR>
FireWire HD, booting into it to ensure it's viable, launching Disk Utility, and<BR>
repairing the disk and permissions. If that doesn't improve things, reinstall<BR>
the latest COMBO update for the machine's OS. If the problem persists, do an<BR>
Archive &amp; Install installation, saving user and network settings. See<BR>
