<div>I am using HP 2.5 driver and seems to set it up for black ink

&lt;carrot_ink@email.carrotink.com&gt;. I have only used a black
refill as yet and am pleased with the price performance. My printer is
an HP 930 c</div>
<div>The trick get Black Ink only printed is to set up the window that
comes up when you select &quot;Print&quot; as follows: 1) In the
&quot;Print&quot;window pull lower box down to &quot;Paper Type and
Quality&quot;, 2) In that window top select &quot;Paper&quot;, 3) Then
in that window select &quot;Quality:&nbsp; Draft&quot; and
&quot;Color: Gray scale&quot;, 4) In the same window select
&quot;Color Options&quot; 5) in the window that opens check the
&quot;Black only&quot; box.</div>
