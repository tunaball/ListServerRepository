<br><font size=2 face="sans-serif">The only caveats I'd add is that it's
best to set the &quot;Queue&quot; preferences to have adjust permissions
(uplead and download) to off. As a default this is ON and has stopped many
a download or upload from happening for us.</font>
<br>

huge set of files (8-10 files - together approx 1GB) and it kept choking.
Good thing I was able to resume after rebooting the app. I downloaded them
in smaller groups (1-2 files at a time) and it handled them AOK. As this
has only happened on 1 Mac I'm not going to write the app off yet as &quot;weak&quot;
but if it happens on another Mac in a similar situation then I'll have
to start making assumptions.</font>
