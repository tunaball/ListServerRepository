As others have mentioned, identifying a machine by e-mail address is highly suspect.&nbsp; It would be the equivalent of saying that a letter you got in the regular mail had to be from a house because the return address written on the envelope was that house's address.&nbsp; There are a number of examples you can find where your e-mail address can be used to send something that isn't coming directly from a machine you have that e-mail set up on that are even legitimate.&nbsp; Online greeting cards would be a good place to start.
<br><br>Also to clarify something else.&nbsp; None of the machines has to be infected by anything.&nbsp; I'm not saying they're not, but if any of the addresses are out on the web someplace (or was submitted when signing up for an account of some sorts), someone else has access to that information.
