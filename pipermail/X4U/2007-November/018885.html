One way I have used, not the most elegant but the only way I figured  
out:

Select the text you want, then under Mail > Services > TextEdit > New  
Window Containing Selection. This produces a TextEdit document which  
you can simply print then close without saving if you don't want to  
keep it or of course Save if required.

HTH
