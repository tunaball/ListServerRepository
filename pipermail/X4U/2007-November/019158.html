I run Parallels on the Mac Pro with XP and Vista virtual machines.   
Windows performance is more than adequate.  (I tell people in jest  
that my Mac is the fastest Windows PC in the company.)  If all you  
need to run in Windows is IE forget Boot Camp and use Parallels (or  
VMWare -- I run it on me MacBook).  You'll definitely want to bump the  
memory if you go the VM route, since you'll be running two OS' at once.
