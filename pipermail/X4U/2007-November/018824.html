On 1 Nov, 2007, at 17:53, Zane H. Healy wrote:

> Is there a way to set the max volume on a restricted account?

If you have set up separate accounts on your Mac, each account  
preferences setting will be separate from the other accounts.

Just set the volume level that you want for the account you want to  
control and then lock the settings using Parental Controls to isolate  
that user account from changing any system preference setting.
