I have an iMac (G5) with four accounts: me, wife and 2 kids. Other  
users staying logged in does not seem to be a problem. I use heavy  
resource software only occasionally, so your results might be  
different than mine. Why don't you start out with them staying logged  
in. If it does not work, change to logging out.

You will be glad that you set up separate user accounts. Keeping all  
software settings and documents separate is a must. Don't know how  
young your kids are, but if they are young, you can also use parental  
controls to build in restrictions.

Hope this helps.

Stan
