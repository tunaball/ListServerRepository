On Feb 17, 2009, at 9:14 AM, Mark Des Cotes wrote:

> I've been trying to figure out how to upload files to an FTP server  
> using the "Connect to Server..." function of my Mac. I can connect  
> to the server without a problem and I can download files from it  
> but when I try to drag a file onto it I get a message saying it  
> can't be modified. I can upload files to it using Fetch or one of  
> my web site creation programs, but I was hoping to be able to  
> upload to it without the use of 3rd party software. I have full  
> access to administer the FTP site but I don't see anything for me  
> to change. Am I missing something or is this a limitation of the  
> "Connect to Server..." function of the Mac?

FTP downloading is built into OS X, but not uploading.

If you want to be able to upload files from your Mac just as easily  
as you can download them, try:

Fire Widget (free)
http://buschfeuer.com/fire/

If you want to be able to upload via FTP from your browser, get the  
FireFTP add-on for Firefox (free):

FireFTP (free)
http://fireftp.mozdev.org/
