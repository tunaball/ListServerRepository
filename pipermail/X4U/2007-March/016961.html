At 17:29 -0800 3/2/07, nk wrote:
>I recall this was  bit easier in OS 9...is there a thing out there  that will tell me the Creator code of an application?

If you're willing to use Terminal.app do:

GetFileInfo  path_to_file

GetFileInfo comes with the Developer Tools but it can also be used by itself.  There's also SetFile which goes the other way allowing you to change type and creator.
