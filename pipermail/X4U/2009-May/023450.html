At 19:08 -0400 4/5/09, John Drake wrote:
>I have our  Airport Extreme setup using DHCP, but not sure how to 
>reserve specific IPs for each computer.
>I'm having the same issue in that my iTunes server keeps getting a 
>new IP as issued by the ISP so then we can't connect.
>Up to now I've been using IP Scanner to find the new IP and then I 
>go in and update.
>Any detailed help will be appreciated.

This depends on the router (or whatever is your DHCP server) 
involved. Some can't even do it. If you tell us the make, model, and 
firmware revision of the router someone with the same type may be 
able to help you.

You wouldn't normally access your iTunes server via the ISP issued 
address unless it's on a different Internet feed. DHCP allocated 
addresses can jump around in much the same way though.
