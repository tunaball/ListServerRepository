There is an option to smooth the output. Universal Access is intended 
for those with poor eyesight whose eyes will antialias the image. 
There's no point in using CPU cycles removing jaggies that the 
intended user can't see anyway. If you can see the jaggies, just be 
glad your eyesight is good enough to do so.
