Charles,

>What, if anything, has to be done after replacing the PRAM battery on a MAC?

Nothing at all, if  you have the Date/Time set to update 
automatically. The first time you start up after replacing the 
battery, the system will go find the correct date and time. If your 
Date/Time System Pref is NOT set to set date and time automatically, 
then you would need to go reset them. As far as I know, that's all 
that might need correcting after replacing the battery.
