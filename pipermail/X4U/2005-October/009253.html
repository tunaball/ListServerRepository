I use both TechTool Pro 4 and DiskWarrior with Tiger on my G5, and  
have found both to be "life savers" when I have had serious problems.  
They are each suited to different kinds of repair and inspection,  
disk surveillance, etc. I use DiskWarrior about once a week to   
rebuild the directory on my main hard drive, and less often on  
several FireWire drives. This keeps things running properly and  
eliminates slowdown from small problems. I have used it to fix hard  
drives that nothing else would be able to do. This is the most  
reliable program of its type for the Mac, and over the past few years  
has never caused any damage on my system. I can recommend it as worth  
the cost, and then some. It should be on every serious Mac user's  
machine. I think that anyone with the need to keep things running  
without glitches should also try TechTool Pro 4. The latest versions  
of these programs are needed for Tiger.
