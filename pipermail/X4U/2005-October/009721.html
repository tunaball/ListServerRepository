I would verify the account set up. Goto Preferences -> Accounts ->  
[Applicable Account] -> Advanced tab
Verify the following
Port -> 995
SSL selected
Authentication -> Password

if it is configured as non-SSL on port 110 you will find occasional  
to no connection.
