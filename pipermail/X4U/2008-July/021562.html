Open XML Converter 1.0 is available at
<http://support.microsoft.com/kb/953823/>, requires MS Office2004 for Mac
update 11.5.0, and note that "You cannot use beta versions of the Open XML

a beta version of the converter installed on your computer, download and
install version 1.0 or a later version of the Open XML Converter."
