You have the option to go to disk utility in the os installer when you  
get to the place were you select the install disk. In one of the menus  
you will see disk utility. You would select the disk reformat in then  
while also selected select the partition tab and there is were you can  
partition it to a GUID partition. Just make sure you have all your  
data backed up, you'll lose everything.
