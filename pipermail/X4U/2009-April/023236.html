That term "standard" is soooo misused. One nice thing is that there 
are so many to choose from. Another is that there is no requirement 
on any vendor to support any of them.

RFC1740 is classified as "standards track" but note that RFC822, 
August 1982, is the only "standard"  for all of email. It's 
replacement, RFC2822, has been "standards track" since April 2001. 
RFC 2045 which defines MIME has been standards track since November 
1996.
