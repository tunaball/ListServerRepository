David:

I disagree. Double paging is bad and especially in small computer  
land as I/O is sooooo slow and the inability to chain I/O requests  
(so that say 20 I/O operations get done but with only one trip  
thought the code to do I/O). I have worked on systems that do a  
"handshake" between the OS and VM so VM does the paging it is a  
really nice and time saving feature. Some OS's know that they are  
running under VM and do this automatically others you have to tell  
them (each OS does it slightly differently).
