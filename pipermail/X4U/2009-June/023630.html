On 6/7/09 4:43 PM, Ed Gould wrote:

> I have been having issues (no email) from a yahoo group that I am on.
> I am pretty sure it is one group as I do not have issues from other
> yahoo groups that I am on.

Also log into that particular Yahoo group and click "Edit Membership" (words
like that) for that group -- each Yahoogroup, even for the same Yahoo ID,
can have different settings. Some groups default to Web Only access, while
others default to Digest email, and others default to Individual Email. That
group might be set to Web Only access, and all you'll need to do is change
the setting.

The other thing I'd try is to make sure you've selected a VALID EMAIL
ADDRESS for that group. If it's trying to send mail to an email address you
never check, you won't get the emails -- they will be going to that other
account, instead.

Good Luck.
