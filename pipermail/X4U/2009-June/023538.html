My standard procedure is to use the Finder "Find" command, and search  
for the string "superduper".  It will find all prefs, application  
support files, the app itself, and other related files that use the  
name.  I then move each one to the trash.  Can repeat the procedure  
using the name of the company "shirt-pocket" which may find a few more  
files - move them to the trash - doing this step risks any other  
software made by the same company.

Once you're sure you've been successful, (i.e. restart)  empty the  
trash.
