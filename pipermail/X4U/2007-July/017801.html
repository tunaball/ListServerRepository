> Someone can explain me the behavior of .DS_Store and .localized files, seems
> they appear when something is made in a subdir or volume
> (partition)...thanks in advance.

The .DS_Store file remembers the Finder window settings for each and every
folder opened in OS X. IIRC, the .localized file only gets populated if you use
multiple languages via the International prefPane. Don't know why you're
asking, but don't muck with them.
