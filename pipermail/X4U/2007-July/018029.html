control-click or right-click on a .PDF file and select "Get Info", or  
click on the file and then Command (Apple Key) + the letter "i"

Select the popup:  "Open with"

Select Preview

Then hit the Change All.. button below that.

This should tell OSX to always use Preview to open files of type: PDF
