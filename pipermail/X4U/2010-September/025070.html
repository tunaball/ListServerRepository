> Is there any freeware out there that can be used to compare the contents of a couple directories?  I'm trying to merge some hard drives, and I know I have duplication.
> 
> Zane
> 

Try "Sybchronize! Pro". You can select two directories to synchronize, and check the option to "show files to copy."

This will bring up a list of every file that is going to be copied in either direction. It shows the file size and last modified dates and times for each file.
