Type this into a Terminal app window:  macerror -XXX, where XXX is the error
number. E.g., for this one, we have:

macerror -36
-> Mac OS error -36 (ioErr): I/O error (bummers)

Alternatively, peruse the various errors.h files that reside on the computer.
