Do the following:

Click on "File", "Import and Export," "Export to a File," select the part of 
Outlook you want to move (Calendar, Contacts, . . .), and select the file 
type (for example, "Microsoft Excel," or "Comma Separated Values"), and then 
the output file name.

There you go.
