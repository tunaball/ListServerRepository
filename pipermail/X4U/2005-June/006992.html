> A new iMac g5 owner is using Verizon DSL in Maryland.
>
> Does anybody know how to setup up her router and or whether she  
> actually needs to install
> any of the software from the CD that came with it?
>
> It is a turn key process for Mac OS X with this company?
>


I use Verizon DSL here in Virginia. Works like a charm. Tech Support  
for Macs is hit or miss with Verizon.

You don't need ANY of the software on the CD. DON'T install it.

1) You need to find out if the connection uses DHCP or PPOE. (When I  
was in Arlington, the WAN connection was PPOE. I am in Woodbridge  
now, and the WAN connection is DHCP)

2) set Router to DHCP or PPOE as needed.

3) Configure Mac for either DHCP or static IP (Local LAN side) and  
make sure you set router appropriately, i.e. either to serve IP  
addresses or not. I have my router's DHCP server shut off, and all  
the computers in the house are all set to static IP addresses in the  
192.168.1 range.

HTH
