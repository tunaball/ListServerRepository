The key argument is support for open standards. Proprietary EDI is 
simply not necessary nor as useful as a published XML schema. Employing 
Firefox as the standard baseline browser makes sense as it runs on 
every platform with more than 100 users. Finally, there is the bleeding 
wound of internet security. Clearly any *nix platform, like MacOS, is 
preferable for the end users from a security point of view.
