Ever hear of the OS/2 effect?
If _Developers_ know they can write code for one platform and have it  
run on other boxes, what's their motivation to write code for the  
other platforms. Soon there are only Windows apps running on  
Macintel. Then OS X becomes an optional shell for Longhorn. Not a  
scenario any of us want to see played out.
