Did you quit and relaunch Admin Utility, or click "rescan"?
If so, time to cheat. Temporarily connect to the base station using  
an ethernet cable to the base stations LAN port (The one with the <-- 
 > symbol). Then relaunch the admin utility and setup the rest of the  
base station stuff - encryption, password, and so forth.  Since this  
is temporary, you can use the ethernet cable that's currently  
connecting the  cable modem to the  WAN port (the round symbol). Just  
remember to put the ethenret caable back when you're done.
