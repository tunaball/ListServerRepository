Try starting up with the Tiger install CD/DVD, launch Disk Utility, and
repair disk. Restart. If that doesn't work, try resetting the PRAM. If it
does, review <http://docs.info.apple.com/article.html?artnum=106693> and
try reinstalling Tiger (Archive & Install)
<http://docs.info.apple.com/article.html?artnum=301270>. Further
information here: <http://docs.info.apple.com/article.html?artnum=106464>.
