I don't know what is really wrong, but it appears that you have an  
improperly defined function. FWIW, I have found that many "java"  
objects found in the Windows world are really ActiveX or similar with a  
java adapter class as a wrapper. Cute, but prone to failure.

Also, you may want to check out this tool:
	<http://chrispederick.com/work/firefox/webdeveloper/>

It is very handy for debugging web pages in FireFox and Mozilla  
Browsers. The stuff it can reveal has some utility in finding probable  
causes of bad behavior in Internet Explorer for Windows.
