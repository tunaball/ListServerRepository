Besides what I mentioned above, I also replaced my router. The old  
router is about 10 years old and the setup is confusing.
Here's what I did to get VNC working through the router from a remote  
location.

Enable Port Forwarding,
Start port: 5900,
End port: 5900,
Server IP: the Macmini's IP address,

In the VNC client enter the router's "Internet IP address" in the  
Host field.
I use a static IP address, not sure how this works with a dynamic IP.
