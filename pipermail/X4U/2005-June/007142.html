Close all Finder windows, then open a folder, say the Applications  
folder.  Pick an application name that is visible, say Address Book.

Initiate the "show desktop" action in Exposé, which slides the window  
off the screen.  Then, while still in Exposé mode, type "Address" and  
hit enter.

On my iBook and iMac, the text label for the Address Book application  
appears, highlighted, on the desktop in the location that it would be  
if the Applications window wasn't hidden by Exposé.

Perhaps this is old news, but interesting anyways.
