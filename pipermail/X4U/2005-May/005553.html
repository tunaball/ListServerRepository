Yes, we are only going to get the benefits of OS X by providing it 
with a lot more memory than the OS of yesterday.

Try using MenuMeters 
<http://www.ragingmenace.com/software/menumeters/>. With this tool, 
you can see graphically in your menu that amount of free and used 
memory along with the pageouts. This should tell you what your real 
needs are based on actual use.

Do note that the addition of memory typically only changes the speed 
at which things get done, not whether or not the answer will be 
correct. And the value of a user's time will always be subjective.
