Point of information:

The early eMacs (which were all G4s, AFAIK) *did* boot into 9.x; I 
know because I have one that I got used exactly for that reason (to 
run Webstar 4.5). I don't know when that capability ceased. The one I 
have is a vintage 2003, 700 MHz; system profiler calls it "eMac 
version 2.1; boot ROM 4.42f1". It shipped with Jaguar. If I recall 
correctly at least one later iteration (the 800 MHz models) also 
booted into OS9.
