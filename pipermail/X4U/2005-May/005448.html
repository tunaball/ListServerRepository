See the "importing audio CD as AIFF" thread, the answer is there. 
iTunes adds to the AIFF file a tag (which you can view with a hex 
editor). If you enter the tag info yourself, the data is stored in the 
tag. If the AIFF is ripped from an audio CD whose data hasn't been 
retrieved from CDDB, the tag consists of the CD's magic number and part 
of the TOC, which allows iTunes to retrieve it at some later date.
