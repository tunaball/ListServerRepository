Wouldn't just unchecking the "Open safe files" box in Safari also do  
this?  Or using a different web browser?  Widgets are just like any  
downloadable program - if any malware is executed by a user, it could  
hose their user space.  The security issue here is that if the "open  
safe files" box is checked in the Safari prefs, a widget will auto- 
install into dashboard, not dashboard itself.

As for wasting system resources, it's strange that even with  
Dashboard running with about 7 widgets, my system is still running  
much faster than with Panther.  Maybe I should chalk it up to the  
erase/install I did with Tiger...
