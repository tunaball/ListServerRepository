posting as a separate item.

============

Previous versions of Mail.app had an annoying bug whereby clicking on  
the curved arrow icon beside a message to which a reply had been sent  
would only bring up the content of the reply if the original message  
was still in the Inbox.

This is resolved in Mail 2.0. Clicking on the arrow now works in  
cases where the original message has been directed by a Rule to  
another mailbox. Excellent.
