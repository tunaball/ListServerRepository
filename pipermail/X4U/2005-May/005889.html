Add to that:

1. Mail doesn't have categories for email. Categories in Entourage can have
a color associated with them which I find useful.

2. You can't edit emails that you have received. While I see this as a
security issue so that you can't falsify emails sent from someone else,
there are practical uses to edit a received email, like removing all the
extraneous promotional info from an invoice that forces it to print on 2
pages instead of one or making an html email into a plain text one.
