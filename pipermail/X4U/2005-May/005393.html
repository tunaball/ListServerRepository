> Q:   An mp3 has specific id3 tag info built into it ... However - AFIK - an

iTunes has an internal database that stores the info. I am pretty suire if
you took that AIFF file to a different computer, it would not see the info
you added. iTunes has several bits of info that don't correspond to ID3
tags for MP3 files ((play count, for example). This data is stored in the
iTunes library file.
