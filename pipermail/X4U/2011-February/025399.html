Macintosh OS X10.7 (Lion) will be dropping support for PowerPC  
applications!

http://www.9to5mac.com/53977/10-7-lion-drops-support-for-powerpc- 
applications
or
http://is.gd/rhzNmX

So, if you are currently running any applications that require Rosetta,
now is the time to start looking for updates or alternatives, and to  
start migrating your data to the new application(s) if you plan on  
updating to 10.7 this summer.

Note that this means that AppleWorks and Eudora, two much-loved (but  
discontinued and unsupported)
applications that many are clinging to, won't run in 10.7.
