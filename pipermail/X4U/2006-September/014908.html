Hi Allen. If no one has handled this one yet, the solution is easy...

On your MacPro, open the Sharing System Preference and select the 
"Windows Sharing" service.

You will see that there is a button to the right that allows you to 
choose which accounts can be accessed from the Windows SMB world. 
Make sure that at least one account on the Mac has been checked and 
you should be off to the races.
