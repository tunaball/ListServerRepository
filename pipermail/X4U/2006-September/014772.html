Actually the dynamic IP address isn't a problem. You can get a DDNS service
Dynamic Domain Name Server (cheap) that puts a little program on your home
machine and this back ground program monitors your WAN IP and when it
changes it updates a server on the internet. So instead of connecting to an
IP address you connect to a name like "myhome35.tzo.com"
