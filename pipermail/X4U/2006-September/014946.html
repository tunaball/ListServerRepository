I use Transcriva (Bartas Technologies).

I record on my cell phone (or directly to the Mac built-in  
microphone) and Bluetooth or email the sound file to the Mac and the  
typist simply imports the file into Transcriva.

It has keyboard control (a la foot pedal of old), to 'rewind (some or  
no seconds)' FF FR, etc. Typed file is plain text for any word  
processor.

I use Pages from Apple which will export directly to pdf, Word, text,  
Web, etc.
