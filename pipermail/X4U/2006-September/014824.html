In "Library/Application Support" in the Firefox folder are files in  
the format "Bookmarks-XXXX.html".  One can delete these but the last  
one.
Each time Firefox is started it crates a new one.  Therefor: keep the  
last one and save space on the HD.

This tip is from the German Macwelt magazine.
