Are you importing the tracks as MP3s?  Apparently the gap is a problem  
with the MP3 format.

See the following threads:
<http://www.macosxhints.com/article.php?story=20020116002112531>
<http://groups-beta.google.com/group/comp.os.os2.multimedia/msg/ 
7cf46c4511624458>

The solution is to import the tracks in AIFF and set crossfade playback  
to zero.  I just tried it with a Donald Glaude CD and it works  
perfectly.  The downside is AIFF results in large files.

I also tried ACC format, but it still had a noticeable gap. I'm going  
to try Apple Lossless now and I'll post the results when I'm done.
