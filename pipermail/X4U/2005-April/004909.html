Hello List,

I have a client who's a web developer and running on a G5 with 10.3.8 on
it.  He's having a problem with Safari not allowing him to view page
source.  The keyboard shortcut will not bring up anything when viewing a
page.  However, most of the time, picking "View page source" from the
menu will bring up a blank window with "Window" as the title.  Now, the
strangest part is, if he reloads the page, then tries again, he's able
to view the source.  

I've already tried repair permissions, but it didn't fix the problem.

nutshell, the user can't view page source from Safari when loading a
page.  However, if the user re-loads the page, Safari is then able to
view the page source.  Any help is appreciated.  Thanks.
