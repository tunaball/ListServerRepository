I missed the first part of this conversation, but every version of Unix, or Unix 
clone, to include MacOS X, has had some type of password expiration.

per the MacOSX FAQ here:

http://www.osxfaq.com/man/1/chpass.ws

The chpass command can be used to modify the "Change:" time, which is when a 
user must change their password by.

Also from the chpass man page on a MacOS X 10.3.8 system, also note:

-----------------------

  This version of chpass is currently limited to changing user database entries
  in the flat file and local netinfo databases.

-----------------------

hope this helps.
