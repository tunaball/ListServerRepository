Alexandre,

The boot delay could be due to font caching issues, a problem which 
is fixed with the 10.3.9 update. Give it a try. Also possible you 
have a corrupt or duplicate System font, which could explain the 
Safari crash. Sounds like you've already cleared out Safari's cache 
files using MacJanitor?

For the other issues, I'd recommend trying a PRAM reset (CMD-Opt-P-R 
on startup until you hear at least one extra startup chime). Yeah, 
this is an unlikely cure-all, but it does sometimes fix system 
sleep/wake issues.

HTH,

Rick
