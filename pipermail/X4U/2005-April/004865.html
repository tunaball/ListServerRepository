If I recall correctly, even when the region code is removed from the 
firmware of the optical drive, there is still another mechanism within 
Mac OS (or DVD player) that records a region & which can be changed 
also only 5 times.

Flashing the drive region-free is a great improvement, but this is 
still a pain. I was about to say "as it requires an o/s reinstall to 
circumvent", but perhaps there's just a preferences file somewhere that 
can be deleted. Last time I tried it on a PC, region-removal was much 
more mature, but clearly there are a lot more people interested in 
hacking that platform.

Ridiculous stuff like region codes just inspires in me a complete lack 
of respect for the entertainment industry, and makes me feel like 
justice is being done by copyright pirates.
