Some Mac OS X features make one wonder what they were smoking in 
Cupertino... You have to use Safari to set the default browser to 
something else, you have to use Mail to set the defaul mail client to 
something else... and you have to use Internet Explorer to set the 
default ftp client. Your setting is to use IE as ftp client. Go to IE > 
Preferences > Network > Protocol Helpers and change the ftp to Finder. 
Alternatively, you can try Default App

<http://www.rubicode.com/Software/RCDefaultApp/>

However, Finder as ftp client has limited capabilities. IMHO, you'd be 
better off using Fetch, Transmit, or Interarchy.
