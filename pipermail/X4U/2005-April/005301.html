So far no issues at all...

I did a complete Disk Util - HD repair with the disk that came with  
the iMac G5 (nothing found) and then a File Permissions repair...  
(Side note - the Panther install disk would not boot on the iMac G5,  
apparently it need at least 10.3.5.)

Then did the Upgrade choice on Tiger installation. I did a custom  
install, eliminating all the language upgrades and many of the  
printer drivers that I will NEVER use...

I clicked "Upgrade" and went out for a meeting... Came back a few  
hours later and logged in to everything working fine.

Spotlight is way cool.

Mail is better, although the list of mailboxes on the left seems to  
have too much line-spacing such that what was view-able without  
scrolling in 10.3.9 now required scrolling to see all boxes.

Epson SP 1280 prints fine.
Safari works fine.
iCal works fine.
BT wireless mouse and keyboard works fine.
Dreamweaver MX04, Freehand MX, Fireworks MX04 all seem fine - so far.
Photoshop CS, InDesign CS both seem fine.
Suitcase v11.0.2 seems okay so far
Audacity ok, Firefox ok, etc.

Once I come upon an issue I will report it... But so far very clean  
and easy.
