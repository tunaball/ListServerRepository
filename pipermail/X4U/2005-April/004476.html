On 04/07/05, Randy Jamerson <rjamerson at ebsco.com> wrote:
>
> Does anyone know of software that will allow me to create a screensaver for
> Mac OS X (take photos, make a slideshow and create a distributable
> installer)?

Why? Put the photos into the Pictures folder in the Home folder and have
them go to System Preferences->Desktop & Screen Saver pref pane, and select
Picture Folder. See Mac Help->Screen Saver->Using your own pictures as a
screen saver. Check at <http://www.resexcellence.com/> for information on
screen savers and associated software.
