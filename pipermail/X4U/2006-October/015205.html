>his harddisks.  When he opens these images, some of them are 
>disturbed.  
>They seems to have a lot of white dots. Like they're overexposed.
>He claims that this is done during the copying.  What can gone wrong?

Simply moving them should not have done anything. But if he's opened 
and saved them again each time, if he didn't save at highest quality 
then he was re-compressing them each time, losing some information 
each time.
