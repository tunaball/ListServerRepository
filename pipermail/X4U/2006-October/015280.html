>>Anyone with a username and password (or who can guess one) can
>>download files from it (and upload to it).
>
>Where is the facility for setting up a guest FTP username and password
>when putting OS X into FTP server mode?

Just set up another normal user called guest. In Netinfo Manager make 
their shell /usr/bin/false (so they can't do a Unix network login) 
and set up a GUI login command to log out for that user. (Don't know 
how to do that, but recently someone here did it by mistake IIRC). 
That way it will be an ftp only account.
