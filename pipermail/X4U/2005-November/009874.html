On 11/2/05, net kat <netkat at comcast.net> wrote:
> I see where iTunes offers whether to check for new podcasts daily,
> weekly, hourly and manually...I have it set at Daily, but the time is
> a curious 7:17am..
> be nice if I could set this to something like 5:50am..a time before I
> get up so that the casts are there waiting for me.
>
> Any thoughts as to whether this can be done?
>

Yup. Easily. The reason why it's set to 7:17 am is because that's the
time that you last checked *before* setting it to update daily.

So do this. It works:

1. At 5:50 am (or whatever time you want it to check), Go to the
settings and set the update preference to "manually". Hit "Okay" to
close. Then click the button in the upper right hand corner to
"Update"
2. When the update is complete, go back to settings and you'll see
that it has the time you did that manual update. Now change it to
"daily" and you'll see that the next update is set for tomorrow at
that time you just did the manual update.
