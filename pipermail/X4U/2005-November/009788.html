Gail,
Here's a workaround.
Connect the computer without a DVD drive to a Mac that does have a  
DVD drive by using a Firewire cable and start the no-DVD unit in  
target disk mode. Place the install DVD into the one that does have  
the DVD drive and run the installer, choosing the no-DVD unit as the  
target of the install.
As they say in France, viola!
