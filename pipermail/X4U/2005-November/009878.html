>> So do this. It works:
>>
>> 1. At 5:50 am (or whatever time you want it to check), Go to the

>> close. Then click the button in the upper right hand corner to
>> "Update"
>> 2. When the update is complete, go back to settings and you'll see
>> that it has the time you did that manual update. Now change it to
>> "daily" and you'll see that the next update is set for tomorrow at
>> that time you just did the manual update.
