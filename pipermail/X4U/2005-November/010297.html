Finder views filesystems.  FTP is File Transfer Protocol, which
is primarily about moving files between your host and the remote
host, and not manipulating files and folders on a remote host.

Actually, I'd like to see Finder have more inclusive FTP support
to allow things like uploading.  But that's Apple's decision and
not mine.  Besides, I'd like to see Apple fix Finder so that it
doesn't lock up so often (stupid Carbon?).

If you can't wait for Apple to add this to Finder, there are
freeware alternatives like RBrowser that does the job.
