I had a similar problem in that Safari, iTunes and the Widgets would  
not work properly after a system upgrade. The fix from Apple tech  
support was to delete the plist files in Main/Library/Preferences/ 
SystemConfigurations and restart. That worked. Unfortunately, this  
was the 3rd call to tech support on this problem. The first two  
resulted it a system and an application reinstallation with no  
solution to the problem, but the last  worked fine.
