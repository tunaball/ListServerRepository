I don't understand the comment "Depend(s) on who starts the copy.. be it 
on my computer or another computer.."
The action can only be accomplished from the host computer. The host 
shows the connection to the server, but the server does not show a 
connection to the host. In other words, the server volume(s) show(s) up 
on the host, but not vice versa.
