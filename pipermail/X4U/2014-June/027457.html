On Jun 3, 2014, at 2:35 PM, Jens Selvig wrote:

> I have a friend who has a HP 8600 pro printer. When she prints to it from her Mavericks 9.3 iMac, the printer doesn’t start printing for about 40 seconds after sending the print job. It acts like the printer doesn’t wake up for 40 seconds! She has a new Apple Extreme router which replaced a linksys router. Oddly, the printer worked OK with the linksys and a windows computer.
> 
> Any  ideas on what would fix this?

Try un-installing the old driver:
http://h10025.www1.hp.com/ewfrf/wc/document?cc=us&lc=en&docname=c02440673

And then install the latest Mavericks drivers:
<http://h10025.www1.hp.com/ewfrf/wc/softwareDownloadIndex?softwareitem=oj-125213-2&cc=us&dlc=en&lc=en&product=4323648&tmp_track_link=ot_search>

or

http://is.gd/7Pu6Kp

In the Print dialog box there is a drop down menu.  
Choose: Scheduler.  
Set the Priority to "Urgent."
