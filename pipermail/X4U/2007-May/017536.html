IIRC, tooltips are on by default in Tiger's Safari. I don't use Mail, so can
say anything about that app. To see if it's ever been changed, type in this
terminal command:

defaults read com.apple.safari WebKitShowsURLsInToolTips

0 indicates off and 1 indicates on. See this hint for details about Safari's
tooltips <http://www.macosxhints.com/article.php?story=20070519173316942>.
