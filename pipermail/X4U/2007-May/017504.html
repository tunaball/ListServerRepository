Am 20.05.2007 um 12:31 schrieb Lawrence Haber:

> I know how to change the format of addresses in Address Book.app   
> from, say, a US address format to one for someplace else, such as  
> Switzerland, for example.  But, this will change all the address  
> formats throughout the entirety of  my address book.
>
> Does anyone know how to have a different address format template on  
> an address by address basis?  I have a lot of addresses from all  
> over the world and I haven't figured out how to make a US, UK,  
> German and Swiss address all look right, all at the same time.

For me it works like this:

- edit contact
- click the popup menu to the left of the address whose format you  
want to change
- go to the submenu 'change address format'
- select the format you like

This just changes the format of this individual address.
