I've used Netgear model WGT624 
<http://www.netgear.com/Products/RoutersandGateways/SuperGWirelessRouters/WGT624.aspx> 
for several years and highly recommend it. It is promoted as a wireless 
router, but I've disabled the wireless and instead use its four ethernet 
ports. Very easy to set up and I've used it with macs and a Windows 98 pc.
