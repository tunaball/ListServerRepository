> Hello, Paul.
> 
> I assume that you are connecting to the Brother via Ethernet. If so, you probably have to set up the printer using IP.
> 
> In the Add Printer dialog, click the IP icon. The default Line Printer Daemon should work. Enter the IP address for the printer (you should be able to get this by printing a config page). If necessary, select the new driver that you downloaded. Optionally, you might want to name the printer (Mac OS will use the IP address as the default name).
> 
> I had to do this in order to connect with my old HP LaserJet 8150dn in Snow Leopard because AppleTalk has been retired. Works fine.
