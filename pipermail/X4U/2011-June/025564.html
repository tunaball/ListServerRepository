Serveral applescritps to rename files in finder.

Scripts: http://www.rafaespada.com/downloads/files/courier/files_finder.zip
Screenshot: http://www.rafaespada.com/downloads/files/courier/files_finder.jpg

f_add_select.scptd - add prefix or sufix in the selected files in finder

f_case_select.scptd - change case in the selected files in finder

f_internet_select.scptd - remove and change characters to use in internet in the selected files in finder (i am from europe and here use server "non standard" characters in normal use there are not compatible with internet servers - áéíóúñç)

f_replace_select.scptd - replace characters in the selected files in finder

file_random_ext.scpt - add a random text prior to extension

file_random_rename.scpt - rename with random text the selected files in finder.
