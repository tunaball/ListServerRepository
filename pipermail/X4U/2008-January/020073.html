Reboot, but don't log in. Leave it until, if you were logged in, the 
space would have reduced significantly. Log in and check the space. 
This will tell you if the space grabber is running all the time or 
only when you're logged in.

Do you have any other machine running on the same network? (If so it 
would allow logging in from that and more diagnostics would be 
possible without the overhead of the GUI).
