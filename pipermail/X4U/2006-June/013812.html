Check out the 5 star MainMenu v1.5.4. Feature laden utility with much 
to offer including the ability to disable/enable the dashboard.

 From Versiontracker...
MainMenu is an easy-to-use menubar item that allows you to run a 
large set of Mac OS X maintenance, cleaning and optimization tasks. 
MainMenu can repair permissions, update prebindings, clean system and 
users caches, empty many browsers caches/history, force empty the 
trash, clean logs, repair disks, and much more. Most tasks provide 
feedback in the log window during execution. The batch task function 
allows you to run many routines at once. Bug reports, feature 
requests, feedback (positive as well as negative) and anything else 
are welcome. MainMenu is currently free.
