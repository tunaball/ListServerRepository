That's the problem with some '.mpeg' files. The person encoding them  
actually created Windows Media files but improperly named them MPEG.

Windows Media Player for Mac will play these, but it has been  
discontinued, thankfully, as it was not very good.

However, since it has been discontinued, MS, it seems, has contracted  
with Flip4Mac to give away their plugins for QuickTime.

Flip4Mac was not free before MS gave up the player market to Apple.
