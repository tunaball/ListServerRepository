> I'm wanting to try out the quicktime streaming server software. But  
> currently I'm not running OSX Server.
> Just running the latest OSX on an iMac G5
>
> Is this possible, or do you HAVE TO HAVE osx server ?

The first result from http://www.google.com/search?q=quicktime 
+streaming+server is http://www.apple.com/quicktime/streamingserver/

That page states:

    QuickTime Streaming Server System Requirements

    - Mac OS X Server v10.4 (or later)

    - Xserve, Power Mac (G5, G4 or G3), iMac, eMac or Mac mini computer;
    128MB of RAM, at least 256MB of RAM for high-demand servers running
    multiple services; built-in USB; 4GB of available disk space.

You should probably look at Darwin Streaming Server, "the open source  
version of Apple's QuickTime Streaming Server technology that allows  
you to send streaming media to clients across the Internet using the  
industry standard RTP and RTSP protocols". http://developer.apple.com/ 
opensource/server/streaming/index.html
This mailing list might be worth a look: http://lists.apple.com/ 
mailman/listinfo/streaming-server-users and these pages http:// 
www.streamingmedia.com/article.asp?id=7705&c=6 http:// 
www.streamingmedia.com/tutorials/view.asp?tutorial_id=143
