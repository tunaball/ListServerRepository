I'd recommend not running any Dashboard Widgets.  Is there anyway to turn
that pig off yet?  It is one of the reasons I went back to 10.3.9 on my G5

use Firefox as it is far more efficient than Safari.  I'm using the first

Safari it would get so sluggish, I'd have to restart it (the memory usage
would also go through the roof).
