> I've got a series of photographs of a model of a building.  I'd  
> like to be able to stitch them together to get a rotatable 3D  
> image.  Any suggestions for (cheap) software that can do this for  
> me?  I searched VersionTracker, but I'm not sure I'm using good

Hey John,

If I'm to read your subject and description correctly, it sounds like  
you either want to:

a) create true 3d geometry from a series of photographs, in which  
case you might want to check this out:

http://www.strata.com/foto3d.asp

OR

b) create a quicktime VR of the scene, which can include a 3d model:

http://www.apple.com/quicktime/technologies/qtvr/

And a list of authoring solutions from the Apple site:

http://www.apple.com/quicktime/resources/tools/authoring.html
