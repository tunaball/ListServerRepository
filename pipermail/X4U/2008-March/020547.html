Basically, Intel Macs partition hard-drives differently than the way  
PPC Macs do. I don't think a PPC Mac can format a drive as Intel- 
bootable, and I don't think one can install an o/s on a drive that  
has been partitioned by an Intel (at least, not so the o/s is  
bootable on an Intel).

Basically, you need to use an Intel to install the o/s on your  
Macbook (ideally by booting the install CD from the machine itself).
