I'm pretty sure that the file "~/Library/Application Support/ 
AddressBook/AddressBook.data" contains user data for Apple's  
AddressBook app. (Note: "~/" means "in your home folder"). I think  
you'll find that if you close AddressBook, rename this file & reopen  
AddressBook then you'll start with a fresh sheet. There's also an  
AddressBook.data.previous file in the same folder.

Outlook is not the same as Outlook Express - they're really  
completely different programs - which is why you couldn't find  
the .vcf options. Generally speaking .CSV files are a good way to  
transfer contacts - these are plain text files with data fields held  
as "Comma Separated Values", so you may find it helpful to open them  
in TextEdit or similar just to check the exported data.
