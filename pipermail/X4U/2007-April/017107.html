The main advantage of A&I is that you can't do a upgrade previous version if
there is no previous version. I.e., if Tiger's installed, then upgrade is
unavailable. If Panther's installed, then upgrade is unavailabe if you wanted
to install Panther. It's an intermediate process between upgrade and E&I, that
latter which requires reinstallation of all 3rd-party stuff.
