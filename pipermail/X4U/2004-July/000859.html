Maris Avots said:

>If using Carbon Copy Cloner impacts system performance, what is the best way
>to move existing system files to a new hard drive?


If you want an exact clone of your system, it's still Carbon Copy Cloner. 
If you just want another system installation, a better way to go, if it 
will work, is to install the system with our original OS X installer 
CD's.  

However, some folks (mostly in the early days of OS X) have reported that 
they cannot successfully create a bootable external drive using their 
install CD's, yet they can using CCC.
