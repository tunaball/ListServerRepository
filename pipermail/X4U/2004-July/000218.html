While I know I really could use some more RAM, I don't think that was 
my real problem.  Part of it turns out to be a zombie 'tcsh' process 
that was trying to take up about 1GB RAM, plus I think the system 
just plain need rebooted.  I decided to do so cleanup to free up a 
little disk space, and then run DiskWarrior.  After rebooting, the 
system seems much happier so far with MS Word/X, Photoshop CS, 
Illustrator CS, InDesign CS, Eudora 6, and Safari running.

Right now the top pig in memory is Illustrator, which isn't to 
surprising as it has by far the most complex document open.  I just 
had my wife log in and start up her main apps, and that barely puts a 
load on the system.  I really think that the OS has a slow memory 
leak in there somewhere, as well as some of the apps.
