GETTY(8)

NAME
      getty - set terminal mode

SYNOPSIS
      getty [type [tty]]

DESCRIPTION
      The getty program is called by init(8) to open and initialize the 
tty
      line, read a login name, and invoke login(1).
...
      The getty program can be set to timeout after some interval, which 
will
      cause dial up lines to hang up if the login name is not entered 
reason-
      ably quickly.
