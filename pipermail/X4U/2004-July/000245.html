On 6-jul-04, at 17:18, Peter Krug wrote:

>> cerrtain website's disallow the caching of QT movies which disables 
>> the
>> "save movie as..." What you have to do is open the page (or movie 
>> file) in a
>> browser that keeps a findable cache (I use Mozilla) and then go to 
>> that
>> browser's cache folder and order the window by size. The QT movie 
>> will have
>> some funny name but will be one of the larger files and will rise to 
>> the top
>> as it loads. Then you can copy that file to wherever you want.
>>
>
> What I do is after the movie completely loads, with the page still 
> open, switch to the finder and search for an invisible file called 
> "QTPluginTemp".  Move it to the desktop and rename it with a .mov 
> extension.  You're done.  This works in Camino and Safari.  This is a 
> simple way to do the same thing that the MacOSX Hints link reported 
> earlier does.
>
