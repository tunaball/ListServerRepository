I'd run the Apple Hardware Test program that came on her install disks,
especially the extended RAM tests. Then, if it passes those, I'd create
another admin account, log into it and see if the crashes stop. Finally, if
the other user doesn't stop it, then reinstall OS X, using Archive &
Install, saving network settings, repair permissions, update via Software
Update, and, when there are no more,
repair permissions again.
