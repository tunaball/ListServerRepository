I have found on several occasions that after downloading and playing a 
quicktime movie on a website, and BEFORE I close the window the 
quicktime movie does save in a directory: Example 
~\private\tmp\501\TemporaryItems\

I will open a find window immediately after the file completes 
downloading, and search for file Modified with todays date, and with 
Visibility = Invisible.  It will show the file.  It should be named 
something like QTPluginTemp6121361. I then navigate to it, drag it to 
the desktop, and you can do with it what you will.

Hope this helps.

John Belen
