The AFP connection protocol is afp://username:password@IP.ADD.RES.S and I
believe the SMB connection protocol is smb://Server/Share

If the OS X machine you're using to connect is in a different workgroup than
your windows box, I believe that goes in there like
smb://Workgroup/Server/Share

http://www.macwindows.com has more info, I'll bet.
