>> How would I go about burning a .MOV file on a DVD so that it would  
>> play on any commercial DVD player?
>
> If you have Toast, go to the Video panel, select DVD-Video, drag  
> the .mov file into the window and click "Burn".
>
> If not, download ffmpegx from <http://homepage.mac.com/major4/>   
> (follow the Download link) and carefully follow the directions to  
> install. Then follow the directions from <http://homepage.mac.com/ 
> major4/dvd_sub.html> to create and burn the DVD.
>
> If you need more specifics, just ask.
