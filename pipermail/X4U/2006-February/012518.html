This is a well-founded fear. FTP is a big hole in your server  
security. The suggestion for webDAV is a good one; seems to be pretty  
safe.

On my Linux servers I use a webshare app to allow access through the  
browser. The advantage is that the webshare app specifically sets non- 
executable permissions to the uploaded file. Only the app and the  
admin can manipulate the files.

I think there is a similar app for Mac OS but I am still a bit low on  
the learning curve to know.
