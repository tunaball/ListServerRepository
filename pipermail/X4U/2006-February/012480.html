The slideshows at
http://www.nantuckethydrangea.com/gallery/HydrangeaGallery.html
were done in Photoshop 6. The automatic navigation was added later by 
customizing the settings/user files in the  Web Gallery folder and it 
is a bit cumbersome, Though a static show is pretty easy to edit and 
understand. You can scale, add photo data, html tags, etc right from 
the application interface or by using customized presets, and batch 
process as many photos as you like.
