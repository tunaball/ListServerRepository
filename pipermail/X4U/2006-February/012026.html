<http://www.macosxhints.com/article.php?story=20050627224051361>

Basically, this command:

defaults write com.apple.screencapture type png

executed in the Terminal app creates the com.apple.screencapture.plist, if it
doesn't exist, and makes png the default screencapture. If you use pdf, tiff,
or jpg instead of png, that's what screencapture's will become.
