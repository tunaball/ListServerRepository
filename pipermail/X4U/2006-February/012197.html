Is there a way to modify the built in rules for Smart Mailboxes to allow
inclusion of an outgoing message if the recipient is the member of a group?

In Apple Mail one can create a smart mailbox . Rules permit inclusion if the
sender is a member of a group in the Address Book, and there is an option to
include sent mail. But there seems to be no way to include a message if the
recipient is a member of a group. It appears to be necessary to add each
group member individually.

So the question is: is there a way to modify the built in rules for Smart
Mailboxes to include an outgoing message if the recipient is the member of a
group? Or any way to accomplish this without adding each group member
address individually?
