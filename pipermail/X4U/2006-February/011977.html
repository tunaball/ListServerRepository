in Safari, if you have some tabs open, then you open a set of book  
marks using "Open in Tabs," those tabs replace what you already had  
open, requiring you to make new tabs and open them again.

in Firefox, opening a set of bookmarks in tabs ADDS those tabs to  
whatever you have open.

much, much smarter, IMO.
