According to applesource (among others)
<http://www.applesource.com.au/how-to/soa/Learn-to-take-screenshots-in-Mac-O
S-X/0,2000451082,339287569,00.htm>:

Mac OS X 10.2 = JPG
Mac OS X 10.3 = PDF
Mac OS X 10.4 = PNG

If you're using Mac OS X 10.4 and want to change the default format, you can
do this in Terminal:

<http://www.macosxhints.com/article.php?story=20050627224051361>

This works in Mac OS X 10.4, not earlier, and was posted before the advent
of Mac OS X 10.5, so I don't have any way to know if it works in Leopard or
not. Before anyone tries it in Leopard, you'd probably want to investigate
further.

There are other screensnap utilities, such as SnapzPro, that let you choose
the format of your screenshots; Paul, is it possible you're using one of
those utilities?
