On Jan 7, 2006, at 7:57 pm, net kat wrote:

> I'm using Text Edit to put web pages with CSS together..I have the 
> prefs set to save pages as plaintext, but when I save, these text 
> files have a Safari icon, and when I re-open them in Text Edit, rather 
> than seeing my toiled-over code, I see it as though looking in a 
> browser!
>
> Any clues what I need to do to keep my code files AS code?

In TextEdit preferences, check "Ignore rich text commands in HTML 
files". You can also toggle this in the Open dialogue when you open 
files from within TextEdit.

But I second Milton van der Veen's recommendation of TextWrangler.

Miche.
