At 7:20 AM -0800 1/15/06, Jim Robertson wrote:
>I have DiskWarrior version 3.0. Alsoft has a free update to 3.0.3, but I
>need to be able to put DiskWarrior onto a bootable disk that will work with
>Tiger. My 3.0 disk won't boot my 15" Al PowerBook or my Dual G5 1.8 GHz
>PowerMac. There is a shareware program named BootCD that isn't yet
>compatible with Tiger.
>
>I know this comes up from time to time, but I'm wondering if anyone has come
>up with a way to make bootable Tiger CDs onto which I could put the latest
>version of DiskWarrior?

Interesting. It seems hit and miss. I know a few people who have made 
a 3.0.3 CD that will not work in Tiger. On the other hand, I made one 
and it does boot my machine. I don't know what the difference may be. 
Have you tried making a 3.0.3 CD on the machine you want to boot? If 
not, try that: go to the computer you want the CD to work on, and 
make the update on that very machine.

As a workaround if that doesn't work, how about installing 
DiskWarrior on both machines, and start up one machine in FireWire 
Target Disk mode and run DiskWarrior from the hard drive of the other 
machine?
