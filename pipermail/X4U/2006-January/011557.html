I had to go figure out what I don't like about iTerm, I remembered 
trying it (actually I've tried several versions).  It works fairly 
well, but can't handle double-width fonts, and is painfully slow. 
OTOH, the keyboard emulation seems to be excellent based on a quick 
test a couple minutes ago.

If I didn't have a modified .xmodmap file that remaps the keyboard 
the way I need, I'd be really tempted to use iTerm.  The deciding 
factor is that xterms are *way* faster.
