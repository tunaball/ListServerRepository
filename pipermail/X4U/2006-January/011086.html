I've burned dozens of different iDVD projects and have had only one  
issue and that was because of a separate audio file that I had added  
to the template. After I fixed that one audio file that iDVD project  
burned fine too.

I almost always save to disk image first and then burn to DVD-R from  
there.

iDVD for me has been rock solid.
