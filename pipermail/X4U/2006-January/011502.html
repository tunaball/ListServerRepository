PPC code will be around for a long while unless code makes specific  
use of features in the intel chips.

If, for instance, you have Windows emulator running natively on the  
intel CPUs then that won't work on a PPC.

If Apple uses new DRM or virtualisation schemes (both currently not  
built into the intel chips uses for the first batch of Macintel) then  
that, too, would not run on PPC (nor on the first Macintels for that  
matter).
