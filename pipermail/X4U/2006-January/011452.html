>> I've never used iChat before. At this point, I want to use it to send
>> a file to another Mac user I know. He also has never used iChat.
>
> Launch iChat. It will walk you through the setup. There is a button  
> to press
> to "Get an iChat Account" since you don't currently have an AIM  
> account.
> Tell each other your account names and then add them to your Buddy  
> List.
>
>> Can someone please give me a quick rundown on how to set this up so
>> we can transfer the file? It is to large to email.
>
> Copied directly from iChat Help:
>
> You can use iChat to send or receive files from your buddies.  
> Unlike email,
> there is no limit on the size of the file you can send, but larger  
> files
> will take longer to transfer.
>
> Select a person in your Buddy List, then choose Buddies > Send File.
> Select the file you want to send and click Send. You can send only  
> one file
> at a time. If the file is large, or you want to send multiple files  
> at once,
> you should compress the files before sending. The recipient can  
> accept or
> reject the file you've sent.
>
> Tip: To quickly send a file, drag it to a buddy's name in your  
> Buddy List,
> or into the message area of a chat window.
