I had the same problem.
I solved it importing the folders in batches and finding the ? 
corrupted? mailbox.
Once I found the mailbox  I used some mailbox archiving software to  
archive the culprit and re-import it in Mail
I'm sorry I don't remember which software was; I found it through  
versiontracker. I have tried some of them until I found one able to  
archive in a format suitable to be imported by Mail.
It might be this <http://www.versiontracker.com/dyn/moreinfo/macosx/ 
26325> or <http://www.versiontracker.com/dyn/moreinfo/macosx/23083>  
or <http://www.versiontracker.com/dyn/moreinfo/macosx/21945>
For sure I did not need to buy in order to use it once.
