Windoze XP speaks BIOS, so it does not install on Intel Macs.

Windoze XP 64-bit speaks EFI, so it can install on Intel Macs.
However, Intel Core Duo is 32-bit, not 64-bit.

Windoze Vista is purported to speak EFI as well as BIOS, so it
can install on Intel Macs (whenever it is finally released).

Intel Macs have no BIOS, instead they have EFI (Intel's version
of a next-generation BIOS).
