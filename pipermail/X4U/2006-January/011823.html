Can I create a bootable backup with ChronoSync?

The short answer is, yes. However, it is not designed as a utility for
cloning an entire hard drive.

To have ChronoSync make a bootable backup you need to do the following:

1. Run ChronoSync with full administrative privileges.
2. Synchronize to a hard drive that has it's "Ignore ownership on this
volume" setting turned OFF.
3. Execute a post-synchronization AppleScript that "blesses" the system
folder on your external hard drive.

____We discourage using ChronoSync in this capacity, however.____
