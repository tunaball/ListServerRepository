On 01/24/06, Crandon David <tabdave at comcast.net> wrote:
>
> When using Carbon Copy Cloner (the latest version) to make a clone of
> my 10.2.8 disk, I always get the following error message:
>
> There was an error during copying 'Network' from source to target:
> Unable to get the attr list on server: 45
>
> I press the "continue" button and the cloning proceeds normally.
>
> Anyone know what the error message means?

IIRC, you've connected to a server that's no longer available. Check
Finder->Go->Connect to Server, click on the little circle icon, and select
Clear Recent Servers. If the problem persists, post to the CCC Troubleshooting
forum at <http://forums.bombich.com/>.
