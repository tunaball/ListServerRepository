I think it's old news that hackers can shoehorn OS X onto a Wintel  
Box, but that's precisely what they have: A hacked, shoehorned,  
pirated copy.

The contention that Apple will wither to dust now bcs it's only a  
matter of time before all owners of Wintel machines will have their  
own hacked copy of OS X is built on bricks of faerie dust.

For one thing, if you're a real Windows devotee, you don't WANT Mac  
OS. If you're truly an OS X fan, and need OS X, you aren't going to  
want some hacked copy on which to run your business. Heck, there are  
problems enough already with legit software running on the boxes it  
was designed for.

So, for this particular "apple is dead" scenario to be actualized,  
everyone who pirates OS X onto a windows box will have one sorry  
experience, probably even worse than having windows itself on that  
windows box!

And what do we all know about cheap Windows hardware, which seems to  
be the BIIIIIG draw for Windows users?

I wouldn't start hitting the Scotch over this one..
