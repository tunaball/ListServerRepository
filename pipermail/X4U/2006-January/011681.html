I'm not sure if you are using a wireless connection when you  
experience this, but I have had similar problems with my connection  
and my iBook.  I have found it easiest to turn off the Airport card,  
then run and turn the card on via Network diagnostics.  I've often  
thought that there must be a better way but, for now, it works.

It may be worth trying Network Diagnostics even if you aren't using a  
wireless connection.

Good luck,
