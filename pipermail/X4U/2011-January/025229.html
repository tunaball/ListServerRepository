Paul,

Generally, on the setup part of the printer display, the physical machine, one can go through the setup and find an item that will display the IP address on every Brother printer I have used.

If you are using a router (Apple), one can get the IP off Airport Utility.  Select router, manually configure button at bottom, click the Advanced icon at the top, then "Logging & Statistics" and bottom button "Logs and Statistics", then, DHCP Clients to find the IP address.

Another way is to launch the brother control center and it should launch the page you reference.  At the bottom left is a "Browse" button.  If your printer is on, that should show up with the DNS name.

Let me know if I need to clarify anything.
