On 10 Jul 2006, at 19:50, Randy B.Singer wrote:
> ...
> There are three products for OS X that can recover lost files:

You omitted:
   http://www.cgsecurity.org/wiki/PhotoRec
and:
   http://echoone.com/filejuicer/

I haven't tested them, but I always consider GPL software such as  
PhotoRec before opaque commercial offerings - a closed code base does  
not benefit the user, IMO. I believe there may be a GUI for PhotoRec.

It is perhaps not clear from the documentation of this software that  
the user may be required to make a disk image if the file has been  
lost on his or her system drive. Booting from a large external drive  
on which OS X has been installed will make this trivial.

Although I found both programs whilst searching for tools to recover  
photographs it should be noted that PhotoRec recognises .doc Word  
documents, .pst Outlook mailboxes (I wish I had know or thought of  
this when called out to a customer as an emergency on Sunday) and  
plenty of other file formats.
