> In Tiger one can set it to a maximum of 1 hour or 1 day.  Not 3 or 4
> hour.

You are missing my point and probably did not follow the link to
macosxhints.com. 

Yes, there is no interval in the graphic user interface between 1 hour and 1
day. But in the actual preference file, it uses seconds. When you set it via
the user interface to 1 hour, the preference file is set to ChangeTime =
3600. (3600 seconds=1 hour)

Prior to Tiger you could edit all your preference files in TextEdit. I had
edited my ChangeTime to 10800. I can no longer do that because Tiger
converted that .plist file into binary format.
