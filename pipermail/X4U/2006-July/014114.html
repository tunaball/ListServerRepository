On Jul 16, 2006, at 2:53 PM, Jerry K wrote:

> I am probably missing something obvious and just need to RTFM a  
> little deeper, but is it possible to create an iso cdrom image  
> using Disk Tool?

One way is to create a DMG image of the CD size you want using Disk  
Utility and then convert it to an ISO format with a free utility  
called ISOlator.

Another option is FireStarter FX (donation ware) which lets you  
directly create an ISO image according to its docs.
