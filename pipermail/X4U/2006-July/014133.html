On Jul 18, 2006, at 7:13 AM, Mark Des Cotes wrote:

> Can this be done using Quicktime Pro?

 From what I can tell, not directly. I don't think QTPro exports into  
mpeg2. You could try using the "Movie to QuickTime Movie" export  
option, select something like "DV/DVCPRO - NTSC" and then have Toast  
transcode again to create the DVD. Note that DV will take a lot of  
drive space (12-13 GB per hour of video). I tend to have trouble  
getting sound to export properly when using QTPro, but that all  
depends on the codec used for sound in the original file.
