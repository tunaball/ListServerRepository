UNIX - and that is OS newXt, server or not, does not store passwords. What it keeps is the result of a one way hash of the password as it was when originally entered.

When a user logs in his password as typed is sent through the same hashing algorithm and the result is compared with the stored hash.

So. . . The server does NOT KNOW the password. All you can do is reset it to a new one.
