> Option-F12 works for me.

If OPT-F12 doesn't work and if you've just got one external drive you could
just fire off this Applescript:

property drawerstatus : true

if drawerstatus is true then
    do shell script "drutil eject -drive external"
else
    do shell script "drutil tray close -drive external"
end if
set drawerstatus to not drawerstatus

Attach it to a key using System Preferences or your method of choice. The first
press will open the tray and the next will close it. Worst case, it takes two
presses if something interferes with the tray.
