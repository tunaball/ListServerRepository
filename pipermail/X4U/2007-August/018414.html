On 18 Aug 2007, at 23:59, Darlene Hess wrote:

> I am hoping someone can help me find software that will allow me to  
> reveal passwords. We often have teachers who have forgotten their  
> passwords over the summer and it would be nice if we could retrieve  
> that information for them, especially for their connection to the  
> server.

If their password is "saved" on their Mac then it can be found in  
Applications/Utilities/Keychain Access. I think that by default it  
(unfortunately for you) requires the user to remember their main  
login password before it will reveal their other saved passwords, but  
perhaps this may be of help to you in any case.

I think it _may_ be possible - when initially setting up a keychain -  
to configure it so that site admins can open it as well as the main  
user.
I think that the KEYCHAIN password is unaffected if another admin  
changes or resets the user's LOGON password (but you might wish to  
check for yourself in the hope that my memory is faulty).
