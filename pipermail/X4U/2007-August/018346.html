For some reason its not always possible to install OS X onto a  
FireWire drive from the OS X installer CD-ROM.  This problem seems  
more common for folks using Intel-based computers.  Make sure that  
the FireWire drive is formatted in Disk Utility as HFS+, with the  
proper Intel partition scheme, prior to trying to put OS X on it.
http://db.tidbits.com/article/8405

The solution is to clone your OS from your main drive to your  
FireWire drive using either of these:

Carbon Copy Cloner (donation requested)
http://www.bombich.com/software/ccc.html

SuperDuper $28
http://www.shirt-pocket.com/SuperDuper/SuperDuperDescription.html
