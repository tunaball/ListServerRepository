At the risk of entering a thread far along...
There are dozens of commercial methods and several shareware ways of copy-protecting 
data CDs and DVDs. There has been quite a set of programs developed to be able to allow 
one to make backups. Unfortunately, some schemes like Hexalock can be really hard to 
bypass. Even more unfortunately, most of the backup software doesn't run on Macs. sigh. 

For me, the best defense is: don't buy it. If forced to buy, complain a lot to the company and 
post online reviews decrying draconian restrictions.

For a fairly comprehensive list of techniques/software, look at:
   http://www.cdmediaworld.com/hardware/cdrom/cd_utils.shtml

Good luck,
