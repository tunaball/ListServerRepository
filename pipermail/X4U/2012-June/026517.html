I know very little about the deep internals of the Mac being a Unix 
and Networks guy, but I wonder if anyone has looked at this idea.

'Hackintosh' is a way to run OSX on generic PC hardware.
Boot Camp is a way to run Windows, Linux etc on Mac hardware. I 
presume it provides the 'BIOSy' links so that an OS intended to use 
PC hardware can link through to the equivalents on the Mac hardware.

Is it possible to install pre-Lion as a Hackintosh on a Mac using 
Boot Camp? As this would be an Apple OS running directly on Apple 
hardware this should not be illegal. There is no emulation.
