To save a complete Youtube film install Perian <http://perian.org>  
and see also WATCH <http://perian.org/#watch> on that site to know  
what to do.
It's free.  It will create a .flv file which one can open in QT and  
then save as a .mov file.  One can just add .flv after the downloaded  
file.
No need for Com-I.
