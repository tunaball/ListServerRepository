At 10:43 PM -0500, 8/26/07, Cheryl Homiak wrote:

>I'd especially like feedback if somebody on this list uses or has

I do lots of installations and I've never found any particular 
problematic situations between ISPs and wireless access points. I've 
dealt with all the ISPs and lots of access points. There are 
differences in configuration process but as far as compatibility, 
there is no conflict that is consistent or unworkable.

(The single exception to all of the above is the very few ISPs that 
provide internet access with private WAN IP addresses.)
