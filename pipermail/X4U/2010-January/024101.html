I hate it when that happens!  :)

First of all, you might try posting your question on the Apple message boards.

Okay, a few questions...

1. How is your WiFi configured in terms of security? Are you using WAP2? Have you changed anything with your access point configuration lately?  (Try rebooting your wireless access point if you haven't yet.)

2. Launch the iPhone's Settings app, and select "WiFi". Is it set to "ON"? Do you have it set to "Ask to join Networks"?

3. If you can "see" your WiFi network in the list, tap the blue arrow on the right. Is it set for DHCP? Try tapping "Renew Lease" and see what happens. 

4. Try tapping "Forget this Network". Then return to the list of Wi-Fi Networks and see if it behaves differently.

I'm not sure if any of this will help, but perhaps someone else will add more...

Keep us posted, and good luck!
