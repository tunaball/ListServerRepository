This can be accomplished by turning everything off (unplugging the 
cable modem), doing the reconfiguration, then turning everything back 
on. The cable modem will slave to the MAC address it sees first.

And don't let them install any software on any computer. You don't need 
it. Go for the "Self Install" option if it's available.
