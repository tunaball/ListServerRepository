Great suggestion. However, if your prefs check out with PT and you still
have the problem, I suggest creating a new admin user account, log into it,
and see if the problem continues. If not, then it's something else in the
original user account. In that case, you should copy all the data files
from the original to the new user, reset the prefs, copy pertinent stuff
from that user's Library folder (except prefs), and eventually delete the
original. If so, then you have some system level corruption and I suggest
reinstalling the latest OS X Combo updater for your particular
installation, run Software Update and bring the system up to date. Finally,
run Disk Utility and repair permissions. The above should fix what's ailing
you.
