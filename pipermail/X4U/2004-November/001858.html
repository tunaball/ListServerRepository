Hello Paul,

The Address Book Data is held at
Users/Library/ApplicationSupport/AddressBook

Shift these files from the PB to the G5.

Make a backup of the G5 Data Files first, in case you have some info in
there you wish to keep.

In Address Book you can make a backup, by going to File Menu/Backup
Database, saves the data with the date in the File name in a location called
Address Book BU. I think the first time you do this you select the Save
Location. Mine is on a different Drive to the Panther System.
