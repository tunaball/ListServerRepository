hi paul,

the cups driver is here:

<http://solutions.brother.com/Common_Download/en_us/macosxlaser/printerindex10.3.html>

the best way is to flush your printers configuration and start from scratch:

	sudo rm -f /etc/cups/printers.conf
	sudo rm -Rf /etc/cups/ppd/*

then restart cups:

	sudo killall -HUP cupsd

now set up your printer again. what we do in large user environments 
is we flush, redo all printer config, then "push" out the 
/etc/cups/printers.conf file and the /etc/cups/ppd folder then we 
restart cups daemon (cupsd). pretty nifty.
