you don't need sharepoints.
You can mount one machine as a server to the other simply by going to  
network and selecting the other machine (virtual disk-named for the  
user) and all other attached and mounted HDs.
Then use the User ID and password of the machine you wish access to.  
This should make everything available so you can drag and drop and  
open docs to your heart's content.
