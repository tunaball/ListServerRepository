On 15 Feb 2007 08:36:58 -0800, "revDAVE " <coolcat at hosting4days.com> wrote:

> Aside from hand copying and pasting - does anyone know a way to save the aol
> address book to a text file or anything?

This worked for me:
Open AOL's Address Book
Select All
Paste into an open MS Excel spreadsheet

You will then have names and email addresses; other info will be lost.

To use this data, tweak and save in CSV format for importing into other
databases.
