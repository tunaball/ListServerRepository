anyone know how to type the symbols for Ctrl Opt Cmd and Shift?

nk

What's wrong with free? Keyboard Viewer, for example.

System Preferences>International>Input Menu
Tick On Keyboard Viewer,
Tick On Show input menu in menu bar.

And if you select "US" in the input method part of the dialog box,  
you get a nifty US flag in the menu bar.

Hold down the special key (option, control, command, shift), and  
Keyboard Viewer shows the characters that correspond to using that  
special key.
Select the keys using keyboard or mouse with the Keyboard Viewer  
visible and the corresponding characters are typed into the current  
document.
