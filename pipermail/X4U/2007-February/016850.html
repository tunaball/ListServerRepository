On 02/20/07, Crandon David <tabdave at ca.rr.com> wrote:
>
> Can someone please refresh my memory about how to change both the
> short and long names of my computer? I'm giving my old G4 to someone
> else, but don't want to completely reformat the hard drive.

Computer name: SysPrefs->Sharing->computer name. Changing the shortname is more
difficult: <http://homepage.mac.com/frakes/MOSXPT/content/shortusername.html>,
<http://docs.info.apple.com/article.html?artnum=106824>, and
<http://discussions.apple.com/search.jspa?search=Go&q=shortname>.

That said, make sure you include all of the installed OS and application
installer disks with the machine.
