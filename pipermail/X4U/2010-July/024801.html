We have an Okidata Microline 320 Turbo printer (9 pin impact printer) that we print tractor fed mailing labels on a Windows PC using FIleMaker Pro.

Has anyone had any experience getting this printer (or any impact printer) running under Mac OS X? My MacBook Pro running the latest of Mac OS X 10.6 sees the printer (connected via USB), and I used a driver I found in Snow Leopard's default set of drivers called "Oki 9-Pin Series". I am also using FileMaker Pro on the Mac. I am able to send a print job to the printer, but nothing but garbage comes out, instead of legible text (obviously, the printer isn't understanding anything that the Mac is sending to it).

We used to use old ImageWriters on older Macs, but would like to be able to print to an impact printer (specifically for the need of tractor feed because of the way our production is setup in our mailroom). I would consider a different printer that would allow me to print on tractor feed for a continuous uninterrupted sheets of labels (1 up - 3.5" wide by 1" label).
