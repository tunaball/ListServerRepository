On Jul 13, 2010, at 2:58 PM, J. R. Rosen wrote:

> I have a PowerMac G5 2.3Mhz (late 2005) running 10.4.11.  OS 10.5  
> can't be found for under $175.00 (highway robbery), and 10.6 is  
> only $30.00 - BUT it WILL NOT run on a non-Intel Mac!
>
> Has anyone heard of a hack to allow G5's to be able to run 10.6?


that's kind of the whole concept behind 10.6: among other things, all  
remaining PPC code was stripped out. 10.6 is designed to run only on  
intel-based macs.

I remember 10 years ago, when OS X first came out, there were hacks  
to enable borderline macs like the 7600 to run OS X through some  
hack, but this isn't like that; it's not about horsepower, it's about  
the code itself and the chip it requires to run.
