Hi Al,

Where downloaded items go is actually governed by the Browser (eg. Safari). If you moved the Downloads folder the Browser is still going to look in the old location (and maybe even create a new Downloads folder if it's missing). You need to launch your browser and go into the settings and re-establish the connection to your desired location.

However, the Downloads folder in your user's folder _should_ be working fine. If you couldn't drag it to the Dock (the right side of the "sidewalk") then perhaps there are permission problems with your downloads folder. I suggest putting the Downloads folder back into your user's folder and running Repair Permissions from the Disk Utility app. If that dun work, try creating a new "Downloads" folder (maybe even run Repair Permissions again.

Best 'o luck!

Coj
