Turn on personal file sharing using System Preferences->sharing. Then  
connect/mount the other machine/account or drive using Finder->Go- 
 >Connect to Server.

The Airport express should allow you to network up to 10 machines  
wirelessly. They will give each machine a different IP address (i.e..  
10.2.1.2). You can use these for the server address.
