Power Point Show or PPS is a tool in Microsoft Office Power Point which is ideal for drafting presentations and slide shows. PPS files can be opened through the program called Power Point Viewer. This software can be downloaded for free. If the files cannot be opened from the Tools folder in My Computer, the user should search for installed PPS files, and through the settings in Advanced Options, these files can be opened.

They also have one for Word. See URL below.

http://en.kioskea.net/faq/190-opening-a-pps-file
