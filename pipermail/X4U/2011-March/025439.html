You can look at these fonts:

http://www.bedfordny.info/html/captcha/fonts/

install any you like and then the kids can select the font in a word
processor (open or libre office would work) and type away, change
size, etc.

I use shift-command-4 which allows you to draw a box around anything
on the screen.  Which you lift off your finger from the mouse it
creates a .png file of the boxed area.

Other mac keyboard shortcuts here:

http://support.apple.com/kb/ht1343
