>On Jul 26, 2011, at 1:55 PM, Jens Selvig wrote:
>
>>  A friend of mine has several Macs that he is 
>>considering upgrading from Snow Leopard to 
>>Lion. He has an apple id and so does his wife. 
>>So the question is, do they each have to 
>>purchase an upgrade or will purchase work for 
>>both computers?
>
>Apple would tell you yes.

Apple would not tell you 'yes'. Apple support 
would tell you to download the program to each 
computer. Having downloaded it once using an 
Apple ID will allow you to download it again on a 
second computer that is using the same Apple ID 
for the App Store at no cost. Apple will allow 
you to install the Lion OS on to all of your 
personal computers that you own and control. 
Apple does not try to get additional purchases 
from the same person.
