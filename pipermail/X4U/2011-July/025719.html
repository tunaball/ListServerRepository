On Jul 26, 2011, at 1:55 PM, Jens Selvig wrote:

> A friend of mine has several Macs that he is considering upgrading from Snow Leopard to Lion. He has an apple id and so does his wife. So the question is, do they each have to purchase an upgrade or will purchase work for both computers? 

Apple would tell you yes. "Rich folks" would say it's only $30. But, you don't have to.

You can download Mac OS X10.7 Lion and *before* you install it, grab the file called "Install Mac OS X Lion.app" - it should have been downloaded to /Applications.

Then use these instructions to create a bootable Lion disc or drive:
<http://www.macworld.com/article/161069/2011/07/make_a_bootable_lion_installer.html>

Or:
<http://www.informationweek.com/byte/howto/personal-tech/desktop-os/231002151>
