On 22 Dec 2006, at 7:08 PM, nk wrote:

> is there a way to export all the contents of Address Book into a  
> text-editor-friendly form?


	There is a fairly simple method built into Tiger's Address Book to  
pretty much do this.  It may not help nk, because I suspect it  
probably does not work in Panther.  Others may find it useful:


	--Open Address Book.

	--Select all entries.   (Or all the ones you want.)

	--Give a print command.  You get a large dialog box.

	--Select "list" for the style.  Choose the other options in this  
dialog box to meet your needs.

	--Hit the PDF button in this dialog box.


	
	A similar process works in iCal.  In iCal I found a bug that  
complicates things.  The same bug might exist in Address Book.  ICal  
does not like to crunch much data.  Give it too many entries for this  
process and it will crash or hang.  I had to break up the data into  
smaller bites then combine the resulting PDFs into one.  That's a  
bother but pretty easy to do.
