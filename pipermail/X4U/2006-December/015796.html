Apple finally added on recent versions of iTunes the ability to 
deauthorize the computer you are currently work from (see iTunes 
Advanced menu), but I have not seen anywhere the ability to 
deauthorize a computer if you are not using it at the moment (e.g. 
directly edit the list of 5 computers).

The only option I see for computers no longer in your possession 
would be to deauthorize all and let each active computer get 
authorized in due course. Since it's an easy process, it's really not 
a burden.
