On 12/14/06 11:55 AM, Mark Des Cotes <mark at astroprinting.com> wrote:

> Is there an easy way to unlock all the files in a folder and it's subfolders?

I'd do it with AppleScript. I just threw this together:

set masterFolder to choose folder

unLock(masterFolder)

on unLock(thisFolder)
    
    tell application "Finder"
        
        set itemList to every item in thisFolder
        
        repeat with thisItem in itemList
            
            set locked of thisItem to false
            
            if kind of thisItem is "folder" then my unLock(thisItem)
            
        end repeat
        
    end tell
    
end unLock

Paste this into a new Script Editor window and run it. It will prompt you
for the folder whose contents you want to unlock and work down recursively,
unlocking as it goes.
