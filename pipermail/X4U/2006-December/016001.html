You can enable root by going to Netinfo Manager and click on Security  
and then Authenticate.  Then, enter your password, click on Security  
again and you'll see that enable root is now an option that you can  
choose.
You do have to remember that using root is very dangerous territory  
if you really don't know what you're doing.  So, please be careful.
