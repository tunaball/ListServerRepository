Apples to Apples, yes I would and did spend the extra money to buy  
the Airport Extreme Base Station, the one that allows a USB printer  
to be installed plugged in so you can print wireless, also has a  
input for a antenna if you want extra range. Not that much more, but  
some. Plus an ethernet jack out for computers with out wireless  
access or ????  It's easy to set up, there is no windows junk to work  
around, the interface on setting up the base station is Pure Mac. To  
me that alone is worth it. Trying to get support for a Mac from any  
other maker, good luck. I have never had problems, what more do you  
want. Is a mac worth more then a PC. Ya I pay a little more but I get  
what I pay for, $60 bucks more and it will run till it's out dated in  
10 years.
Trouble, you get to call Apple, where they speak Mac, just to be sure  
to get a level 2 tech but it's pretty easy and windows machines work  
fine also.
