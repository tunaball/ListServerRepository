On 12/20/06 6:39 PM, Jeff Porten wrote:

> 1) Force the MacBook to think it has an external monitor, even though
> it doesn't.
> 
> 2) Use VNC on the PB to create a full screen view of the virtual
> external monitor.
> 
> Voila, the PB becomes an external monitor.  Problem is that 1) is
> impossible AFAIK, and 2) is difficult as VNC doesn't discriminate
> between monitors.  But it's one way to skin this cat.

So, can you:

1) Boot the MacBook into Target Disk mode;

2) Plug it into the PowerBook using a Firewire cable;

3) Restart the PowerBook, selecting the hard disk in the MacBook as the boot
disk?

It would be as if you're running the MacBook -- apps, OS, etc. -- on the
PowerBook. You wouldn't have BOTH monitors, though, and perhaps that's the
hang-up in this scenario.
