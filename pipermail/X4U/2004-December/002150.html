On Dec 16, 2004, at AM 05:22, Paul Moortgat wrote:

> Is it possible to save a screendump (comm shift 4) as a jpg file 
> instead of a pdf one?

Hi Paul.
Just use the Grab Utility. It's located inside /Applications/Utilities.
Launch it, and use CMD Z to get a screen dump. All screenshots are 
saved as TIFF files. These then can quite easily be converted to JPEG 
format using Photoshop, Graphic Converter, etc.
