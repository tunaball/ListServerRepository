On 12/15/04 1:22 PM, Paul Moortgat <paul.moortgat at pandora.be> wrote:

> Is it possible to save a screendump (comm shift 4) as a jpg file
> instead of a pdf one?

Not directly, but here's the quickest way I know how:

1. Add Control to your screenshot key command (Command-Shift-3, 4). This
will put the captured image on the clipboard instead on the desktop.

2. Open Preview

3. Select New from Clipboard from the File menu

4. Select Export from the File menu and choose JPEG as the format.
