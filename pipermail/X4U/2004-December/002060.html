I am using HP 2.5 driver and seems to set it up for black ink only. 
However I do have a color cartridge in place. I have a do it your 
self refill I obtained from refill from "Carrot Ink" -- 
<carrot_ink at email.carrotink.com>. I have only used a black refill as 
yet and am pleased with the price performance. My printer is an HP 
930 c
The trick get Black Ink only printed is to set up the window that 
comes up when you select "Print" as follows: 1) In the "Print"window 
pull lower box down to "Paper Type and Quality", 2) In that window 
top select "Paper", 3) Then in that window select "Quality:  Draft" 
and "Color: Gray scale", 4) In the same window select "Color Options" 
5) in the window that opens check the "Black only" box.

My order from Carrot.com follows FYI. The same kit may be used for 
the HP 930 and HP 970 cs, I believe.
