At about 5:29 PM -0500 on 12/20/04, Kevin Lee Allen wrote:
>I am expecting a new powerbook, I am sure my questions have been 
>addressed, but I could not find specifics in the archives.
>
>here goes. I basically want to move my home directory from my 
>desktop to my laptop. I assume I can copy away and set things up 
>tweaking each step, but is thee an automated way to do this? I am 
>looking to use the laptop as my primary web tool, so I want all of 
>my permissions, bookmarks and passwords to migrate nicely.
>
>This is specifically true of my iTunes music which i currently have 
>on several systems, i would like to move these files, without 
>re-importing from the CDs
>
>I would also like to set up[ a virtual ID for my wife. I think I 
>need to do this via .Mac, but I would like her to be able to use her 
>desktop as a primary system, but have her e-mail and bookmarks 
>mirrored on the PB.
>
>Thoughts, ideas, suggestions welcomed.

If you start up the PB in target-disk mode, you can connect it with a 
FireWire cable to your desktop, which will see the PB as a Firewire 
drive. Then you can use Carbon Copy Cloner (free/donationware)

<http://www.bombich.com/software/ccc.html>

to clone your entire Users directory to the PB, if you want.
