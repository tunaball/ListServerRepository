> There is no permanent fix that I know of. This has been an ongoing

> find a temporary fix. I can't remember where I got it, probably
> MacHints. Anyway, copy the following into the AppleScript editor and
> save it. Run it every time you switch users. I keep mine in my dock
> to make it easy. You'll notice that the whole script simply launches
> DMProxy.
> do shell script "/System/Library/Frameworks/
> ApplicationServices.framework/Versions/A/Frameworks/
> CoreGraphics.framework/Versions/A/Resources/DMProxy"
