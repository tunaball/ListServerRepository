At 8:26 PM -0500, 4/5/06, Robert Nicholson wrote:

>What is protected about running Windows native on the same drive as 
>OSX? Surely a virus on Windows is not restricted to writing to the 
>windows partition.

For the same reason that the Mac can not write to an NTFS partition 
(the standard format of a Windows XP drive), the Windows OS will not 
be able to write to the Mac partition (unless someone is going beyond 
what Apple is providing.) But more importantly, it must be recognized 
that just having a Windows OS will very much make the user 
susceptible to every Windows virus and spyware product that exists. 
Thus, that user will likely have a virus on their computer within 10 
minutes of having installed the OS and connecting to the Internet 
unless they have proper anti-virus software. This is just the reality 
of the Windows world.
