> "The default column width is determined by the first Finder window you set
> up in column mode. All new Finder windows use the same window size and
> column widths, which is fine."

That article is about the Column view in Finder windows. The Column view in
open/save dialog boxes is different. AFAIK any width changes to the columns
in open/save dialog boxes cannot be saved at all. You have to resize them
every time. And unlike Finder windows, double-clicking the column divider at
the bottom does not auto-expand the column.
