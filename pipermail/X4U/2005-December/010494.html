What do people think on the subject, has Tiger reached the point it's 
worth upgrading to?

I bought Tiger when it came out, but downgraded back to 10.3.9 after 
using it for a couple of months when my system disk died. 
Realistically, the only thing Tiger really buys me is better font 
support in X-Windows (this is actually quite important for me).  On 
the downside, I loose the ability to access classic Appletalk servers 
(no longer as important, as my OpenVMS server I access that way is 
now running a newer version of Samba that will work with Mac OS X).
