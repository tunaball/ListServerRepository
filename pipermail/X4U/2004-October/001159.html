> (2) Also, html messages have to be downloaded by clicking a link that it's
> okay to download pictures. I'd prefer that a setting be made to let them

Go to Preferences -> General Preferences -> Security

Under "HTML Options" check "Automatically download pictures from my Address
Book contacts."  For messages from mailing lists if you create a Mailing
List Rule Entourage will download pictures in them also.

As for problem number (1), I don't think I've ever had that happen.
