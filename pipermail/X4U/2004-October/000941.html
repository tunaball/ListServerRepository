On 4 Oct 2004, at 18:13, Steve Talkowski wrote:
> Is it possible to install the Classic environment without having to 
> reinstall OSX from scratch?

You can install classic whenever you want. It will not interfere with 
the OSX.
  If you have it, just install it from the DVD (open the DVD and there 
is a file called 'install applications' or something like that).
On the other hand, if you have a separate OS9 CD, install it and then 
choose the OSX system as startup disk.
