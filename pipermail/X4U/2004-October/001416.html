Eugene Lee wrote:

Be sure to disable any 3rd-party software that modifies the keyboard
input system (e.g. any key macro programs, things that re-map the
function keys, etc.).  Also, try moving your ~/Library/Preferences
folder to the desktop, reboot, then log into your main user account
and see if the problem persists.  If it doesn't, then one of those
preferences is corrupt and likely the cause.
