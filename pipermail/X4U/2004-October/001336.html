>It appears that there is a maximum size for mail. Is this true?

As has been pointed out, different servers have different max sizes 
set.  Something else to keep in mind is that if you attach a file to 
the email, it's going to be converted from a binary file to a ASCII 
file, that encoding adds considerably to the size.

Also, as I always like to point out, know who you're mailing the file 
to.  If they're on a dialup, they most likely won't take getting a 
5MB picture very kindly.  I prefer to put large files on a webserver 
and send the person the URL.
