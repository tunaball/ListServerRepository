On 10/05/04, Richard <cms01 at tampabay.rr.com> wrote:
>
> Where can you tweak the "User Agent" info for Safari?
> So, instead of a website saying hey your on a Mac,
> I want it to say, Hey your on a PC.

Safari's User Agent Option, which tells the website that you're using
another browser, is available under Safari's Debug menu. To activate the
Debug menu, quit Safari, launch the Terminal application, type the below
command in the Terminal window, and press return:

	defaults write com.apple.Safari IncludeDebugMenu 1

Quit Terminal, launch Safari, and note that Debug is now the last Menu
item. Select User Agent from that menu and you can select one of these
options:

Mozilla 1.1
Netscape 7.0, 6.22, and 4.79
Mac MSIE 5.22
Windoze MSIE 6.0
Konqueror 3

To deactivate the Debug menu, type, in Terminal:

	defaults write com.apple.Safari IncludeDebugMenu 0
