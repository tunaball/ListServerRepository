On Mar 5, 2005, at 3:14 PM, Anne-Marie Concepcion wrote:

> I set up a .Mac account a few weeks ago and so have an iDisk icon in 
> my finder windows. I remember testing dragging over files when I first 
> got it going and it worked fine and can still them in the iDisks 
> fodlers.
>
> I haven't used it since but just tried to back up some files from my 
> local drive to it and it won't take them. I'm trying to drag a simple 
> text file to the Backup or the Documents folder on the iDisk.

> Whenever I try it I get this error:
> The operation cannot be completed because you do not have sufficient 
> privielges for some of the items.

The Backup folder is reserved for the Backup application available to 
.mac members.  You should be able to drag & drop to the Documents, 
Movies, Music, Pictures, Public folders.  If you still can't I would 
eject the iDisk, logout and login, and try again. You might be able to 
fix it by ejecting the iDisk, and relaunching the finder, rather than 
logging out.
