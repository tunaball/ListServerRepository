Then, something's amiss. I don't use OnyX, so can't respond to the problem
you're seeing.

SysPrefs->Sharing stores the computer's name and you can change it.
SysPrefs->Accounts->Password stores the user's name (which you can change)
and the user's short name (which can be changed, but is a difficult
procedure: see <http://www.macupdate.com/info.php/id/16620> for details).

The home folder gets its name from the user's short name. From what you
provided, you selected the user's short name to be the same as the
computer's name. That may be what's causing your problem. You might want to
change some of those names and see what happens. This may be quicker than
creating a new user and moving everything from the old account to the new
one.
