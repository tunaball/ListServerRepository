I don't know about the warning but you can move your OS9 Netscape
bookmarks/mail etc...to the OSX Netscape/Mozilla or Firefox for that matter.
All you have to do is copy the folder inside your Mozilla default profile
folder on OS9 to same one in OSX the default profile folder doesn't have
your name on it but just an alphanumeric series of characters like the one
of the end of this example. You'll have to create a default profile in the
OSX Netscape first then replace that profile folder with the one from OS9
and restart the browser. For what it's worth I would recommend
Mozilla/Firefox over Netscape, it's the same basic browser only far more
developed than Netscape and you should be able to import all your OS9
Netscape stuff there no problem.

Home Folder/Library/Mozilla/Profiles/Default/ rfyiqe7s.slt

As for the hard drive I would recommend putting in the biggest one you can
afford unless you've got another machine you'd like to put it in.
