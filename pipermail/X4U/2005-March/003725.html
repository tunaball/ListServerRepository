Andrew T. Lynch wrote:
>  I want to scan in a two page document and send it as a single 
>fax.  Is
>  there a simple way to do this using the built in fax 
>capabilities?
> When I scan in two pages in image capture, I get two tiff 
>documents.

If you have Acrobat you can create a single pdf by importing 
both tiff images. You can then send the pdf as a single fax. Or, 
perhaps you can import the two tiff pages into a Word doc and 
send it as a fax.
