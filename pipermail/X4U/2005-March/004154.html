On Mar 22, 2005, at 10:10 AM, Fabian Fang wrote:

> On Mar 22, 2005, at 8:30 AM, Robert A. Virzi wrote:
>
>> Any way to make it go flush right?  Or for that matter, pushing it  
>> to the top or bottom of a side-mounted dock?  It has always  
>> annoyed me that the default is centered, when this is really not a  
>> great place on my cinema display.  If onyX does this, I'm there!
>
> OnyX does this easily.

-------------------

> defaults write com.apple.dock pinning start

The original defaults write command posted earlier does this. Use  
'start' if you want the dock to pin to the left or 'end' to pin to  
the right. If your dock is on the side, start to pin at the top, and  
end to pin at the bottom.
