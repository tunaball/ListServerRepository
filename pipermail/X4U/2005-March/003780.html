> TextEdit, tired Word(showed 4 characters), tried Resorcerer (crashed)),
> and maybe it is unnecessary?  Everything seems to be working fine,

It's a standard Console log file stored for historical purposes; there are
up to ten of them. The system rotates log files everytime it runs
maintenance tasks. Launch the Console app in Utilities, select File->Open
Quickly->/Library/Logs/Console/brian->console.log.1-that will open it up.
You can definitely throw all of them, except the one entitled console.log,
away.
