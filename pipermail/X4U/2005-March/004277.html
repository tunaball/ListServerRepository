> It's on each users computer. I'd like to set it up so that one 
> computer creates green folders by default, another creates orange 
> folders by default, and so on.

The only way I can think of to do this is to have a background 
AppleScript or cron process check the Users folder every so often and 
set the colour of the folders. Better still, check the Desktop (and/or 
wherever the designers tend to stick the new folders) and just change 
the new folders in these places.

If you want some help with the AppleScript, contact me directly.
