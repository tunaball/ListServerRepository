On Mar 9, 2005, at 12:22 PM, Sean Collins wrote:

> Some of the ethernet capable Laserwriters used that bizarre Apple 
> mini-ethernet connector, call AAUI.  This requires a dongle adapter 
> to convert it to a standard RJ-45 plug.

If memory serves this was not an Apple only solution, Sun and probably 
others did something similar for a while. It made sense at the time 
because NICs and the convertors for the various Ethernet cable types in 
use at the time were still quite expensive. This was just before the 
world settled on 10/100BaseT cables.
