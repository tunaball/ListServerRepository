Pagemaker PC has a Publisher converter.  If you know someone with Pagemaker
PC, you could have them convert it and then you could open the Pagemaker
file on the Mac. Of course, it's probably going to cause the file to reflow
and look nothing like it did in Publisher.

There is no way to open Publisher directly on a Mac.  If you want to support
that file type, you need to buy a PC and Publisher.  It is not an expensive
program.
