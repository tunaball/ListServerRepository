is much clearer to me now.

Roughly:
router = device that allows mult. computers to access same DSL 
connection. DSL routers usually come with NAT.
(ethernet) hub = device that allows mult. ethernet-connected devices 
to access network assets connected to the same hub; kind of like a 
USB hub
switch = "smart" ethernet hub that does faster connections between 
devices hooked into it

I'll get a fast, cheap switch (as recommended here), connect it to 
the MIH 120 and leave the MIH120 in place. My MIH 120 is a 
router/hub/switch but it's a slow switch. So I'll just use it for a 
router.

about right?
