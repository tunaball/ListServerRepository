Some Linux apps are x86 specific.  The main app I use Linux for is largely
written in x86 assembly language.  Not something you can easily port to PPC
Linux.  

Overall, software written for Linux tend to work best on x86 Linux rather
than something like PPC/MIPS/Alpha Linux, and the often can't easily be
ported to another Unix varient.  Which isn't to say that there isn't a*lot*
of software on Linux that can be, especially the really popular applications
