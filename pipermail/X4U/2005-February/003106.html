I don't understand your reluctance to install 10.3.7, but, that said,
create another admin user account, log into it, and see if that fixes your
problem. If so, then something's amiss in your old user account. Slowly
copy the preference files from the old to the new until you identify the
culprit-or reset things the way you want them in the new account and
eventually delete the old one. If not, then run the 10.3.5 Combo updater
and report back.
