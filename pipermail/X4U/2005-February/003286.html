We have a number of removable FireWire drives that shuttle between 
three workstations. The workstations are visible on our network and, 
with the appropriate username and password, they are easily mounted via 
the Network icon in a system window.

When a user logs in for the first time on a workstation, the attached 
external drives are also visible and mountable over the network by 
logging into the workstation and choosing the drive. Once the initial 
user logs out and A) logs back in, or B) another user logs in, the 
external drives are no longer visible to others on the network as they 
were before.

Why is this? What is the workaround?

Is it possible to have external drives mountable over the network if no 
one is logged into a workstation?

Personal File Sharing is turned on in the preference pane.
AppleTalk is checked as "Active" in the Ethernet portion of the Network 
pane.
