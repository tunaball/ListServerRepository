After having file writing issues to two different FW external HDs while 
iSight was plugged in as well, I make it a practice to unplug the 
iSight whenever I need to run an external FW HD. If I needed "always 
on" external HDs, then I would look into a powered FW hub that the HDs 
could be plugged into. Don't know if that would solve it, but it might.
