Hmm, working fine over here.  I just happened to download it last night 
and also found this great hint on www.macosxhints.com to transfer all 
my Safari bookmarks (and folders!) over.

Open terminal and type:

defaults write com.apple.Safari IncludeDebugMenu 1

(changing 1 to 0 disables the hidden menu)

Go to the Debug window and export Safari bookmarks to a file.

Open Firefox.

Bookmarks > Manage Bookmarks > File > Import

Done!
