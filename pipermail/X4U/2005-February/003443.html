You could always use the command line:

sudo asr -source / -target /Volumes/nameofcomputer

I do it all the time. I make sure I'm booted from the source disk and 
then type the above command, replacing nameofcomputer with the actual 
name of the hard drive you are copying to.

Or there is another utility called SuperDuper!
