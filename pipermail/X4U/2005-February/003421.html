OK, what it is saying is that something bad 
happened _outside_ of the java environment as 
such, I believe.

With this kind of error, I'd suspect that the 
Java app is calling a function in a compiled 
library (what the windows people know as a dll) 
and not finding it.
So either you need to add the library to your 
path, or you have a version-incompatibility of 
some kind.

Maybe this is enough for you to start making good 
guesses; if not, you really need to contact the 
developer's tech. support. They should be able to 
help based on the error message above.
