Ed, one more time, I suggest you reinstall SuperDuper. It's actually 
a wonderful program and I have no idea why you had such a problem 
with it. But anyway, download it again. 
<http://www.versiontracker.com/dyn/moreinfo/macosx/22126>
and activate it, and then delete any schedules you might find in it 
(left over from old support files you never found) by clicking the 
Schedules button and then highlighting and deleting any  listed 
schedules.

After that, I'm sure you can uninstall the program if you still wish.

In my opinion, though, it is a very useful and nearly elegant program 
that does exactly what it advertises, and your time might be better 
spent reading the user manual. It is a superb program for what it is 
designed to do, and it's possible that your troubles with it stemmed 
from not reading the basics of how to use it, which would be too bad, 
really.
