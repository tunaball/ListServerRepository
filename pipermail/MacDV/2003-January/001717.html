The "Ken Buns effect" has been discussed several times on this list, 
but maybe I don't completely comprehend what the effect is.
  As I understand it, the "effect" is to be able to "animate" a still 
photo, with the ability to pan across the photo and to  zoom in or out.

There are several programs which can do this:

Zoom and Pan from Virtex software
Slick motion from GeeThree software. (This has been pulled from their 
web site......????)
Photo to Movie from lq graphics
Still life from granted software

I like Still Life because I can dump in lots of photos, it 
automatically does cross fade transitions like iphoto,
and I can easily do "the KB effect" on whichever photos I chose.
The program will output to DV or Quicktime.

It seems that people on this list have had good results  with all 4.

This effect is apparently included in iMovie 3.
