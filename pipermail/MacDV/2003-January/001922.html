I've been using a Canon ZR-40 since last Spring, and haven't had any 
problems with it (other than the fact that I can't capture to a 
FireWire hard disk -- I have to capture to my internal).

One little design quirk is that  it has no Record button for 
recording in VCR mode -- you have to use the remote to do that (or a 
command via FireWire, like iMovie sends when it exports to the 
camcorder).

I've used its analog-to-digital pass-through mode to capture  old 8mm 
video from an old analog camcorder, and I've used its 
digital-to-analog ability to simultaneously export to the camcorder 
and several daisy-chained VHS decks.
