I use both Plus Optimizer which comes with Disk Warrior and also 
TechTools Pro (3.0.8)  I like both.  I think DW's Plus Optimizer is 
more gentle and doesn't move files if it can't recognize them.  It 
then gives you a list of these files and asks you to report back so 
that they can update the software accordingly.

I'm using  OS 9.2.2.  But I have read on another list that to 
defragment your start-up disk, as you have to startup from the DW 
disk anyway it still works with OS X.  If you are optimizing other 
partitions on your HD then you can use the installed version without 
trouble.  I don't have OSX yet so I am only reporting the discussion 
from another list.

Anyway I won't touch Norton software for this or other purposes as I 
have had too many nightmares with it.
