If you regularly have all of the mentioned apps opened at the same time, 
2GB of RAM will help prevent the system from thrashing the hard drive 
when it has to do a virtual memory swap.  So in that sense it will make 
going between apps faster.  But encoding video, processing photos, etc. 
probably won't speed up much.

Hope that helps?

- Mark
