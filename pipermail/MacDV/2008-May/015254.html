If you use the Book templates, you can add text to a photo.
Then, using the Print to PDF, you can re-import the book pages with  
text back into iPhoto.
Use these re-imported pictures for your slideshow.

Start by creating an album and putting all the pictures you want to  
use in it.
Then, choose Book and pick one of the templates. You can alter the  
font, size, and page layouts (one photo per page, two, different  
backgrounds, etc)
When you have all the photos laid out with text, choose Print, then  
using the dropdown, PDF to iPhoto.

Make sure you see the window to Save, which sometimes gets behind the  
main iPhoto window.
