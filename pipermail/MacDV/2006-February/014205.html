On 1/18/06 5:54 AM, "Aaron" <macuser at aarons.fastmail.fm> wrote:


> What is the exact name and location of the component when installed?

/System/Library/QuickTime/QuickTimeMPEG2.component

Lot's of programs let you access and manipulate mpeg2. I like Mpeg
Streamclip and ffmpegX. The current version of the playback component works
with QuickTime 6.0.4 and later, I think. I use it sometimes, but I don't
like the fact that if the mpeg file contains no multiplexed audio track, the
playback component takes a really long time to display the window for the
video track.
