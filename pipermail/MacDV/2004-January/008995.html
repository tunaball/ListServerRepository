On Tuesday, January 13, 2004, at 06:34 AM, Gordon B. Alley wrote:

> It makes no difference which FireWire port you connect your camcorder 
> to. You select the target disk when you create the new project in 
> iMovie. The following discussion is referring to iMovie 3.0.3, but 
> earlier versions behave similarly in this regard.
>
> When you click the Create Project button, or select New Project... 
> from the file menu, a file dialog window appears. Mine defaults to my 
> Documents folder, which resides on my internal hard disk.
>
> If you would like to create the project folder (which will contain the 
> captured clips) in a different folder or on a different disk, you must 
> navigate to that disk in this dialog before you click its Save button. 
> If the disk browser doesn't appear in the dialog, you can click on the 
> disclosure triangle button beside the Save As text box.
>
