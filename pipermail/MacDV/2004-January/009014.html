The file you have is a Microsoft Word document and if it has a picture, it is embedded in the file. If you know someone with Microsoft Word, here's how to extract images from the file: Go under the File menu of Word to "Save as HTML". Not only will any text in the file will save in HTML format, but any pictures will save as Jpegs.
