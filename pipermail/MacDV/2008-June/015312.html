Use Share>Export Using QuickTime>Movie to QuickTime Movie - under  
Options
choose Video.
click Settings. Choose>Compression Type: DV/DVCPro - NTSC
Compressor Quality: Chooose: Best. Click OK
next, Sound.
Make sure it sez, Sample Rate: 48.000 khz
Sample size: 16 bit
Channels: Stereo (LR)
if not, click Settings and change

Click OK

in the Save dialog box, give it a name and a destination (Movies works  
well)
