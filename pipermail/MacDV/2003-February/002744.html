On Friday, February 7, 2003, at 12:20  PM, Randy Wilson wrote:

> 1. scan the photos and save them as high quality JPEG's, perhaps
> putting them all in an iPhoto archive;

Actually, if you're interested in saving the digital images and 
archiving them for later use, the PNG format is better than jpeg.  The 
jpeg format is inherently lossy, PNG is not.  And you still get a 
fairly compact file using PNG.  Use jpeg for posting to web pages and 
other uses where file size is more critical.  Whether iPhoto can handle 
the PNG files is an open question, since I have yet to fool with the 
program...  (I'm still using the serial cable on the 6100/G3 to 
download stills from the Olympus D-600L :-)  If not, Graphic Converter 
can certainly handle batch conversions for the times you need to use 
the archives as jpegs.
