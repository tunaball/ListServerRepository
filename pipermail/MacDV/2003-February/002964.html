It's pretty easy to move the files from the ReplayTV to the Mac as long 
as you have the Replay connected via Ethernet. The software has a 
command line interface, but if you have ever used UNIX or DOS, it 
shouldn't be too tough to work with.
This site has some discussions related to Replay and Mac: 
http://www.designwork.net/cgi-bin/discus/discus.cgi

And you can find the MacReplayTV software for copying the files here:
http://homepage.mac.com/rnc/

There is also software which allows you to copy digital photos to the 
Replay unit, something which is not offered by SonicBlue (PC only).

Once you move the files from the Replay to your Mac, you can play them 
back fine using the MPEG plug in for Quicktime. You can also save them 
in MPEG4 format for CD playback. What is more difficult is demuxing the 
files and making them ready to put on DVD. I have not had success with 
that process yet.
