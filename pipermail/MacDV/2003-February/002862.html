Cutting and pasting a clip has no effect as to were it is located on 
the hard drive.  The file always stays intact that is why a cut and 
paste happens so fast and an undo is so fast.  All you do by cutting, 
pasting or moving clips is tell the computer what part of the file to 
play, were to stop and then move the head to the next point ect..  Most

modern hard drives are so fast that it can be played in real time and 
appear as a contiguous  file rather than a bit from here and piece from

there which it really is.
