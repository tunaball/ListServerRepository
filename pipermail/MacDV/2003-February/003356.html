Little Snitch is, for all intents and purposes, an ip firewall.  But, 
it makes it very easy to set up the rules.  It basically intercepts any 
incoming or outgoing ip traffic and asks you what you want to do with 
it.  You can allow or deny on a one-time basis, or set up rules to 
always allow or deny the traffic, all based on the application 
requesting/sending the data.  It makes it very easy to turn off the 
stupid "phone home" functions of some of the newer programs being 
released.

http://www.obdev.at/products/littlesnitch/index.html
