A while back someone posted this helpful list of "what FCE can't do" and I
saved it.  I don't know if it's accurate but I think it was from a pro list,
so here it is again FYI... (apologies to the original poster).

No Media Manager
No EDL export (but you can open an Express project in FCP3)
No 3rd party Hardware support
No other codecs supported.
Timecode on captured clips will display as 00:00;00 (though the real TC will
be imbedded into the clip, you just will not be able to access it)
No Keyframe on Filters
Can't keyframe in the Motion tab (must use Keyframe in the Canvas)
Doesn't support 3rd AE party plug-ins
No logging, you can Capture Now, Capture Clip and something called Capture
Project
No RS422, only firewire
No offline RT
No 3 way color correction
No Cinema Tools
No audio OMF export
No edit to tape
No batch export
Doesn't have FXScript
Only 32 levels of undo
