>Hi: Run a program called Activity Monitor.
>
>Applications > Utilities > Activity Monitor
>
>Type Command - 1 to show the activity window.
>
>Choose the process: iTunesHelper and Stop the process.
>
>Now you should be able to install iDVD with a custom install.
>
>The installer mistakes iTunes Helper for iTunes.
>
