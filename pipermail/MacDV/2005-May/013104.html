On May 18, 2005, at 10:30 AM, Seilo Karl wrote:

> How exactly does one create a disk image.  Can you share the steps 
> involved, please?
>
>

Create your iDVD project.
  Select  :save as disc image" from the file menu.
  A disc image will be created (put it on the desktop).
Double click the disc image any time you want to use it.
drop the resulting folder in the Toast window or onto a blank DVD to 
burn.
  When you're done eject the folder by trashing it.
This does save massive space compared to keeping a project online and 
time compared to copying a disc.
