Although it's not really a waveform editor, you *can* apply filters and 
effects to the audio tracks, so it *should* be able to do what you want. 
  Actually, the compressor in Soundtrack is really nice (borrowed from 
Logic), as is the EQ -- so it should be able to clean up existing 
recordings.  What it doesn't do is allow cut/copy/paste of the actual 
audio waveform -- well, you probably *could* do it somehow, but it 
wouldn't be easy (since that's not what the app is really designed to do).

If you want a good basic wafeform editor, you should check out 
SoundStudio (used to come pre-installed on "pro" Macs, but don't know if 
it still does).  http://www.felttip.com/products/soundstudio/

Anyway, if you want to buy it, check the going prices on Ebay first to 
make a fair offer.  ;-)
