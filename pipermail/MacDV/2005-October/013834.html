Hiya,

If you ever want to repair file permissions I suggest you leave the 
receipts alone.

Disk Utility uses this folder to determine what permissions should be set 
for what application.

This folder should be quite small in size any way. If you're that 
desperate for space then I suggest you get another drive.

Other things you can do to alleviate space is to run Monolingual (check 
versiontracker.com) to remove unwanted languages. you can also remove 
unwanted Printer files (esp. PPDs).

Oh, the /Library/Packages folder holds any software you have downloaded 
via Apple's Software updates. While it is good to have these for future 
use, if needed, once they are installed they can be removed. This folder 
could be large in size.
