On Sunday, July 27, 2003, at 07:00 PM, Gary Fujihara wrote:

> Working on soundtrack for a FCP4 movie.  MP3s are too compressed and 
> lossy when rendered.  Can anyone suggest a good CD ripper for the Mac?

You can use iTunes, or even the Finder.

For iTunes, just set up your preferences for either aiff or wav (I 
normally use aiff).

For the Finder, insert disk, select track (they show up as aiff files) 
and copy them over like you would any other file -you don't get any 
metadata doing it this way though.
