I am using a Canopus analog converter box to bring in
VHS family movies into FCE to edit. Some of these VHS
tapes have live music, others have soundtracks.

When I capture them, I get drops or sound gaps --
always in the same places. On some tapes there are
never any drops. There are never any dropped video
frames. The sound is fine when I play the tapes on the
VCR.

So far I've:

1.  Checked to make certain my Canopus box is set to
48hz.
2.  Checked my Easy Set Up to make certain it is set
to DV Converter - 48hz.
3.  Reduced the number of Audio Tracks down to 4.
4. Set Audio Playback to High.
5.Added an Audio Effect, Rendered and then removed it,
and re-rendered it.
6.  Added an Audio Effect, rendered etc.
7. Did the same by adjusting the db level one or two
points and rendering.
8. Played with every Audio Effect (this was actually
quite fun).
9. Restarted the computer.
10. Switched the audio cables (checked the
connections)
11. tried exporting back to tape -- the gaps now
appear on the copy.

This is a brand new computer and my first capture, so
I do not think the hard drive has fragmented yet.  I
am running FCE 1.01 and using OS X 10.2.6

Any other suggestions before I go out and buy an
external hard drive?  If I can save myself another
$200, I'd be grateful.

Would also love to understand  why the drops happen in
the same spot each time, no mattter how many times I
capture.
