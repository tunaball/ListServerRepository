If everybody involved has Macs running OS X and broadband, then I'd 
HIGHLY recommend iChatAV and iSight. Absolutely incredible.

iChatAV works with other FW cameras as well, but at present it's Mac 
(OS X) only.

If you have a mixed set of participants (iow, some of them are running 
PCs), I'd still STRONGLY recommend FW webcams or FW videocams over 
anything else (and broadband!). You can try using Squidcam on the Macs. 
PCs usually have some crappo vidconf software either built-in or comes 
with the camera.
