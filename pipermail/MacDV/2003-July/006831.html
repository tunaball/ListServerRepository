I apologize for posting a Final Cut Pro answer without stating that.

I haven't done any work using iMovie and an external HD, so it's quite
likely that you don't have any trouble using two ports at the same time, but
working in Final Cut, you must use a daisy chain. The application is not
able to send information up and down both ports simultaneously. The
communication between the software and the hardware is established when you
launch the application. If you change the FW video device (converter box,
camera, or deck) you should do it with the application closed, then connect
the new device, then relaunch the app. Some people just launch log and
capture or Edit to tape, but that's not a reliable way to establish the
communication with the video device.
