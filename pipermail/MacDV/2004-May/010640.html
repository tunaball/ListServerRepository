I've had the same problem. If you are using Panther, the file
IOSCSIParallelFamily.kext is the problem (In my case). My Agfa Duoscan SCSI
scanner worked up until 10.2.8. When I upgraded to Panther, it stopped
working. I replaced the Panther IOSCSIParallelFamily.kext file with the one
from 10.2.8 and my computer could see and use my scanner again. I booted
into OS 9 when I changed out the files. I've heard (but don't know if it's
true) that changing out the files logged in as root or while booted from
another OS X disk may not work. Don't know why.
