>> QuickTime 5 should be able to play an AVI as long as it uses a
>> codec that is available both in the AVI format and QuickTime (like the
>> good-ol' Cinepak codec).  If it's a "DivX" AVI, you'll need some QuickTime
>> addition (codec plug-in) to handle it, or a utility to convert it into a
>> QuickTime file with a different codec.
> Thanks,
> How would I make these determinations and  how  would  I get the  codec

If you can communicate with the person that created the file, then you could
find out what codec it uses.  If not, you can still install the codec and/or
try a conversion utility to see if it works.  And there are usually
instructions to insall the QuickTime plug-ins (with the software, on the
developer's web site).

Hope that helps.
