The biggest thing is not to let the interface scare you.
Mostly everything that FCP can do can be done at least 4 different ways.
Menu, Icon,(button) Contextual menu, or direct drag.
There are a number of interface tweaks you can make to simplify the  
look or put more tools on the timeline.
There are books from  bookstores, DVD's and tutorial CD's, and some  
online tutorials ( dvcreators.net )
