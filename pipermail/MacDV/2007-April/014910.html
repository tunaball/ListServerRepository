following electronic message:

>
>I created my movie in iMovie and now I want to burn a DVD that I can
>pop into my DVD player and watch on TV.  No iDVD themes, no funky
>music, just the project I created in iMovie.  Any advice?
>

If you have Toast, just use the DVD option on the Video pane, with menu
set to None and AutoPlay turned on.

Alternatively, use ffmpegX and use one of its DVD Presets to encode the
content amd create a DVD image file -- then burn that with Disk Utility.
