CMD Shift 3 captures the entire screen
CMD Shift 4 captures a portion of the screen.

There's another combo that allows you to capture the active window too (but
I don't remember it right now).

Note, that these captures will create a PDF file (If you are running Jaguar)

A better way to do it is to use SnapZPro.
