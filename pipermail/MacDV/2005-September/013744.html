On Sep 23, 2005, at 3:19 PM, David DelMonte wrote:

> Forgive if this has been discussed before.. but is there any way to  
> capture quicktime video streams

Just read this in Macworld. Good for Tiger. For previous versions of  
the OS, I think the download ends up in a tmp folder while it's  
streaming in Quicktime.

Use Safari to get to the web site. Start the download and wait until  
the Quicktime window opens and starts playing the video. Go the  
Quicktime/Window/Show Movie Info and copy the URL listed under  
Source. Paste that into a Safari window and the streaming video will  
now play within the Safari window. Open up Safari's Activity window  
and Option Double-click on the source file listed in the window. The  
video stream will now download.
