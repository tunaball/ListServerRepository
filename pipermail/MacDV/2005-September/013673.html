Two other possibilities if you don't have third party software. 
1. Use the Panther Upgrade disk to start up with the C-key. Then run disk 
repair and repair permissions from the Disk Utility on the Upgrade CD/DVD.
2. Run FSCK. Start in Single user mode by holding down Command-S while 
starting. You get a white type on black screen. When you get the curser to type, 
type in fsck -fy and hit the return key. It will run for a few minutes and tell 
you if repairs were done or if the drive appears OK. If repairs were done, then 
rerun fsck until it says the drive is OK. Then type reboot and hit return.

You can also just try a restart if you haven't done that or a shut down and 
restart.
