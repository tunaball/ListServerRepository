I do this all the time using Toast, it is simple and works every time.  
In Toast 6 you go to the copy menu insert your original, press the 
record button and select the number of copies you want, after the 
original is copied it will ask for blank disks the number of times 
selected.  I have never created a dud using this method.  If you think 
that you may be asked for more copies shortly save the image and you 
can skip the recording portion of the process next time.
