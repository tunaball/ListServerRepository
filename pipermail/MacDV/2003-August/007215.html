It's mostly the LiveType live fonts and Soundtrack Loops.
If you install the apps without installing the fonts and loops, it's only
about 1 gig. But you won't be able to experience the fun and creativity of
LiveType and Soundtrack.

Btw, you can install the libraries on an external harddrive. For Soundtrack,
just install the library on the external and then import when you need the
loops. 
For LiveType, you must create an alias in Libraries/Application Support.
