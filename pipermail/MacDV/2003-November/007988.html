> Because the standards aren't perfect?  Because of software bugs (as

Also because not all DVD players can handle "burned" vs. "pressed" 
disks. Unfortunately, it even comes down to the media itself sometimes. 
Its possible the brand of DVD media you have won't work with the Dell, 
but if you burned the same movie using a different brand of DVD media 
it might work. If the DVD works in a stand-alone DVD player, I'd tend 
to blame the Dell drive or the combination of the Dell drive and that 
brand of media.

Also, before filing a bug report with Apple, I'd check with Dell to 
make sure their drive can read homemade DVDs.
