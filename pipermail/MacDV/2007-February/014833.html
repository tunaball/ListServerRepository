First a couple of observations/questions.

What is mpeg streamline? Do you mean MPEG StreamClip? If so, MPEG  
StreamClip does not create .wmv files. What did you create using MPEG  
StreamClip?

I downloaded the file you have, and it is an MPEG2 file. Is this the  
original file that you are converting from?

Use MPEG StreamClip to convert that to an AVI, or use Flip4Mac to  
convert to a .WMV. It's possible that FFMPEG will also convert to AVI.

 From reading the other responses, you do not have the luxury of  
expecting your recipients to be able to install QuickTime or anything  
else, for that matter.

For that then, you have to encode to your audience, WMV. All Windows  
machines can play WMV, just like all Macs can play QuickTime files.

Buy the Flip4Mac studio that will allow you to export to WMV and be  
done with it.
