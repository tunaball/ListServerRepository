On 21/04/2010, at 6:18 AM, Derek Roff wrote:

> I don't know if it will work with your MPEG files, but with .dv files, QT 7 Pro will let you merge multiple files into one.  I just tested it, and this worked:
> 
> Open the first file in QT 7 Pro.  Move the playhead to where you want the new video to appear (usually the beginning or the end).  Drag the file icon for the second video file over the current video window, and release the mouse.  The second video will appear in the timeline with the first, in whatever spot you chose.  Choose File>Save As, and give the file a new name, to avoid replacing the original video file with the new combined version.
> 
> Repeat with additional files.  Note that the second file is not opened in QT 7, or in anything else.  The file icon is dragged and dropped, as though you were moving it to another folder in basic file management. I use the column view of the files when I do this, but I suspect any view will work.
> 
> So far as I can tell, both files end up at full quality and no re-encoding.  MPEG files have some special properties, so this may not work with them, but it's worth a try.
