On Apr 18, 2010, at 9:00 PM, Conlon Brett wrote:

> Anyone know what setting can be ticked to keep the 16:9 anamorphic  
> ratio so it plays on the computer as 16:9? I was hoping QuickTime  
> Pro 7 (the only QT Pro I've paid for) would have a tickbox in the  
> properties window but it doesn't. Any other suggestions?

You can use Dumpster.  Change the top left value of the track matrix  
to 64/45 = 1.422222.
This will make it play right on a Mac.
I don't know if all the hardware out there that plays mp4 will read  
the aspect ratio metadata.
