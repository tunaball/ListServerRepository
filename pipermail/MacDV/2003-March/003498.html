My experience is similar to yours it all works.  Even the Ken Burns 
Effect works fine when rendering new pictures in the timeline but 
crashes the program when trying to change a previously rendered effect. 
  Best way around this is delete the picture from the timeline then 
reimport it and everything is honky dory.  Pictures imported after the 
update can be rerendered using the Ken Burns effect with no problem.  
This is a little more work but only for projects started prior to the 
latest update.
