I discovered that, if I use Toast 6 to create a DVD, I can "burn" a 
disk image file instead of a physical DVD (instead of clicking the 
burn button, select "Save as Disc Image..." from the File menu). Then 
I can mount that image to take a look at the result before burning 
the actual DVD and as many copies as I need.
