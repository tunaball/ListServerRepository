When using iMovie clips in FCP I ususally just import the whole media 
folder inside the project folder. The project itself is of no use and 
you can only import the actual media.
When you drop it into the timeline in FCP it will show a need for 
rendering.
It is a VERY small audio render and nothing to fret about.
Works for me.
