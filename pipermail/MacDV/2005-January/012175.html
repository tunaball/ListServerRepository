>> IIRC, I believe that the only way to fix this, is to open the clip in
>> quicktime pro, extract the audio, upsample the audio and then replace the 12
>> bit audio with the upsample 16 bit audio.
>> 
>How does one "upsample" and what program do you do this with?

In my last message I mentioned QuickTime Pro, so I will use that as an example.

When the audio file is open in QuickTime Pro, choose Export from the File menu.

Then choose Sound to AIFF from the Export dropdown, then click the Options button.

In that dialog box you can choose what Rate - in kHz - and what Size  - in bits - that you want.

If the original sound file was in 12 bit, choosing 16 bit would upsample the audio.
