On a Mac, right?

Create the artwork (in Photoshop, etc.) then copy it.

Select the file you want to put a custom icon on, and "Get Info" 
(Apple-I) on it.

A little 'Info' window will appear, upper left hand corner of 
which is the current icon.

Select that current icon by clicking it once, do "Paste" 
(Apple-V), and you're done.
