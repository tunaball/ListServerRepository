Two things you might consider:

- iMovie 2.1.1 can't export to QuickTime if Roxio Toast Titanium 
5.0.1-5.1.1's Toast Video CD Export 1.0 is in iMovie's 
Resources:Plugins: folder. Toast Titanium 5.1.2 installs Toast Video 
CD Export 1.4 which fixes this. With new Toast versions this is not 
an issue.

- QuickTime export may fail if the ~/Movies folder is missing. 
Re-create that folder if that is the case.

BTW, I skipped iMovie 3 because it was so buggy.
