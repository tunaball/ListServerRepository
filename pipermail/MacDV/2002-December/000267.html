Outlook OS9 only exports as a text file.  I believe export is under the 
file menu.  Jaguar address book does not import text file addresses.

I imported the Outlook text file into Netscape, (import addresses under 
file menu, Internet Explorer doesn't have this capacity).

Netscape  exports addresses as a .ldif file,  which Jaguar address book 
can import.  Worked fine.
