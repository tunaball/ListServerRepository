Hmmm... OK.  I thought the issue was that you couldn't play the movie 
by itself on the PC, so you tried to put it in a PDF as kind of a 
"container".  So why does it have to be in a PDF file?  Is it for a 
presentation?  My guess as to why it won't play in a PDF on a PC is 
that QuickTime on the Mac is a system-level extension (allowing 
playback anywhere), where on the PC, it is not integrated into the OS 
the same way.  I guess the same could be true for the Windows Media 
format -- it's part of the OS in Windows, but not on the Mac.  Maybe a 
Windows Media file would work in a PDF on Windows?
