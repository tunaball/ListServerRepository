Yeah, I've noticed that with a lot of the newer options becoming 
available to create these kinds of discs on Mac OS X, the documentation 
is sparse or missing altogether.  The problem (of sorts) is that these 
apps are created (or ported) by programmers, not user interface and 
documentation specialists.  I guess they figure if they could figure it 
out, then you can too...  not the best assumption to make for most 
users. (myself included!)

On the bright side, more compression apps are coming out that have some 
sort of GUI to ease the MPEG creation/transcoding process.  I don't 
have any specific links for any foolproof software just yet, but when I 
do, I'll be sure to share the info with the MacDV list.  ;-)

- Mark
