I've been playing with creating SVCDs on the Mac (Mac OS X anyway) for 
a few months, and it's coming along (slowly but surely).  I have 
created a couple of SVCDs through very tedious processes, but the tools 
are starting to mature so that it's gradually becoming easier.  
Granted, there isn't a really intuitive point-and-click/drag-and-drop 
interface to do this, but most technically savvy users will be able to 
follow along.

Here's a link to "RNC’s Guide to SVCD on Mac":

http://homepage.mac.com/rnc/

More specifically, how to create an SVCD from a DV QuickTime file with 
MediaPipe and MissingMpegTools:

http://homepage.mac.com/rnc/svcdXc.html

Hope at least some of you find this info helpful.
