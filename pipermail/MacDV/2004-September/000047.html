On Saturday, September 18, 2004, at 07:58  PM, lloyds wrote:
>
> 2)  In exporting a video from FCE, using the path for QuickTime
> conversion, when it asks for "size," what kind of numbers should I
> enter? (Inches? Inches on the screen?  Pixels?)

If you are making a DVD, you shouldn't export using the path for 
QuickTime conversion. You should export as a "Final Cut Express Movie" 
and do not make it self contained. Then import this FCE Movie into your 
iDVD project.

In cases where you DO want to make QuickTime movies, you can use the 
"size" parameter to control the final file size. Most of the high 
resolution QuickTime presets, such as "DSL/Cable High," use a player 
screen size of 320 X 240 pixels. If you keep everything the same except 
decreasing the player screen size to "Custom 240 X 180 pixels" you'll 
have a movie with a much smaller file size, but still a pretty 
watchable screen size.
