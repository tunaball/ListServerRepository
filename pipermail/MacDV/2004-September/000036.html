This is a known bug with v. 2.0

First of all, use the Go To Next Gap (Shift G) to find the gaps in your
project and delete them. This will only work if the gap is in both the video
and the audio.

2nd, upgrade to the latest version of FCE, 2.0.3 so you don't have this
problem anymore.
