> The only way I know to make this happen (if you actually want it to) is
> to assign scripts to the 'Hang up phone' event.
> 
> In the Synergy tab in System Pref's click on 'Phone Events' and then
> (beside the words "Also, run the following when :") change the
> drop-down menu so it reads "Hanging Up" - any scripts or actions you
> now see in the 'Selected Scripts' pane will run automatically when you
> hang up.
> 
> If this pane has action scripts in it (specifically related to iTunes)
> you might want to remove them - if it's empty.....then I have no idea
> what's wrong!  :-)
> 
Wow thanks it only took me like 2 months to figure that out, I spent a lot
of time staring blankly at the prefs panel.
