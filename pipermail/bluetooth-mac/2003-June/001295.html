Hi Bill

There are several Bluetooth adaptors for parallel-enabled printers. 
However, as yet Mac OS X does not support printing over Bluetooth. The 
only way to get it to work is to send each file to the printer via the 
Bluetooth File Exchange app - cumbersome to say the least. Apple will, 
without doubt, add this ability in a forthcoming release of its 
implementation of Bluetooth... we have to remain patient.

Check www.expansys,com for some adaptors - if you're still interested.

Good luck!

Steve
