> From palm.com:
> Here are the steps to setup up a Bluetooth HotSync operation:
> 
> Creating a Connection to the desktop system:
> Tap on the Prefs icon in Applications Launcher
> Tap on Communication
> Tap on Connection
> Tap the New onscreen button
> Enter a name for the connection in the field labeled Name:
> Tap the arrow next to Connect To:
> Select PC from the drop down list
> Tap the arrow next to via: and select Bluetooth
> Tap the box labeled Tap to Find next to Device:
> Select the name of the PC from the displayed list
> When asked if you want to add the PC to the Trusted Device list, select no
> On the Edit Connection screen tap the OK button
> Tap the Done button to get back to the main Preferences screen
> A connection to the PC has now been established
> 
> Setting up the Mac Desktop
> 
> Open the HotSync Manager software
> In the HotSync Controls tab, verify that HotSync enabled is selected
> In the Connection Settings tab, click on the checkbox next to
> bluetoŠync-port 
> Exit out of HotSync Software Setup
> 
> Setting up the handheld for HotSync
> 
> From the Applications Launcher, tap on the HotSync icon
> Tap on the Menu icon in the Graffiti area
> If the Option menu did not drop down, tap on Options
> Select LANSync PrefsŠ
> Tap on LANSync 
> Tap OK 
> Tap on Local above the HotSync icon
> Tap on the box below the HotSync icon and select the BT Connection that was
> created in the above steps
> Tap the HotSync icon
