If you can't wait for possible Maccompatibility for pcmcia-bluetooth: it is
possible to build in USB-bluetooth (you need some technical skills). See:

http://www.macitynet.it/immagini/servizispeciali/bluetoothinsidee/
(source: appleinsider.com)

They built an USB bluetooth adapter inside an Tibook, including cables. You
will loss one external USB port and dependent on the way you build it, will
only get a range from 60 cm - 2 metres due to the titanium.
