If you're getting a kernel panic on a different user and you didn't 
copy anything over from the first user's Preferences, my guess is that 
there's a problem with you OS X installation since you've eliminated 
everything but the core.  Unfortunately, Palm Desktop installs some 
parts to the System and I'm not sure that the Uninstall option actually 
gets rid of everything.  Of course, a remote possibility is that you 
have a corrupt User Data file.
