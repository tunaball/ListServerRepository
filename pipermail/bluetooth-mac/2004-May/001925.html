Has anyone been successful setting one of these up?  It seems as if it 
should work since both OS X 10.3.x and the Belkin adapter support the 
necessary protocols including Hardcopy Cable Replacement Protocol 
(HCRP).  While the Bluetooth Setup Assistant can see the adapter, it 
won't finish the installation since it can't find a driver (Belkin 
doesn't supply one).

Belkin showed this product at last January's Macworld Expo in San 
Francisco and promised that it would work with the Mac.  Unfortunately, 
now that it's shipping, they're NOT supporting the Mac.
