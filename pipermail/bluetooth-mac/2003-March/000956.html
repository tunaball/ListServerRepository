Hi,

Bluetooth file exchange works fine both ways, and I've successfully 
used it as a modem over both GSM and GPRS. iSync doesn't currently work 
with the P800 but the web is covered with rumours that it's under 
development. It uses SyncML so it shouldn't be too hard??

As for how do I find the phone overall - I'm biased, but I think it's 
great. I've got Doom, SSH, VNC, ICQ, Lemmings and Opera on there. What 
more do you need from a phone? ;-)
