KOZAK Imre Oliver <ki023 at hszk.bme.hu> wrote:
> There is a new utility in Jaguar 10.2.4 update in the
> Applications/Utilities folder, called Bluetooth Serial Utility.
>
> It can be used to set up Bluetooth services for paiered devices
> similarly to Windows Bluetooth drivers. For exaple you can define a
> Serial port for the T68, an OBEX Object Push port for the Tungsten
> synchronization and such.

Although the Bluetooth Serial Utility *can* be used independently, it's
really made to be used via the "Setup New Device" button in Bluetooth System
Preferences. When you do it this way, the newly set-up "port" is available
immediately.
