For an interface, I've found USB to be fine for MIDI.  I use a couple of
different cheap controllers that have direct USB connections (Edirol
PCR-30, MAudio Oxygen-8) and also the Yamaha UX-16 cable which
is a cheap USB-to-MIDI in/out cable.

If you want the same interface to handle both audio and MIDI there
are some USB solutions but I think they tend to have latency issues
for audio - not sure about the newer ones (anyone tried Mackie Spike?
is it shipping yet?)  There are a few firewire audio options for under
$500, the really nice ones cost more...
