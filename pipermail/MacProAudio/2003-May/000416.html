I helped a guy get his imic working with Cubase SX, I think the trick 
was to go to the Speech Preference Pane>Speech Recognition>Listening, 
and select iMic USB audio.
I've been using SX and it's great!  My criticisms are: The USB dongle 
(Don't forget adding a USB hub for those I Book sessions.)  It freaks 
if you move the video window between monitors on a dual monitor system 
(which you almost need.)  I don't like if you bring up two projects 
that it brings up ALL the windows for both projects (messy!) The only 
other thing I've run into is trying quantitize Midi events to video 
frames (you can work around it, but...)
On my wish list would be Audio Units support (unlikely) a better free 
VST reverb, And I really really wish you could run Reason as a Rewire 
master and Cubase as the slave instead of the other way around. The New 
Reason Reverb is SWEEET.
I'd say buy the software now, Steinberg has always had a pretty liberal 
upgrade policy.
Interface wise,  l love the Emagic EMI 6|2m.
