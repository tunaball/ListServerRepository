Aldo: which Mac OS are you using? OS9 or OSX. In either case, you need 
to select the input device that you are using. If everything is working 
properly with the USB Swissonic, you should see the device when you go 
to the Sound Control panel (in OS9) or the Sound Module in System 
Preferences in OSX. Then select the appropriate input device.

In both cases, you will have the option to check the signal level.

If the USB device doesn't show up, there's probably a problem with the 
drivers. In that case, go to the supplier's web site and see if they 
have Mac OS drivers.

Hope this helps.
