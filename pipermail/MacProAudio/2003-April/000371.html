Hi

Here's how to do it in Peak:

Open the file, select all, then go to the DSP menu and choose Convert 
Sample Rate.

this will allow you to specify a new sample rate without changing the 
pitch or length of the file. It may take some time to process a long 
file.

In Cool Edit on a PC the option is Convert Sample Type which (if I 
remember correctly) is in the Edit menu. It also lets you change the Bit 
Depth and number of channels of the file in the same dialogue. These 
come under different menu options in Peak.

Do note however that DVD's can often be 96kHz
