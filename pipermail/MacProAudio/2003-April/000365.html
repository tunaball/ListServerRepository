Hey, Daavid. I tried the O02 and realized that I was just more accustomed
to working on the screen itself, so downgraded to an mBox. I'll pick up the
questions I can:


>Any inkling when OS X version of LE is coming out?

It should be included. If not, get with Digi. You're entitled to it at no cost.

>
>Does LE work as an editor without the control surface connected?

No, which was one of the deciding factors in downgrading, since I wanted to
edit using my Powerbook. With either the 002 or the mBox, you need the
hardware connected, and the mBox was lighter and smaller. One person has
suggested to me using a separate partition for ProTools Free, another that
I keep both DAE files and install the appropriate one in the system folder
when i want to use PT Free on the road. My solution at present is that I
keep LE/mBox on my main production puter, and use Free on the Powerbook.
