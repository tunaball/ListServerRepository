I have made a small partition of 1 GB where I have put the system and
another that contains the rest.
If I was to rebuilt a whole system I will add a 3rd partition containing
Logic's files and Audio files. In the second partition I will put the
applications.

Why ?
- The system does not change very often so the disk does not have to be
re-organize.
- Same for the Application part.
- You do not need to boot on CD to re-organize the Applications partition
and the "Logic" partition
- It's more easy to backup one partition at a time
