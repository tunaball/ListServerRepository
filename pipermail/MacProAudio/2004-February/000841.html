Bond,

Free Windows Media Player for Mac OS X:

http://www.microsoft.com/windows/windowsmedia/software/Macintosh/osx/ 
default.aspx

Safari doesn't support Real player, but you can use Internet Explorer  
on the same computer with Safari if you need to use real player.  Many  
people who use Safari (like me) keep I.E. around because there are  
still a few websites that Safari doesn't work with.

Free Real player for Mac is available here:

https://order.real.com/pt/ 
order.html?ppath=cpromac040702a&type=macbeta&src=010604realhome_1_2_2_1_ 
8_3

Free I.E. for Mac:

http://www.microsoft.com/mac/products/internetexplorer/ 
internetexplorer.aspx?pid=internetexplorer

Scott
