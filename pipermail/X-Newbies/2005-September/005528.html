On Sep 13, 2005, at 8:12 PM, Aldo Johnson wrote:

> I bought a new Mac recently. I've got my old mailboxes in my backup  
> HD. All the inbox.mbox and stuff files from my mail accounts.

The easiest way to do it is using the Migration Assistant in the  
Applications/Utilities folder. Connect the old Mac via firewire in  
target mode, and then run Migration Assistant - it will copy over  
whatever part of your old user account that you want on the new machine.
