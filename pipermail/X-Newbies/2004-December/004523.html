At 9:01 PM -0500 12/31/04, Al Poulin wrote:
>In the user accounts, all except Zoo Tycoon run.  The error message 
>is:  "Zoo Tycoon cannot be played from a locked or write protected 
>volume.  You may want to re-install Zoo Tycoon on an unlocked volume 
>to fix this problem."

Sounds like the app has placed preferences, licensing or other files 
in a location only accessible/writable by the installing user. 
Probably within the user's Library hierarchy. Since anywhere within a 
user's space is usually inaccessible to other users (or at least 
un-writable), it's preventing use of the app from those accounts.

You could try finding the relevant files, placing copies in the same 
relative location in the "Shared" user account hierarchy, then place 
aliases in that relative location each ordinary user's hierarchy. Or 
even put copies in the location for each user (skipping the Shared 
user approach).

Or call the developer/distributor and ask them for a solution.
