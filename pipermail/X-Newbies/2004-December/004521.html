The normal advice is to download the latest driver and other software 
from the scanner vendor's web site.

You did not say what brand scanner you have, but with my Canon CanoScan 
N1240U I did fine with the latest driver and Toolbox available from the 
Canon Web site while running 10.3.5 and now 10.3.7.  I was a bit 
perplexed until I realized that the driver had to be loaded in the 
Plugins folder in the Toolbox folder in Applications.  This was moving 
from OS 9.2.2 directly to Panther.
