In the Bombich Forums is a very helpful thread on how to properly  
prepare your external HD before cloning to it so that it will be  
bootable:
http://forums.bombich.com/viewtopic.php?t=4084

Beyond that, all that you need is SuperDuper, if that is your backup  
program of choice.  (And it is an excellent choice.)

In the event of a misfortune, you can just startup from the backup  
and use it like you would have the original disk.  You can also clone  
the backup disk to a new disk using SuperDuper.
