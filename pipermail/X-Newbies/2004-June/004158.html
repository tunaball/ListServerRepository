Here's another tip: Use Disktracker to catalogue what's on the disk, 
then let Disktracker save it as a textfile, and print that. I don't 
know which makes the better result, BBedit or Disktracker, but 
Disktracker is a very good cataloguing software so I thought I would 
tip you guys about it.

Search on http://www.versiontracker.com/ for info on Disktracker (or 
any other program or function), it's always a good place to look for 
information at.

Disktracker can be tried for free, for a period of time.
