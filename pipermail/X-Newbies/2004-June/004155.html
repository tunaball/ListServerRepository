>How does one print a list of the contents of a folder in system X?
>For example if one has burnt a folder to CD then you want to print out the
>contents to stick on the outer case.


If you have BBEdit then drag and drop the folder from the Finder onto a
BBEdit window. After a while to scan the folder you will have a nice
tabbed listing of all the contents of the folder. Then just print it.
