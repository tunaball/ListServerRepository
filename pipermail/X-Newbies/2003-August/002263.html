We have seen some cases where index files get unusually large, which 
could lead to a slowdown like you're seeing. It's a bit tricky to track 
this down, though. If you quit Mail, then go to the Finder and navigate 
to the Library/Mail folder in your home directory. You should see your 
mailboxes organized roughly like you would see them in the mailboxes 
drawer. Find the mailbox that you're having trouble with and ctrl-click 
on it and choose Show Package Contents. Move aside (or you can delete) 
any files with the word index in the title. The next time you launch 
Mail and open this mailbox, the index will rebuild itself. See if that 
resolves the problem.
