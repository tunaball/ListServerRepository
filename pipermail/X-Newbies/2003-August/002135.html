> Yes, you can move it to your global Application folder

In fact, under OS X it's recommended to keep your applications in the 
/Applications folder. Only applications that should not be available to 
other users should be stored in ~/Applications. Only applications 
stored in the /Applications folder provide services. Storing Safari 
therein should enable the "Open URL in Safari" service.
