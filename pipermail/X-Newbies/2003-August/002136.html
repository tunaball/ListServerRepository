To avoid clutter with Apple's shipping apps, you might consider putting
your personal apps in a subfolder with the global Applications folder,
e.g. "MyApps" (or /Applications/MyApps).

: Only applications that should not be available to 
: other users should be stored in ~/Applications. Only applications 
: stored in the /Applications folder provide services.

This is technical not quite correct.  Services are available from apps
located in /Applications, /Network/Applications, and ~/Applications.
