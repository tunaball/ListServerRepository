The Tally T8006 is a low-end color laser printer that uses Tally 
Compressed Printer Language (CPL).  The T8006 cuts costs by using the 
host computer's CPU and memory to rasterize print images.  The T8006 
doesn't have the hardware that most printers include (including Tally's 
other, more expensive, color laser printers) to translate and rasterize 
traditional printer command languages. 

GIMP-Print doesn't have a driver that will allow you to use a CPL 
printer.  

See:
http://gimp-print.sourceforge.net/p_Supported_Printers.php3

The T8006 essentially requires a proprietary driver for OS X that can 
only be written by the manfacturer of the printer.  I'm afraid that your 
friend is out of luck.
