On Saturday, August 2, 2003, at 11:51 AM, Walter Draeger wrote:

> I am using the Apple Mail email system. I was sent an email I can't 
> delete. Each time I try the system freezes up. <snip> How can I delete 
> that frozen email. Using 10,2 on iMac.

Two things you can try.

First, select the mailbox containing the offending message and select 
Mailbox->Rebuild Mailbox from the menu. That might fix it.

Second, see the little bar that splits the top part of the Mail window 
(the list) from the bottom part (the message)? If you place your mouse 
pointer over it, it will change icons and you can drag it all the way 
to the bottom. That way there is no message display. IIRC this prevents 
the message from being read when you select it from the list. Now try 
to delete it.

Hope that helps.

-Mike
