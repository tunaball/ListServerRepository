> I don't want a bunch of people scouring preferences looking for an 
> option that doesn't exist in preferences.

Ah, the good Samaritan. Too bad the preference does exist. 
Mail>Preferences>Viewing>Display images and embedded objects in HTML 
messages. Turn it off, and the cricket croaks. (Actually, not quite. If 
a message has been viewed once with the option on, the picture is 
cached, so it will be displayed even when the option is turned off. But 
with the option turned off, messages from new senders will not display 
the pic.)
