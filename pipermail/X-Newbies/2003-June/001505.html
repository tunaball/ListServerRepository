wrote:

> How do you rebuild the mailbox when it crashes the program before you 
> can even get to the menu?

Quit Mail. Go to ~/Library/Mail/. You should see a folder named after 
your e-mail account. Inside there should be an item named "Inbox.mbox". 
Make a backup copy of this item somewhere else, then trash it.

Launch Mail; it should work fine, and Mail will create a new blank 
Inbox.

To recover messages from the damaged inbox, open the backup copy with 
Ctrl-Click>Show Package Contents command. Inside there should be a file 
called mbox. Open it with a text editor such as BBEdit. (Note that you 
might be able to find on the net a copy of BBEdit Lite 6.5, which still 
works fine with Jaguar, and which was the last version of an superb 
free text editor.) TextEdit might also work.
