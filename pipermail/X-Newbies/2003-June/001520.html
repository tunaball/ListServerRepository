Hello

I solved similar  backup "problems" by installing automatic backup 
programs compatible with OS 10.2  -Jag

The first Program is Daja Vu (shareware). I have it configured  to auto 
open at 8PM every  day  and mirror my complete Home File into  an 
external FW drive.

The second  Program set is File Synchronization ( shareware) and 
Cronnix ( freeware Timer)) . I have it configured  to auto open and 
backup some selected, valuable  personal files at 10AM every  day into 
the same external FW drive

I am using both just in case one dies on me!

Use Google ( or Sherlock) to locate applicable Web Sites for download 
and take a free trial to see if it suits your needs.
