Yes, I did this before we had DSL at home and it does work.  After it is
completely downloaded, copy the raw file to your Mac.  Then unstuff it using
Stuffit on your Mac.

While you're at it, take this opportunity to learn how to use the built-in
Samba server to transfer files via Ethernet.  You'll impress your friends.
Writing a CD to transfer a file is quite primitive.
