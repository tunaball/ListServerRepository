On 17/2/03 TheMacintoshLady wrote: 

>If .pdf files were made in X, to be opened in AR 5 for X, how does one 
>make them readable on OS 9, in AR 5 for 9?
They should open in 9 unless the 'X' created file has no resource fork. 
Even so, if you leave the .pdf extension on the filename, provided that
your file extension mapping in File Exchange (in OS9) maps it to Acrobat
Rdr then you should be OK.
