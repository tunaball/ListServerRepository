Do you often need to boot into OS 9?

If so, consider at least two partitions - the first w/OS 9, the second
w/OS X. Why? Hold down the "d" key to boot into OS 9. Hold down the "x"
key to boot into OS X. Simple. No Start-up disk BS.

How? "d" instructs the computer to use the first working system folder to
boot from. "x" directs it to find OS X and boot into it.

HTH
