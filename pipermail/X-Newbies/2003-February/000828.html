On Mon, Feb 17, 2003 at 01:45:28PM -0500, TheMacintoshLady wrote:
: 
: If .pdf files were made in X, to be opened in AR 5 for X, how does one 
: make them readable on OS 9, in AR 5 for 9?

It depends on what OS X app created the PDF files.  Some apps (I'm
guessing things like Adobe Acrobat) will set the type & creator codes
("PDF" and "CARO" respectively) that OS 9 looks for.  If that's the
case, then you need do nothing.

If those PDF files do not have the .pdf filename extension, OS 9 will
have no idea that those files are PDF files.  If they do have the .pdf
filename extension but do not have the type & creator codes set, use the
old OS 9 Internet control panel (aka Internet Config) to map ".pdf" to
Adobe Acrobat.
