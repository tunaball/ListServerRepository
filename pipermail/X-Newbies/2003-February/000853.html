> Is it any good and how can I use it with OS 9 machines?

It is excellent! You can use a new Airport Extreme Base station with an
older OS9 running machine that has the original Airport functionality
(Airport Card built in). But the older Airport communicates a lot slower. If
all you have is old OS9 machines, pick up an original Airport system.
