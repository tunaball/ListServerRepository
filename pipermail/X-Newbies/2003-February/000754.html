I use the fax on OS 9 without any problem, so haven't attempted to learn
the fax routine on OS X. But regarding memory, according to the Mac OS X
Missing Manual, OS X doesn't allocate memory to specific programs like
OS 9.  It just takes whatever it needs, on the fly, as it needs it,
including from virtual memory

A further review of the Missing Manual shows no mention of fax. And the The
Little Mac OS X Book by Robin Williams mentions using Fax software for
sending fax's - so apparently X doesn't have it.'s own fax program.  Zippy
