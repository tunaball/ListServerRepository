Wrong.
New machines do not ship with an OS 9 CD. OS 9 is installed as part of 
the 'additonal software' restore process.
See http://docs.info.apple.com/article.html?artnum=42929 for details.

New machines either ship with OS X cds and a set of restore cds, or for 
even newer machines, an install / restore dvd, which has both the 
Jaguar installer and an app to restore the rest.
