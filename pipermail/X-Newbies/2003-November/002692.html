Another way to list files is to drag the folder icon into an open iCab 
window, provides you with a list that can be printed or saved as pdf.  
The folder and file names are live links.

So, and this is really handy, you can navigate to any folder or file on 
your hard drive or mounted drives, or even open many of the files 
themselves.
