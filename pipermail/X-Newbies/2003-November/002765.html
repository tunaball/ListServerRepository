> The pc has software like
> "listmaker" and "HTML directory" which allows you to make lists of 
> items in a
> particular folder or drive. Is there such software/freeware/shareware 
> for the mac/

There are plenty. BBEdit, a powerful text editor (which you should have 
in your tool kit if you do any kind of web work) will do it; the free 
Lite version still works with Jaguar (has anyone tested it with 
Panther?).

There are also dedicated cataloguing utilities (e.g., CDFinder); or, if 
you're of the lean-and-mean persuasion, go the CLI route. Start with 
man ls in Terminal.
