> .xls is an Excel file. Excel will open them, but not much else.

Except for Mariner Calc, AppleWorks, ThinkFree Office, NeoOffice/J,  
icExcel and about half a dozen other programs or office suites, that is.

Gail's problem may be that she's trying to use Stuffit Expander to  
open an .xls file -- that won't work. Appleworks will be able to open  
the file IF she uses the following procedure.

1. Open Appleworks.
2. Go to File -> Open.
3. (this is the part people forget) CHANGE the pop-up button near the  
bottom of the open dialog box from "AppleWorks" to "All Types". NOW  
you will be able to point the dialog box to your .xls file and it  
will open.
