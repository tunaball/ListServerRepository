As has been said, OpenOffice or NeoOffice/J will open these files.

OpenOffice has just hit version 2.0 which should offer better support 
for excel formats and so on.

Not sure when NeoOffice/J will use the same base code, it is currently 
based on OpenOffice 1.1.

NeoOffice/J is far easier to use/set up, and looks nicer, OpenOffice on 
Mac OS X needs Apple's X11 server
package installed, whereas NeoOffice/J uses a mix of Carbon and Java for 
the UI.

Both are a bit memory/cpu intensive I find though.
