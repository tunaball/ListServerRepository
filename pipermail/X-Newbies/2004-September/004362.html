There is an issue with the 10.3.5 upgrade and 3rd party ram. Remove 3rd
party modules, install the update and reinstall the ram.

You can also run the hardware test CD to check for iffy ram. There is also a
3rd party app to check ram at versiontracker.com. I'm sorry but I don't
recall the application's name offhand.
