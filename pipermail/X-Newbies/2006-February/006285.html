What outgoing mailserver are you using on your wife's office computer? 
is this a home office using the same ISP that you are using for the 
computer where you set up info at ...?  If it a other office it may be that 
the ISP is not allowing port 25(the outbound mail port) except to a 
specific set of mailservers.  Have her enter the same outgoing mail 
server as others in the office are using.
