Its not a virus, its a trojan horse. IOW you have to be tricked into  
downloading it, unzipping the file, then running the program in an  
admin account or give it an admin password. There's no way to protect  
a system against an admin user intentionally running a bad program.

In the end, its no different than the other rootkits etc we've been  
hearing about for a long time (the last one was masquerading as a jpg  
file IIRC).

Bottom line is if you don't know where an application came from,  
don't run it.
