Aldo said:

>Is there a V.92 modem driver for the mac?


There is no universal V.92 modem driver that will allow you to use any 
Windows USB V.92 modem with a Macintosh.

I don't know the technical reasons, but you can't purchase a V.92 USB 
modem meant for a Windows computer, tweak the initialization string, and 
have it work with your Macintosh, like you could with earlier modems.  

So for now the only two choices for a Macintosh-compatible external USB 
V.92 modem are the recently released Apple modem, which only works with 
OS X 10.4, and which is a soft-modem (i.e. it performs poorly and drags 
down your Mac too), or the Best Data V.92 USB external fax modem for 
Macintosh.
