On Feb 6, 2006, at 10:54 AM, Chris wrote:

> Hi all,
>
> A permissions/ownership question.  Is ownership by 'system' the  
> same as
> ownership by 'root'.
>
> For example if I want to edit sshd_config I am told that it is  
> owned by
> root and do I wish to unlock it.  If yes, I put in an admin  
> password and
> off we go.  However looking at the ownership and permissions, the file
> is owned by system with group ownership wheel, and I have read only
> access, which seems fair enough.
>
> But are system and root the same in this context?
>
> cheers,
>
> Chris


They are the same. In Terminal, you'll see root. In the finder, and  
system dialog boxes, you'll see system. It's translated for the non- 
unix types.
