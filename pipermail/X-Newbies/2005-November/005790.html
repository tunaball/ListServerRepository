On Wed, Nov 02, 2005 at 07:38:06AM CST, Brian Durant <globetrotterdk at gmail.com> wrote:
: 
: For some reason OS X doesn't provide virtual desktops and multiple
: clipboards like Linux. Is this a BSD thing or an OS X thing? What have
: the lists knowledgeable readers done about this issue? There seem to
: be shareware alternatives for the missing operability's but are any
: worth recommending?

It's not a technology issue.  It's a user experience choice.
Apple decided to keep it simple, with a single desktop and a
single clipboard.  If you do need multiple virtual desktops,
one of the earliest is CodeTek VirtualDesktop:

	<http://www.codetek.com/index.php>

As for multiple clipboards, I can imagine how that would be
useful in a multiple virtual desktop situation.  Try asking
the CodeTek folks to add the feature to the next release.
