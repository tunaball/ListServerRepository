Without Toast 7:

Use ffmpegX to convert the AVI to a DVD image. This will take a while  
-- a long while.
The DVD image can be burned using Disk Utility.

(disclaimer: I've not personally tried this and so don't know for  
sure that it will work.)

With Toast 7:
1. Drop AVI into Toast's DVD window.
2. Wait for conversion (another long wait)
3. DVD burns.
