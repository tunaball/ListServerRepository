I am running 10.3.2 on my Ti Book, 2 eMacs and an iMac.  All can not only 
print to, but also use the utilities to clean heads, print the test page, 
etc., my 740, all without touching Classic.

Epson Printer Utility  V 1.67 is correct but mine is in the 
/Applications/Utilities folder.  Printer setup utility V 3.0 is also 
correct and is also in the /Applications/Utilities folder.  Don't give 
up.  Try moving the printer utility to the utilities folder.

Did you install the additional (Epson) printers when installing Panther?

Good Luck.
