Don't really understand what you're asking.

If you're wondering if you can transfer your mail from OE to a mail 
program that runs in Panther, then the answer is "probably." Mail (the 
default mail program for OS X) can. You might also want to look at 
Entourage, the OS X version of Outlook Express (more like OE's big 
brother).

If you're asking if Safari can be made to "hand off" mailto: links to 
OE running in Classic, I would say no.

If you're asking if you can run OE in Classic under Panther, then yes I 
think you can ... but why?
