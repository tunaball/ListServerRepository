What I have done several times is to set up the new system on the new 
machine as far as establishing your login account (but not internet 
configuration). Then put this machine in firewire target mode and use 
carbon copy cloner to copy your old user folder to the new system. 
Then go to Library on the old machine and use the finder to copy 
preferences, application support items and other specific items. You 
will still have to reserialize a few applications, but most will work 
fine.
