Newbies should also not be afraid of using Help and searching the Apple 
Knowledge Base. For instance, searching Mail Help for "offline" or 
"can't receive email" retrieves information very much to the point. The 
same information is available in the Apple KB articles ID 151509 and, 
especially for troubleshooting, ID 151535.

The OS X Help is not the best thing since sliced bread, but, 
nevertheless, it can be used. As for the Apple KB, the AppleCare 
channel in Sherlock is plugged right in. You pay for these facilities 
when you buy the OS -- so why not use them?
