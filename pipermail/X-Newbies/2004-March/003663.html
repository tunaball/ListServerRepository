On Sunday, Mar 28, 2004, at 04:55 Canada/Eastern, Chris Walker wrote:

> [...] I now want to convert some of my favourite Icons from 9 to X.  
> [...]
> Does anyone know if a simple conversion is possible? [...]

Not in most cases; usually Mac OS 9 icons do not have the resolution or  
quality available in OS X, so they need working over. They are stored  
in various types of resources (e.g., ICN# or icns), which can be  
accessed with ResEdit. In OS X, icon data is stored in .icns files,  
which can be created with various applications (e.g., IconComposer).  
There are various tutorials describing the process (e.g.,  
<http://www.scene24.net/projects/osxicons/> or  
<http://www.macdevcenter.com/pub/a/mac/2001/05/24/ 
aqua_design.html?page=1>).
