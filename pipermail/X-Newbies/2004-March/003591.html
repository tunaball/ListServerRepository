On Mar 18, 2004, at 12:57, Louis Cornelio wrote:

> Acrobat seems to have hijacked all my print preview funstions! ( like
> from appleworks!)
> Any way to undo this?

Select a pdf file, do Get Info. In the 'Open with' area, select 
Preview. Then, click on the 'Change All'. It will prompt you by asking 
if you want to open all files of that type with Preview.
