: check my internet settings? Maddening... any thoughts?

iTunes contacts CDDB.com to identify any newly inserted Audio CD and
return with useful info (disc name, track titles, artist, etc.).
The process will not work if you're not online, iTunes has a problem
contacting CDDB's servers, or your Audio CD is still unidentified.
