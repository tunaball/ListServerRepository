The master password for the keychain is your administrative password,  
unless you have changed the admin password recently. Sometimes,  
particularly when the admin password is set to (nothing) or just  
spaces, the keychain gets confused.

This is EXACTLY why Apple and I both recommend that you NOT set your  
master password to (nothing) or just spaces!

The solution to this problem has been described: boot from your OS X  
install discs, ignore the offer to reinstall your OS, go to the  
menubar and choose the option to reset password. This will reset your  
admin AND keychain password to whatever you've now selected (which  
will NOT, I trust, be blank).
