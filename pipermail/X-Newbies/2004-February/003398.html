> I recently bought a used iMacDV400 that came with Jaguar installed and no
version of OS 9. My question is this. Whats the bess way I can install a
version of OS 9 on the hard drive which has not been partioned?
>
> TIA for all help
>
Got another drive or CD that has a copy of OS 9? If you do, you can simply
drag the OS 9 System Folder on to the OS X disk. Note; that this will work
for installing OS 9 but not for OS X. When you have the System Folder on the
OS X drive open the OS 9 preferences panel while in OS X and point Classic
at the OS 9 System folder. Tell it to rebuild it's desktop and you should be
good to go.
