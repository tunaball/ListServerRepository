I copied my swap files to another partition with the help of SwapCop. 
This was a year ago (under 10.1.5), so I leave it to you to find if 
the program is still valid. But I strongly hope that you know what 
you are doing before you start moving swap files around, as this is a 
very fundamental part of how the OS works. This is not something to 
do on a whim... It worked fine for me, though, but when I updated my 
OS to 10.3 the swap disk was yet again the same as the OS, but this 
time the disk was much larger so now I haven't moved it again. Also 
it seems that 10.3 treats swapfiles different than 10.1.5. These can 
grow much bigger than 74 MB...
