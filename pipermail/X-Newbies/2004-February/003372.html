1. Make sure you have removed all no-Apple extensions and control 
panels from the Calssic system folder.
2. THen remove the extensions that OS X puts in:
Classic
Classic support
Classic Support UI
3. Start up Classic and it will put those back in. Probably one of 
them is corrupted.
