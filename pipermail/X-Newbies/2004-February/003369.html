To help narrow things down a bit more, IIRC things get loaded in the 
following order
	The stuff that's "loose" in the System Folder (or is that last? not 
sure off the top of my head).
	The stuff in the Extensions folder
	The stuff in the Control Panels folder
And unless you've done something to tell the system different, they 
will load in alphabetical order (extensions a-z, then control panels 
a-z).

The reason I mention this is that you stated it hung just before it 
finished. That tells me it probably got all the way through the 
Extensions folder and hung somewhere near the end of the Control Panels 
folder. The other thing to note if you have the whole startup window 
showing, is which icon was the last one? Can you tell what it 
corresponds to and work from there? Keep in mind not all extensions 
display an icon when loading.

Hope that helps narrow it down.
