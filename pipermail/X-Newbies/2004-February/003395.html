On Monday, Feb 9, 2004, at 17:55 Canada/Eastern, Robert wrote:

> The help file in Safari where the plugins are listed. It doesn't show 
> there but shows correctly in the terminal with java -version.

You mean, the Installed Plug-ins list. Double-check it. You probably 
have more than one entry for Java. Under Mac OS X, you may have 
different versions of Java co-existing on your machine. Applications 
which require specifically an older version will use the older one; 
applications which do not have a specific version requirement will use 
the newest version.
