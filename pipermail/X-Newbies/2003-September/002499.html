On Wednesday, September 24, 2003, at 01:45  PM, Eugene Lee wrote:

>  I wonder
> if it's possible to write an AppleScript that launches Internet Connect
> to get a dialup connection, then launches Mail.app to download mail, 
> and
> finally disconnect the dialup connection.

IMO no Applescript is necessary. It can all be done using the PPP 
options in the Network System Preference (button at bottom of the PPP 
tab).

First, just set the PPP options to connect automatically. Then Internet 
Connect will connect when Mail needs it.

Next, in the same PPP options pane, tell it to disconnect if idle for 1 
minute, or whatever you want it to be. That way 1 minute after Mail 
finishes receiving/sending email, it will disconnect.
