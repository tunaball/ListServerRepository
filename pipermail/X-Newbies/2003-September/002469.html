Did he also run it on the server? If he isn't the admin on the server when
he brings a document to his hard drive the permissions may not be right. You
need to check out how file permissions are controlled and inherited on the
server as well. This area can be really confusing because simply moving a
file from one directory to another can affect how they are set depending on
who moves it. Try doing a google search on "Unix permissions"; see what you
get for explanations, careful, it can make your hair hurt.
