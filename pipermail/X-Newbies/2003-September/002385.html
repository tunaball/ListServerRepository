Often, the failure of an application to open is due to a corrupt  
preference file, or problems with privileges on support files. Many of  
the preferences are stored in  
/Users/<yourusername>/Library/Preferences/com.apple.internetconfig.plist

You might want to try backing up that preference file and move the  
original out of the Preferences folder. Then try launching IE and see  
if it opens. I've had this file get screwed up before and it caused all  
sorts of headaches until I deleted and recreated it.
