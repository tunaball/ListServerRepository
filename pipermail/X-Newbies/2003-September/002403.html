It doesn't hurt to trash preferences.  The system just recreates them.

It might be easier to use Dragster to reset your preferences.:

http://www.ifthensoft.com/dragster.hqx

Or, if you want to test and see if the problem is being caused by a 
corrupted user preferences file before you do anything else, you can 
create a new user on your system, log out and back in as the new user, 
and see if the problem has gone away.  If it has, the problem was indeed 
a corrupted user preferences file.

For how to do this, see:

http://www.macattorney.com/tutorial.html
