>>>How does one, after install a Mac OS X app,
>>>remove it, completely


Often you have to make liberal use of the Find feature to snag all the 
parts of an application in OS X.  But you can easily get around this with 
some free utilities. 

You can use Bills Of Materials Reader on the installer package of an 
applicatin to read where it installs all of its files:
http://mitglied.lycos.de/simonloo/old_idisk/bomreader102.sit

DesInstaller is a simple tool that reads the receipts generated when you 
install a pkg file with Apple's Installer. Every file installed by this 
package is removed, even if it has been modified, and archived if you ask 
the DesInstaller to.
http://krugazor.free.fr/software/desinstaller/DesInstaller.php
