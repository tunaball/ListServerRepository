I have an iMac dv 400, g3 and  an external firewire drive (Maxtor 40 
gig) which contains my osx2.2. I boot regularly off this drive. The 
internal drives contain OS9  and Yellow Dog Linux.  By holding the 
option key down when rebooting I  can choose which drive to reboot to. 
This didn't work too well in previous versions of OSX but now works 
like a jewel in the latest OSX revision.
Hope this helps, Bob
