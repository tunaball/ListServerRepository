FWIW I use Onyx to repair permissions and run the cron scripts before
doing an update.  I'm not a Unix guru but I can see a logical reason why
this should be done.

If permissions get changed then presumably the installer will not
necessarily put in the correct updated files unless it is told to ignore
permissions and do it regardless.  Hence you could end up with a partial
update with potential problems down the line.

That was my reasoning anyway.
