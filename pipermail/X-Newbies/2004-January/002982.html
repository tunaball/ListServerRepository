In order to get the 740 printer drivers, you have to specifically 
select them thru the Custom install of Panther.  They do not 
automatically get installed even if you do a clean install.

See <http://docs.info.apple.com/article.html?artnum=25662> and 
<http://docs.info.apple.com/article.html?artnum=25588>  for more info 
on how to install them at this point.
