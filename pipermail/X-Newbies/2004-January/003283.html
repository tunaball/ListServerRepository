Well, almost that easy, but you need to make sure things are set 
correctly. Since you say that your base station is an Airport Extreme 
(G) then either a B or G PCI card can be used. After installing, make 
sure the card settings match the router (SSID, channel number, WEP). 
That should be about it.

Good luck,
