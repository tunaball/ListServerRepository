>In the Mac lingo  is the G5 dual considered a 'Server' or is there actually different OSX
>software for servers and 'regular' Macs as it concerns OS X 10.3 or 10.3.2??
>

>Kevin W

There is Server software and there is Client software.
OS X Server is server software and OS X is client software.

There are Servers and there are user computers.
Xserve is a server and the Power Mac G5 is a user computer

The G5 is a processor that can exist in either type of computer.

The larger question is 'Can a client computer (i.e. a Power Mac G5) be used as a server while running OS X. And to that, there are many viewpoints. And the bottom line answer is 'It depends'.
