Are you saying you are using a Mac at work (lucky you!). If yes, you 
should've used the Download Only command (rather than Install). 
However, with v10.4.1, updates can include only specific files for the 
machine you're downloading on. So you should use a browser to download 
the update package from

<http://www.apple.com/support/downloads/>

and use that one to update your home Mac.
