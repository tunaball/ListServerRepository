> Good question. Wish I knew the answer.
>
> A clunky and not very useful workaround is to remove items from the 
> Help Center. The Help Center list is kept in 
> ~/Library/Preferences/com.apple.help.plist. Each application which 
> uses Apple's help framework has a key in that file, and the file can 
> be opened with any text editor (but preferably with 
> PropertyListEditor); remove the key, and the item won't appear in HC 
> and won't be searched. The key will be added back to the plist when 
> you invoke help from the respective application.
>

This works as expected in 10.3. You can choose to search only a single 
application's help library, or all libraries.
