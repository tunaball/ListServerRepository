If you want to do this without using Terminal, Mail.app will do the 
trick.

Launch Mail. Choose File-->Import Mail  (choose "other" when prompted 
for the type). Select the folder on your hard drive which contains the 
pine mailboxes.

When Mail has finished importing, Launch Eudora. Use the File->Import 
function. Tell it to import from Mail.app.
