On 06/05/2004, at 2:46 am, J wrote:

> If one changes their account name slightly, where are the prefs to 
> delete so that the screensaver with the white apple on it  displays 
> the correct name?

If you're talking about the Apple-shipped screen saver Computer Name, 
that actually displays the Computer Name rather than the current user 
(though that'd be handy in some cases if it did.)

You can change the computer name in the Sharing preference pane, the 
change takes effect as soon as you close that pane or tab out of the 
Computer Name field. Note that if that computer is sharing a printer or 
files, connections may need to be re-established on the computers 
connecting to it.
