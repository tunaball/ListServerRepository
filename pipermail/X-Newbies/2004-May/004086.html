There's no reason to even mention Filemaker Pro, since you are trying to
get MySQL working.  Since MySQL is local, the hostname is "localhost".
To create a user account, do something like this in the MySQL command
line interface:

	shell> mysql --user=root mysql
	mysql> GRANT SELECT,INSERT,UPDATE,DELETE,CREATE,DROP
	->     ON dbname.*
	->     TO 'username'@'localhost'
	->     IDENTIFIED BY 'password';

where "dbname", "username", and "password" are yours to invent.

	http://dev.mysql.com/doc/mysql/en/Adding_users.html

Then create database itself:

	mysql> CREATE DATABASE dbname;
