In the Finder, choose Preferences from the Finder menu. At the bottom 
there is a "Select..." button for "Languages for searching file 
contents". If you click that button, a new window pops up saying  
"Select the languages you want to use when you search the contents of 
files. Select fewer languages to make indexing faster and use less disk 
space." followed by a list of languages with checkboxes.

The default has several languages checked. To make it faster, deselect 
all the extras. I don't know for sure if de-selecting all of them is 
equivalent to turning indexing off completely or not, but it would seem 
like that should do it.

Might be worth a try.
