Regarding system maintenance, besides Macaroni, you can use MacJanitor, 
which is Freeware and runs the three maintenance tasks when you want. 
You can also do it by Applescript or in the terminal. One can edit the 
"Cron" table where these Unix tasks are scheduled or install a unix 
program like Anacron, which will run Cron tasks that were missed when 
the computer wakes from sleep. These last few are more advanced 
techniques. For those interested, check out http://www.macosxhints.com 
where one will find a slew of excellent tips and techniques for 
managing OSX. I list these techniques (and there may be others) to 
demonstrate that OS X does give a significant number of ways to deal 
with many issues. As I've evolved from a Classic Mac power user to an 
OS X power user since the days of the OS X beta release, I have become 
more and more impressed with the usefulness of this flexibility. Given 
that this is the Newbies list, I won't clutter the list with the 
code/instructions for these techniques. E-mail me off list if you have 
an interest in learning more.
