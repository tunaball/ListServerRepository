> Have tried fresh downloads, dragging install icon to desk top first, 
> [...]

You've tried everything except what you should begin with. Mac OS X 
enforces a strict filespace separation: system - local - user - 
application - network. The first step is to decide where the problem 
is. Create a new user account, and try it there. Same problem? The 
issue is not in the user space. Works? The problem is in the user 
space. And you take it from there.
