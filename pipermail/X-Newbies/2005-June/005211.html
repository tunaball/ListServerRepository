Importing files to mail from that folder will not work, those are not  
the .mbox extension files mail uses. Your files are in  
user/"myname"/library/mail.
When I get info on the mail folder in cache it is 20 MB file size,  
whereas a single mbox file like my inbox is over 200 MB. If you right  
click an mbox you will find it is a package (view package contents)  
with the actual mbox file inside along with some indexing files.
