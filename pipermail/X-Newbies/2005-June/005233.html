If you're working in an office with PCs, there might be some value in  
knowing what's a virus so that you don't accidentally spread it. And  
if you exchange Word documents frequently via email, you might want  
to check for macro viruses. If you're a home user, and you don't  
email MS Office documents, then a virus scanner is probably a waste  
of time and money.
