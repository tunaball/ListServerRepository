> etc have loaded.

That's to be expected. Whatever interfered with Toon, will continue to 
do so, but at a different point.

The next stage is to determine which app or combination of apps causes 
the problem, using Activity Monitor. Quit all open apps, start Activity 
Monitor, then identify in the process list your startup items (the ones 
which would not appear in the new account). Then go recursively through 
the list. Force the first one to quit, using the Quit Process command, 
then try launching Toon. And so on. This should allow you to find the 
troublemaker.

It's a good tool to have in your kitbag. As to font caches, however, 
since they are not the problem, leave them alone. The golden rule: If 
it ain't broke, don't fix it.
