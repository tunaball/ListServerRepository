> From: Kurt Streeter <kurt.streeter at yahoo.co.uk>
> I am having trouble accessing my .mac mail account. Since upgrading
> my account to a fully paid account (as opposed to the 60 day trial),
> I have not been able to access the account. All the other mail
> accounts in Mail are fine, but the .mac one gives the following  
> message:
>
> There may be a problem with the mail server or network. Check the
> settings for account ".Mac Account" or try again.
>
> The server error encountered was: The connection to the server
> "mail.mac.com" on port 993 timed out.
>
> I'm not too sure if this is something that I have changed while
> upgrading the account, but any help would be appreciated.

Yes, the problem is your fault. :)

You cannot connect to mail.mac.com on port 993. The correct port is 110.
While you're at it, check your smtp port -- it should be port 25.

Sounds like you changed over from GMail -- they use oddball settings  
like port 993.
