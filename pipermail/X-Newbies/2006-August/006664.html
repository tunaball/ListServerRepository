Find the preferences folder under:
Users/~you/Library/Preferences
	Open the Appleworks folder and move the preferences file to the trash.
Go to the documents folder:
	inside the Appleworks folder find the folder Recent Items.
	delete the contents. (When this folder gets full it slows down  
Appleworks)
These files should be deleted automatically but they aren't.

Start appleworks
it should work then.
