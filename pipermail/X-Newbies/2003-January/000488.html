You must install iMac Firmware 4.1.9 BEFORE Mac OS X 10.2

BTW 1.. According to Article , the imac Firmware Update 4.1.9 is 
available on the  Mac OS X Install CD Disc 1

Installation disc and  reinstall OS9.
