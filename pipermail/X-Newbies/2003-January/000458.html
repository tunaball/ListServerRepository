On 5Jan03,  Chris Foote <foote at chem.ucla.edu> wrote:

>To configure the browser that Eudora uses, option click
>on a link, then select the application.

Also on 5JAN03, Vincent Cayenne <vcayenne at mac.com> wrote:

>Open Eudora Pro, then go to a message that has a link.
>Double-click the link while holding down the option
>key. In the dialog box that appears titled "please
>choose an application to do http", navigate to your IE
>5.2.1 and select it, making sure that the "Always use
>this application" is checked.

Thanks guys for the advice. Option clicking a link and selecting IE 
5.2.1 in the manner you suggested worked like a charm. Now when I 
click on a link, Eudora launches my OS X browser whether the browser 
is running or not. Thanks again.
