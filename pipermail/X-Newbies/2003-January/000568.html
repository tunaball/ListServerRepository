on 03/01/24 16:53, Chris Beamis at beamis at drizzle.com wrote:

> Is there a convenient way to make it so that I can disable the modem
> from dialing out while I'm using the machine?
> 
> 
In System Preferences > Network >
Set "Show:" to "Network Port Configurations"
Un-check "Internal Modem".
