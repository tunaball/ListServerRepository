> Hi,

> Today I was using Finale (a music notation program) in OS 9, because,
> believe it or not, Coda Music Technology is still working on carbonizing it.
> 
> Anyhow, I don't have an OS 9 printer driver for my current printer, so I
> couldn't print.  But I was able to save it as a "postscript" (.ps) file.
> 
> Is there any way to print this postscript .ps file in OS X?
> 

Someone told me verbally to "send it to a postscript printer."  We have a HP
laser printer here, so I opened my Print Center, opened the laser printer
window, and dragged the .ps file from the Finder to it.

It started printing.  It just worked.  OS X is amazing.
