On Tuesday, Apr 27, 2004, at 11:35 Canada/Eastern, Chris Walker wrote:

> [...] If I turn file sharing on, does this leave port 22 open so that 
> I can SSH
> into my machine?

File Sharing and Remote Login (SSH) are different kettles of fish. 
Whether you enable file sharing or not is irrelevant to SSH. Inspect 
your Firewall settings, you should see there which ports are the 
defaults for which services.

> Following on from this, aren't I leaving a nice big open door for 
> hacker attacks via port 22?

Big door, yes. Open door, no. It's locked, and the lock is pretty 
sound. That's the point of the exercise. Check the OpenSSH FAQ.

Try <http://www.openssh.org/macos.html>. I don't how good any of them 
are, I use the default tools.
