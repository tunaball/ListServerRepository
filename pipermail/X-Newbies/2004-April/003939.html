I'm the head of a very large Mac user group for attorneys, and this 
question has come up often, because attorneys used to love WP/Mac.  Most 
have long since switched to Microsoft Word.

DataViz's MacLink converters do an okay job translating WP/Mac files, but 
they are terrible if your files contain complex formatting.

Another option is to save the orginal file in RTF format within WP/Mac.  
(WP/Mac runs nicely in Classic under OS X, but some folks have reported a 
nasty file corruption bug.)  However, this solution has the same problem 
as with the DataViz product.

A really nice recent solution is to download and use the new FREE 
opensource OS X version of AbiWord.  You may even like AbiWord enough to 
use it instead of Microsoft Word.  But in any case AbiWord includes some 
very effective translators.  It can open and save as Word, WordPerfect, 
and several other formats.
