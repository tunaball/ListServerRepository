: Florin Alexander Neumann tapped out this message on 7/23/2003 2:37 PM

: >Simple. Don't do it. There's a good reason why you shouldn't mess with 
: >the System folder, especially if you're a newbie. There are plenty of 
: >other places to put new fonts if you need them. OS X is not OS 9.
: 
: I assume the person wanted to add some fonts to their font folder, so why 
: stop there? Why not tell them how to do it? Is it that difficult?

Because the original person wanted to "drag the new fonts to the system
library's font folder".  Modifying anything inside the System folder is
a severe no-no in OS X, as it should be.  If the person needs to add new
fonts to their box, they should drop it in the global library font folder
/Library/Fonts or the user's library font folder (home/Library/Fonts).
