That's correct. Stuffit Expander will not fix the problem of AOL not 
understanding MIME extensions. However, it does fix a problem where AOL 
has never done a good job of handling multiple attachments in the same 
message. If you send an outgoing message in AOL, you'll notice that if 
it has more than 1 attachment, AOL will automatically zip it into 1 
attachment.
