Hi Deb,

Try removing all other USB and Firewire devices, and have only the
following configuration:

	PowerMac G4
	Apple Pro Keyboard
	Logitech Wiresless Optical Mouse (to keyboard)

First, connect the keyboard directly to the 17" Studio Display and
see if you have the sleep problems.  If not, also try and connect the
keyboard directly to the G4.

Try to establish a baseline, a stable hardware configuration where
everything works.  Then start introducing other devices into the
picture, one device at a time.
