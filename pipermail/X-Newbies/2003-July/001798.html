>I can save the doc to the desktop then open Word and go file open and then

Select one of the files on your desktop (don't open it), then use the 
"Get Info" command from the File menu. Reveal the "Open With" area 
and choose Microsoft Word from the list. Click "Change All" to set 
Word as the relevant application for all files of that type.
