Had the same problem before. I was given the solution below, and it 
worked great. Don't be afraid of opening the terminal window for this. 
Just make sure you type it right.

HTH

Begin forwarded message:

> Open the Terminal. Type:
>
> sudo rm -i
>
> Then drag the file into the Terminal window. The line should look like:
>
> sudo rm -i ~/Desktop/StubbonFileName
>
> Hit Return. That should do it.
>
> If you can't drag the file into the window, simply type the second 
> Terminal
> command, substituting StubornFileName for the name of the file.
>
> Hope that helps.
>
> Bill
