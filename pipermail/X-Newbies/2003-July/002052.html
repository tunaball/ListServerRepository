Depending on the context, .scr can stand for a screen saver, a script, 
or even for a format (derived from PCX) for DOS screen shots. As to 
.pif, usually (not always) it stands for Program Information File, 
which is basically a collection of settings required to run a DOS 
programme under Windows. Some malwares use .scr and .pif files to 
spread themselves.
