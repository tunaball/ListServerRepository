On Nov 2, 2008, at 7:35 AM, Mark Taintor wrote:

> I've been using an Apple 16/600 printer for many years and it's been  
> very reliable, if a bit slow, especially on large PDF files.
> Lately, it's been having lots of paper jams. The sheet will start to  
> come out and get stuck, which makes the bottom several inches get  
> quite crinkled.
> I'm wondering if it's time to get a new printer. Do you think it  
> would be cost effective to have this one serviced?


This is a well-known problem with the LaserWriter 16/600 PS and  
similar Apple printers.  You can fix it yourself with the following kit:
<http://www.fixyourownprinter.com/kits/apple/K66>

I have done it on a 16/600 PS and a 630 Pro.  It requires a little  
dexterity, but not too difficult.
