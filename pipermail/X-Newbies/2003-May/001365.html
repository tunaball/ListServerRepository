It depends where they came from. OS 9 requires a "resource fork" that 
gives information about the associated application. OS X doesn't use 
this (and may not create it, I'm not sure). OS X actually makes an 
extension, but hides it. If the extension gets messed up, (like .jpg 
1), the file won't be recognized, although you can always force the 
application to open it.
