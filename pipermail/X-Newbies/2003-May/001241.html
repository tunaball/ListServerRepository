I'm using 10.2.5 and am printing fine to a Windows network as well.
Can you give us more information about your printer settings?

I presume you are using IP printing to connect to your network
printer...
1) You need to make sure that you have configured your PowerBook to an
available IP Address from your Windows Network.
2) This can be done either via DHCP or manually.
3) Launch Print Center and verify that your printers have the correct IP
information set on them in Print Center.
4) To test connectivity, try to ping the printer from your PowerBook
using
Network Utility.

Let us know if you need more help and information.
