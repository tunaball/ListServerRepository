Would you try the Pic Blanc tunnel or the Glacier de Sarenne if you 
were a novice skier? This is definitely a non-newbie subject. IMHO,

mostly troubleshooting, rather than maintenance. I'm not sure a novice 
user needs to run them on a regular basis.
