In any case, as others have said, its an exceptionally bad idea to 
tweak around with the System folder's contents, and the first thing to 
do with this kind of post-upgrade flakey behavior is to validate the 
hardware - particularly memory. Memory that tests good can be not good 
enough to run without flaking out.
