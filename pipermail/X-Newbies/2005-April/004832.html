On Apr 11, 2005, at 06:25, Kirk McElhearn wrote:

>> A small, but irritating issue for me. I am using a great desktop
>> background that I downloaded from www.deviantart.com but every time I
>> reboot my G5, behind the log-in dialog box, is my old default desktop
>> background. That of course changes as soon as the system starts and my
>> desktop appears. I can't seem for the life of me to be able to get rid
>> of that default desktop background showing up at startup. I would like
>> to have the same image as is my desktop background. Anyone run into
>> this stupid little irritating issue before?
>
> While you might be able to change it by delving into the internals of 
> the
> System folder, it's probably not advised.

Actually, it's quite easy and painless.

Go to HD>Library>Desktop Pictures and find the file named Aqua Blue. 
Rename it to Old Aqua Blue (or your choice). Then, drop into that 
folder the picture that you want displayed behind your login window and 
rename it to Aqua Blue.  Done.
