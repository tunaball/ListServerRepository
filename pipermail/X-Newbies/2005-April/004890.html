On 4/18/05, Alex <lists at lexial.ca> wrote:
> 
> On Apr 18, 2005, at 3:46 AM, Brian Durant wrote:
> 
> > In all of the instructions for iTunes, I find that they refer to being
> > able to burn audio CDs by clicking on the button in the upper right
> > hand corner of iTunes. Unfortunately my button says "import"[...]
> 
> That's because you selected an audio CD as a source. The button is
> dynamic and context-sensitive: it changes depending on the selected
> source. iTunes cannot copy directly an audio CD -- you need to import
> the CD first, then create a playlist and add items to the playlist.
> When you select a playlist (rather than an audio CD), the button will
> change to "Burn Disc".

OK, got it. So that would mean that whenever I want to copy a CD, I
need to change my import prefs in iTunes so that iTunes rips in AIFF
format, then drag the CD icon in the sidebar down to the white area to
create a playlist just for that CD and then select all of the ripped
files and then burn to CD. Sounds like iTunes works better with
creating compendium CDs. I guess I'll use Toast for copying the few
CDs I want copied for my CD Walkman.
