hi everyone,

i've gone and got myself a problem for which there are a few reports 
already on the web, i.e. my pismo won't wake up properly after being 
put to sleep, although for the *second* time after a new start. i'm 
running 10.2.5 with 768MB RAM ...

the symptoms are:
i). the first sleep and wake cycle is no problem,
ii). the second sleep cycle results in a dead display upon waking, with 
no response to anything,
iii). this happens when slept at the log-in screen as well as when 
logged-in as a user,
iv). this isn't a problem under 9.2.2.

obviously this is a major pain in the rear. there are a few leads on 
the web (tickling the display brightness controls, hitting the power 
button and then S to force sleep), but none of these have any effect. 
apparently the pismo is not the only machine to suffer from this. PRAM 
reset and all the usual stuff has had no effect either.

short of doing a full install, anyone have any further advice?
