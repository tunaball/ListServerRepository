So, you are upgrading from 10.1 to 10.2, right? Still in the first 
partition?
Start up in 10.1, make sure that the machine is not set to go to sleep 
in that OS, in that 9 is not running during the install it's sleep 
settings should not matter.
On a long shot, if that does not work, strip everything 'aftermarket' 
out of the machine. As silly as it sounds with my beige it would seem 
to 'go to sleep' minutes before the install completed. I stripped it 
back to stock, including having only 96MB or RAM, less that what X 
needs to be able to run. Then the install worked. Then I reinstalled 
the extra RAM to bring it up to 640MB
