Read your email regarding OSX on a PBook. I finally
was succesful in installing Jaguar on my Wallstreet
(1998) PBook but it took quite a few hoops.
The 2 biggest (and these are critical)issues is that
you need to partition your drive with the first
partition NO bigger than 8 Gig and OSX to be installed
there. It's the4 only way on that laptop.
Also, just as critical, remove any third party RAM
that adds more ram than came with the book. Seriously,
this was learned after weeks of failed installs. You
can put the Ram back in no problem after OSX is up and
running but it will not install in any additional RAM
configurations.
I'm currently running 10.2.5 on my G3 Pbook with the
additional RAM and a CPU upgrade, no problem.
