Whoa. Appleworks runs under OS X or under OS 9 using Classic. Classic  
isn't supported in Leopard, but that's not an issue with Appleworks.  
Appleworks will run under OS X by itself (one of the few programs that  
will run in either OS without installing separate versions).

So Appleworks will run in OS X. I know it will, I just did it.

It will also run in Snow Leopard, because all it will need then is  
Rosetta (which, you'll recall, allows PowerPC code to run on an Intel  
processor.)

However, Snow Leopard itself, AFAIK, will require an Intel processor,  
so Judy will have to get a spiffy new 'Book if she wants to run Snow  
Leopard.

But = right now = Appleworks will run on her Powerbook with Leopard.
