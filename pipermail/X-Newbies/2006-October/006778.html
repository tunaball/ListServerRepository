Okay, you need to be able to run OS X to run iWorks and Tiger.  You 
should be able to run OS X just fine on any Mac that has at least a 
400Mhz G3 PowerPC processor.  You can run Tiger if you purchase it and 
then send in the disks to Apple and exchange the DVD disk for CD-ROM 
disks.

Have a look at:

http://www.baucomcomputers.com/

There are a couple of very inexpensive used laptops that fill the bill.
