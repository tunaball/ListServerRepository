On 10/11/06, Mike <mliii at earthlink.net> wrote:
> That's easy- you can access a disk, application, file, or folder from
> another folder or the desktop by creating an alias to it.
> In the Finder, open a window showing the item.
> Choose File > Make Alias.
> If you want, move the alias to a more convenient location.
> To create an alias for an item and relocate it in one step, press and
> hold the Command and Option keys while you drag the item's icon to
> another folder or to the desktop. Be sure to release the mouse before
> releasing the Command and Option keys.
>
> If you want to find the original item of an alias, select the alias
> and choose File > Show Original.
