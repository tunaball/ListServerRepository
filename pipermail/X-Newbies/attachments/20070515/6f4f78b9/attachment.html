<FONT COLOR="#0000FF">&gt; I recently purchased a used iBook G4, and the vendor pre-loaded it with Tiger. <BR>
&gt; I have no experience with OS X (my desktop runs 9.2.2), and <BR>
&gt; so I have some questions:<BR>
&gt; <BR>
&gt; 1) I know that Tiger runs non-X apps under Classic. How do I tell if Classic <BR>
&gt; is installed, and if it is not, how do I install it?<BR>
</FONT><BR>
Go to System Preferences &gt; System &gt; Classic to find out. <BR>
If it is not there, drag the System Folder from an existing MacOS 9 installation (preferably for your specific model) onto your Root directory &nbsp;-- doing a direct install of 9 onto a previously existing OS X installation will screw up OS X. Then (you may have to restart first) open the Classic control panel to configure it (you will be coached). &nbsp;&nbsp;&nbsp;<BR>
<FONT COLOR="#0000FF">&gt; 2) Is it OK to create subfolders in the Applications folder?<BR>
</FONT><BR>
Yes, BUT some applications don't like it; some will work fine in a subfolder.<BR>
MacOS 9/Classic expects to find its applications in a folder named &#8220;Applications (Mac OS 9)&#8221; at the root level. Create one if it is not there.<BR>
<BR>
<FONT COLOR="#0000FF">&gt; 3) Is the firewall in Tiger good enough security, or should I be looking for a <BR>
&gt; third-party firewall? How do I activate Tiger's firewall?<BR>
</FONT><BR>
It is plenty good. &nbsp;Go &nbsp;System Preferences &gt; Sharing &gt; Firewall<BR>
<BR>
<FONT COLOR="#0000FF">&gt; 4) What's the easiest way to fax under Tiger?<BR>
</FONT><BR>
Buy PageSender. Nothing else comes close.<BR>
www.versiontracker.com/dyn/moreinfo/macosx/14537<BR>
