Using html is not irresponsible, nor is posting over 5 KB. Holding a
posting to a thread, using size as the only criteria does not in my

arbitrary method for withholding postings to the list. The original
posting was only 165 bytes over the arbitrary limit set by the
moderator and was held for 54 minutes before being allowed into the
list "sandbox". It seems to me to be more an issue of being anal
retentive than being a responsible poster to the list.
