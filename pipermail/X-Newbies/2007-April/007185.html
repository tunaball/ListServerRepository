You would have to purchase 10.4 ( http://www.apple.com/macosx/ 
tiger/ ) to get from 10.3.x to 10.4.x - but I would just wait for  
10.5 ( http://www.apple.com/macosx/leopard/ ) which should be out  
really soon now (May - June-ish).

Installing it is super easy, and rarely causes any problems. To get  
the latest updates for the OS you have, pick "Software Updates" from  
the Apple menu. This also very rarely causes any issues for an  
average user, so you don't have to constantly ride herd on all  
updates to prevent this or that thing from blowing up.
