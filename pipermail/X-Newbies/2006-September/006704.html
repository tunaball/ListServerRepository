How to extract Sand from a MacOS 9 install CD:

1. You will need an install CD and the freeware application TomeViewer,
which works only under MacOS 9 or Classic.
http://www.versiontracker.com/dyn/moreinfo/macos/4561

2.  Sand is inside "Mac OS 9.1:Software Installers:System Software:
Mac OS 9.1:Installation Tome"

3.  Once TomeViewer  is installed, double click Installation Tome and
TomeViewer should launch; a window will open showing all of Installation
Tome's contents.

4.  Select Sand and choose Archive:Expand... From the menubar.
You will be asked where you want to put it.
