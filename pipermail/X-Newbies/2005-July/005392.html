Daniel Carter said:

>now, once again, I wonder how crucial it is that I update firmware on 
>my Lacie d2 80GB hard drive (is LaCie Update Tool v1.3.3 the proper 
>download?)? what precautions should I take?


I think that that question is best asked of LaCie.  There were some nasty 
problems associated with drives that used the Oxford 911 chipset, and 
LaCie provided updates to handle them.  According to MacFixIt, there were 
even some problems that apparently were unique to LaCie.

However, none of us know exactly what is in the LaCie firmware updaters, 
only LaCie does.  So you can either take it on faith that you need to 
install the firmware updaters that they say that you need to install, 
following their instructions exactly, or you can ask them why you need to 
install them, and if there are any risks associated with doing so.
