What sort of file did you send them?  Was it a file that you saved as a 
PDF?  PDF (portable document format) is a universal format that allows 
anyone to read or print the file, but these files are really not made to 
be edited.  (It's minimally possible using the full commerical version of 
Acrobat, but that's not what PDF's are designed for.)
