On Sunday, November 24, 2002, at 09:05 AM, Tim Collier wrote:

> Correct me if I'm wrong, but isn't that the way ALL installation CD's 
> have always worked?  You can install OS 9 from an OS 9 Installation CD 
> but you certainly can't run the OS.  It's amazing at how 
> 'misinformation' is taken as gospel.
>
> Tim

It's not misinformation. You can run the OS, just not all the extras.

Minor examples:
You can open and use simpletext, and create /edit files if necessary.
If you have resedit, you can use it.

Extremely useful examples:
You have a working finder, with which you can trash / move / backup 
files, even to most USB / Firewire drives.
You can backup your files to a network drive, while booted to the 9 
install cd. You can even do so via IP - if there is a dhcp server, 
you'll get a valid IP address.
