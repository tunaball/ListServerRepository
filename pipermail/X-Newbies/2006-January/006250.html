Try reinstalling the system from the install disks that came with the
machine.  Under the options menu do 'archive and install' DON'T do a
reformat.  This should put a fresh system on the machine although you
will lose any haxies if you installed any.  Everything else should be OK
unless the disk is really hosed.

Similar thing happened to me with a G4 and the above method sorted it.
