> 	I'm trying to find the Mac equivalent of a Widoze clipboard utility 
> called Clipmate.

Butler is a file launcher, but the it has this function as well. It is 
extremely customizable, compared to other pasteboards I've used.
It works for text&graphics and there is a plaintext paste option as 
well.
This pasteboard is actually the only part of Butler I use (for the rest 
I find Launchbar well beyond any file launcher).
It is actually a beta (though I've never experienced any bug), so I 
guess It'll became a shareware at a certain point.
