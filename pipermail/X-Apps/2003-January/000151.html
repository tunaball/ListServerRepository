Same here (grabbed IE bookmarks automagically).

I nonetheless imported instead bookmarks from OmniWeb. Alas, not an 
automagic process. What involved: generate bookmarks.html from OmniWeb. 
Open it with Safari. Open a second Safari window and show bookmarks. 
drag&drop (very nicely done I must say) bookmarks from Omniweb 
generated html (opened in 1st Safari window) into corresponding 
bookmark folder in 2nd Safari window. If needed, create additional 
bookmark folders and/or nested folders in Safari.

Takes some time but a chance of reorganizing you bookmarks ;-)
