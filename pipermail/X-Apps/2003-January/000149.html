You can make Safari read them again by resetting a defaults
flag, using the Terminal:

  defaults write com.apple.safari IEFavoritesWereImported 0

Check that it's now zero:

  defaults read com.apple.safari IEFavoritesWereImported 

and relaunch Safari. Should read them again. This is easily
applescriptable...
