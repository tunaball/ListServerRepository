On 11/10/03 10:18 AM, "Warren Pollans" <warren at meyer-pollans.net> wrote:

> QUESTION:  Can I safely delete the old "Microcsoft Office X" folder?  Is there
> any other cleanup that needs to be done?

All of your personal configuration should be in the Microsoft User Data
folder in Documents or in files in Library folder so you should be fine
deleting the application folder *except* if you created any customized
templates. Those are in the Word folder and should be saved before you dump
the folder.
