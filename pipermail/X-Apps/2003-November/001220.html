> Why are the file name extensions changed when a file is opened by a
> different application.

This is usually the function of (and controlled by) a given program's 
preference settings. In the case of Tex-Edit Plus, for example, go to 
the "save" dialog tab in T-E's prefs and deselect the approriate check 
box (or select, depending on what you want it to do)...
