This actually doesn't make any sense.

The current DiskWarrior CD includes OS versions up to 10.2.6. Any 
machines that require 10.2.7 or later (including all G5s and the most 
recent PowerBook G4s) will not boot from that CD. (This happens all the 
time with DiskWarrior. They prepare a new CD, and you can order it for 
a nominal fee.)

The DiskWarrior application will run in Panther, and can diagnose 
another drive, but it cannot complete the replace directory operation. 
It will run fine in 10.2.8, and can diagnose and repair a drive with 
Panther installed. For example, you can keep one partition on a drive 
(external or internal) with 10.2.8 and DiskWarrior, start up from that 
partition and repair another partition which contains Panther.
