I read somewhere that apparently it is not possible to index folders 
that enclose
previously-indexed folders when the cumulative number of enclosed files
exceeds 20,000. Afraid I can't attribute that info as I did not record 
where it came from. But it seems accurate to me. I have dragged my 
Documents folder into the Find search pane and when I do a Contents 
search in the Documents folder the Contents Index for that folder is 
updated automatically. I once tried to index my hard disk [to include 
all my Apps etc] but ran into some problems. I think to index your Hard 
Drive you will first have to delete the index for your Documents 
folder. There is a work around I found which I thought a bit too 
complex for me but if you wish to contact me off list I could send it 
to you.
