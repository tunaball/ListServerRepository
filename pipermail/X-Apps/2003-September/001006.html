If you look under the File Menu in IE, there's a menu item named "Work Offline". There's no equivalent menu item in Safari, and searching the help for "offline" produces one irrelevant hit.

So it would appear IE supports offline viewing, Safari doesn't.

If you're online, even if the page is in the cache, failing to check for an update is IMO a bug. If you've selected "Work Offline" in IE, then it shouldn't check.

As far as I can see, both browsers are working as designed. If you need offline browsing in Safari, download the site in Page Sucker or an equivalent and access it from your hard disk, not the cache. Use the bug reporting function in Safari to request the feature.
