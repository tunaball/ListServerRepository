We have a MacSense router, which is basically a rebranded LinkSys, and we
can connect to it, but the links don't work, as others here have reported.

The problem is technically not with Safari, but with the router. If you
examine the source code of the router's admin page, it doesn't contain
proper HTML for the links. Since Safari is pretty standards-based, it
doesn't work. I emailed Apple about it using the bug report form (since
they'll probably respond before LinkSys or MacSense would bother to change
the HTML in the router's firmware.

Strangely, Chimera also doesn't work with the router -- it never refreshes
pages after changes are made.
