Robert,

Hopefully when you say you saved the data in your User folder you mean 
that you saved/backed your entire user directory ((i.e. 
/Users/yourname/), which would mean you got all of the folders therein. 
  If so, in your User folder there should be a folder 
/Users/yourname/Library/Addresses that will have the 
AddressBook.addressbook file and the Addresses.addressBook folder in 
it.  That's where your personal Address Book data should be.

If nothing else you could do a Find File for AddressBook on the backup 
and see what it turns up.

HTH
