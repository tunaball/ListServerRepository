It depends ;) It has a LOT of features; far more than any other file utility
on the market. If you need or want many of those features, it's definitely
worth it, as to get all those features otherwise, you'll probably spend more
than the price of File Buddy buying several different utilities.

On the other hand, if you don't need all of File Buddy's functionality, you
can probably get by with a different, cheaper utility (like FileXaminer).
