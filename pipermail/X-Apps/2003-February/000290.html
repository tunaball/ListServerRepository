Just because I'm not paranoid enough at times, and because I've grown 
used to using my own machines as guinea pigs:

Yep.

It only does CD labels for an Avery 5931/8931 CD insert, but it works. 
You need to install some parts in your ~/Library/iTunes/Scripts, and 
some in Appleworks' starting points, and there's some fonts too. It 
also sits for a few seconds between progress indicators, so it looks 
like it's hung. If you're patient, and wait a few seconds, it'll finish 
up nicely.

Note - you do need to have a CD insert (that you just installed) open 
in Appleworks before running the script.
