On Sunday, December 22, 2002, at 03:11  PM, Judy Antipin wrote:

> I'm having a problem with TextEdit not launching in 10.2.2  (10.2.3
> after update). I've repaired permissions, used disk utillity repair 
> from
> the install disk,  ditched the plist and even tried to open it from
> within Terminal. Each time it 'flexes its muscles' in the dock, then
> disappears. If I click on a .txt document nothing happens. I've checked
> Apple support pages and Macintouch and can't find anything.
>
> Judy Antipin

Try this Judy.  Go to /Users/your user name/Library/Preferences/ and 
move this file to your desktop.
com.apple.TextEdit.plist  [This file will automatically be recreated 
the next time you launch TextEdit.]

Then try to launch TextEdit.  If that works, go ahead and put the file 
you moved to the desktop in the trash, and empty the trash.  If it 
doesn't work put the file back in preferences.  You will get a dialog 
saying the file exists, do you want to replace it?  Say yes.

Good luck
Tom K.
