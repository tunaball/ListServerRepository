I've been reading the list for about two weeks and have gotten some 
valuable feedback on several issues so far.  I work for a school 
district and am the project leader for our OSX implementation next 
school year.  The issue I'm writing about today is with AppleWorks. 
In some cases AW6 files that behave normally on an OS9 machine will 
behave quite differently in OSX. Most often when asked to print, they 
print out blank pages. I've discovered that the page setup is 
sometimes set to 'none' which if you look isn't even an option.  Once 
the page setup is corrected, then the margins go all wacky and read 
450. Needless to say, this is going to cause some serious stress for 
our teachers if we are unable to clearly explain what to do or how to 
prevent it from happening. At this point the only guess I have is 
that all of these files were originally AW5 files translated into 
AW6.  Has anyone else seen this problem?
