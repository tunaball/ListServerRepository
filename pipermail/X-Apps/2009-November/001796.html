>FFx takes up too much of the Downloads Folder.  When I ask for it to  be "Moved to Trash", I am told this can't be done "because it is  open.   Please send me exact instructions as to what I must do to  delete FFx completely. Be sure you are not executing Firefox from a disk image that you downloaded long ago. It's general practice to copy applications from disk images like that into application space in the file system.

There might be an installer that should be run or some drag-me-to... readme file.

That Downloads Folder that's getting full can be inside of that disk image which is limited to a whole lot less than the total disk capacity.

If Firefox is running it should show up in the dock. If so click on it and there ought to be a quit item in the menus.
