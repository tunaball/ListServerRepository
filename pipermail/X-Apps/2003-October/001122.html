Ideally the new Mail.app in Panther will allow you to show custom mail 
headers in Mail.apps Index view.

ie. I filter my mail on the server as it arrives and I can encode 
useful information in custom headers and rather than putting this 
information into the body I'd rather see it in the index view. So it 
would be real nice if Mail.app could associate a list of custom headers 
with each mail folder like it does now WRT to the supported headers 
that you can show on a folder by folder basis.

I just want to be able to make the Index view aware of X- headers that 
I can add to certain mail messages as they arrive.

currently I'm using a mailbox as a log file and I'd like to be able to 
see at a glance what happened with the mail by adding this information 
into an X- custom header and to be able to see this header in Mail.app.
