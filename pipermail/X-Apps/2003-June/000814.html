On Wednesday, June 25, 2003, at 01:30  PM, mac2 mail wrote:

> Is there any way to save or capture a Real (.rm) movie for playback at 
> a later date,

A quick search at the real.com site found this:

  <snip>
1. Hold down the Control button, and click the link to the presentation 
in the Web page.
  2. From the pull-down menu, select the option Download link to the 
disk.
  3. Select the folder in which you would like to save the file, and 
click Save.

If the file appears in the Save As window with the extension .ram at 
the end of the name, it is a random access memory (RAM) file and cannot 
be downloaded. Also, it is not possible to save RealAudio or RealMedia 
presentations that are embedded in Web pages.
<snip>

Otherwise, they state, it is the choice of the author as to whether 
they want the file to be saveable, for copywrite purposes.
