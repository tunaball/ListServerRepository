On 21/12/2003 5:18 pm, "L. Cornelio"  wrote:

> 
> How do I change my preferences so that iTunes does not always open audio
> mp3 files? - I prefer a different app.


Try selecting a mp3 file and doing a "get info" on it.
There is a section called "Open with"
Select your preferred app from the drop down
Just below that is a button that says "change all"
Click it.

A shorter way is to use the contextual menu's "open with".
If you hold down the alt key it changes to ALWAYS open with - I'm not sure
that it changes every file though.

> 
> How do I change my preferences so that iPhoto does not open when I
> connect a camera?
> 

Easy! Select the Image Capture application (usually in the Applications
folder if you haven't messed with it!) In the Preferences there is the
option you want! "When a camera is connected open <pop-up>"

Hope this helps
