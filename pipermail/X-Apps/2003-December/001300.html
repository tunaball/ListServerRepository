Will they both run and not crash each other?
Sure

Can you set it up so when you click  on a link in an email it will open in
Safari?
Yes (do this in the Safari Preferences -> General -> set default browser)

Can you set things up so Outlook Express will open a new email when you
click on a link on a web page?
Yes (do this in Apple's Mail program (go figure) open Preferences -> General
and then set the default mail program to whatever you like).

Is there anything else you'd want to do?  Apple Mail uses the Addressbook
and that does contain URL's which (when filled out) will appear in Safari as
a bookmark menu and this would not be there if you use Outlook but that's
pretty trivial.
