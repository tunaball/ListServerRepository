I'll endorse Gretchen's suggestion, I've used UrlManager Pro a lot, tho 
not so much since settling on Safari as browser of choice. It doesn't 
let you synchronize between browsers, unless that is new for OSX, but I 
find it useful to create a bookmark folder specific for each client, so 
when relevant topics or links come up, I can put them there. The spiffy 
thing is that you can then sort and export as html, so making links 
pages or just sending the info to a client is real easy.
