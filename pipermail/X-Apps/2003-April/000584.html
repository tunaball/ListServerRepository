I use URL Manager Pro, but I've been experimenting with Bookit (shareware,
currently in beta), which imports from all kinds of browsers and lets you
organize them all there, and then export the identical file back to each
browser (or to other computers).  I find I keep saving bookmarks in
different browsers instead of remembering to launch UMP and always save
there.

Bookit includes Safari, which is why I got it in the first place, because
Safari does not export its bookmarks at all! UMP can't read them yet,
either. I wanted a way to integrate all the nice bookmarks I'd saved in
Safari for use in other browsers.

Another tool that may help in that latter regard is "Safari Bookmark
Exporter", which just came out on VersionTracker. It's free, and imports
Safari bookmarks and exports them for Internet Explorer, Camino/Chimera,
OmniWeb, or Mozilla/Netscape.
