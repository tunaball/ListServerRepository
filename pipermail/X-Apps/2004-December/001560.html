It would not share an IP address. Every IP addressable device must have its own IP address else it wouldn't be addressable.

The only way that you will be able to print to a printer is to either share it via the PC's printer sharing capability, or to have a printer driver on the Mac and share the printer as a network printer. This would be only if the printer was able to be connected to the network as opposed to your PC computer.
