That file is DEFINITELY used by Internet Explorer. If you are having
problems with IE's prefs, try trashing that file (or moving it elsewhere
temporarily, moving back if nothing changes). You'll need to reset a couple
of other things, like default Mail program, in the Internet Preferences
Panel, but not very much.
