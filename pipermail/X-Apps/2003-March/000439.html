Maybe IE really doesn't work well with all web sites, maybe your bank 
changed something on its site.

As a long-time slagger, I haven't kept up my copy of MS-IE, I'm at 
version 5.2.1. I experimented with a few preferences and had no trouble 
keeping them between sessions. I looked and looked and looked in vain 
for an MS-IE preference file to see where it's stored. Finally, I did a 
cmd-F search on "date modified", sorted the hundreds of files by date 
and discovered that someone made changes to

	~/Library/Preferences/com.apple.internetconfig.plist

at the very time I made my IE preference changes. So, I it could be MS 
is using Apple's Internet preference file for its own purposes. Perhaps 
more than one application is using that preference file and reverting 
the changes you make. I really don't know. Try a web search for info on 
Explorer preference file problems.
