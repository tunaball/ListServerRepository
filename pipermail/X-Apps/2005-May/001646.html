When I first installed Panther OS X 10.3 with Safari 1.2.4, I goofed 
and let Safari run to the Apple site for whatever it wanted to do.  
After that it would not load the IE Favorites.  So I quickly 
reinstalled Panther, did not let Safari run by itself, and had the OS X 
version of IE 5.2 import all my favorites from the OS 9 version of IE 
5.1.  After all that, when I opened Safari, it automatically brought in 
my Favorites flawlessly.

I believe that is the only successful, easy scenario.  All others are 
very laborious.
