Before you launch a virtual PC from the "Virtual PC List" window,  
select a virtual PC, if you have multiple operating systems installed  
such as a version of XP and a Linux version, choose "Settings...",   
select "Network Settings", then "Enable Networking" and choose the  
"Shared Networking" to use the same settings from OS X or "Virtual  
Switch" to assign a different IP to the virtual PC. Most cases you'd  
use Shared Networking.
