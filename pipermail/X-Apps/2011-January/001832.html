In Mail's View menu on Leopard you have the option to view

	Message -> Long Headers *OR* Raw Source.

The first option should be helpful.


By editing Mail's preferences in "Viewing" you can create a Custom way  
to "Show header details": Drag "Attachments" to the headers you want  
to see. Then you'll have more information on the attachment(s) and an  
option to use Quick Look.
