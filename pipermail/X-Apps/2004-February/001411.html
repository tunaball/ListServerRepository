I was putting a PDF form with some javascript together yesterday (in 
Acrobat), and as I tweaked and re-saved it, I noticed the file got 
bigger and bigger every time I saved it. At first, but just a few dozen 
k, but after a while by 500k each save. Fortunately, it occurred to me 
to try doing a "save as" to a new file, and that does seem to reset the 
process, so I got back from a 2 MB file to a 160k file. Don't know if 
that works in every case, but it does seem to help Acrobats problem 
with growing PDF files.
