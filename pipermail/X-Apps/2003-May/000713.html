It sounds as if you have a corrupt file-perhaps the icon is damaged 
or something. The easiest way to remove it, if you can't click it, is 
with the terminal. Open the terminal, and type cd (directory the file 
is in; unfortunately, you'll have to learn enough unix to figure out 
how to get to the directory). Then "rm (filename)".
I can't think of a way to do this with the finder if clicking the 
file crashes the finder.
