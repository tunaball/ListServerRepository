Not sure if this was mentioned already, but Alsoft has begun shipping
Diskwarrior 3 today.  From VersionTracker, changes are:

- Native Mac OS X directory rebuilding for Mac OS X Extended disks, as well
as disks running under earlier versions of the Mac OS.
- Significant speed improvements- support for journaling
- a component that uses a drive's built-in S.M.A.R.T. technology to monitor
drive hardware, giving users the opportunity to back up data when hardware
is in danger of failing.

Supposedly those who pre-ordered it months ago will be receiving theirs
shortly, and those who order today should receive theirs by the end of the
month.
