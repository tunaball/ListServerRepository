> \\Can anyone refer me to a Jag compatible software and/or tutorials to 
> generate a personal web page?
>
> I wish to display personal pictures from my digital camera for "free" 
> viewing

You might check out Stone Design's software <http://www.stone.com/>, in 
particular PhotoToWeb and Create. PhotoToWeb is specifically designed 
to take (nested) folders of photos and create the HTML needed to 
display them. Create is a WYSIWYG HTML generator; build your document 
as you would in a drawing or page layout program and then generate the 
HTML to display it.
