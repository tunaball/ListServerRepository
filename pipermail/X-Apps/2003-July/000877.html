Al D'Amanda <aldamand at comcast.net> wrote:
> How/where do you select a default to open Desktop Volumes always in
> column view ?
> I have checked "open new widows in column view" in Finder Preferences.
> But double clicking on any desktop volume icon always open contents in
> List

That setting ("open new windows in column view") applies only if you've
never opened/closed a particular window before. Once you open a particular
window, changed the view settings, and then closed it, it will use those
settings over the default. Chances are that sometime in the past, you opened
the window for the volume in question, and then closed it while it was in
list view.

The way it *should* work (and I emphasize "should" because it doesn't
always) is that you should open the volume in question, switch the view to
Column view, and then immediately close it. This should "set" the view to
column view.
