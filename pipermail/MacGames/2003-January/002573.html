>I recently upgraded the video card in my G4 tower to an ATI 8500. Now I've
>lost the ability to play several older games. Specifically I now get a black
>screen instead of the game screen. I can hear the video in the background.

Did you also upgrade QuickTime when you did this?  The latest version (6.1)
apparently changes the way that some full-screen system calls work, and
several games now just get those black screens.

If that is your problem, the only fix is for the game developer to release
a patch.  It's apparently a simple fix (several games have been updated
already), but there are probably some games out there that will never be
compatible with 6.1.
