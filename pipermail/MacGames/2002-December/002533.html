Hi Marcus,

Yes, it is possible. You need to find out what ports these games use, and
then make sure that your router is doing NAT and port forwards those ports
to the hosts machine. How this is done depends on your setup. I don't have
an ABS so I can't tell you if it is possible to port forward with this,
but it should be somewhere in most router's configuration utility.
