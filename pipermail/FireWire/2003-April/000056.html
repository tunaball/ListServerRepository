Your new problem sounds like a compatibility issue between drivers

If you formatted the drive using Apple's generic drivers and now you are
using 3rd party drivers, there may be some voodoo going on there which is
causing the problem (at least this is what we have seen as the source of the
problem consistently in the past)

So, you should be using whatever drivers you used when you formatted the
drive
