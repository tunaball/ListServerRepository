Like Eric, I find that it is easiest to set up separate playlists for each iPod and
then synch only to them. We use a single, large Library and each person can decide
which songs to listen to at any one time.

An alternative is to have one iPod synching to an OSX library and another to an OS9
library, Problem with this is the extra space required to have two separate
libraries. Another problem is that the OSX version of iTunes will over time be more
feature-laden than the OS( version.
