There was a script written (CD to Signature) by an excellent script
writer for CTM Development's PowerMail and Apple's iTunes. I have it set
as a crontab to run every minute. If there is no music playing, it
doesn't explode, it gracefully adds the line you see t the end of my
email here. Cool stuff!

I asked him if he would post it to the "Scripts for iTunes With Other
Apps" page. (<http://www.malcolmadams.com/itunes/scripts/scripts05.shtml> )

He thought it was a good idea. Perhaps it will show up there soon.
It can currently be found on <http://homepage.mac.com/wayneb/powermail.html>

There is also a script written for Apple's Mail that does something
similar. I don't know about Entourage and what may or may not be out
there for that. Good luck. Have fun!
