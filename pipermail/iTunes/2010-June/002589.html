Becca,
     There's a decent explanantion of what Bonjour does right here:

http://en.wikipedia.org/wiki/Bonjour_(software)

but in general, it advertises services on a network, such as printer sharing.  I think every Windows network printer out there has Bonjour turned on by default and has Bonjour in its driver package.  I have seen it on Windows print server devices as well.  Pretty cool technology.

If you want things to work "automagically", it's very useful and zero configuration.  If you prefer to do things the slower, harder, more manual way, there must be a way to uninstall or remove it by editing the Windows registry.
