It's been a while since I imported my mp3s, and I only had to do it 
once, so I may be recalling this incorrectly:

File --> Import. From with the Open Dialog Box, if a folder (directory) 
is selected, iTunes will examine it and recursively examine all 
sub-folders, importing every appropriate file it can find.
