I find that in iT4 this preference set to zero no longer perform 
premature crossfade as it did in previous versions. This setting is 
best for all music because breaks between tracks are built into songs 
that are meant to have breaks at the end of each track. Compilations 
and symphonies now play seamlessly with no breaks nor premature 
crossfade in iT4 for me. The delay mentioned below is an unusual 
situations involving a server that the writer admits happens only 
occasionally. I think Apple has fixed the problems with this setting in 
iT4.
