> When you delete a song from the library it is only an ALIAS you are

This isn't true, Kunga. As long as the file is in your designated music 
folder, you have the option for iTunes to move it directly to the 
trash. If the song is being accessed via a playlist, holding down the 
option key will bring up the option dialog. OTOH, if the file is _not_ 
in the designated music folder, only the iTunes entry will be deleted 
as you stated.
