My wife and I share a library as well, but we only use one computer  
for synicing iPods. iTunes will manage each iPod separately. It just  
works.
I have tried to access an iTunes library on one computer from another  
over a network, but never had much luck. The "guest" cannot write to  
the library files, which makes iTunes unhappy.
It may be possible to do what you want, but to make it easy, just use  
one computer for both iPods.
