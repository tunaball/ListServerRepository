Try this: In iTunes, select all the CDs relating to the book. Then do  
Command I (Get Info). Click on "Info" tab. Down at the bottom, notice  
a check box next to "Part of a compilation". Check that box, since you  
have all items of that book selected, it will change the selection for  
all at once. Test and tell us of that makes a difference.

Another thing to try if the first suggestion doesn't work. Look to see  
the difference in file type between the two (i.e. on is mp4 & other is  
mp4a). If there is a clear difference between the two, try renaming  
the files that don't work to the format of the ones that do. Let us  
know.
