1. I believe Audion saves playlists as separate files but itunes does 
not.

2.  For backing up iTunes Library file, I just copy the "iTunes 4 Music 
Library"  file to another drive or you can also duplicate the file and 
put the duplicate in a different folder on the same drive.

3. FYI  I routinely move iTunes Library files around between a number 
of Macs in my house and they work just fine - right down to the 
statistics (Play Count, Rating, Last Played date). the only caveat is 
that you have to be very disciplined  in maintaining a valid path from 
iTunes to where your music is stored (I use a separate HD  on a 1.42G 
G4DP that all my other machines access).
