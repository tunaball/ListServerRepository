Ted,

If you drag the iTunes folder into the Library of iTunes, it should 
only import the songs that are not in iTunes already.   You should not 
get duplicates from this process.  You can then list the Library by 
"Date Added" to see what new songs (old songs that have been re-added, 
in this case) there are.

Dragging a folder of music that is not the iTunes library folder will 
cause duplication.
