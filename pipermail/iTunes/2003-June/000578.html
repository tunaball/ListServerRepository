On Thursday, June 5, 2003, at 08:16  AM, iTunesList wrote:

> I presume I'll lose all my playlists.

On the Doug's Applescripts site you can download a set of Scripts 
called "Restore Library 2 Step".  The first one will write the names of 
Playlists associated with a song in the comment field.  Then when you 
delete the Library file and reimport the files, you can run the second 
script, which will rebuild your playlists.  It does not work for Smart 
Playlists, but those are pretty easy to remake.
