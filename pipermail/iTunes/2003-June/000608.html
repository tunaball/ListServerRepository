> How do I share my library with another user on the same computer,
> without copying all the songs?
>
> I've set up the library location for the second user as the main user's
> iTunes folder, but nothing shows in user 2's library.
>
> I'm sure there is an obvious way to do this, but what is it?
>
> Thanks
>
> Andrew

In the Preferences>Advanced Menu, set a common location.  It might be a 
network share, it might be local on one computer, you decide.   Oh 
wait, that is a network question.

OK, quick answer.  be the main user of the iTunes, set permisssions of 
the main user's directory accordingly, then as the secondary user, use 
the advanced menu to not use the secondary local account, but the one 
of the main user path.

I run mine off of a network so others can chime in on local user versus 
secondary user.  This is all very do-able if permissions are set 
correctly I would think.
