Hi Becca:

You need to use iTunes to convert the tracks to AAC.  Select the  
tracks and choose "Convert to AAC"
under the "Advanced" menu.  This will not destroy your mp3 files it  
will create new mp4a files.
Once this is done fine the files that were converted and change the  
mp4a to mp4b.
This will make the files bookmarkable.  For any Mac subscribers there  
is an apple script that does this
automatically, but with the PC I think you need to change mp4a to mp4b  
manually

http://dougscripts.com/itunes/scripts/ss.php?sp=makebookmarkable
