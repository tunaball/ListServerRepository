Hi list,
Came up with what I think is a good way to combine two libraries from
different computers which have many songs the same.

I first synced my smaller collection to my larger one using ImSafe,
checking, ignore if file already exists, and then dragged my new coalesced
folder onto itunes which only updated newly added songs.  It seems to have
worked pretty well with no duplicates that I notice so far.

Hope this helps.
