On Mar 3, 2007, at 12:48 PM, listLover wrote:

I keep my music divided into two libraries.

i would like to have the "copy song files to library location" work  
when 1 library is loaded, but not when the other one is.

I think I'm SOL, but curious is anyone knows if there's a way to do  
this.

The current version of iTunes lets you create and choose a library.  
Start iTunes with the option key down and create or choose a library.  
If you have the same song in both libraries, you can use iTunes ->  
Preferences -> Advanced and deselect "Copy Files to iTunes Music  
Folder..." so that the songs will stay in one place.
