Have you tried dragging the file into a Toast window and done a burn  
from there?

If you don’t have Toast, you could try it from Disk Utility. Create a  
Disk Image in Disk Utility on your Desktop the same size as your file  
(or perhaps a tad bigger), make it write a read/write disk image.  
Mount the image by double clicking it and copy your file into its  
window. Then burn the disk image in Disk Utility. It might work.
