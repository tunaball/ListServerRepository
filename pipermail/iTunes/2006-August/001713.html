Mildred, you don't mention what computer you have, or what operating 
system. Vague questions will only produce vague answers, but  I'm 
willing to guess a little.

I have an iMac G5 running OS 10.4.7. I've never seen a folder named "My 
Pictures" on any Macintosh computer. I'm guessing that you have some 
version of Windows (XP, ME, 98). iPhoto is a program that is bundled 
free with Macintosh systems, and is not available for Windows machines.

You may need to read the manual that came with your iPod to find out how 
to transfer pictures from a Windows machine to an iPod (assuming it can 
be done). You may be able to get help here, if you specify what computer 
and operating system you are using.
