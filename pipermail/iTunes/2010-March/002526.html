On Mar 17, 2010, at 12:48 AM, Becca wrote:

> I've got my son's computer set up with iTunes, and family sharing is set as allowed in his permissions. I've got my computer set up similarly.
> 
> I can see his iTunes library from my computer, but not my library from his -- and while I can see his library, I can neither copy to or from it.
> 
> Is this the way sharing is supposed to work, or do I have something set up wrong? I'd like to be able to put music from my computer onto his iPod.

Make sure that on both computers, in the Sharing prefs, you have checked the options to both share and look for shared libraries.

Then, read the help about Home Sharing and set that up correctly; it's not the same as seeing his library (and being able to _play_ his music).
