Terry:  I have purchased several audio books on cd and ripped them  
into iTunes, joining the tracks.  I found an apple
script named "Make Bookmarkable" at http://dougscripts.com/itunes/   
This installs a script under the script menu in iTunes.
You then just select the ripped book and run the script.  It changes  
the suffix to m4b and installs it in the audio books
folder.
