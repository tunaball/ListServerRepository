> Bianca was writing about the album's folder - not it's listing in 
> iTunes. You need to have the Create file names with track number 
> checked in the Importing Preference Pane to get that part straight. 
> File names are different than the file tags that keep things in order 
> within the iTunes interface.

I can't recall if the track number column is automatically populated 
when you import, but you don't need that column, nor do you need to 
incorporate the track number into the file name. Another way of 
handling it is to choose the Date Added as the sort criterion, then 
reverse the sort. Of course, that's only useful if you are filtering on 
the album name.
