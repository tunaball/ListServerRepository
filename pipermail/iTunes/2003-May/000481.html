I have always found the "date added" column useful for weeding these 
kinds of things out of the library.  All of the duplicates should have 
todays date and about the same time, so it should be easy to sort by 
the date added and then select either all the old or all the new stuff 
and delete it.

There are some Applescripts for removing duplicates and for removing 
listings that don't have a file anymore.  I don't know which situation 
you are in exactly, but they may help as well.
