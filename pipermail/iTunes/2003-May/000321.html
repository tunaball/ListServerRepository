Tony wrote:
> An item on my iTunes wish list has been keywords [sub-genre] akin to
> iPhoto's keywords.

I would suggest instead using the "Comments" field.  You can then use 
the "contains" option for the smart playlists, and you'll be able to 
add an arbitrary number of keywords to each song.

That way, for example, you can mix genres.  For example, "Weather 
Report" could have both "rock" and "jazz" in its comment field and 
Shania Twain could have both "pop" and "country" in their comments.  
I'm sure you can see how this can be much more flexible and easier to 
read.

Hope this helps,

	-- Todd
