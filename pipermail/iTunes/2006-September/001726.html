Ah, this one I can do because I do this all the time. Go into your [old] music folder --> iTunes then copy your iTunes Library and your iTunes Music Library.xml over to your new iTunes folder & voila! just like the move never happened.

The one thing I would do first is make sure I am running the same version of iTunes on each machine to ensure compatibility between the items being moved. I don't know if this matters, but in case it does, doing so would avoid the issue.
