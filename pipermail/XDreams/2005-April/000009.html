To save time with different sharing and port configurations, I propose that
the Sharing attributes be saved within a location set up by the user. You
might, for instance, have three variations of a "Home - DSL via Airport"
location with differences in the Sharing configuration (perhaps one could be
set up to allow Remote Access, another no sharing at all, etc.).
