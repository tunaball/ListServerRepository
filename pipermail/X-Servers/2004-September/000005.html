The KDC requires DNS to have both forward and reverse lookups setup  
correctly, did you set up a reverse record?
Here's the bad news, once that KDC fails to start, about the only way  
to fix it is to reinstall OSX Server. Actually, I have not seen a way  
to fix this yet without reinstalling the server operating system. I've  
called several people at Apple and received the same answer.
