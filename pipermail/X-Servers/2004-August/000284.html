> For what it's worth, I finally figured out what my problem was.  The 
> hwmond daemon wasn't running.
>
> From a discussion in the Apple X Server forum, I added the line
>
>     hwmond:respawn:/usr/sbin/hwmond # Hardware Monitor daemon
>
> to the /etc/watchdog.conf file. Now Server Monitor works!
>
> Note that this was a clean install (twice!) of OS X 10.3.3.  Why 
> wouldn't hwmond be included in this file (or wherever it belongs)?
