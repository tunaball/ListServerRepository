The best means to upgrade the server is to join the maintenance 
program. You should be able to do it at any point prior to the release 
of the next OS and get 3 years of upgrades to the server OS. It is the 
same price as 1 copy of the server software. This is basically a hidden 
cost in my opinion, since it really is a requirement with the frequency 
of the updates and the fact that previous versions of the OS really are 
not being updated even for security concerns. Even so, the Xserve is 
still a very nice deal for a server.
