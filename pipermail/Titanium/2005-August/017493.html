What you're asking for is a Network Attached Storage which is a "server"
dedicated to providing nothing but storage. Another type of system is a
"Storage Area Network" using fiber switches connected to drive arrays
specifically made for network storage and which appear as addessable
storage.

What you've concluded is correct. You need a server of some type dedicated
to "sharing"" a storage device over the local area network.
