> So there's no way to connect my Airport Extreme Base station to my  
> firewire ac-powered hard drive (no adaptors, etc.)?

Hard disks have file systems which require an operating system of  
some sort with the proper file system drivers.  The AirPort Base  
stations do not run an operating system with HFS+ or UFS file system  
support.
