No, this is incorrect - all PowerBook G4s and white iBooks use the  
same size plug.  The PowerBook G3s and colored iBooks use the larger  
plug.  The confusion comes from the fact that the early G4s shipped  
with the same shape "yo-yo" (round) power adapter as the G3s, but  
with the smaller plug.  Later G4s came with the square white brick  
that is still used now.

It is correct that it will work fine, though with a few minor issues  
besides just taking longer to charge - it is (in theory) possible to  
use enough power that you will actually deplete the battery while  
using one of these older power adapters.  In practice, I've never  
actually seen this.  However, because of this possibility, the  
computer will not boot or wake up if the battery is very depleted:  
instead, you'll have to let it charge for about 10 minutes first.

Trust me on this, I'm a former Ti/500 owner, a current 1 GHz 17"  
owner (and I still use a square brick that I got for my Ti with the  
Al), and an Apple-certified technician. ;-)
