On  12/18/02 23:21, MSwrote:

> This may sound a bit peculiar, but if anyone knows what files to delete
> in order to get rid of all the auto-fill-in web sites that pop up in
> the browser's address line

Some days ago someone posted the same question in another forum I visit. I
answered with this excerpt from a message posted on the Usenet by user
Richard Grieve:

<quote>
If you're using Classic, I'd kill the following...

Mac HD:System Folder:Preferences:Explorer:
Mac HD:System Folder:Preferences:Internet Preferences

If you're using OS X, the equivalent files are...

/Users/username/Library/Preferences/com.apple.internetconfig.plist
/Users/username/Library/Preferences/Explorer

NOTE: the Explorer folder on both pathways above contains your
Favorites.html file (your bookmarks).  Make sure to copy this file to a
safe location first, then drag it back into the corresponding location
once IE has automatically regenerated a new Explorer folder.

<end quote>


Hope it helps.
