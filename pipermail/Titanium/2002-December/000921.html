On Fri, Dec 13, 2002 at 08:09:29AM -0800, Douglas Hawkins wrote:
> Being NEW to Mac networking, I was wondering if a MAC
> GEEK would mind informing me if X uses the base model
> of the OSI layers to encode and decode information. I
> ask this because I have ran across a game ( Star Wars
> Galactic battle grounds and Clone wars) that even
> though the game is the same, I cannot play against PPL
> useing wintel machines With my MAC. I have written
> ASPYR to get more info and ask for a patch, But they
> havent written back yet. Any Info is  appreciated.
> 


Yes, the TCP/IP stack in OS X adheres to the OSI 7-layer model.
The incompatibility you're facing is probably somewhere within
layers 4-7.  I.e., it's something to do with how the game
deals with client-server interaction, rather than how OS X puts
bits on the wire and adheres to common network protocols (layers 1-4).
