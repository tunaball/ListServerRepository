on 12/1/2002 3:26 PM, Charles J. Thompson at charlesjt at cox.net wrote:

> It more likely means that Readiris is a Cocoa application,  which is
> defined by Apple as follows:
> "The Cocoa application environment is designed specifically for Mac
> OS X-only native applications. "

That is not necessarily true -- most likely the app is Mach-O, which is Mac
OS X-only.

Whether they use the Cocoa APIs, Carbon APIs, Java APIs, or AppleScript APIs
really makes no difference -- the only real difference is whether it was
compiled for Mach-O (which will only run on Mac OS X) or CFM (which will run
on Mac OS X and Mac OS 9).
