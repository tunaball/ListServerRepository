On  Sat, 21 Dec 2002 21:22:05 +0800 Don Jose Romulo Davies
<davies at quickweb.com.ph> wrote:

> I upgraded to 10.2.3 on my Ti-500. I encountered no problems during
> setup, however, i noticed that everything runs slower, from booting to
> launching apps. What can i do?

After the first boot, it is normal to have a very slow system.

Run the Permissions repair in the (freshly updated) Disk Utility. Also it
would be wise to run both the /sbin/fsck -y from shell and the sudo
update_prebinding -verbose -root / from terminal.

I have never had a problem detected in the shell routine, and this time was
no different. But after the prebinding repair and the permissions repair I
find that my system is running about the same as it did before the update.

About fragmentation: I always run DiskWarrior from OS 9 every month or so
and no matter what anyone says, I find that things run much more crisply
after rebuilding the Volume Directory - both in OS 9 and OS X. I can't wait
until the OS X version of DiskWarrior comes out.
