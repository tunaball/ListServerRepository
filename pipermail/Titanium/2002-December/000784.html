>Is there a way in OS X 10.2.1 to set a default application to startup when
>opening from a file of a certain type? Thanks for your input.

Git info on the document; go to Open With; choose the app you want. 
If you click "Change All" it will set that as the default app for all 
similar files. For example, the default app for .JPG is Preview, if 
you select a .JPG file and select Photoshop, then select change all, 
every .JPG file will open in photoshop instead of preview when double 
clicked.
