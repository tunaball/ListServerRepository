All you guys who keep advocating that OS X should be ported to x86 are 
forgetting something, I think.  OS X would run like a POS on x86.  Case 
in point - ever tried the x86 port of Solaris?  OS X is an operating 
system that is designed for high-performance hardware, as is Solaris. 
The x86 port of Solaris is proof-of-concept and nothing but a learning 
tools release for those that won't buy the native hardware.  x86 belongs 
to Windows.  I'd leave it that way.
