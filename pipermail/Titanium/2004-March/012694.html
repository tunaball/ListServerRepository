>Just got an Al 15" and keychain password prompt keeps popping up 
>with Safari, Eudora, etc.. How do I turn it off?
>

Tom, Sounds like you're not happy with the default Safari settings. 
Go to the Safari Preferences' autofill tab and uncheck the "other 
forms" box.  If you like, check the User names and passwords if you 
want that sort of functionality.  Eudora should only require 
authentication once.

Hope this helps,
