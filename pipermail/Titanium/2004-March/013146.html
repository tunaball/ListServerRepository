hundreds of apps will do this, im not sure which one holds the crown but:

if you drag a WAV into iTunes, then select the file and go to Advanced > 
Convert to AAC, that will make a file that is comparable to an MP3 with 
better quality. DVD Audio just selected this format for the future so it's 
not a bad thing to get used to but I'm sure others with have varying 
opinions.

also a program called "shorten" does it.
