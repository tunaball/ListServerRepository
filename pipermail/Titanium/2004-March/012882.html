The reference file doesn't contain the movie, only references to the 
clips. That's why it's so small. Dragging the reference file into Toast 
makes the video clips available in Toast. I'm not familiar enough with 
Toast to know whether it copies the actual clips or just uses the 
references.

Note, however, that Quicktime files can also be self-contained movies 
where all the video is within the Quicktime file itself. But that's not 
the case with iMovie projects.
