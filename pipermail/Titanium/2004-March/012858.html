The project files don't work with Toast. The reference file it speaks 
of is the Quicktime movie file. Each iMovie project directory has a 
project file, a movie file (reference file), and a media directory that 
contains the individual clips that make up your movie. Try dragging the 
movie file into Toast - that should work. Alternatively, you can go 
into the media directory and drag individual clips into Toast if you 
wish.
