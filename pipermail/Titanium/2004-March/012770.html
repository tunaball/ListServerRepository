WindowServer is one of the core services of Mac OS X, it's the parent 
of the Finder and other GUI applications. As it's name indicate, it's 
the window server part of Quartz Compositor, the layered compositing 
engine that lets Mac OS X share pixel on the screen between several 
windows in real time.

Try to log out and log back in with as few windows as possible. Are you 
using Fast User Switching? You could also have a look at the preference 
files (in ~/Library/Preferences/ByHost and /Library/Preferences). Good 
luck,
