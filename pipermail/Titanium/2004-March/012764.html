Knowledgebase I assume, but would appreciate more information :-))

I searched with Google and ended up with this answer from 
<http://forums.macrumors.com/archive/topic/59082-1.html>:

" WindowServer draws everything you see on the screen. It acts as a 
mediator between applications that want to draw on the screen and the 
graphics hardware. When an application wants to draw something, it 
establishes a connection with WindowServer, tells WindowServer what to 
draw, and WindowServer handles the drawing / compositing / Quartz 
Extreme hardware acceleration to finally produce what's seen on the 
entire screen."

So no virus and don't delete it since it seems to be an essential file 
if your computer have Quartz Extreme. (Mine doesn't so I can  perhaps 
feel less envious knowing that I don't have that function slowing down 
my system. ;-)) *grin*
