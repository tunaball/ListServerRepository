On Jan 4, 2005, at 9:21 PM, Dick Grable wrote:

> OK, I give up. How does a person access SWAT as root?

SWAT = Samba Web Administration Tool.  Therefore it's accessed via the 
HTTP (web) protocol using a web browser on TCP port 901.  127.0.0.1 is 
the loopback address (localhost).  The :901 part tells your web browser 
to query port 901 instead of the standard TCP port 80.

By using the address: http://1127.0.0.1:901 in Safari (or what ever 
your favorite web browser is), it will communicate with SWAT.  A little 
authentication dialog will automagically slide out of the top of the 
window.  Where it says "name" you type in "root".  Where it says 
"password" you type in the root password for the system.  If your 
system doesn't have the root password enabled, you open a Terminal 
window and type in 'sudo passwd root' to set a root password for the 
system.
