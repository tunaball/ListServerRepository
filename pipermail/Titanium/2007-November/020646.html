I found this a useful list of features that have been lost in Leopard:

			http://leopard-lost.blogspot.com/

miss the feature of knowing the connection speed.
I wonder why this feature was dropped?  is there any workaround?

I had also found that the Leopard Address Book no longer enables me to  
send SMS from within it, nor to dial up with Skype.
It seems that a number of the useful functions of Address Book have  
been removed.  I cannot imagine why this was done.
How do we campaign to get these features back in an update?

So for those of us with laptops on the move, Leopard has apparently  
lost some very useful functionality, for no obvious reason.
