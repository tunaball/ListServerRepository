> You should do a fresh installation of OS X.

BEFORE you do that -- check to make sure that the machine isn't "waking up"
when the screen is closed.

I just dealt with this situation.  Every time I had the machine plugged in
and closed the lid, the computer would wake up and go to sleep every second.
When it was waking up, it was looking for a screen to display on, wasn't
finding one, sending a "screen resize" of (0,0) to all the apps, which then
crashed because they ended up doing a divide by zero.  If I had the screen
saver up (and requiring a password to wake up), things were better because
the screen saver doesn't really care about the size.

The *actual* problem?  The lower right of my keyboard was flexing up *just
enough* so that one of the keys could be pressed by the closed screen,
causing the machine to wake up.

Solution: "straighten" the keyboard in that area so that the screen didn't
hit the keys.
