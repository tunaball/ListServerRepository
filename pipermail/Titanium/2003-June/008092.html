I wish there was, but we've discussed this before.  The idea is to reverse
the action of the "fn" key so that they will function like "regular"
function keys when the "fn" key is _not_ pressed, and do the rarely-needed
powerbook hardware stuff (volume, brightness, etc.) when the fn key _is_
pressed.  There is only one way to do it (thanks to Joao Carlos de Pinho):

Boot into OS9
Control Panels
Keyboard
Function Keys
Check: "Enable Hot Function Keys"
Uncheck: "Use F1 thru F12 for Hot Function Keys"

I submitted this to submitted to http://www.apple.com/macosx/feedback/, and
I hope they fix it someday.  Otherwise I'll never be able to buy one of the
new powerbooks that won't boot into OS 9.
