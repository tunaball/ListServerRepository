You may want to take a look at this document, which
provides some very basic info on what you want to do.

http://docs.info.apple.com/article.html?artnum=19652

It's fairly easy to shares files.  If you're running
10.2, you can share files via SMB to and from the
powerbook and the desktop.  If you're running 10.1.X,
it has a SMB client built-in, but not a server.

As for printer sharing, you may not have very good
luck.  Windows 2000 has Unix print services, but 98SE
doesn't.  You could take a look at a program like
Dave.  Depending on how technical you are, there's
some other options as well.  Something for instance
like:

http://www.xlr8yourmac.com/OSX/cups_printing_in_jaguar.html
