Hi Steve,

I've got WDS working here. The key points are:

1. Both base stations need to be the Extreme versions.
2. Both base stations need to be on the same channel
3. Both base stations need to have the WDS option enabled ("extend 
wireless network (using WDS)")
4. Each base station needs a unique IP address

Hope this helps,
