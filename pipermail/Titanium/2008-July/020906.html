Read,

Trevor's memory was accurate. In 2004 a member of this group provided  
a web address (http://superdrive.cynikal.net/) from which the user  
could download patches able to update the firmware in UJ-815 and  
UJ-816 superdrives released before April 2004. The instructions were  
somewhat elaborate but the superdrives were rendered region-free. PAL  
format was not specifically mentioned, but apparently those disks  
played also.
