John - I think what Alsoft say is that if you 
have a bootable CD for your currrent Mac, then 
you can update it to another bootable disk which 
will also have the DW3.3 application.  The 
updater does not change the OS X version, only 
the Disk Warrior application.

I hit that problem myself when I moved from my 
(stolen) Ti-book to this Al-book, which only 
boots with 10.2.something.

I think in this case (update to 3.3) you have to 
update the DW application to use it with Panther, 
irrespective of whether the resulting CD you make 
is bootable or not.

What I would do in your case is perhaps boot from 
an external FW drive, then use the new 
(non-bootable) CD to run DW to fix the boot 
disk/partition on your Powerbook.
