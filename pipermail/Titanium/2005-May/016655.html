This is now totally seamless with Windows.  All you need is Bonjour in 
the Windows box, which you can download here (look for the Windows link 
on the lower right corner):

http://www.apple.com/macosx/features/bonjour/

This provides ZeroConf networking on the Windows box, which Macs 
already have.  The two machines don't even need to be on the same 
workgroup, all you need to know is the machine names.  Say yours is 
Trevor's PowerBook, and the Win box's name is Dell-Laptop.  In Finder 
you use Connect and type in:

smb://Dell-Laptop.local

On the Win box in Windows Explorer you type in:

\\Trevor's PowerBook.local\

It "just works" with Bonjour.  Without Bonjour, the setup on the Win 
box to connect computer-to-computer with SMB is non-trivial.
