John;
Keep in mind that this will put back any system corruption you might have. I
would do a clean install of the OS and only copy back your data files. That
way you will have eliminated any possible source of corruption. It is
entirely possible the operating system or preferece files became corrupted.
Tech Tool does not test for these problems, nor does Apple's Disk Utility.
Good luck. Let us know.
