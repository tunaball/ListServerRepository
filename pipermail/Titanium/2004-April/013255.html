Christopher,

It was a damaged preferences file for KeystrokeRecorder X. But if you need a
"click" instead of a "boop" you can try this:

System Preferences > Universal Access > Keyboard > Slow Keys ON, ³Use click
key sounds² checked (and set to short).

Not quite as annoying as the booping I had, but close!
