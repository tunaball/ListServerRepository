On 15 Apr 2004, at 21:53, TM Buffalo wrote:

>   and it keeps asking for my password and will not accept the right one

this sounds a lot like it could be a problem with your mailserver or 
connection to it. Sometimes if the connection gets interrupted or the 
mailserver does not respond incorrectly, mail user agents (incorrectly) 
assume this is a password issue....

Try connecting to your mailserver using another mail application 
(Mozilla, Entourage etc) and see if you get the same problem. If you do 
it's this issue.
