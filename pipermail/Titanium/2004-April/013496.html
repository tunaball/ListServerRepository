On Apr 20, 2004, at 1:39 PM, Joseph W. Chance wrote:
> Can anyone tell me how to fix this one.  I keep my iTunes library on 
> an external drive.  While importing some CD covers, it (Panther) 
> crashed.  When I re-booted, my iTunes Music Library file is no longer 
> valid.  I tried making a new alias and moving it to the User/Music 
> folder, but it still says that it is not valid.  Along the way, 
> something happened to the music library folder on the external drive.  
> Any suggestions how to return the playlists to their proper working 
> order.  The music is still there, just the library file does not work. 
>  Thanks.

You can delete the library files, restart iTunes, and it will make a 
new one.  But you will lose the playlists you've made and have to 
re-add the music to iTunes.
