Yes.  The Save command saves a list of all processes, plus the various 
items from the tabs at the bottom of the window.  It gets saved as an 
XML property list.  It's not AppleScript-aware, so if you wanted to 
schedule it regularly, you'd have to use something like QuicKeys or (if 
you have the patience to learn it), GUI AppleScripting, which is much 
more difficult than just a plain AppleScript.
