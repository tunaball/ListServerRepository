> This has caused me to lose my playlists *UGH* many times, but I now 
> backup
> the library file weekly.
>
> However, there is now a built in recovery tool of sorts.  Move your 
> library
> file out of the itunes folder (in ~/Music/), and also move out the 
> itunes
> music library.xml file.  Open itunes, it will be empty.  Go to file, 
> import,
> and import the xml file.
>
> All will be well.
