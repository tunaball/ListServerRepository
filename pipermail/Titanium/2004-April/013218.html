On Apr 4, 2004, at 6:36 AM, Murray E. Milligan wrote:

> I need to know please, where I can find the Safari favourites and Mail 
> mailboxes I currently have so I can save them to CD-ROM and re-insert 
> them into the newly installed 10.3.

They're in the Library (in your user account).  The Safari bookmarks 
are found at:  library/Safari/Bookmarks.plist.  You can save this file 
and then replace the new one.  (You can also sync the bookmarks on two 
computers by copying this file into the Safari library folder on 
another computer.  I don't know what the mailbox files are called, but 
they would be in library/Mail.
