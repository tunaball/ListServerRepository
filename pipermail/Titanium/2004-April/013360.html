Tarik Bilgin typed this message on 4/16/04 9:46 AM:

> I am writing up some documents atm, and as usual pulling my hair out
> with MS Word and instead using Textedit.
> 
> Textedit does the job -- but I find it a bit limited.
> 
> Can anyone recommend a:
> 
> 1) Decent word processor with a nice Cocoa GUI?
> 
> 2) A Tex front-end (Texshop was recommended) which could help me to
> learn Tex. Failing that a pointer to some help files or a book would be
> good.

This is where PathFinder shows its true worth. It has a decent built-in text
editor and pdf viewer among other tools. It is really a Finder replacement
(actually a Finder supplement) but it replaces many more things as well.

Also, Tom Bender's Tex-Edit has a lot of neat utilities built in as well and
has been around for about as long as I can remember. It comes with some
pretty handy scripts as well and just keeps getting better and better.
