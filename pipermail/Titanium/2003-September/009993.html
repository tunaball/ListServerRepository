As with many things, and as you probably know, you would be in better shape 
if you had something set up beforehand. There is also a case of someone 
recovering a iMac with Timbuktu. 
<http://www.wired.com/news/mac/0,2125,50025,00.html>, if you had that 
installed. If they are stupid enough to use your dialup account, you might 
have an opportunity to trace the call with cooperative ISPs and police.

The IP address is variable anad assigned according to the network the machine 
is connected to. It's ethernet address (MAC address) is fixed and available, 
but it is only visible in the local broadcast domain. They would have to 
connect to the same network you were on for you to see it.
