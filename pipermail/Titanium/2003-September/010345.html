So far I have followed a personal procedure before each update and 
(luck?!) never got problems:

I run Diskwarrior and repair privileges *before* any Update. I only use 
COMBO updater, after the update I repair privileges again.

I have a Ti 667MHz and apparently the 10.2.8 main problem appears with 
dual G4 desktops. Other models (laptops as well) show minor glitches 
that - honestly - I have seen all the time with people using the 
incremental update.
I believe the incremental would work flawlessly with an untouched 
system, with no third party add-ons and with all system files that will 
not undergo any change not modified and/or *corrupted* in any way. The 
COMBO is a sort of force-update that - my experience - have kept so far 
all minor glitches away.

10.2.8 here alive and kicking.
