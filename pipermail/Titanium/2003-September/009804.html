At 10:43 AM -0400 2003.09.06, Tom Klinkowstein wrote:

>Any way to make the the names of folders and files appear in a smaller font size in X?

Open a folder, perhaps your hard disk icon. Select "View Options" from the View Menu. Change the Text Size there. Make sure that you have selected this setting to apply to "All Windows" in that View Options dialog.

You can individually change the view options of certain folders by first checking the radio button for "This Window Only" and then making changes. These changes won't be reflected in other windows.
