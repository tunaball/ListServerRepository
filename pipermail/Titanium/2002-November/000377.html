on 02/11/30 08:39, Dan Zawadzki at danz at inin.com wrote:

> Does anyone know if there is a way to disable the embedded TiBook function
> key mappings for F1 through F6?  I want to use these keys in other
> applications but the Mac firmware always takes over and uses the default
> screen and audio controls unless I press the function key to null the
> default settings.
> 
Dan,

I don't know a way to disable them, but that's what the "fn" key in the
lower left of your keyboard is for.  If you hold down "fn" while hitting a
function key, the signal will go to the frontmost application instead of the
screen, audio, etc.  At least, it works that way for me.
