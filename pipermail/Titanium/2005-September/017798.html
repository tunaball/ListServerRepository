With the Apple firmware no.  If you get an 845 from somewhere like OWC with
the straight factory firmware, yes it does.  I installed one in my Ti 1
GHz...works great. 

There is no firmware flash available yet for the 845 drive, but work is
being done on this so that there will be both a full factory firmware
version you can flash it to and a region free version as well.  There will
also be, as usual, a firmware flash to return it to Apple specs.
