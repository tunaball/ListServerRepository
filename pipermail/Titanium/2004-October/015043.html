I will bet he is not able to see the HD when booting off the install CD to
do that.  Try cmd/s on booting and see if you can run fsck.
Otherwise, I would get a copy of Disk Warrior, boot from it, more often than
not it will see a HD in this condition where it will just not mount, run it
and then reboot.  
This should repair it enough to boot.  After that, run cocktail or at least
disk utility and if all goes well, it should be fine.
I have had this same thing happen one time and this did the trick.
