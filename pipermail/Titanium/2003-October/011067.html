Gosh Frank.  There's really nothing to be nervous about :-)  If you 
have everything you think you might like to have backed up on the 
external drive, the reinstallation of a fresh Mac OS is really simple 
and straight forward.  Much easier than Windows, IMHO.

I did a fresh install on my 800 awhile back because I had Debian PPC 
Linux on one partition, got tired of playing with Debian, and wanted 
the partition back so I could put OS 9 on it.  Except there was no 
driver partitions for OS 9, so that was impossible.  I ended up just 
copying my files to the network server, deleted the partitions, and 
started over fresh with the software restore CD.  When I got it running 
again and logged on, the first thing that popped up was the Software 
Update.  So I did that, copied my stuff back from the server, and was 
back where I started except no linux install and all my disk is now 
available to Mac OS X/OS 9, which are now side by side on the same 
partition.

Worked great, and was really simple.
