Anyone else played with Panther's implementation of Panther?

It seems so much more straight forward.

I have my SE T68i set for Bluetooth on, plug in the D-link BT adaptor,  
fire up Address Book, and "boom" the little bluetooth symbols lights up  
blue. Before under Jaguar I had to go through all sorts of pairing  
nonsense, sometimes even after waking from sleep. Now it seems quite  
automatic.

Still learning about Mail too and little things, like clicking on the  
reply arrow to a message you have replied to, and it takes you to that  
message you replied with - not sure if that is clear, but it is very  
helpful.

More little things being discovered a long the way...
