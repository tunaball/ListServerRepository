hehe with all the AOL bashing nobody answered his OS 9 dilemma. When 
you initialize to do an OS X install you have one chance to get this 
part right. Under the erase tab in Disk Utility there is an option to 
install Mac OS 9 Drivers. Without this checked you can not boot from an 
OS 9 install disk and install onto your now OS X only hard drive. I'm 
afraid the only option I have found is to redo the initialization. 
Maybe someone has an easier solution as I have had a few customers do 
this and then stick me with thew complete redo. Good luck.
