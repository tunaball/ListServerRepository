I think the problem is that your Library folder is in the User folder. 
Inside HD/Users you should have a Shared folder, your username folder, other 
usernames and that is it. The Library folder resides in 
HD/Users/yourusername. YOur folder is one level up and that will cause 
problems.
There is also a Library folder on the HD level but that is different.
The finder will let you just move the Library out of your username folder so 
maybe you moved it accidently.
Move it back, logout, log back in, (or restart) and then try it.
