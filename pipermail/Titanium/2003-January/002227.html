>>Any professional photographers out there using a PB to download images while
>>travelling?  What are you using to download and browse your photos prior to
>>Photoshop?  iPhoto, Photoshop Elements, iView Media Pro or what?

Photoshop 7 has an excellent built-in File Browser;  one of the nice 
things about it is that it saves changes and attributes to a file 
that is saved when you archive the images... that way if you, say, 
burn your images to a CD, you can pop the CD in and see the same 
changes/attributes in the File Browser, yet the original images are 
archived.
