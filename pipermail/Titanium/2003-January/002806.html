Did you know....

that if you connect a second external display via VGA to your G4 400 Ti 
running 10.2.3, that Apple refuses to launch with the error:

"DVD Player encountered a serious error. The current machine or system 
configuration is not supported. [-70013] "

If you disconnect the display the problem goes away :)
