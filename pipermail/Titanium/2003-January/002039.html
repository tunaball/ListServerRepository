First out, Diskwarrior fixes the directory and not necessarily the 
desktop file, although it can fix custom icons, apparently.

What did the fixit reports say? Anything of note?

My question is, how old is the computer? Also, have you moved a lot 
of data on and off that computer recently? I'm guessing the disk 
could be severely fragmented.

Since you've run both DW and NDD, I'd suggest running either 
PlusOptimizer or Speed Disk. I tend to like Norton Speed Disk, but be 
very careful. Only run this, if you have a recent version of Norton 
Systemworks.

However, before doing anything, make sure you have a recoverable 
backup of your files & system.

That's all I can infer based on the presented information.

Hope this helps and good luck!
