On Thursday, January 30, 2003, at 04:30 PM, Loren Schooley wrote:

> Does OSX have a pop up message program? I can SSH into my buddies 
> machine,
> and now I want to send a message to his desktop. Possible?

No.

But...

If you log in as root or as the logged in user, you can use the open 
command to open an arbitrary application.

If you are both using the terminal, you can use write to send each 
other messages.
