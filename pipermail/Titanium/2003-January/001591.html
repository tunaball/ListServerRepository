Jim - there is a LOT about this (including from me) in Archive of this list.

In a few words: DVD (disks) are (normally) region-coded.  These are 
geographic regions.
1 for USA, 2 for Europe etc.  The DVD player has to have its region 
set to match the disk it is playing.
The region can be reset a few times, then it is locked to the last region.
If you only play disks from region 1, it is no issue.
If like me you have DVD from all regions, it is a pain.....

HOWEVER, there are kind and competent people who have written free 
code to make the players region-free, or code to reset the counter to 
5 again, when needed.

This code definitely works on the Rev A Ti-book (as I have), where my 
drive is now region-free.
Perhaps this code will appear for later Ti-books, but I do not know 
specifically.

I can re-send other info, if you have any further needs and cannot 
find what you need in the archives.
