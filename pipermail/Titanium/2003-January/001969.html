Understood.  That's why I posted to link to RsyncX.  With the GUI front 
end, it makes the job of creating a shell script very easy for a non 
unix-oriented user, and creates a clickable icon that does the job.

Also remember that some are attracted to OS X *because* it has the unix 
tools we are used to working with.  There's nothing wrong with that 
either, and it doesn't hurt a non unix-oriented user to know that there

do not care to use them.
