>I seem to recall reading somewhere that OS X Server just flat won't run on a
>PowerBook...

buzz wrong, running Server X.2 on a PB G4 667 DVI
a client upgraded his G4 to an xServe and had no use for his OS X 
Server software anymore, got it for a about $100. Runs just a nice as 
regular X. I wanted it mostly to use my own smtp server. I know I 
could have enabled sendmail in a regular X but I tried and gave up 
after an hour (didn't have more time at the moment and then found 
this deal). Sendmail is indeed picky on ip changes and often required 
to be restarted or even reboot the machine to get it working again, 
same for the Server manager.
Conclusion : the functions already in regular X work just like in 
regular X, the server extras can be picky on ip changes requiring to 
restart.
