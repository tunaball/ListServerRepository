That's right. There is a good TurboTax Mac OS X version. However, there 
have been some complaints that the Windows version  has some additional 
features not in the Mac version. Some people have needed those features and 
have been using the Windows version on Virtual PC for several years. Now 
there is a license enforcement scheme required and you can't register and 
use a Virtual PC version of TurboTax, probably because of the need for a 
hardware serial number or something like that.

So, as something to beware of, if other vendors decide to go to license 
enforcement schemes tied to hardware on Windows software, it could be a big 
problem for Virtual PC usefulness.
