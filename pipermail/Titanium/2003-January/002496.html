I keep seeing people throwing around the name Power4 for what apple may 
use as there new cpu. The desktop version of the Power4 is so far called 
the PPC-970. There are some MAJOR differences between the server 
Power4(multicore) and the PPC-970(unicore). I have posted a link here so 
that people may read a little bit more about it. This read is somewhat 
dry to those who aren't tech geeks.

http://arstechnica.com/cpu/02q2/ppc970/ppc970-1.html

This helps explain some of the differences between IBM's current PPC-970 
desktop project and the currently available Power4(multicore).

-Rock
