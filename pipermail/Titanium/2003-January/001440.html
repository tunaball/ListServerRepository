You said: "..set up an account in Entourage with the Exchange server's IP
address or name (tried both)"

Can you ping the exchange server IP? Are you on the same network? Go to
shell.  Here is my example:

lorens-Computer:Loren:~> telnet 155.12.22.37 110
Trying 155.12.22.37...
Connected to trinity.aus.prodigycorp.net.
Escape character is '^]'.
+OK Microsoft Exchange POP3 server version 5.5.2653.23 ready
quit
Connection closed by foreign host.
lorens-Computer:Loren:~>

If you can do this from your shell, Entourage will work.
