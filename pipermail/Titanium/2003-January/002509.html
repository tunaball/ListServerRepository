I don't.  I run it on FreeBSD and OpenBSD.  I never migrated over to
GUI-based MUAs for a number of reasons, inculding HTML email and 
too many GUI MUA authors thinking code autoexecution is a good thing.

The only time I use a GUI MUA is when I'm checking an IMAP mailbox, and
then only when I control the server as well. :)

However, if I were to run mutt on OSX, I'd simply build it from source
rather than install a percompiled version.  I prefer to know how
something's compiled and linked, and have a copy of the source used to
do it, whenever possible.  I'm one of those strange luddites who'd be
perfectly happy with a WYSE terminal.
