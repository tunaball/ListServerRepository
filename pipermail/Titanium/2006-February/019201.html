Vince Sabio forwarded this notice about a Mac list server:

Those of you who subscribe to the Mac-related discussion lists at
listmoms.net (Mac-L, PowerList, X-Tech, etc.) may have noticed that the
lists have been down since January 27th. The cause of the outage was a
failure of the primary server's system disk. The server has been temporarily
removed from the co-location facility where the listmoms.net servers are
housed, the disk is in the process of being replaced, and the server will be
re-commissioned as soon as the OS and associated software are restored. The
system is fully backed up on a daily basis, so the lists should return to
full operational status as soon as the server is brought back on line, which
is anticipated to be sometime early next week.
