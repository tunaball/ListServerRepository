I do have to provide one warning about linux in VPC, and if you have 
limited disk space.  Any of the RPM distros will take up 1.5 GB for a 
base install.  Like wham bam thank-you-m'am, there you go, 1.5GB used 
up.  You can install Debian with less than 200MB, and a fully 
functional KDE3 desktop with Debian on 560MB.  I sent you, by private 
email, a screen shot showing Debian running with XTerm open, and I ran 
df at the command line to show you how much disk space I used to to 
install it.

If you want to ever try Debian, don't be afraid to email - I'll send 
you all the gory details of how to get it running :-)  You don't need 
to even burn a CD.  You can install it direct from a downloaded 186MB 
ISO netinstall image.
