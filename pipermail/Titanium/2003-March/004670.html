The network settings are stored here:

/var/db/SystemConfiguration

To access this folder, select Go to Folder... in the finder and enter 
/var/db. Move the directory to your new machine and put them in the 
same location. You will have to show info on the containing folder and 
temporarily change the privileges for it, so that you may modify it 
(using the Get Info window). Then change them back when you made the 
change.
