Do you get the crashes with any other programs?  If it's a somewhat 
universal thing, have you added any RAM?  If you bought the cheapest 
RAM you could find, it's also the most likely to fail.
Otherwise, try the usual things; deleting preferences, reinstalling the 
application...  If it's really bothering you, you could try either 
making a new user (see if the crashing still occurs) or doing an 
archive and install of OS X to see if some of the system files are 
corrupted.
