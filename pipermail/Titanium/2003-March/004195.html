Yes, it is an alternative to dial up.  I have been using it that way 
for several months.  From the speed tests at dslreports.com I often 
see 100-110 kbps down and 30-40kbps up.  This is with CDMA2000-1x 
service branded as "Express Network" by Verizon and as "PCSVision" by 
SprintPCS.  Sprint likes to discourage this use with phones, but the 
Novatel Merlin PCCard works.  Ping times are on the long side so 
complex web page downloads can be slower, and I don't think that it 
would be the best solution for gaming, but it sure is convenient.
