Michael Bigley <wakinyan at fuse.net> writes:
> Using existing Airport cards in your Macs you won't notice much speed
> difference

Actually, if you're using a mixed network of Airport and Airport 
Extreme computers, the more you use the Airport computers on the 
network, the slower the AE computers will go (though it'll still be 
faster than plain airport).  This is because every time an Airport Mac 
wants to use the radio waves, everybody else has to slow down to 
802.11b speeds.  Using only Airport computers on an Airport Extreme 
network will be the same speed as using them on a plain Airport network.
