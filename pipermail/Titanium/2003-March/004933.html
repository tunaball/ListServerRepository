I assume you've opened up System Preferences->Desktop and used the pane 
that appears on the second monitor to change that background in 
addition to the pane that shows up in system preferences, correct?  
Each background is independently controlled.

If you've done this, then try deleting the preference file that stores 
this, log out, and log back in.  The preference file is called 
com.apple.desktop.plist, and is stored in the Library folder in your 
user folder.
The reason why the picture still appeared after you removed it from 
your computer is that it is cached inside ~/Library/Caches/Desktop/.
