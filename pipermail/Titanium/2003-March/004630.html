The Hardware Test CD doesn't run the Mac OS, it runs it's own OS that is
_very_ closely tied to the hardware that it goes with. In other words, you
can't boot a PowerBook G4 Hardware test in a PowerMac G4, for that matter,
you can't even boot it in a PowerBook G3 (FireWire) even though they are
virtually the same machine.

This is done to make absolutely sure that it isn't a software issues that
could be causing errors.
