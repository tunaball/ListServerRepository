It may be worth talking a step back and noting that typing stuff into 
unix shells without understanding exactly what the command can do can 
be dangerous...

in a unix filesystem, the root directory is known as "/".

so /etc a directory under this.

/etc/rc.common is a file called rc.common in /etc

I recommend "Unix in a Nutshell" and "Unix system administration" 
(O'Reilly) as good starting points
