If you are thinking of the .DS_Store files -- there is one of these in 
each directory on your Mac OSX volume.

Any directory you mount on a OSX machine (even via SMB) will get one of 
these files. I think it may be where the OS stores info about position 
of files within the window in Aqua -- I'm not sure about that though.

In any case, it's perfectly safe to delete the file, a new one will be 
created each time. You can assure the sysadmin that the file is small 
and harmless.
