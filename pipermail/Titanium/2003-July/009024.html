Lisbeth,

Sorry I have been away for a few days and did not see your follow on post.
The answer to your question is yes. You can use both at the same time.

Broadband Optimizer changes the default packet size to 64K bytes from some
lower number usually 32K Bytes. This affects every packet received.

Speed Download uses a completely different approach to speeding up file
transfers by opening up to 6 ftp sessions at once and downloading 6
fragments of the file in each stream and then reassembling after download.
The speed comes from downloading 6 parts of the file in parallel rather in
serial fashion. Of course the Broadband optimizer will optimize each stream
packet size as well.

Hope this helps.
