All this talk about the fragility of the Titanium is amusing. I've got a 400
that gets used every single day. It looks surprisingly good for the amount
of use it gets.

First thing I got when I got the computer is a Zero Halliburton briefcase.
So that eliminates any chance of the laptop getting banging up during
transport, unless I got run run over by a tank.

I did have to replace the battery, but I don't expect batteries to last
forever, no matter how much they cost. The funny thing is that the battery
is the only operating expense I've had so far.

Otherwise the paint still looks good, aside from being worn in a few areas
from sheer use. Bottom is scratched up but I don't spend all my time looking
at the bottom.

Still no dead pixels.

A pretty good for a first revision Apple product.
