> You can use Cocktail to start your drive in Journaling mode. Just 
> click, and
> that's it.

Or, if you're unwilling to download and / or install Cocktail for just 
this feature, you can always revert to the terminal.app and type 'sudo 
diskutil enableJournal /'
This will prompt for the superuser (root) password and allocate the 
journaling file and enable journaling in 10.2
Here are the results for my ti667 (infrared):

[arjan-boss-computer:~/] arjanbos% sudo diskutil enableJournal /
Password:
Allocated 8192K for journal file.
Journaling has been enabled on /
