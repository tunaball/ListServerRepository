> Hi!
> Anyone have this happen? Possible causes?
> 
> Powerbook 15" OS 10.2.6: When upstairs in our house, it will not
> connect to the
> Airport network (the base station is upstairs). BUT when downstairs
> (another base
> station is there - but we can move between them), it WILL connect.

David,

Make sure that both base stations have the same name and are at least 3
channels apart. Also both should not be giving out ip numbers. One should be
just an ethernet bridge and the other will be your router and DHCP server
unless you have a separate router in which case both airport base stations
will be ethernet bridges.
