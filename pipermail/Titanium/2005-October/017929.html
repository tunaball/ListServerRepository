> So I've got a 1.67 ghz pb with 1 gig of ram and it was running very  
> very slowly so I recently
> added SPOTLIGHT=-NO- to /etc/hostconfig rebooted and it seems to be  
> running a lot faster now.
>
> Has anybody turned off spotlight had this experience?
>
> One problem is search in Mail.app isn't enabled anymore but I maybe  
> able to live without that.
>
> Ideally I'd like be able to update my indexes nightly and turn off  
> real time updating if this is the culprit for the slowness I was  
> seeing.

I'm a brand new convert to Mac, having just gotten a TiG4 1GHz w/  
512MB. I have to admit that spotlight turned on and everything that  
it is still a lot faster than the PC I was working on previously. I  
never really noticed any 'slowness', aside from running Photoshop/ 
Illustrator which seems to be pretty pokey on so little ram. The only  
thing I've really wound up doing is getting rid of all the Dashboard  
widgets. I did not particularly appreciate each individual widget  
eating up something like 10MB of memory.

Given the amount I actually use spotlight though, I might bother  
trying this.
