I had to have both of the 512mb chips replaced in my albook after 
discovering severe memory related problems with tons of corrupted data 
on my hard drive. As can be expected Apple does not stand behind any 
memory it has not authorized to be installed. The source of the old ram 
very quickly replaced it with "better" name brand life time warranteed 
ram. It seems that some brands have a reputation for quality that is 
based in real experiences.
