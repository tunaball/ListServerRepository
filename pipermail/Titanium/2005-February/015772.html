Yes, but I believe the display is still lit, right?  I think if you 
read the older manuals it recommends to not run the computer with the 
display shut in FireWire Target Disk mode, as it may overheat.  They've 
left this tidbit out of the newer manuals.  For whatever reason, the 
cpu runs at 100% load in FireWire Target Disk Mode, and the computer 
will get extremely hot after extended use as a Target machine.  I 
believe, without checking, that they recommend having the power brick 
plugged in in FireWire Target Disk Mode because of the elevated power 
requirements.  Keep that in mind when using this feature.

I don't think I'd recommend running the machine with the display shut 
if you're using it as a Target host.
