If reinstalling from the combo updater does not help consider doing an
archive and install or it it has been a long time since you formatted the
drive, zero out the drive and reinstall everything from scratch. It is
possible you have some preferences that have become corrupted or corruption
in the operating system itself.
To check for corrupt preferences, and lots of other things, download Onyx
and under Unix utilities run the plutil command for both users and system
and it will let you know if any preferences are corrupt.
You can run this from terminal also but Onyx does so much more I thought I
would mention it.
