Seymour,

This happened to 3 of the 30 users I support.  Those 3 folks did not 
repair permissions before applying the update.  Nothing I did 
(perimissions, fsck, PRAM, NVRAM, Diskwarrior) allowed the computers 
to reboot.  The always hung right before the login window at "Waiting 
for Apple File Service".  Hopefully you won't have this problem, but 
from my user's experiences, it is likely.  An archive and install was 
the only solution.
