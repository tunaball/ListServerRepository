Presumably, it can't find System Profiler.  If you moved it or the 
Utilities or Applications folder, that could be the problem.  It should 
be located at /Applications/Utilities/System Profiler.  It's also 
possible that your copy is somehow corrupted - it takes more than 
something with the right name in the right place for the button to be 
active.
If you can't get yours to work, you might try using an application like 
TomeViewer to extract it from the 10.3 install disks, or copy it from 
somebody who has a working copy.
