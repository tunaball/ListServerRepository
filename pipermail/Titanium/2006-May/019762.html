Sometimes Verizon releases phones with Bluetooth OBEX file transfers  
enabled and then a subsequent version has it disabled. For example,  
the initial release of RAZR phones in Dec of 2005 had OBEX enabled,  
probably by mistake. All subsequent versions had it disabled and the  
code was completely removed from the phone's firmware.
