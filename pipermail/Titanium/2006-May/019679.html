> Is there any security risk for hte College if they open port 143 ?

If they direct it to the Exchange mail server only, there should be  
no problem. Only IMAP commands will be recognized and responded to on  
that port by that one server. They still may not do it, however,  
citing a overly restrictive policy. In that case, see if they have  
VPN access you can use. It will add an extra connection and  
authentication step, but should get you into the interior network.
