On May 25, 2006, at 10:01 AM, Glenn L. Austin wrote:

> Or Entourage, which does work with Exchange servers, both MAPI and  
> WEBDAV. (I've used it on both)

Entourage can NOT use MAPI; it is IMAP/POP and WEBDAV only. IMAP must  
be enabled on the Exchange server in order for Entourage to fetch  
mail. WEBDAV and LDAP are used for Calendar and scheduling.

Outlook Web Access (OWA) is another way to get your mail from an  
exchange server using a web browser. It's not the greatest, but it  
does get you in.

Some good info on Entourage and exchange is available here:

<http://www.entourage.mvps.org/index.html>
