I was really happy to read about ucontrol  until I found that it does not
fix the F5 key.  There is another way, which was contributed to this list by
Joao Carlos de Pinho about a month ago:

Boot into OS9
Control Panels
Keyboard
Function Keys
Check: "Enable Hot Function Keys"
Uncheck: "Use F1 thru F12 for Hot Function Keys"

Unfortunately, you have to re-do this any time you install an OS update,
such as 10.2.4 --> 10.2.5.  But that's not too bad since you've already
restarted to do the update anyhow.

And, it fixes ALL your function keys.
