At 16:30 -0400 27-5-2003, Michael Bigley wrote:
>>If I had this problem, I would reboot in OS9, and rebuild the 
>>desktop - this re-connects files and their applications.
>
>Removing this file does the same thing and saves the rebooting:
>~/Library/Preferences/com.apple.LaunchServices.plist
>
>This file also can be deleted if the problem persists:
>~/Library/Caches/com.apple.LaunchServices.UserCache.csstore
>Though it is related more to startup issues, I believe. Note that 
>deleting these files will remove any previous Open With 
>customizations you have done, and restore Finder settings to default 
>as well.
>
>That is for Jag; running 10.1.x you have to remove these files:
>~/Library/Preferences/LSApplications
>~/Library/Preferences/LSClaimedTypes
>~/Library/Preferences/LSSchemes
>
>Which no longer exist in Jag.
