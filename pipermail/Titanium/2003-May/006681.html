You should be able to just set everything up as if you were going to 
copy the disc (insert DVD, select the Copy button), and then choose 
Save as Disc Image in the File menu; this should save it on your hard 
drive for burning as much as you like.  Then to burn it, just choose 
Disc Image under "Other" and you're all set.
