I installed it and cured my volume fluctuations flaw.

Concerning iTunes Library sharing, why not using simple File Sharing?

This is common Unix stuff using symlinks: if I am connected to a server 
and have access to its iTunes Music folder - say it is in Public folder 
and the server as File Sharing on, then the only thing I need is to 
have inside Music>iTunes a symlink called iTunes Music pointing to the 
mounted volume containing the server iTunes Music folder. Bingo, iTunes 
believes it is a local folder and does not complain.

Sure, you cannot use the iTunes GUI directly but it is truly little 
effort. \
