try discarding this file then restarting:
  ~/Library/Preferences/com.apple.LaunchServices.plist
this file will not be recreated until you do an "open with". For 
example, select a .pdf file, then get info (Cmd+I); select Open With 
and choose the app you prefer (Preview, Acrobat, Acrobat Reader); 
then click the Change All button, if you want all .pdf files to be 
opened in the same app.
