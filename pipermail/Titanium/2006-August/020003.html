-----------
Don't you have the Hardware test CD which came with the Ti? I should  
try to start with that before I install things. I also thought it was  
possible to use Techtools from its CD and not install it.

Don't assume that the RAM is fine because it is Apple original. If  
you can borrow the right kind from another Ti I should swap and test  
with that and mostly test that the RAM is seated correctly.

Don't forget my little "First Aid check" Have the correct screw  
driver and check that all screws are tight, and if 800 has the  
"loose" keyboard check that the screws _and_ magnets are tight and in  
their correct places. A loose magnet can cause all sorts of odd  
behavior if fallen down in the TI.
