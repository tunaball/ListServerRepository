I'm surprised no one mentioned IPSec virtual private network
(VPN) yet.  I don't have time to get into the details, which
there can be a lot of, but there's a VPN (client, at least?)
built in to OS 10.2  And you can buy hardware boxes (eg
Linksys, D-Link) which you use instead of a cable/dsl router
and can set up various different secure "tunnels" between
various different clients and the file sharer.  You have to know
your LAN's public IP address, and your specifice LAN computers'
LAN-private IP addresses.
