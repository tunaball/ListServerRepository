I found the answer to the problem. Alsoft "recommends" that you run
DiskWarrior 3 from the base level of the Utilities folder inside the
Applications folder. The thing is, as I found out, you MUST run it from
there. If you try and run it from a folder inside the Utilities folder as I
did, you hit the wall with the kextload crash problem.
