>> Is there a way to translate the appleworks database to office 
>> database for use on a windows platform?
> ----------
> The best you can do is to start Excel on your Mac and in the Data menu 
> choose Get External Data (or something like that. I have a Swedish 
> version :-) ) In the alternative you'll choose "Text import" and find 
> your AW database in the Open Dialog Window.
>
> Excel will feel that your database is suitably divided and you will 
> get each post on a new line and each field in a new cell. After that 
> you can add an additional line on top and enter the field name if you 
> like.
>
> This Excelfile is now easily transferable directly to Windows and 
> there possible to import to FileMaker Pro or you can make it to a FM 
> Pro file before forwarding to Windows.
>
