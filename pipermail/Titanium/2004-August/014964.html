I have had great success with the Demo version of Solid
Thinking 5.5  (www.solidthinking.com). You will be able to
open and view almost any common 3D file. As a demo file,
however, you will not be able to save. For reference, I
rotate the images, then use Screen Grab to capture various
views in 2D (tiff). In addition, you can merge several
files together to create an assembly, if necessary.
