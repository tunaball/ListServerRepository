1) Not how it looks in my mail. If the folded thread contains an unread 
message the thread will be marked with the status bullet.

2) If you sort the mailfolder by Message Status (or  whatever it is 
called in your US version) you will get all unread mail either at the 
top or bottom. Easiest is to click on the Status Bullet in the Columns 
Raw. To change the sorting order click again. Then you can read the 
unread messages after each other. :-) (But only in one mail folder at a 
time of course)
