<div>I have installed Tiger last week on my Al-1.25, with no problem.&nbsp;
20 minutes.</div>
<div><br></div>
<div>One surprising - to me - change is that the screen capture now
saves the image as PNG.</div>
<div>Why on earth did that get changed from PDF ?</div>
<div><br></div>
<div>From Macintouch (contribution from Neil Lee) I picked up this
command:</div>
<div><br></div>
<div><font face="Courier" size="-3"
color="#000000"><x-tab>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</x-tab>defaults write com.apple.screencapture type pdf</font></div>
<div><br></div>
<div>which changes the default to PDF from PNG (need to log out and in
after this Unix command is entered in Terminal).</div>
<div><br></div>
<div>I thought others might find this helpful.</div>
