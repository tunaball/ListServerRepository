It's been a while since I looked at the USB vs. Firewire tests, but if
I remember correctly Firewire wouldn't really be all that much faster
for filling a 2 or 4GB Nano.&nbsp; It seems like it was around 8-10GB
where the Firewire really started being noticeably faster than
USB.&nbsp; Assuming this is true I would think this may have been part
of the decision to leave it out.&nbsp; Perhaps the cost outweighed the
benefit.&nbsp; And don't forget that the for it circuitry takes up some
room and real estate is probably at a premium in there.&nbsp; So I
guess I was in a way disappointed that there is no Firewire support
with the Nano, but I got over it quickly.&nbsp; Now, if it ever goes
