I used Carbon Copy Cloner to copy from my TiBook 500 to an AlBook 1.25.<BR>
Boot the new Mac in Target disk mode and use CCC to clone the TiBook<BR>
drive to the AlBook. I did an archive and install after the cloning, to<BR>
install any AlBook specific files. Not sure if the last step was<BR>
necessary but it was quick and painless.<BR>
