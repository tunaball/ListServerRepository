Thanks, I think I'll pass on the developer preview.  If I had a dev box 
here I'd try it, but every Mac I own is currently in "mission critical" 
mode and can't afford to mess one up :-)

Overall, I must say I'm fairly happy with Java apps as far as speed.  I 
think it's the actual difference in system bandwidth between the 
PowerBook and Power Mac G4 that causes the dramatic difference in 
speed, rendering the shapefiles in ArcExplorer.  The Power Mac will 
render in 2-3 seconds the same file that takes the 1.25 GHz PowerBook 
20+ seconds to do.  It's possible I need to drop in the full 2GB of RAM 
in my PowerBook because the data won't fit into 1 GB RAM, and the Power 
Mac has *dramatically* better thruput on the disk I/O system when it 
needs to page RAM in and out.  I haven't updated the RAM in my PB - 
seen the prices on those 1GB modules?  Cripes, I could justify buying 
another PowerBook before sticking that much money in RAM.
