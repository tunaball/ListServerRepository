Hi Gina (and everybody...),

I'm using exactly this Printer (Brother HL1270-N) with Panther and it 
works very well.

There is only one application making troubles with printing, which is 
Finale 2002 (a OS9 music score writing software), when trying to print 
in Classic mode. It always gives a postscript error. (But I suppose, 
the problem is coming rather from Finale than from Panther).

The printer driver for OSX can be downloaded from
http://solutions.brother.com/hl1200/download/drivers1270n.html#macOS

Hope that helps,

best regards,

Johannes
