>
> Hey ya'll does anyone know how to save an entire website
> to a folder with all its text and images?

If you use Camino.
Just save as "HTML complete". It will save the website as html and a 
folder with all its content (jpg, etc). But this will save just the 
current webpage

With M$ Explorer you can download multiple levels of the website. But 
you don't have access to the source material (I actually don't know if 
there is a way to do it).

But you can also use softaware like sitesucker. This will just download 
the website with everything to the folder of your choice. And you can 
specify how many levels of the site you want to be downloaded.

Finally, I've to say that, if I want just to keep a copy of the website 
(such us receipts), I prefer to print it as pdf.
