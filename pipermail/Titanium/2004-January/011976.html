> Has anybody done any benchmarks on different powerbook models to see 
> what they're capable of?
>
> Background info:
> 1.25 Ghz G4 AlBook w/1.0 GB RAM.
> I'm working with ESRI shapefiles in ArcExplorer.  Polygon files seem 
> to render pretty quickly, however, point data files are very slow.  
> Some of the point files I'm working with have ~148,000 data points 
> within a few minutes of latitude/longitude.  It'll take the PowerBook 
> up to 30 seconds to render a file, and every time you do a query on an 
> underlying layer, it has to re-render the map, which is time consuming 
> and quite slow.
>
In an other mail you said that the app is a java app. This is a known 
weak point of the Apple Java Virtual Machine that is running your java 
app. So taking a honda power generator and a G5 won't really solve your 
problem.

Rumours are that the 1.4.2 upgrade will solve many graphics based 
performance problems. This 1.4.2 is currently in Developer Preview 2, 
so it's there for you to try out. But be advised that there is no other 
uninstall for 1.4.2 than re-installing the complete OS.
