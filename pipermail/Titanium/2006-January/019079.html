David Ferleger said:

>1. Will any brand of Mac OS X run on this G3? Since there is no CD  
>drive, can it be installed via Ethernet?
You have a wallstreet most likely. Question is if it's a WS I or WS II
(also called PDQ). CPU speed 233Mhz w/o cache, 250Mhz and 299Mhz is WS I
and 233Mhz w cache, 266Mhz and 300Mhz is WS II. 
Panther work better in my personal experience than Jaguar with XPostFacto
3. It may also work with XPostFacto 4 and Tiger, but I'm not sure as I
upgraded to titanium 1.5 years ago.
You'll find lots of info on wallstreets in the user forums of XPostFacto.

if you want to run OS X.
