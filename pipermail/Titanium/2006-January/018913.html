On Jan 13, 2006, at 7:42 PM, Alex wrote:

> Best of all Wine is Opensource and FREE!!!

And that makes it good?  WINE is buggy, a moving target, unsupported,  
and at best provides only partial support for applications considered  
"fully supported" in WINE.

If you run Macs and need a Windows compatibility layer, Microsoft  
already provides a (fully supported) answer - VirtualPC.  If you run  
Intel Macs and need Windows, what did you buy the Intel Mac for?  Too  
many people pick the operating system based on religious beliefs,  
platform evangelism or what have you.

You pick your application suite first, because it's what gets the  
work done.  THEN you pick the operating environment to run it.  If  
you need Windows software you should be running Windows, not Mac OS  
X.  Switching back and forth between operating environments is a  
productivity killing nightmare.
