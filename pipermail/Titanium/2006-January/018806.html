For instance, if you have a LaCie drive you would go to www.lacie.com  
and head for the download page. You will find the most up to date  
firmware update for the FW800 drives. If you run the updater (which  
is a stand-alone application) it will search for your drive and read  
the firmware version that is installed in it. If it needs to be  
updated, it will go ahead and apply the update. If it doesn’t need  
the update, it will simply alert you to the fact and quit.

I believe other drive manufacturers have similar resources for download.
