You may have got the problem which occured for quite some people  
using Apple's program update where iTunes 6.0.2 didn't seem to sit

problem for some.

installed the program update successfully on an Albook, with a very  
clean 10.3.9 system so it seems it is some third party install which  
causes the trouble.
