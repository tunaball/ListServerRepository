On Jan 12, 2006, at 11:16 AM, Loren wrote:

> Will Windows or i386 unix's work on this Duo Core thingy?

Windows or x86 unices will run on the processor, but they won't run  
on a Mac with an Intel processor.  The Mac don't have a BIOS.  It  
instead uses Intel's EFI (Extensible Firmware Interface).
