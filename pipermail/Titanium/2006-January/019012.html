It seems you mean repair permissions. Probably you mean that you get a
message every time that says  "we're using special permissions....".
This is normal and is no reason to run permissions again.
Permissions repair should always be done booted from the System you use
daily, so don't boot from a CD/DVD, unless it's the exact incremental
version as your System, i e 10.4.1 for example. The reason being that
permissions and their settings typically need to be updated in
incremental versions (as new files are added to the system and other  
changes).

If you mean actual disk or file structure problems, it can be a good  
idea
to repair while booted from CD/DVD though. This could solve problems  
that
isn't repaired when booted from the system you're repairing.
