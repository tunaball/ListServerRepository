Hi!

I have similar sound problems with my 667 15" TiBook and Dual 1.25 MDD.  It
seems that there was a change in the audio driver in X from 10.3.6 onwards
(Core Audio).  From what I can gather, this was still an issue with the
developer builds of 10.4, but here's hoping its fixed by the time Tiger is
unleashed!

Only fix I have found so far is to go back to 10.3.5, before Apple changed
the audio driver.  Which kind of sucks if you are using software that
requires 10.3.6 or later......
