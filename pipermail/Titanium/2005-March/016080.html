Do you have the latest Airport software installed ?  Are you running 10.3.8?

In System Preferences / Network, select Airport.

Tick the box "show in menu".

Then select that new menu item, and turn the Airport card ON.

That is how you can see if it is working or not.

Some grey bars should appear in the menu, which go black if there is 
a wifi source nearby (eg in Starbucks). You do not need to pay to 
connect in order to see the signal strength.
