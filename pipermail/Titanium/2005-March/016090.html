This sounds like the network port has been deleted.  While in Network 
preferences, go to the "Show" pull-down and select "Network Port 
Configurations".  See if something gets auto-detected, and if not, 
choose "New..." and see if you can add a port of type "Airport".  Then 
it should show up in the interface list.
