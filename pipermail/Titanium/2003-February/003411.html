>With the old (non Extreme) base stations, you can set up multiple ABS in a roaming network to cover a larger area if you connect them all to Ethernet.

Yes, Connecting via Ethernet allows you to use the ABS as a Bridge.

>With the Airport Extreme ABS, you can network multiple ABS in a Wirelessly Distributed Network (where only the main station needs a Ethernet connection) to do the same, but you can't use a non-Extreme ABS.

Correct.

>I am assuming you can use a Extreme ABS in the first kind of network. And access the USB printer.

Yes, continuing to use the ABS in a wired bridge network does not disallow the use of the USB printer sharing.
