just runs in the background.  It takes about 8 hours to do a dvd-- so I'll
start one when I get to work, then start another when I get home.  I hope to
have my whole dvd collection divxed within a couple months.  The only
problem with the latest release is it thinks it's already expired, so you
have to set your date back to use it (thus the old date on all my emails)
