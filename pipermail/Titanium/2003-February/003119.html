More RAM would probably help.

FWIW, I use a 400 MHz Ti with OS X 10.2.3 and consider myself 
something of a "power user". I would *like* a faster machine of 
course, but can't really justify it as this machine really is fast 
enough for all my day-to-day stuff. The only exception I can think of 
is that I did some iMovie work over the Xmas holidays and then I 
really could have benefited from a much faster machine, but the rest 
of the time it's fine.

Next time you machine seems slow try running top in a Terminal window 
and see if something is sucking up your CPU cycles (some applications 
eat CPU bandwidth even when they are not doing anything - tsk, tsk, 
bad programmers, no biscuit).
