For web developers who use your Apache webserver with PHP on your Mac 
to develop PHP webpages, be advised that this latest upgrade replaces 
your /etc/httpd/httpd.conf file for Apache and disables PHP for you, 
without warning.  This is unacceptable for any upgrade on any unix 
system, although whoever wrote the upgrade scripts were intelligent 
enough to at least save a copy of the old config file named 
httpd.conf.applesaved in the same directory.  I was developing a PHP 
front end to a MySQL database, and downloaded the upgrade while working 
on my PHP scripts.  I was somewhat irritated that the machine had to be 
restarted after the upgrade installed, and even more irritated after 
restart when my webserver configuration was broken.
