There are three PreferencePanes folders, in the main Library, your 
Users Library, and in System/Library.

I don't use the Eriksson, but, if you ran an installer, the easiest 
way to see if it loaded up correctly seems to be to open System 
Preferences, and see if it is in the bottom row of 'panes'. If it is, 
then it really doesn't matter which PrefPane folder it's in. My guess 
is that it will be in System/Library/PreferencePanes, because that is 
where "Bluetooth" is located, but the second place to look would be 
/Library/PreferencePanes.
