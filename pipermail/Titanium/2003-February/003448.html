Try creating a new user on the system, log in as that new user, 
configure your Mail program, and then test it.  If it works ok from the 
new user account, you have a corrupted mbox file someplace in your 
~/Library/Mail directory.  It baffles me as to why AppleMail is using 
mbox format for message storage.  mbox has a long history of problems, 
and with the advent of Dan Bernstein's Maildir (which positively rocks 
and which was part of his qmail project ), mbox is long ago obsolete as 
a mail storage format.  The "other" unix GUI mail client, KMail, uses 
Maildir.
