>The dock becomes visible only if you go with the pointer in the dock location, it does not become visible at its will, only at your will ;-) If you start applications via an icon on the desktop or recent items, or if you use a whatever AppLauncher the dock does not show up because of those operations: only if you go with the pointer on the side he is hiding.
