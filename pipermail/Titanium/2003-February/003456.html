When the uptimes.net project was still running, I had an FTP server on 
there running FreeBSD 2.1.6 with an uptime of 1,363 days when the 
project finally went offline due to finanacial and server resource 
problems.  The FTP server was put online in January of 1997 and taken 
offline in July of 2002 with no reboots - total uptime was 1,997 days 
when it was finally retired, with a cpu usage of 36% over it's total 
life.  It was running a 133 MHz Pentium processor with 32 MB of RAM, 
and 3 Seagate 2.5GB IDE hard disks.
