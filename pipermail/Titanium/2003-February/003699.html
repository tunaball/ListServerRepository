I get to Path Finder when I click the desktop. Once Path Finder is launched
the only way I can get to the Apple Finder is to click its Icon in the dock
(but why would you want the Apple Finder once you have Path Finder running?)

By the Way, Path Finder is way cool. The most powerful features are it use
of Terminal and the ability to navigate and perform tasks as Root without
logging out and in again. You can create disk images, convert picture
images, secure delete items and on and on... It has an interface very
similar to SNAX but with decent looking icons. It puts a huge Trash Icon on
the desktop where it is supposed to be and gives you an application/document
menu.
Now, if only it would stop crashing when I try to use the Command/Shift/G
(Go to) command.
