>Do you guys know anyone using this in a dual monitor setup? How does that
>work?
>
>Does each Virtual Desktop contain the Ti/External monitor real estate? Or
>just the external/main?

I just tried it out, and it works perfectly. The little "pager 
display" gets extended by the extra real estate of your second 
monitor, and you can move windows from one screen to the next in the 
pager. So I guess each virtual desktop gets the combined real estate, 
is a way to look at it.
