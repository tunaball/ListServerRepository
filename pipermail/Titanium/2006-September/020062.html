I've had errors like that crop up from time to time.  the disk passes  
the S.M.A.R.T tests, but comes up with directory errors that cant be  
fixed.  the ONLY way I've managed to recover is to reformat the disk  
(which solves the errors) I picked up TechTool pro and have its  
automatic diagnostics set to check and repair directories every 4 hours.

 From what I can gather, its a small error that if its caught early  
enough that it can be fixed but left alone, the errors compound till  
they arent able to be fixed.  Since installing techtool pro, I've not  
had any more problems

good luck!
