>What is your favourite FTP client to use on OSX on your Ti?

Transmit, the new version supports SFTP, which most sensible ISPs 
will or should be moving to. It is also has a fabulous user 
interface, let's you know what is going on during your file 
transfers; allows you to edit files on the server (dangerous, but 
handy for minor tweaks). You can also view jpegs on the server in 
case it has been a while and your naming convention now makes no 
sense to you ;-)

You can drag and drop from the finder or you can set the default 
window to your "Current projects" folder (or what ever you want) for 
quick transfers within the app.
