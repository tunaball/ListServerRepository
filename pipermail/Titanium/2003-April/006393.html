Sorry to say, it does not work, at least on with the Lexmark 
printer/scanner/fax/copier. When the printer is connected through the 
new Airport, it connects using Rendezvous, which is actually a 
separate choice under the print center. The software is looking for a 
scanner on the USB port so it can't find it.  Either software must be 
re-written or the Rendezvous printing function re-designed to ghost 
the USB port.
