Well, if you want to figure out what is going wrong yourself, you can 
try removing one stick of RAM at a time to see if one or the other is 
causing the problem; this would allow you to use your computer more 
normally until you have time to repair it, but ultimately it will have 
to go back to Apple for either a logic board or RAM replacement.  Three 
beeps means that none of the RAM passed the power on self-test.  The 
strange part is that this is the *power on* self-test, and doesn't run 
when the computer wakes up or is rebooted.  It shouldn't be given just 
by opening the LCD.
More info about the POST beeps:
<http://docs.info.apple.com/article.html?artnum=58442>
