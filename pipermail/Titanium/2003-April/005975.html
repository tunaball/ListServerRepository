the OmniGraffle installer [on the original .dmg file] creates the 
folder in Application Support in the Home Library, that being in the 
Users directory, then <yourname>, Library/Application Support:

Users/<yourname>/Library/Application Support/OmniGraffle/Palettes.

Try doing a search, Jerry. Command-F and type : Maps.graffle    or

all of them here, and I never had to manually create a folder in 
Application Support. They never write to the main Library, either, so 
have a look, using Cmd-F, and let us know what turns up.
