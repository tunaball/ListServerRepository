At 8:28 PM +0100 4/15/03, Richard Edwards wrote:
>
>I've converted a CD track to MP3 using ITunes but alas mislaid the CD ....
>Can I convert it back to a CD Track and if so how?

First, make a copy of your MP3 song. You don't want to go to AIFF and back to MP3 again because it introduces another generation of MP3 conversion loss.

iTunes --> Preferences --> Importing --> Import Using --> AIFF Encoder

Then Advanced [menu] --> Convert Selection to AIFF
