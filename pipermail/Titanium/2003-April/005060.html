Hooked to a DSL modem, your Ti is more than likely using pppoe to get 
an address, dns server, and default gateway.

In Windows, right click the "network places" icon and select 
"properties".  Right click the "local area connection icon" and select 
"properties" (VPC emulates an Intel NIC).  Select TCP/IP, then click 
the properties. If you set "Obtain An IP Address Automatically" windows 
will use DHCP to get an address. Do not set a gateway or any other 
settings. Windows should get an address from the VPC emulator and use 
NAT to use your Mac's connection to the internet.
