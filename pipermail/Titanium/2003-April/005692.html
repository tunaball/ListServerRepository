I use Path Finder (a superior Finder substitute). You don't have to set up
any accounts or log in and out.

1) With Path Finder all you do is create a duplicate of the Path Finder
application. 
2) Then you select it (highlight it) and go the "Command" menu in your open
Path Finder and select "Open in Root."
3) The duplicate Path Finder will launch and you now have root access.
4) Quit from the original Path Finder and go on with your 'rooting about.'
5) When you are finished, just quit from the duplicate Path Finder and
relaunch the original.

Simple and slick IMHO.
