This only works in OS X.

Launch Apple DVD Player and select Window > Show Info from the menu bar. Go
to the chapter or special feature which has the audio you want and not the
Title number, Chapter, and Audio.

Go to macupdate.com and search for osex and mAC3dec.

Osex allows you to extract the audio file into an .AC3 file. Be sure to
de-select the option to grab the video and subtitle parts of the DVD. Select
output to "elementary streams" so that each audio track is in a separate
.AC3 file. Then select the Title number, chapter, and Audio track, and click
"Begin". Osex will create an .AC3 file that will contain you audio.

Then use mAC3dec to convert the .AC3 file into a 2 channel stereo AIFF file.
Afterwards, use the mp3 encoder of your choice to convert it into mp3
format.
