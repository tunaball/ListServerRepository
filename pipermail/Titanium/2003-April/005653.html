A possible solution to this is to use VirtualPC on the Mac. I have one
Windows only application that I use fairly frequently, but which didn't
merit my upgrading my old Pentium II, so I recently installed VPC on my Mac
with Win 2K, and have the best of both worlds. Additionally, I find that
Win2K runs with more reliability / stability on my Mac through VPC then it
did on my Dell!
