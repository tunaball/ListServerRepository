I use a few, for different purposes:

curl (http://curl.haxx.se) - because it's quick and dirty when I have a 
URL to download. As it is a command line program I can put it to the 
background and have it continue downloading my file even if I log out 
or close Terminal.app. Also handles http and others.

ncftp (http://www.ncftp.org) - because it is command line and has 
terrific tab completion when browsing an FTP site.

Fugu - Not strictly FTP, but SSH filetransfers. Great for encrypted 
connections to servers.

Transmit (http://www.panic.com) - The greatest Cocoa interface for a 
very nice and full-featured FTP client.
