Hi!

I actually purchased the OG 3 Pro upgrade just an hour ago!

The instructions on the page do work for me, to install the stencils. I 
used the Library -> Application Support -> OmniGraffle -> Palettes that 
descends from my home directory, but I guess the global library will 
work as well...

I advise you to verify the spelling of the directory names. Also note 
that you have to restart OmniGraffle to be able to find the newly 
installed palettes.
