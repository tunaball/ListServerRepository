There is an easy way. Download Path Finder. Duplicate it and launch the
original application. In Path Finder select the duplicate copy so it is
highlighted, scroll down the "File" menu and select "Launch as Root." The
duplicate of the original launches with root privileges. Quit from the
original application.

Now you have total root privileges. When you are finished doing whatever
damage you can think of, quit from the duplicate Path Finder. You can then
launch the original Path Finder or continue working in the Apple Finder.
