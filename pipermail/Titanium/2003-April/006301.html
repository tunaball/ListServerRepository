XXL paused, thought it over, and spoke thusly:

>
>Is there anything like DVDack that resets the DVD region identity counter
>that will work in OS X so that the DVD drive does not become irretrievable
>set to one region?
>-Joel

If you have an RPC-1 type drive in the Mac, then use an app called 
"Region X". On launch it shows the current state of things, with the 
region as selectable' and the number of changes remaining, 
selectable. So, if you're globe trotting, you can switch regions on 
the fly, and when home, reset to your home region. Simple.

There's a 'Classic' app, that may have an X version, called DVDInfo, 
that will tell you if you have an RPC-1, or RPC-2 drive. And, of 
course, if the drive is an external Pioneer, then you'd need DVD 
Enabler 1.5 in order to operate all the Apple apps [FCP, etc] as if 
it were internal.
