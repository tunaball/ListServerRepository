Not sure if this is what you mean, but the Mail app. only shows the 
number of new emails that are in your "in" box. If you set up rules to 
transfer new mail into a folder you've created it will not let you know 
in the dock, it will however still make a sound when it shows up, if 
you have that feature enabled. This has always been the case, I've had 
list serve folders for the whole time I've been using os X (3+ years) 
and I've never seen the dock update the mail count if emails were 
transfered to a folder via a rule.
