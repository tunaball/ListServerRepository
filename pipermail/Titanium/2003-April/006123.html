I've explored several stand alone programs and using page layout apps 
like illustrator to make cd labels based on everyone's suggestions and 
heres the solution i've come to...

bare templates are pain in the neck, sure you can get the most amazing 
results with illustrator and these templates but the learning curve is 
huge and for my needs was over kill

next i tried two apps. "discus" and "click and design 3d"

click and design was feature packed but has limited text layout tools.  
the text tool is limited to one line per text box, a pain when dealing 
with circular text boxes.  But the worst complaint is the program is 
slow, very slow, to user interface clicks and redraws.

discus is great, its has many paint, drawing, and text layout tools.  
The user interface is very fast and simple.  This is the hands down 
best program and the best way to turn out labels with minimal time and 
effort.  The best feature is that you can import track lists from 
Itunes or toast and have the text appear on the label!!

both apps include templates for cd labels as well as jcard, icards, and 
the mini 3 inch cds plus more....

i encourage all of you to test them each and tell what you think
