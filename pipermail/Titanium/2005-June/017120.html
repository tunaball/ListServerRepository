On Jun 11, 2005, at 11:16 AM, Robert Ameeti wrote:

> Where did you hear that it would be a Phoenix BIOS?

Not would be - _IS_.  The P-IV dev boxes have a Phoenix BIOS.

> And when did you last have open spec architecture from Apple?

PowerPC.  You're not a developer so it may be immaterial.  But if you 
get interested, please refer to the PowerPC Reference Platform (PR*P) 
and the Common Hardware Reference Platform (CHRP) specifications 
relating to IEEE 1275-1994 Open Boot Firmware for Power, which has been 
adopted by Apple (as well as IBM).  It's the first (and only) 
non-proprietary open-standard boot firmware that is usable on different 
processors and buses.
