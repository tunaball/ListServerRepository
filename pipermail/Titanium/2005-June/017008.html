Try this:

Go to the Advanced tab on the QT System Preferences and in Transport  
Setup, choose “Custom.” Then in Transport Protocol, select HTTP. Then  
in Port ID select “Other.” Type in ’7070’ and click OK. Now see if  
you can get the video. If not, go back to default settings.
