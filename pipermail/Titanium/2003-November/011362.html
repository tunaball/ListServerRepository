On Nov 17, 2003, at 9:31 PM, Benn wrote:

> And you'll want a / for each different path, not \.

I should've mentioned that.  The backslash is an escape character, so 
if you have a directory in the root filesystem with two words, like 
'System Directory', on the unix command line you would type 'cd 
/System\ Directory'.  So if the name has just one word like 
SystemDirectory, then you don't use the escape character and the 
command would be 'cd /SystemDirectory'.

Otherwise all unix paths use a forward slash.
