--On Wednesday, November 12, 2003 09:59 AM -0800 "Michael :-}" <msherwood at mac.com> wrote:

> Could someone post the command string for doing an fsck on my 500 TiBook?

I believe it is fsck -y

The -y switch does an automatic "approve" for all changes fsck proposes to make so you don't have to keep punching "y" continually. Run it multiple times until there are no more errors.

If you want, you get the same thing by booting from your OS X install disk and running Repair Disk in the Disk First Aid tab of  Disk Utility.
