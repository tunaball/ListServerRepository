>> He has these applications "unexpectedly quitting" (Entourage, Internet

>> similar to the prefs file in OS 9?  If not is there some
>> troubleshooting measures I can take to get these applications from
>> quitting on him.


Get rid of the "normal" template in Office Templates. It will recreate
itself.

Also look for a corrupt font. If third party fonts have been added to the
system, that could cause MS apps to crash. If no third party then replace
all the system fonts with new ones from a system install disk using Pacifist
or some other program.

These are a couple of easy steps that may cure the problem.
