I would maybe try using this hint:
http://www.macosxhints.com/article.php?story=20030526100608901
But make sure you have the preference file backed up, and make note of 
what the value you changed was.  This tip has you change the kHWMin 
value, but also check to see if the kHWMax value is 1 as well.  If it 
were somehow set to 0, then it would make sense that the keyboard 
wouldn't light up, I guess.
