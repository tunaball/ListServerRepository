In Jaguar it was possible to use the external keyboard for booting into
single use mode (s/cmd key), start up from a CD or DVD (C key), eject disks
on startup (mouse down or media eject key) and change from OS 9 to OS X by
holding down the X key.

Since installing Panther I have tried all these techniques and none of them
will work any more. You have to open the lid and use the Powerbook keyboard
instead (and in the process, unfortunately the Powerbook screen rather than
the external monitor).

Bummer!
