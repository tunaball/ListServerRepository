It wasn't a security flaw at all - it was by design.  And Mac OS X is  
just as susceptible as Windows.

On an affected Sony CD there's a "Start.app" Mac application in  
addition to the Windows installer.  The user gets a prompt for an  
admin username/password, then exits.  It installs two kernel  
extensions: PhoenixNub1.kext and PhoenixNub12.kext.
