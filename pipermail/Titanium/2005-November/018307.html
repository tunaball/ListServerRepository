> which antivirus program do u use on your TiBook .... if you use one?

I use ClamXav, which is free, and has been good at identifying viruses that
I have received from PC users that I don't want to send back out through
e-mail. 

Apple bundles this software (the opensource "ClamAV", command-line version)
with their Mac OS X Server software <http://tinyurl.com/bwq9n>.

Download via VersionTracker <http://tinyurl.com/9sdns>. It is somewhat
cumbersome to setup, but not difficult.
