One list I'm on sends a results message to unsubscribe requests.
If the address the unsubscribe was sent from doesn't match the
address subscribed as, you can see that from the "unsuccessful"
results message.  Simple, adds some security against malicious
unsubscribes sent by 3rd parties, etc.
