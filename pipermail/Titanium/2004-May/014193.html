Another reason to keep OS9 is that if your OS X gets hosed, you can at least 
boot the machine and run some disk diagnostics or repair, or perhaps save off 
some files. Since you are fortunate enough to have a dual-boot system, this 
is best done if you put OS 9 in a separate 2-4GB partition and also have your 
home directory in a separate partition.
