Yes it is a hidden file as denoted by the '.' before the file name. You can
use several ways to see these files:

-use terminal and the ls command (for more info check man ls)
-use a third party program (I believe Cocktail has a simple box to check and
then all hidden files are visible)
-use BBEdit or BBEdit Lite to find and view such files

Probably others as well...

Don't forget that hidden files are hidden for a reason! The usual caution of
think twice (or more times beforehand) and (then and only then) act once
applies.
