No virus' as such, but there is a serious security flaw that has recently
been discovered:

http://www.macintouch.com/i.2004.05.18.secunia

Apparently it can be triggered by clicking on a link at a remote web site
that can launch the Help application on your computer and run an applescript
which is hidden in the link. If the author is nasty enough it can do all
kinds of damage on your computer. You can download a patch that can deal
with this problem at:

http://homepage.mac.com/olliewagner/dgtgf.dmg

It is well written with a test built in and an uninstaller (if for some
reason you want a remote site to be able to run a script on your machine
with or without your knowledge).
