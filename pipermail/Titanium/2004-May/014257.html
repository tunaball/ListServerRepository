You can do a regular Find for .DS_Store, with Find visibility set
to "all".

As a caution, if you're not familiar with command line stuff, if
you use the "rm" command in Terminal window, use "rm -i", which
will give you a prompt telling you what it's about to delete
before it deletes them.
