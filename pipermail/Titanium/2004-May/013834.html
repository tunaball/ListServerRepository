Hi,

I've used Norton's for quite a while now (Ver 2 or thereabouts) and 
NAV back when it was SAM and both have been fine.

Two things though:

1. I never install all the add ons (disk light, save disk etc.).

2. I only use it either off the CD or after booting the troublesome 
machine off another volume.

BTW: The last I heard the latest Norton's SystemWorks (Ver 3.) will 
be the last to include Norton's Utilities. It appears that Symantec 
has stopped development work on it. NAV will continue though.

I suspect the "specialist" utilities (Disk Warrior, Tech Tool Pro, 
etc.) became better at their niches than Norton's could as a 
generalist.
