Some things I've noticed that can help Word go faster or crash less:

1) Acrobat 6 Professional places a .dot file in Word's startup items folder
that yields a PDF creation toolbar in Word after it starts up.  It causes
Word to load slower than molasses in January.  Delete this .dot file and
don't let Acrobat restore it.

2) If you have a lot of fonts, turn 'em off to see the difference in how
fast Word launches.

3) Make sure you've got the latest Office X patch applied (10.1.5 I
believe).  I noticed better speed in Panther of all the Office components.

4) Turn off spell and grammar checking.

5) Don't paste RTF or HTML text copied to Word's clipboard - I have seen it
choke on such a simple thing (paste in a plain text handler's window then
recopy and paste in Word).

6) as stated, kill the prefs from time to time.  I don't know if Word has a
prefs file on OSX like the "PPC Registration Database" pref that it had
under OS9, but if it does, kill that too.

7) The best way, of course, is to never launch Word.  Sigh.
