Can you hook up an external CD burner and see if it plays from that 
device?  If it can, then it is a hardware limitation of the internal 
drive.  Maybe this is a firmware issue?  You didn't by chance flash 
the firmware so that you could burn DVD's at 2X?

If it doesn't play on the external, maybe hook the external up to a 
computer that does play it from the internal drive.  If it plays in 
the external on the other computer, maybe you have a software issue.
