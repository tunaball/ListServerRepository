I also operate a Mac in an almost-exclusively Windows/Exchange environment. 
However, I obtained Thursby's Dave a while ago and have kept it updated. 
Though OS X has improved its integration abilities into Windows networks, 
Dave does seem to provide a more natural integration with network login, and 
easy browsing and access to print servers and file volumes. It costs more, 
but it works well and you should be able to get tech support with problems 
like you're having now.

A disadvantage is that I don't have a lot of expert experience with OS X only 
Windows networking to help those without Dave.
